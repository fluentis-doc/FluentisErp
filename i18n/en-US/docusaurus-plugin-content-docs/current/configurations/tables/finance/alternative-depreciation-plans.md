This table enables the user to define the depreciation plans that has to be valorized in the multi-year depreciations simulation.



| Function | Meaning |
| --- | --- |
| Code | Depreciation plan code |
| Description | Depreciation plan description |
| Division | Management flag on the only one active division  |

There is a percentage command that has to be applied to all categories, not still inserted in the selected depreciation plan, in order to simplify the insertion of category details. 



| Function | Meaning |
| --- | --- |
| Category | It is the category where the user can apply the depreciation plan |
| Description | It is the category description where the user can apply the depreciation plan |
| Application Percentage  | It is the percentage to apply |

 RIBBON BAR: the ribbon bar represents the Form menu, that is the area in which it is possible to operate on previous recordings or generate new ones. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to look for a depreciation plan through the fields: code filter and description |
| New Plan | Button to insert a new depreciation plan |
| New Detail | Button to insert the depreciation plan detail, where the user is located |
| Delete Plan | Button to delete the depreciation plan, where the user is located |
| Delete Detail | Button to delete the depreciation plan detail, where the user is located |
| Add Missing Categories | Button to assign the percentage for all missing categories automatically |
