The table enables the user to organize the objective depreciations.



| Function | Meaning |
| --- | --- |
| Type | Identification code of the objective depreciation type |
| Description  | Description of the objective depreciation type |
| Incremental Oper. | Management flag as â€˜incremental' even for the objective data (see the relative flag of the fixed asset category) |
| Calculate Days of Use | Calculation flag for use days that will calculate days of use of the fixed asset in the current year instead of use all the accounting period as in the normal calculation |
| Perc. | Standard percentage to apply |
| Fixed Percentage | With this active flag the percentage will be fixed without the possibility to personalize it asset by asset |

RIBBON BAR: the ribbon bar represents the Form menu, that is the area in which it is possible to operate on previous recordings or generate new ones. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to look for the objective depreciation type through the fields: type and description |
| Insertion | Button to insert the objective depreciation type |
| Delete  | Button to delete the objective depreciation type where the user is located |
| Preview  | Button to perform the preview |
| Print | Button to print |
