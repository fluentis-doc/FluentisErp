---
title: Charge Element
sidebar_position: 31
---

The charge element table enables the user to encode some elements in order to associate them to the chart of accounts for statistical purposes.



| Function | Meaning |
| --- | --- |
| Code | Identification code of the charge element.  |
| Description | Charge element description. |

RIBBON BAR: the ribbon bar represents the Form menu, that is the area in which it is possible to operate on previous recordings or generate new ones. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search  | It is a command that is necessary in order to look for charge element through the fields: code and description. |
| New | It is a command to insert a charge element. |
| Edit | It is a command that is necessary in order to edit the charge element, where the user is located. |
| View | It is a command that is necessary in order to view the charge element, where the user is located. |
| Delete | It is a command that is necessary in order to delete the charge element, where the user is located. |
