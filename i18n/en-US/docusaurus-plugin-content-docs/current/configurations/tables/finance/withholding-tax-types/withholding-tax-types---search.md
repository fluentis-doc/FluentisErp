title: Withholding Tax Types - Search
---

RIBBON BAR: the ribbon bar represents the Form menu, that is the area in which it is possible to operate on previous recordings or generate new ones. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search  | Button to look for the deduction type through the fields: type and description |
| New | Button to insert a new deduction type |
| Edit | Button to edit the tax type, where the user is located |
| View | Button to view the deductions type where the user is located |
| Delete | Button to delete the deduction type, where the user is located |
