The table enables the user to organize the fiscal deductibility types in order to link them to the desired depreciation type within fixed asset category.



| Function | Meaning |
| --- | --- |
| Code | Identification code of deductibility type |
| Description | Description of deductibility type |



| Function | Meaning |
| --- | --- |
| Start Date | Start date of deductibility type, where the user is located |
| End Date | End date of deductibility type, where the user is located |
| Deduc. Perc. | Deductibility percentage to apply to the deductibility type, where the user is located |

RIBBON BAR: the ribbon bar represents the Form menu, that is the area in which it is possible to operate on previous recordings or generate new ones. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to look for the deductibility type through the fields: code filter and description |
| New Type | Button to insert the deductibility type |
| Delete Type  | Button to delete the deductibility type, where the user is located |
| New Detail  | Button to insert a new detail of deductibility type, where the user is located |
| Delete Detail | Button to delete the deductibility type detail, where the user is located |
| Preview  | Button to perform the preview |
| Print | Button to launch the print |
