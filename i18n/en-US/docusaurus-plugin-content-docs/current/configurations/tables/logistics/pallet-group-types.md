The Pallet Group Types table is a basis table in order to use Pallet. In particular it is possible to specify the Pallet groups in order to identify Pallet according to progressive number, production job order number, purchase order number and production order number.

The Pallet Group Types table consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to apply search filters within database of Pallet group types on system. |
| Insert | It enables the user to insert a new Pallet group type. |
| Delete | It deletes the selected Pallet group type. |

FILTER AREA: The filter area contains the list of the possible data types thanks to which it is possible to perform a selection. In this case the search filter corresponds to Pallet group type and description.

RESULT GRID: the result grid represents the list of Pallet group types which correspond to the filter data, above mentioned. After having specified the values through which it is possible to perform a Pallet group types search, the user can push [Search] button on the related Ribbon and will obtain the list of Pallet group types on the result grid.

Apart from searching Pallet group types on system it is possible even to add one or more Pallet group types. In order to do this it is necessary to click on the first free record (or to push the [insertion] button on toolbar). In this way it is possible to indicate Pallet group type (Alphanumeric code),Pallet group description and group, by choosing among:

- Progressive number;

- Production job order;

- Purchase order;

- Production order.
