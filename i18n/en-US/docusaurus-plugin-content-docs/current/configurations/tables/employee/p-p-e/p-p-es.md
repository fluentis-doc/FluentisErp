The table is located at **P.P.E.**.

In this table, it is possible to code the *P.P.E.* (Personal Protective Equipment) items that are used to categorize the *P.P.E.*.   
Some examples of *P.P.E.* usage can be found in the management sections:
> **Tasks** - in the *P.P.E. to be delivered* tab;   
> **P.P.E. deliveries/returns**;   
> **P.P.E. delivery/return history**.

The table allows for the insertion of new records or the search for existing ones to view, modify, or delete them.

#### Search for P.P.E.

The form consists of a filter area and a results area. Once all desired filters are set, simply click the **Search** button to display the results within the results grid.

#### Inserting P.P.E.

To insert new *P.P.E.*, it is necessary to click on the first empty row in the grid or press the **New** button.   
For the new record, at least the mandatory fields required by the program must be filled in: **Code** and **Description**.

## P.P.E.

It is the list where the main information of the *P.P.E.* is entered.   
The list consists of the following information:   
> **Code**: this is the code of the *P.P.E.*.   
> **Description**: this is the description of the *P.P.E.*.   
> **Category Code**: this is the code of the *P.P.E. Category*.   
> **Category Description**: this is the description of the *P.P.E. Category*.   
> **Risk category code**: this is the code of the *P.P.E. Risk Category*.   
> **Risk Category Description**: this is the description of the *P.P.E. Risk Category*.   
> **Monthly validity**: this is the number of months of validity of the *P.P.E.* under normal usage conditions.   
> **CE**: indicates if the *P.P.E.* is CE marked (European Certification Mark).   
> **Active**: indicates if the *P.P.E.* is still valid and usable.   
> **Use**: free annotations on the subject.   
> **Prescriptions or recommendations for use**: free annotations on the subject.   
> **Available documentation**: free annotations on the subject.   
> **Notes**: free annotations.

### Personal Protective Equipment Used in Tasks

It is the read-only list where all the *Tasks* that use the selected *P.P.E.* are displayed.   
The information displayed is entered and managed in the *Tasks* management. 
The list consists of the following information:   
> **Task code**: this is the code of the *Task*.   
> **Task description**: this is the description of the *Task*.   
> **Use**: annotations related to the specific *Use* of the *P.P.E.* if used for the specific *Task*.   
> **Prescriptions or recommendations for use**: annotations related to the *Prescriptions or recommendations for use* of the *P.P.E.* if used for the specific *Task*.   
> **Notes**: free annotations.
