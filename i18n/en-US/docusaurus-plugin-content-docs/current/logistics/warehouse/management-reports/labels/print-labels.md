---

The Print Labels procedure has been created in order to enable the user to query easily system and to obtain quick labels to use in warehouse.

The Print Labels procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Preview | It recalls the procedure to view the print preview. |
| Print | It prints the labels report. |

INSERTION GRID: the insertion grid has the task to insert items for which user wants to print labels. In order to inser an item into the grid it is possible to type it from keyboard or to use item help that can be activate through a double click on code field. It is recommended that using items help, it is possible to insert through multiselect. Once having inserted the desired items it is possible to specify the number of labels to be printed for each item.

The window enables the user to print a standard report or to print a customized report (after having mapped it in reports management field). If the user wants to print a customized report it will be necessary to choose the same one from printout tab in print parameters.
