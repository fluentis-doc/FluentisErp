title: Search Physical Inventory
sidebar_position: 2
---

The search physical inventory procedure enables the user to select the system inventories. The inventories are necessary in order to group one or more inventory lists.

The physical inventory procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions regarding to the preexistent inventory rather than the creation of a new one. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to apply search filters within database of the inserted inventory. |
| Insert | It refers to the procedure to insert a new inventory. |
| Edit  | It refers to the procedure to edit an inserted inventory. |
| View  | It refers to the procedure to view an inserted inventory. |

FILTER AREA: The filter area contains the list of the possible data types thanks to which it is possible to perform a selection, a research on the total of data inserted in system. Typically Filters have 'AND' condition and it is possible to specify more filter criteria at the same time. In this case it is possible to filter per inventory number, from/to date, open or close, description and/or operator.

RESULT GRID: the result grid represents the list of inventories that belong to data above mentioned. After having specified values through which to obtain an inventory list, the user by clicking on the button [Search Picking] on the related Ribbon, will obtain the list of the desired inventories in the result grid.
