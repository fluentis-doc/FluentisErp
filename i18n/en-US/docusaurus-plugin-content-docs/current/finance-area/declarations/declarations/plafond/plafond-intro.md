The search Plafond procedure has been created in order to enable the user to easily search   previously inserted plafondsin order to view,  edit and delete them if necessary. Through the search procedure it is possible to perform all necessary operations regarding document, after having obtained the necessary user's rights.

The search plafond procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions about a pre-existing document, rather than to create a new one. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search Plafond | Button to apply search filters within database of the inserted plafond |
| New Plafond | It refers to the procedure to insert a new plafond |
| Edit Plafond | It refers to the procedure to edit a previously inserted plafond |
| View Plafond | It refers to the procedure to view a previously inserted plafond |
| Delete Plafond | It performs the procedure to delete a previously inserted plafond |
