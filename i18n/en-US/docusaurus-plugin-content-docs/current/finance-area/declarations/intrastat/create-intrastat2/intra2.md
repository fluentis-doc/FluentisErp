Below these general data there is a list of model header fields: as far as possible the basic data are compiled according to what inserted on 'Company' table. Then it is possible to manage the presentation and activity cessation flags or VAT number variation as well as the subject section delegated to the presentation manually.

At the base of the window there are summary fields about bis, ter, quater and quinquies sections of model; lastly the section related to possible periodicity exchanges that are on declaration in use.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Save | Button through which it is possible to save the declaration. |
|  [Automatic Creation](/docs/finance-area/declarations/intrastat/automatic-creation-intrastat1/automatic-creation)  | Button to refer to the guided procedure of data retrieve from purchase area. |
|  [Create from Records](/docs/finance-area/declarations/intrastat/create-from-records-intrastat1/create-from-records-intrastat1-intro)  | Button to refer to the guided procedure of data retrieve from ledger records. |
