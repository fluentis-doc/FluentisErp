The mask includes: filter (in the upper part of the mask) on which it is necessary to indicate the detail accounts on which search the maturity values that have to be transferred, a grid of maturity value selection and, at the bottom,Â  the accounting template setting, record/accrual dates and the detail account on which re-open the maturity values. In particular, the accounting template has to include maturity values closing and opening flags. The result is the detection of the selected maturity values closures. The maturity values will be re-opened with the same original references but on the detail account of destination.

RIBBON BAR: the ribbon bar represents the form menu, that is the area on which it is possible to perform actions. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to search the maturity values related to the detail account that has to be transferred on another detail account. |
| Accounting | Button for the maturity values giro accounting. |
