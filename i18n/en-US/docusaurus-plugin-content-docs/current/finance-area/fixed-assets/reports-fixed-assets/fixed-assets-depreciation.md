Through this report it is possible to extract details of depreciations calculated according to parameters indicated in the mask.

It is important to signal the peculiarities of standard reports that can be linked to the procedure:

1) The default print performs the hypothetical calculation of depreciations that can be applied according to settings of fixed asset category: that calculation does not consider the residue to be amortized

2) The “depreciations calculated” report (CE_FixedAssetDeprCalc.rpt), that it is recommended always to linked to the process, is dedicated to data extraction as calculated by the program or inserted manually within fixed assets.

The report with fiscal data is particularly interesting for customers who use this characteristic: it will totalize the non-deductible depreciations amounts.

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions linked to the procedure in use. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Preview | Button to perform the print preview. |
