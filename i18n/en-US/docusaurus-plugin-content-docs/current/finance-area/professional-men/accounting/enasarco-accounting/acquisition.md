Once having set, at the base of the mask, accrual record dates, accounting template to be used and possible management in a single record, it will be sufficient to click on accounting button: if everything is successful, the selected lines will disappear from the grid and summaries and it will be possible to view the result preview.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search | It searches the compensations list for which to account Enasarco tributes. |
| Accounting | It performs the accounting of the selected Enasarco tributes. |
