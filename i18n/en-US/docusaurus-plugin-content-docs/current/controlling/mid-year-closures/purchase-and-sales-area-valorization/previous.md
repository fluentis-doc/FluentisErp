RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search | It performs the search of accountings performed |
| Valorization Property | It shows a report with the result of the selected accounting |
| Restore Valorization | It deletes all the selected valorization |
| Restore Record | It deletes the selected record |
