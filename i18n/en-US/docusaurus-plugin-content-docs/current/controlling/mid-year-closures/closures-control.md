The mask is dedicated to control imbalances assigned to CC/PC. Remember that the accounting templates enable the unblock at the origin of the possibility to valorize in a unbalanced way the financial accounting with the analytical accounting, but in case that this is not used, the procedure can be necessary in order to identify possible errors on balances, that otherwise should be correct manually.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search | It searches the detail accounts according to filter parameters set. |
| Preview | It shows the print preview of data viewed. |
| Print | Print the viewed data. |
