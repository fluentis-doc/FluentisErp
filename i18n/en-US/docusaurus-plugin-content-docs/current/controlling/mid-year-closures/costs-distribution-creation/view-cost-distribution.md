In this mask on columns, it will be possible to view different profit centres, on the contrary in the tree section below there will be possible to view first the profit centre balance and then the various encoded groups with values of cost centres divided by different profit centres. The final amount gives the margin for each profit centre.

Through the View Distributions mask it is possible to print data or to export excel data.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Preview | It performs the print preview of the distribution in visualization mode |
| Print | It prints the distribution in visualization mode |
| Export Excel | It export in excel the distribution in visualization mode |
