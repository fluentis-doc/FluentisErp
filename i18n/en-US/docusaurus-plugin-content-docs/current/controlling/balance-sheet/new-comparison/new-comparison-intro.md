Once having clicked on menu of 'New Comparison' it will be necessary to set the reference pattern and the comparison description: once having saved, on the grid below the user will insert the list of reclassifications to be compared in the following 'Variances' section, on which it is possible to set the visualization of the variance percentage from column next to this or from a fix column to be inserted in the related drop-down menu, and the visualization of detail (detail accounts, CC, PC, job orders) which valorize the structure of the reclassified balance sheet.

Lastly within mask there is an export procedure of data on excel.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Save | It saves the settings of comparison. |
