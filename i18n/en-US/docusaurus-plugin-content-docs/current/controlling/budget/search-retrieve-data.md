The search retrieve data procedure has been created in order to enable the user to search easily the retrieve that have been previously created in order to view, to edit and to delete them, if necessary. Through the retrieve data procedure it is possible to perform all necessary operations about document, after having obtained the necessary user's rights.

The search retrieve data procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions about a pre-existing document, rather than to create a new one. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to apply search filters within database of the inserted data retrieve |
| New | It refers to the procedure to insert a new data retrieve |
| Edit | t refers to the procedure to edit the selected data retrieve |
| View | It refers to the procedure to view the data retrieve |
| Delete | It performs the procedure to delete the data retrieve |
