title: Search Purchase Orders
sidebar_position: 2
---

The search Purchase Orders procedure has been created to enable the user to search easily previously inserted orders in order to view, to edit and to delete them, if necessary. Starting from Purchase Orders Search it is possible to perform all necessary operations regarding document, after having obtained the necessary user's rights.

The Search Purchase Orders procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions about a pre-existing document, rather than to create a new one. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | Button to apply search filters within database of the inserted purchase orders. |
| New | It refers to the procedure to insert a new purchase order. |
| Edit | It refers to the procedure to edit a previously inserted purchase order. |
| View | It refers to the procedure to view a previously inserted purchase order. |
| Delete | It performs the procedure to delete a previously inserted purchase order. |
| Execution Property | It performs the procedure to create reversal document related to the selected purchase orders. |
| Duplicate Purchase Order | It performs the procedure to create a document similar to a purchase order selected on grid below. |
| Convert Currency | It performs the procedure to transform the selected document in a “Currency” conveniently specified. |

FILTER AREA: the filter area contains the list of possible data type thanks to which it is possible to perform a selection, a search on total of inserted orders. Typically filters have always “AND” condition and it is possible to specify more filter criteria at the same time. Filter data give the possibility, thanks to [Search Items] button, to search for data of single items inserted on purchase order.

RESULT GRID: the result grid represents the list of purchase orders which correspond to filter data, above mentioned. After having specified values through which it is possible to obtain an orders search, the user, by clicking on [Search Orders] button on the related Ribbon, will obtain the list of the desired orders on the result grid.
