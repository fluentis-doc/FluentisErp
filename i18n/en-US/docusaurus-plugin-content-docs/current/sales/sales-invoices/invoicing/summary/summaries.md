Within Summaries section user can obtain all the main information about the whole document.

**Final Notes**: it is a free field, which can be compiled by the user thanks to 'Codified Notes Help' command;

The **Total** section has the following fields:

**Items Gross Amounts**: represents the amount of all items values; 

**Total Discount Applied**: represents the total value of discounts applied without final discounts;

**Items Net Amount**: Items Gross Amounts - Total discount applied; 

**Total Final Discounts**: represents the value of final discounts; 

**Charges Not Documented**: represents the value of charges inserted in Charges grid; 

**Amount Origin**: Items Net Amount – Final Discounts + total charges not documented + total charges documented; 

**VAT**: represents the amount of values contained in VAT summaries; 

**Total**: Amount Origin+VAT.
