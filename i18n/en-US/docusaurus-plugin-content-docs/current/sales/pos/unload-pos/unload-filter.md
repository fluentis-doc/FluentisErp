In this tab, the user has the possibility to search and to transfer POS on warehouse, according to the available filter criteria. The record date corresponds to the POS date:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions linked to the procedure in use. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search | It enables the user to search POS to be unloaded. |
| Unload  | Button to perform the unload of POS selected from warehouse according to unload algorithms. |

FILTER AREA: the filter area contains the list of possible data types thanks to which it is possible to perform a selection. Typically filters have always “AND” condition and it is possible to specify more filter criteria at the same time.

RESULT GRID: the result grid represents the POS list which correspond to filter data, above mentioned and which fulfill the necessary conditions in order to be unloaded from warehouse. After having specified values through which to obtain a search of POS, the user, by clicking on [Search] button on the related Ribbon, will obtain the list of desired POS on result grid. In addition there is the possibility to select one or more POS to be unloaded in this list.
