title: Delivery Note List
sidebar_position: 2
---

This procedure is necessary to print a delivery note list according to specified filter criteria (delivery note type, number, date, customer); in addition there is the possibility to order by customer, date, number or delivery note type.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Preview | Button to perform the print preview. |
| Print | Button to print according to filter set and printer configuration. |
