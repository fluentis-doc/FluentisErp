This procedure is necessary to print an already inserted delivery note according to specified filter criteria (delivery note type, number, date, customer, etc).

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Preview | Button to perform the print preview. |
| Print | Button to print according to filters set and the printed configuration. |
