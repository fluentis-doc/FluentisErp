In this tab the user has the possibility to search and transfer the delivery note that have set the 'Controlled' flag inside.

**Warehouse Record Date**: it enables the user to indicate warehouse record date, except from 'Create record with document date' setting, within sales delivery notes parameters, 'Unload' tab.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search Delivery Note | Button to search the delivery notes to be unloaded. |
| Unload Delivery Note | Button to perform the unload of the selected delivery notes from warehouse, according to unload algorithms and parameters inserted on Parameters tab. |

FILTER AREA: the filter area contains the list of possible data type thanks to which it is possible to perform a selection. Filters have always 'AND' condition and it is possible to specify more filter criteria at the same time.

RESULT GRID: the result grid represents the list of delivery notes which correspond to the filter data above specified and that fulfill the necessary conditions to be unloaded from warehouse. After having specified values through which to obtain a search of delivery note, the user, by clicking on [Search] button on the related Ribbon, will obtain the list of the desired delivery note on result grid. In addition, there is the possibility to select one or more delivery note to be unloaded in this list.
