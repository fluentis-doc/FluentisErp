It gives the possibility to manage the item variants in a different way compared to the one proposed on item line. For example, at the level of item line, it is possible to insert the total quantity for that matrix and after this tab, it is possible to define different quantities of more variants of the selected item and for variant and attribute, defined as size and colour management. This operation is possible only if all item variants have the same price as the item as regards the standard configuration.
