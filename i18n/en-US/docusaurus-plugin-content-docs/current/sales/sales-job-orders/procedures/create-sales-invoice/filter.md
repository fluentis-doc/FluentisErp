According to filter criteria specified (customer, item, number, date, period for the intervention creation), on Filter tab it is possible to view all items of job orders which contain data for the intervention request (view Job Orders-Items-Intervention Request Tab). At the bottom of the window, in Parameters section, the user can insert the request status and issue date of the same one (this date could be different from the proposed date). In order to conclude the procedure, the user has to click on 'Transfer' button.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search | It enables the user to search items which contain data for the intervention request. |
| Transfer | It enables the user to perform the transfer of the selected items. |
