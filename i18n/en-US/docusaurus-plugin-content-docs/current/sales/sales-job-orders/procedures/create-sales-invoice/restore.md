It enables the user to view the operations created through this procedure. In order to delete a request, the user has to select and to click on 'restore' button. In this way, the intervention request will have the 'not executed' status.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search | It enables the user to filter the intervention requests created. |
| Restore | Button to delete the selected intervention request. |
