It gives the possibility to manage the item variants in a different way compared to the one proposed on item line. For example, at the level of item line, it is possible to insert the total quantity for that matrix and then in this tab it is possible also to set different quantities for more variants of the selected item. This operation can be performed only if all item variants have the same price as the item price during the standard configuration.
