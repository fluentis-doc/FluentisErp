**Item**: it shows the matrix description and possibly the description of the item variant; 



**Agent**: it is possible to propose the value of commission assigned to agent for this item sale. The percentage can be changed;

**Job Order**: it is the sales job order associated to document;

**Mark**: it represents the item mark, taken by its register or item price list; 

**Notes**: it offers the possibility to insert notes for each item.

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following: 



| Function | Meaning |
| --- | --- |
| Delete Agent | Button to delete the agent of the inserted item.  |
