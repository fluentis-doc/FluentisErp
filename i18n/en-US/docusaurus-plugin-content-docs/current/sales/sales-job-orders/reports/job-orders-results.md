This procedure is necessary to print the job orders result report, according to specified filter criteria (job order type, number, date, customer, etc).

RIBBON BAR: it represents the Form menu, that is the area in which it is possible to perform actions. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Preview | Button to perform the print preview. |
| Print | Button to print according to filters set and printer configuration. |
