**Transfer**: the button allows you to *partially or fully cancel a document*. With this function, it will be possible to create a reversal document that will *decrease stock* (in purchases) / *increase stock* (in sales).  

> *Procedure*:  
1. select the document to cancel and click on the **Transfer** button. The application will open a pop-up window with all the item lines of the source document;  
2. fill in this form: the **Type of reversal document** you want to create and, for each item, specify the **Quantity to be reversed**, the **Reversal template**, and activate the **To be reversed** parameter;  
3. click **OK** to confirm the operation.
