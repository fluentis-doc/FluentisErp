title: Search Advance List

---

The search advances lists procedure has been created to enable the user to search easily the previously inserted advance lists in order to view, to edit and to delete them, if necessary. After having obtained the necessary user's rights, through the search procedure it will be possible to perform every single operation.

The search advance lists procedure consists as follows:

RIBBON BAR: the ribbon bar represents the Form menu, that is the area on which it is possible to perform actions linked to the procedure in use. The possible features list is the following:



| Function | Meaning |
| --- | --- |
| Search Lists | Button to apply search filters within database of the previously inserted advances lists. |
| New List | It refers to the procedure to insert a new advance list. |
| Edit List | It refers to the procedure to edit a previously inserted advance list. |
| View List | It refers to the procedure to view a previously inserted advance list. |
| Delete Lists | It refers to the procedure to delete a previously inserted advance list. |

**For further information about the following Reports and Procedures, refer to the related Menu Items.**



| Function | Meaning |
| --- | --- |
|  [Reports](/docs/treasury/advances-collections/reports)  | It refers to the print procedure related to Advanced/Collected Invoices List. |



| Function | Meaning |
| --- | --- |
|  Collections Management | Button to manage collections. |
|  Collections Advances Accounting) | Button to create the ledger record of collections advances. |

FILTER AREA: the filter area contains the list of possible data types thanks to which it is possible to perform the selection. Typically filters have always 'AND' condition and it is possible to specify more search filter at the same time.

RESULT GRID: the result grid represents the list of advances lists that correspond to filter data, above mentioned. After having specified values through which to obtain a search of advance lists, the user, by clicking on the [Search] button on the related Ribbon, will obtain the list of the desired advance lists on the result grid.
