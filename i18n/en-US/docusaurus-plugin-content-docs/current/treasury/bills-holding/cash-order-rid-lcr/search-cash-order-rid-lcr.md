The Search Cash Order/Rid/Lcr has been created in order to enable the user to search easily the electronic files created on bills portfolio module, in order to view the created file or to delete and enable again the creation of a new file.

RIBBON BAR: it represents the Menu form, that is the area in which it is possible to perform actions linked to the procedure in use. The list of features is the following:



| Function | Meaning |
| --- | --- |
| Search Cash Order | It searches the list of electronic files created. |
| View Cash Order | It performs the search of files created according to filters set. |
| Delete Cash Order | It deletes the status flag 'issued file' and needs the confirmation of file from the stored network path. |
