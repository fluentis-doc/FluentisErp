Desde este formulario es posible ejecutar la impresión que resume las facturas anticipadas<!-- fatture anticipate -->.

Se encuentran disponibles los principales elementos de filtro como el tipo de factura<!-- tipo fattura -->, el banco de anticipo<!-- banca di anticipo -->, el cliente<!-- cliente -->, así como las referencias al documento de anticipo<!-- distinta di antipo -->.

Además, es posible limitar la visualización solo a las facturas ya anticipadas<!-- fatture già anticipate -->, cobradas<!-- incassate --> o por anticipar<!-- da anticipare -->.
