El formulario se abre a través de la ruta **Ventas > POS > Búsqueda POS<!-- Vendite > POS > Ricerca POS -->**.

Para la explicación de las funcionalidades comunes a los documentos de venta, consulte la sección [Ventas<!-- Vendite -->](/docs/sales/sales-intro).

### Botones específicos<!-- Pulsanti specifici  -->

> - **Creación DDT**: el procedimiento permite crear un DDT directamente desde uno o más POS; aparecerá una ventana emergente donde ingresar el Tipo de DDT a crear y el cliente al que se debe emitir el documento.     
> - **Creación de facturas<!-- Creazione fatture -->**: el procedimiento permite crear una factura directamente desde uno o más POS; aparecerá una ventana emergente donde ingresar el Tipo de factura a crear y el cliente al que se debe emitir el documento.
