El formulario se abre a través de la ruta **Ventas > Tarifas de venta<!-- Vendite > Listini di vendita -->**.

Para la explicación de las funcionalidades comunes a los documentos de venta, consulte la sección [Ventas](/docs/sales/sales-intro).

El botón específico de esta pantalla de búsqueda es **Cerrar tarifas<!-- Chiudi listini -->**: con este botón se muestra una ventana emergente en la cual ingresar una fecha de finalización de validez para la tarifa<!-- listino -->; después de esta fecha ingresada, la tarifa<!-- listino --> ya no será válida. Si existen documentos para las tarifas de venta<!-- listini di vendita --> seleccionadas, con fecha posterior a la fecha de finalización ingresada en la ventana emergente, para estos documentos no ocurrirá el cambio de fecha y se señalarán los documentos con la siguiente información:
- Tipo de documento de venta<!-- Tipo documento di vendita -->
- Código y Descripción de tipo de documento<!-- Codice e Descrizione tipologia di documento -->
- Número de documento<!-- Numero documento -->
- Cuenta<!-- Conto -->
- Fecha de documento<!-- Data documento -->
- Fecha de finalización de la tarifa de precios<!-- Data di fine listino prezzi -->
