En esta pestaña se pueden definir datos generales del proyecto<!-- progetto -->, como la **Divisa**, las modalidades de **Envío<!-- Spedizione -->**, una posible **Fecha de cierre<!-- Data di chiusura -->**, la **Nación<!-- Nazione -->**. El **Estado de despacho<!-- Stato evasione -->** se vuelve necesario para indicar que el proyecto<!-- progetto --> ha sido despachado en otros documentos, mientras que las **Anotaciones del cliente<!-- Annotazioni cliente -->** traen algunas referencias para el cliente<!-- cliente -->, como un posible proyecto<!-- progetto --> relacionado o notas iniciales.

![](/img/it-it/project-management/projects/header.png)

El encabezado<!-- Testata --> contiene a su vez varias pestañas que presentan los datos presentes por defecto en la ficha del cliente<!-- anagrafica cliente -->, pero que también pueden completarse libremente: Pagos, Descuentos, Agentes, Destino<!-- Destinazione -->, Extra Data, Permisos de Usuario.
En esta última pestaña es posible listar los usuarios y sus respectivos derechos de Solo Lectura (Read Only) o Lectura y Escritura (Read Write), si en la tabla de Tipos de proyecto<!-- Tipi progetto --> está activada la casilla **Seguridad del proyecto habilitada<!-- Sicurezza del progetto abilitata -->**; de lo contrario, todos los usuarios tendrán derechos de Lectura y Escritura (Read Write).
