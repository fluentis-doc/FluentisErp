En este formulario es posible ingresar todos los datos de un nuevo **Proyecto**.<!-- Progetto -->

**Tipo de proyecto<!-- Tipo progetto -->**: contiene el tipo de proyecto<!-- progetto --> seleccionado de la tabla Configuración > Tablas > Gestión de Proyectos > Tipos de proyecto<!-- Configurazione > Tabelle >Gestione Progetti > Tipi progetto -->;

**Cliente**: contiene el cliente al que está vinculado el proyecto<!-- progetto -->;

**Año/Número/Fecha<!-- Anno/Numero/Data -->**: contiene el año, el número y la fecha del proyecto<!-- progetto -->;

**Título del proyecto<!-- Titolo del progetto -->**: se asigna la descripción del cliente, pero el campo es libremente modificable.

Hay botones específicos en la barra de acciones:

![](/img/neutral/common/update-project.png) Actualizar<!-- Aggiorna -->: actualiza los datos del proyecto<!-- progetto --> según otras operaciones realizadas sobre el mismo en otros documentos (por ejemplo, la confirmación de las Intervenciones que crea la Declaración de Actividad por Proyecto<!-- Interventi che crea la Dichiarazione Attività per Progetto -->).

Según los ajustes del tipo de proyecto<!-- tipo progetto -->, se visualizarán determinadas **pestañas<!-- tab -->**.
