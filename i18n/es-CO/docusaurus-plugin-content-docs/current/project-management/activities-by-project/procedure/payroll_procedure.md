Gestión de proyectos<!-- Gestione progetti --> > Procedimientos de Actividad<!-- Procedure Attività --> > Cálculo de datos de nómina<!-- Calcolo dati paghe -->

Desde esta pantalla es posible calcular automáticamente los datos para los recibos de nómina<!-- cedolini paga -->.

Las horas y los gastos (solo aquellos marcados como reembolsables) insertados en los recibos de nómina<!-- cedolini --> se tomarán de las declaraciones de actividad<!-- dichiarazioni attività -->, independientemente de si se ingresaron directamente en Área de Gestión de Proyectos<!-- Area Gestione Progetti --> > declaración de actividad<!-- dichiarazione attività --> o si fueron generados automáticamente por las intervenciones con estado Aprobado. Por lo tanto, será fundamental que cada declaración de actividad<!-- dichiarazione attività --> tenga asociada una categoría de actividad<!-- categoria attività --> con el correspondiente código de nómina<!-- codice paghe --> configurado.

Por defecto, el período a calcular corresponde al año en curso y al mes anterior al actual.
