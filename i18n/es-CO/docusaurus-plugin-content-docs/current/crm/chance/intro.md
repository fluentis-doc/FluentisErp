El módulo permite la gestión integrada de las negociaciones comerciales<!-- trattative commerciali -->. Permite monitorear el ciclo de ventas<!-- ciclo di vendita --> de forma estructurada, comenzando desde la fase inicial de generación de leads hasta el cierre de la negociación<!-- trattativa -->. Las oportunidades se utilizan para recopilar información relevante sobre clientes potenciales, vinculando esta información con otras áreas de la empresa, como el marketing.         
Desde un punto de vista técnico, las oportunidades permiten rastrear cada interacción con el cliente, vinculándola a documentos específicos, cotizaciones<!-- preventivi --> y actividades planificadas. Los datos recopilados a través de este módulo son utilizados para generar previsiones de ventas<!-- previsioni di vendita --> basadas en parámetros estadísticos y probabilísticos. Además, integran funcionalidades de gestión de flujos de trabajo<!-- workflow management --> para automatizar determinados pasos del proceso de ventas<!-- processo di vendita --> y proporcionar notificaciones en tiempo real sobre los cambios en el estado de la oportunidad.         

**Oportunidades<!-- Opportunità -->**       
El módulo de Oportunidades<!-- Opportunità --> de Fluentis ERP permite gestionar todo el ciclo de vida de una oportunidad comercial, desde la creación hasta el cierre.       

**Mapa de oportunidades CRM<!-- Mappa opportunità CRM -->**       
Esta funcionalidad ofrece una visualización geográfica de las oportunidades activas, utilizando la información geográfica presente en la ficha<!-- anagrafica --> de contactos CRM. Es posible filtrar las oportunidades por tipología, usuario operador o probabilidad de éxito.     

**KanBan Board de progreso de oportunidades<!-- Opportunity Progress KanBan Board -->**     
Esta función muestra las oportunidades divididas por probabilidad de éxito, utilizando un KanBan Board. A través de filtros avanzados es posible aislar conjuntos específicos de oportunidades, con columnas que representan diferentes porcentajes de éxito. Cada columna muestra el número de oportunidades, su valor neto total y detalles individuales de cada oportunidad.       

**KanBan Board por etapas de oportunidad<!-- Opportunity Stages KanBan Board -->**      
Similar a la anterior, esta funcionalidad divide las oportunidades por etapas del flujo de trabajo<!-- workflow --> asociado. Cada columna del KanBan Board representa una etapa del flujo de trabajo, con información sobre el número y el valor neto de las oportunidades.      

**KanBan Board de progreso de facturación<!-- Invoicing Progress KanBan Board -->**       
Este procedimiento permite monitorear el estado de las oportunidades, pedidos<!-- ordini --> y facturas en un periodo de tiempo definido, dividido por meses. Los documentos pueden visualizarse de forma agregada o individualmente, con detalles económicos y enlaces para la modificación directa.
