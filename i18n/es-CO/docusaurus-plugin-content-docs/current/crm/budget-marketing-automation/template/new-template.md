El procedimiento para la creación de un **nuevo Template**<!-- nuovo Template --> permite definir cuáles serán las características del Template que luego se utilizará en el proceso de envío de Newsletters, como imágenes, texto, hipervínculos y merge data<!-- merge data -->.
El formulario está compuesto por dos secciones.

La sección superior permite definir la siguiente información:

**Nombre**: corresponde al código con el que se identificará el Template<!-- Template -->;

**Descripción**<!-- Description -->: es la descripción del Template<!-- template -->;

**Objeto de negocio**<!-- Oggetto Business -->: es el objeto de negocio de Fluentis para el que se podrá utilizar el Template<!-- Template -->;

**Tipo de data source**<!-- Tipo di data source -->: es la tipología de data source<!-- data source --> sobre la que se basa el Template<!-- template --> (Simple, GroupBy y SQL Query);

**Data Source**: es la fuente de datos que utilizará el Template<!-- Template --> para la compilación de los merge data<!-- Merge Data -->.

La parte inferior del formulario prevé una serie de pestañas con las distintas funciones para construir el Template<!-- template --> y la hoja de trabajo donde se construye efectivamente el Template<!-- Template --> y visualizar gráficamente el resultado:

![](/img/it-it/applications/crm/budget-marketing-automation/template-file.png)

A continuación, las pestañas presentes en el formulario.

**Archivo**<!-- File -->

Esta pestaña permite realizar las operaciones básicas del Template<!-- Template -->, como abrir una nueva hoja de trabajo, abrir otra hoja de trabajo y la vista previa de impresión del Template<!-- Template -->.  
**Nuevo**<!-- New -->: abre una nueva hoja de trabajo;  
**Abrir**<!-- Open -->: abre una nueva hoja de trabajo;  
**Guardar como**<!-- Save As -->: permite guardar la hoja de trabajo con un nombre en el pc;  
**Imprimir**<!-- Print -->: permite imprimir la hoja de trabajo;  
**Vista previa de impresión**<!-- Print Preview -->: permite mostrar en pantalla la impresión de la hoja de trabajo como vista previa;  
**Deshacer**<!-- Undo -->: elimina el último cambio realizado en el Template<!-- template -->;  
**Rehacer**<!-- Redo -->: restaura el último cambio eliminado en el Template<!-- template -->.  

![](/img/it-it/applications/crm/budget-marketing-automation/file.png)

**Inicio**<!-- Home -->

Esta pestaña contiene los botones para las funciones básicas relacionadas con la edición de texto del Template<!-- Template -->, como el tamaño de fuente, listas, alineación del texto, etc.  
Incluye las funciones básicas de los editores de texto más comunes, como Microsoft Word.  

![](/img/it-it/applications/crm/budget-marketing-automation/home.png)

**Insertar**<!-- Insert -->

Esta pestaña contiene los botones para insertar diversos elementos en el Template<!-- Template -->, como tablas, imágenes e hipervínculos.  
**Tabla**<!-- Table -->: permite insertar una tabla dentro de la hoja de trabajo;  
**Imagen insertada en línea**<!-- Inline Picture -->: permite insertar una imagen en el cuerpo del documento;  
**Imagen**<!-- Picture -->: permite insertar una imagen en el cuerpo del documento;  
**Marcador**<!-- Bookmark -->: permite insertar un marcador dentro del Template<!-- template -->;  
**Hipervínculo**<!-- Hyperlink -->: permite insertar un enlace a una página web o a un marcador del Template<!-- template -->;  
**Encabezado**<!-- Header -->: permite insertar el encabezado del Template<!-- template -->;  
**Pie de página**<!-- Footer -->: permite insertar el pie de página del Template<!-- template -->;  
**Número de página**<!-- Page Number -->: sirve para indicar el número de página en el encabezado o pie de página del Template<!-- Template -->;  
**Cantidad de páginas**<!-- Page Count -->: sirve para indicar el número total de páginas del Template<!-- template --> en el encabezado o pie de página;  
**Cuadro de texto**<!-- Text Box -->: permite insertar una caja de texto en el Template<!-- Template -->;  
**Símbolo**<!-- Symbol -->: sirve para insertar un símbolo especial dentro del Template<!-- Template -->.  

![](/img/it-it/applications/crm/budget-marketing-automation/insert.png)

**Merge data**

En esta pestaña están los botones necesarios para la gestión e inserción de merge data<!-- merge data -->.  
**Insertar campo de merge**<!-- Insert Merge Field -->: El botón permite insertar merge data<!-- merge data --> seleccionándolos de una lista basada en el Data Source<!-- Data Source --> al que hace referencia el Template<!-- Template -->;  
**Ver datos combinados**<!-- View Merged Data -->: el botón permite ocultar/visualizar los merge data<!-- merge data --> insertados en el Template<!-- template -->;  
**Mostrar todos los códigos de campo**<!-- Show All Field Codes -->: el botón sirve para mostrar los códigos de los campos utilizados como merge data<!-- merge data -->;  
**Mostrar todos los resultados de los campos**<!-- Show All Field Results -->: el botón sirve para mostrar el resultado de la extracción de los campos utilizados como merge data<!-- merge data -->;  
**Primer registro**<!-- First Record -->: permite posicionarse en el primer registro extraído de los merge data<!-- merge data -->;  
**Registro anterior**<!-- Previous record -->: permite posicionarse en el registro anterior al actual extraído de los merge data<!-- merge data -->;  
**Siguiente registro**<!-- Next record -->: permite posicionarse en el registro siguiente al actual extraído de los merge data<!-- merge data -->;  
**Último registro**<!-- Last Record -->: permite posicionarse en el último registro extraído de los merge data<!-- merge data -->.
