El filtro de ejecución de reportes<!-- filtro di esecuzione report --> se encuentra en la ruta **Calidad > Calibración de instrumentos > Instrumentos de medición<!-- Strumenti di misura --> > Procedimientos > Lista de Instrumentos de medición<!-- Elenco Strumenti di misura -->**.   

El filtro de ejecución de reportes<!-- filtro di esecuzione report --> permite buscar los *Instrumentos de medición<!-- Strumenti di misura -->* para poder imprimirlos.   
Los reportes predefinidos son:
> **Registro de Instrumentos de medición<!-- Registro Strumenti di misura -->**: lista de los *Instrumentos de medición<!-- Strumenti di misura -->* agrupables por: *Instrumento*, *Categoría*, *Proveedor* (de compra), *Ubicación*, *Usuario*, *Vencimiento de garantía* y *Vencimiento de calibración*;   
> **Registro de Instrumentos no sujetos a calibración<!-- Registro Strumenti non soggetti a taratura -->**: lista de los *Instrumentos de medición<!-- Strumenti di misura -->* donde no se prevé la *Calibración<!-- Taratura -->*; para la impresión de este reporte no afecta el filtro *Estado - No sujetos a calibración: Todos, Sujetos a calibración, No sujetos a calibración<!-- Stato - Non soggetti a taratura: Tutti, Soggetti a taratura, Non soggetti a taratura -->*.
> **Ficha de Instrumento de medición<!-- Scheda Strumento di misura -->**: ficha de los *Instrumentos de medición<!-- Strumenti di misura -->*.
