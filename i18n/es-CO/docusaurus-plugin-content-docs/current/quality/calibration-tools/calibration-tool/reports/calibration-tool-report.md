El filtro para la ejecución de reportes se encuentra en la ruta **Calidad > Calibración de instrumentos<!-- Taratura strumenti --> > Calibración de instrumentos<!-- Taratura strumenti --> > Procedimientos > Listado de Calibraciones de instrumentos<!-- Elenco Tarature strumenti -->**.   

El filtro de ejecución de reportes permite buscar las *Calibraciones de instrumentos<!-- Tarature strumenti -->* para poder imprimirlas.   

De manera estándar está disponible la siguiente reportería:   
> **Registro de Calibraciones de Instrumentos<!-- Registro Tarature Strumenti -->**: listado de las *Calibraciones de instrumentos<!-- Tarature strumenti -->* agrupables por: *Informe<!-- Rapporto -->*, *Instrumento<!-- Strumento -->* y *Realizada por<!-- Effettuata da -->*.
