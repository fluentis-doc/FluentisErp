La tabla se abre desde la ruta **Tablas > Ventas > Tipos de listas de precios de venta<!-- Tipi listini di vendita -->** y se utiliza para la codificación de listas de precios<!-- listini --> genéricas de venta que se diferencian justamente por este código de las listas de precios<!-- listini --> personalizadas por cliente, las cuales tienen como código la cuenta subcuenta<!-- conto sottoconto --> del cliente.

Para ingresar nuevos registros es necesario hacer clic en la cuadrícula en la primera fila vacía o presionar el botón **Nuevo<!-- Nuovo -->**. Los campos a completar son:     
- **Código/Descripción<!-- Codice/Descrizione -->**: código y descripción de la lista de precios<!-- listino --> general. El código ingresado debe ser único.
- **Precio con IVA incluido<!-- Prezzo Ivato -->**: si está activo, esta casilla indica que la lista de precios<!-- listino --> se creará con precio con IVA incluido, por lo que en sus artículos no estará presente la columna Precio, sino solo la columna Precio con IVA que mostrará directamente la suma entre Precio e IVA.
