La tabla se abre a través de la ruta **Tablas > Ventas > Tipos de Documento EDI<!-- Tipi Documento EDI -->** y se utiliza para determinar las propiedades de un documento EDI<!-- documento EDI -->.

Permite insertar nuevos registros o buscar los ya existentes para visualizarlos, modificarlos o eliminarlos.

El formulario se compone de un área de filtro y una de resultados. Una vez configurados todos los filtros deseados, simplemente haga clic en el botón **Buscar<!-- Ricerca -->** para visualizar los resultados dentro de la cuadrícula de resultados.

Para poder insertar nuevos registros es necesario hacer clic en la cuadrícula sobre la primera fila vacía o presionar el botón **Nuevo<!-- Nuovo -->**.

**Tipo**: código de tipo EDI<!-- codice tipo EDI -->.

**Descripción**: descripción del Tipo EDI<!-- descrizione Tipo EDI -->.

**Tipo num.**: tipo de numeración EDI<!-- tipo numerazione EDI --> asociada al tipo EDI.

**Tipo pedido**: tipo de pedido de cliente<!-- tipo ordine cliente --> para la creación del documento EDI<!-- documento EDI -->.

**Tipo previsión de venta**: se configura automáticamente la previsión semanal para la creación del documento EDI<!-- documento EDI -->.

**Predeterminado**: si está seleccionado, marca el tipo EDI<!-- tipo EDI --> por defecto para la creación automática de documentos EDI<!-- documenti EDI -->.

**Notas**: notas relativas al tipo EDI<!-- tipo EDI -->.
