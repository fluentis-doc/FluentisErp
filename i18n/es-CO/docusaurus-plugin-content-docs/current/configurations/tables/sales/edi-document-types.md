title: Tipos de Documento EDI
sidebar_position: 19
---

La tabla se abre a través de la ruta **Tablas > Ventas > Tipos de Documento EDI** y se utiliza para determinar las propiedades de un documento EDI.

Permite insertar nuevos registros o buscar los ya existentes para visualizarlos, modificarlos o eliminarlos.

El formulario se compone de un área de filtro y una de resultados. Una vez configurados todos los filtros deseados, simplemente haga clic en el botón **Buscar** para visualizar los resultados dentro de la cuadrícula de resultados.

Para poder insertar nuevos registros es necesario hacer clic en la cuadrícula sobre la primera fila vacía o presionar el botón **Nuevo**.

**Tipo**: código de tipo EDI.

**Descripción**: descripción del tipo EDI.

**Tipo de numeración**: tipo de numeración EDI asociada al tipo EDI.

**Tipo de orden**: tipo de orden de venta para la creación del documento EDI.

**Tipos de pronósticos de ventas**: se configura automáticamente la previsión semanal para la creación del documento EDI. 

**Predeterminado**: si está seleccionado, marca el tipo EDI por defecto para la creación automática de documentos EDI.

**Notas**: notas relativas al tipo EDI.

Para todo lo que no se detalla en este documento sobre el funcionamiento común de los formularios, consulte el siguiente enlace [Funcionalidades, botones y campos comunes](/docs/guide/common).
