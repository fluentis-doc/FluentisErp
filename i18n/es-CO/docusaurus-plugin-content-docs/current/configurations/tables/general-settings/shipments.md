La tabla se abre a través de la ruta **Tablas > Configuración General > Envíos** y se utiliza para definir el tipo de envío, es decir, si es por tierra, mar, aéreo, etc.

Permite insertar nuevos registros o buscar los ya existentes para visualizarlos, modificarlos o eliminarlos.

El formulario consta de un área de filtro y una de resultados. Una vez configurados todos los filtros deseados, basta hacer clic en el botón **Buscar** para visualizar los resultados dentro de la cuadrícula de resultados.

Para poder insertar nuevos registros, es necesario hacer clic en la cuadrícula superior en la primera fila vacía o presionar el botón **Nuevo**.

#### Campos específicos

**Código/Descripción**: indicar un código único para el tipo de envío y la descripción correspondiente.

**Código Intra**: indicar el código oficial correspondiente que se utilizará para la declaración intrastat.

Para todo lo que no se detalla en este documento sobre el funcionamiento común de los formularios, consulte el siguiente enlace [Funcionalidades, botones y campos comunes](/docs/guide/common).
