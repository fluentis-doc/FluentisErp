Las tablas del área de Tesorería se refieren a los diferentes módulos de dicha área.

**Estas contienen los datos base y las configuraciones que serán utilizadas en todos los procedimientos operativos.**

:::danger[Atención]
La ausencia de gestión de estos datos base podría impedir el uso efectivo del ERP en el área involucrada.
:::

:::note[Ejemplo]
Un ejemplo de datos base lo representan los Tipos de Efectos y los estados de efectos, necesarios para la emisión de los efectos y todas las operaciones posteriores.
:::

Una parametrización adecuada y necesaria permitirá tanto definir la gestión del módulo correspondiente como la posibilidad de aprovechar determinados automatismos.

Las tablas de Tesorería se encuentran en **Configuración > Tablas > Tesorería**.

El área de tesorería se beneficia en gran medida de la activación del procedimiento de [**Fast Start**](/docs/guide/fast-start).

Se recomienda consultar el enlace anterior para obtener más detalles.

En particular, la página enlazada se refiere al detalle de las tablas impactadas y, en caso de proceder con una configuración manual, ofrece una lista de verificación para la configuración.
