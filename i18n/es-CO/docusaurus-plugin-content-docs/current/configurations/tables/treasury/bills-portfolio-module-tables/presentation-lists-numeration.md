Para definir una numeración es suficiente establecer el año de referencia, el código de la numeración, su descripción y el número inicial<!-- numero di partenza -->.

La tabla es una de las que se gestionan automáticamente en el procedimiento de generación de contadores del nuevo ejercicio contable<!-- esercizio contabile -->, que se encuentra dentro de ‘Herramientas'-‘Utilidades'.
