:::tip[Inicio Rápido (FAst Start)]
La tabla está relacionada con el procedimiento de [**Inicio Rápido (Fast Start)**](/docs/guide/fast-start).

En caso de que desee configurar manualmente, consulte la lista de verificación de la página enlazada.
:::

Para definir una numeración, es suficiente con establecer el año de referencia, el código de la numeración, su descripción y el número de inicio.

La tabla se encuentra entre aquellas gestionadas automáticamente en el procedimiento de generación de contadores del nuevo ejercicio contable, que se encuentra dentro de ‘Herramientas’ - ‘Utilidades’.
