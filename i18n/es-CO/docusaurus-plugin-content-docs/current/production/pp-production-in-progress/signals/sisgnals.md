:::important Para qué sirve<!-- A cosa serve -->
El módulo de **Declaraciones de Producción** de Fluentis es una herramienta esencial para gestionar y monitorear eficientemente el ciclo productivo. Cada declaración comienza con una apertura, seguida de posibles reportes de avance, suspensión, reanudación y cierre del proceso. Los operarios pueden ingresar fácilmente datos cruciales como cantidades producidas, descartadas, materiales empleados<!-- materiali impiegati -->, personal involucrado e indicar lotes y números de serie, asegurando una trazabilidad completa de las operaciones productivas; además, dentro del módulo también está integrado el control de calidad. Este enfoque estructurado optimiza la eficiencia, mejorando el control de las actividades y la gestión de los recursos.
:::

El módulo **Declaraciones<!-- Dichiarazioni -->** permite gestionar las declaraciones de producción<!-- dichiarazioni di produzione -->.
