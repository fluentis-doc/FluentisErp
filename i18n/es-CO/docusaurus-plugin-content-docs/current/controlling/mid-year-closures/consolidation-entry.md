Este formulario sirve principalmente para buscar y visualizar los asientos<!-- scritture --> generados por el procedimiento de *Consolidación<!-- Consolidamento -->* del módulo de cierres intermedios<!-- chiusure infrannuali -->.

Es necesario seleccionar desde la respectiva combo box la consolidación<!-- consolidamento --> de referencia procesada para visualizar los asientos<!-- scritture --> correspondientes.

Al igual que para las Registraciones extracontables<!-- Registrazioni extracontabili -->, desde esta ubicación también será posible completar las operaciones de consolidación<!-- consolidamento --> con asientos<!-- registrazioni --> completamente manuales (por ejemplo, para las participaciones entre empresas del grupo).

También estos asientos<!-- registrazioni --> prevén una causal contable<!-- causale contabile --> de referencia: respecto a los asientos<!-- registrazioni --> normales, cada línea de contabilidad requiere indicar primero la sociedad de referencia de la propia línea.

:::tip[]
Para el procedimiento de configuración y ejecución de la consolidación<!-- consolidamento --> vea [**aquí**](/docs/controlling/mid-year-closures/procedures/consolidation)
:::
