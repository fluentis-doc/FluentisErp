Desde este formulario es posible extraer los diferentes tipos de operaciones relacionados con los activos fijos<!-- cespiti --> seleccionados en la cuadrícula.

El formulario se compone de tres partes:

- la zona de filtro en la parte superior (**[para más detalles ver aquí](/docs/finance-area/fixed-assets/reports-fixed-assets/yearly-fixed-assets-purchase)**);

- la cuadrícula de las categorías de activos fijos<!-- categorie cespiti --> que permite, previa activación del indicador<!-- flag --> **Categoría de activo fijo<!-- Categoria cespite -->**, seleccionar la categoría (o las categorías, presionando la tecla Ctrl) deseada. Si el indicador<!-- flag --> no está activo se extraerán los datos de todas las categorías;

- la cuadrícula de la derecha permite seleccionar el tipo (o los tipos, presionando la tecla Ctrl) deseado. Con el indicador<!-- flag --> **Todas las operaciones<!-- Tutte le operazioni -->** activo se considerarán todos los tipos.
