El "Spesometro" se ha modificado profundamente a partir del año 2017. Desde ese año, de hecho, se prevé el envío trimestral a través de un sistema de intercambio similar al utilizado para la facturación electrónica a la administración pública.

Por lo tanto, aunque siguen estando presentes el formulario y las funcionalidades en cuestión por motivos de retrocompatibilidad y para una posible consulta, ahora existe un nuevo comando específico denominado **[Spesometro 2017 facturas trimestrales](/docs/finance-area/declarations/declarations/spesometro-2017-quarterly).**<!-- **[Spesometro 2017 fatture trimestrali](/docs/finance-area/declarations/declarations/spesometro-2017-quarterly).** -->
