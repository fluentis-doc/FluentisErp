La sección 'Ter' de las declaraciones Intrastat contiene las rectificaciones relativas a declaraciones de periodos anteriores, relacionadas con la cesión de bienes<!-- cessione di beni -->. La sección se completa automáticamente mediante el procedimiento, pero se permite su modificación e integración manual.

En la ribbon bar está presente el botón para agrupar las filas en base a los datos de la contraparte.

| Función | Significado |
| --- | --- |
| Guardar<!-- Salva --> | Botón a través del cual guardar la declaración. |
|  [Creación automática](/docs/finance-area/declarations/intrastat/automatic-creation-intrastat1/automatic-creation)  | Botón que llama al asistente para la recuperación de datos desde el área de ventas. |
|  [Creación desde registros<!-- Creazione da registrazioni -->](/docs/finance-area/declarations/intrastat/create-from-records-intrastat1/create-from-records-intrastat1-intro)  | Botón que llama al asistente para la recuperación de datos desde los registros de contabilidad. |
| Eliminar<!-- Cancella --> | Botón para eliminar la fila de detalle seleccionada. |
