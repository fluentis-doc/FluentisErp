La pestaña Intra 1<!-- tab Intra 1 --> contiene los datos de la portada del modelo: los datos básicos se completan según lo ingresado en la tabla de Sociedad<!-- tabella Società -->.

Quedan para gestión manual las banderas de primera presentación y la de cese de actividad o variación de número de IVA<!-- partita IVA -->, así como la sección del sujeto delegado para la presentación.
