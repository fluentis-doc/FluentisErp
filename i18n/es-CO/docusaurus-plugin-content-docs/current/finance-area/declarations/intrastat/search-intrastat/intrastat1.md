title: Intrastat1
sidebar_position: 2
---

Formulario para la búsqueda específica de los modelos INTRA 1 y la inserción de un nuevo modelo (tanto manualmente como mediante procedimiento automático).

En la barra de opciones (barra de herramientas) también están presentes los comandos para la impresión del modelo; es necesario seleccionar de manera individual en la lista desplegable las diferentes secciones a imprimir además de la portada.

| Función | Significado |
| --- | --- |
| Buscar | Botón para aplicar los filtros de búsqueda a toda la base de datos de las declaraciones Intrastat1 previamente ingresadas. Posibilidad de búsqueda por tipo de resumen, período, año y número de presentación. |
| Nuevo | Llama al procedimiento para insertar una nueva declaración Intrastat1. |
| Modificar | Llama al procedimiento para modificar una declaración Intrastat1 previamente ingresada. |
| Eliminar | Elimina una declaración Intrastat1 previamente ingresada. |
| Exportar a archivo | Botón para la creación y simultánea exportación del archivo electrónico, con guardado del archivo en un directorio elegido por el usuario y en la gestión documental de Fluentis. |
