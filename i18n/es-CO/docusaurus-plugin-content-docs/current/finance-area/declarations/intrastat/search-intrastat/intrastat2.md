title: Intrastat2
sidebar_position: 3
---

Formulario para la búsqueda específica de los modelos INTRA 2 y la inserción de un nuevo modelo (tanto manualmente como mediante procedimiento automático).

En la barra de herramientas (barra de herramientas) también están disponibles los comandos para la impresión del modelo; es necesario seleccionar por separado desde la lista desplegable (combobox) las distintas secciones a imprimir además de la portada.

| Función | Significado |
| --- | --- |
| Buscar | Botón para aplicar los filtros de búsqueda en toda la base de datos de las declaraciones Intrastat2 previamente ingresadas. Existe la posibilidad de buscar por tipo de resumen, período, año y número de presentación. |
| Crear nuevo | Llama al procedimiento para ingresar una nueva declaración Intrastat 2. |
| Modificar | Llama al procedimiento para modificar una declaración Intrastat 2 previamente ingresada. |
| Eliminar | Elimina una declaración Intrastat 2 previamente ingresada. |
| Exportar a archivo | Botón para la creación y simultánea exportación del archivo electrónico, con guardado del archivo en un directorio a elección del usuario y en la gestión documental de Fluentis. |
