Esta pestaña muestra la lista de registros<!-- registrazioni --> de tipo intracomunitario, asignados a sujetos definidos como **intrastat** en el registro maestro<!-- anagrafica -->, **que tengan la pestaña Intrastat completada (dicha pestaña es visible si la causal contable<!-- causale contabile --> tiene el indicador INTRASTAT activo).**

La selección se confirma presionando el botón **Siguiente<!-- Avanti -->** presente en la cinta de opciones<!-- ribbon bar -->, que llevará la visualización a la segunda pestaña de Parámetros<!-- Parametri -->.

| Función | Significado |
| --- | --- |
| Búsqueda<!-- Ricerca --> | Botón para buscar la lista de registros contables de tipo intracomunitario. |
| Seleccionar todo<!-- Seleziona tutti --> | Permite seleccionar todas las filas de la cuadrícula de búsqueda |
| Deseleccionar todo<!-- Deseleziona tutti --> | Permite deseleccionar todas las filas de la cuadrícula de búsqueda |
| Siguiente<!-- Avanti --> | Transfiere las filas seleccionadas a la pestaña de Parámetros<!-- Tab Parametri --> (donde se visualizará la vista previa de las filas del modelo intra para completar los posibles datos faltantes) |
