Esta pestaña muestra la lista de facturas<!-- fatture --> de tipo intracomunitario emitidas a sujetos definidos como intrastat en el registro maestro (**que estén impresas y contabilizadas en el período de referencia de la declaración**).

![](/img/it-it/finance-area/declarations/intrastat/automatic-creation-intrastat1/invoices-filter/image01.png)

La lista de artículos dentro de cada documento no mostrará las filas vinculadas a tipos de IVA<!-- aliquote IVA --> definidos como 'No intra' en la tabla de Tipos/modalidades de IVA<!-- Aliquote/modalità IVA -->.

La selección se confirma presionando el botón **Siguiente<!-- Avanti -->** presente en la barra de herramientas, que llevará la visualización a la segunda pestaña **Parámetros<!-- Parametri -->**.

| Función | Significado |
| --- | --- |
| Buscar<!-- Ricerca --> | Botón para buscar la lista de facturas<!-- fatture --> de tipo intracomunitario. |
| Seleccionar todo<!-- Seleziona tutti --> | Permite seleccionar todas las filas presentes en la cuadrícula de búsqueda. |
| Deseleccionar todo<!-- Deseleziona tutti --> | Permite quitar la selección de todas las filas en la cuadrícula de búsqueda. |
