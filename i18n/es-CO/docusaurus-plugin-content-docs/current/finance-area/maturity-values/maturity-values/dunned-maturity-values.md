Búsqueda de partidas reclamadas

 

El procedimiento de búsqueda de las partidas reclamadas<!-- partite sollecitate --> permite al usuario buscar las reclamaciones<!-- solleciti --> previamente ingresados con el fin de visualizarlos, modificarlos y eventualmente eliminarlos.

| Función | Significado |
| --- | --- |
| Buscar<!-- Ricerca --> | Botón para aplicar los filtros de búsqueda sobre toda la base de datos de reclamos<!-- solleciti --> ingresados. |
| Nuevo<!-- Nuovo --> | Llama al procedimiento para ingresar un nueva reclamación<!-- sollecito -->. Normalmente, la reclamación<!-- sollecito --> se crea con el procedimiento de creación automática; por lo tanto, una inserción manual es muy rara y menos eficiente. |
| Modificar<!-- Modifica --> | Llama al procedimiento para modificar una reclamación<!-- sollecito --> previamente ingresado. |
| Visualizar<!-- Visualizza --> | Llama al procedimiento para visualizar una reclamación<!-- sollecito --> previamente ingresado. |
| Eliminar<!-- Cancella --> | Ejecuta el procedimiento para eliminar una reclamación<!-- sollecito --> previamente ingresado. |

**Para la información relativa a los informes<!-- Stampe --> y procedimientos que siguen, consulte las respectivas opciones de menú<!-- Menù -->**<!-- Per le informazioni, relative alle Stampe  e Procedure che seguono, si rinvia alle rispettive voci di Menù -->

| Función | Significado |
| --- | --- |
| Imprimir intereses<!-- Stampa interessi --> | Llama al procedimiento para imprimir los intereses por retrasos en el pago. |
| Imprimir cartas de reclamación<!-- Stampa lettere di sollecito --> | Llama al procedimiento para imprimir las cartas de reclamación<!-- lettere di sollecito -->. |
