Este formulario permite buscar o ingresar manualmente los pagos<!-- pagamenti -->.

Se recuerda que el sistema de partidas<!-- partitari --> de Fluentis prevé que una partida<!-- partita --> esté abierta o cerrada, respectivamente, si NO está asociada o está asociada a un pago<!-- pagamento -->.

En este formulario se destaca, en particular, en el área de filtro, el **flag Pagos<!-- Pagamenti --> no asociados a partidas<!-- partite -->**, muy útil a nivel de diagnóstico para identificar pagos<!-- pagamenti --> huérfanos de la respectiva partida<!-- partita --> y por lo tanto considerados datos no congruentes que deben corregirse.

En caso de que se ingrese manualmente un pago<!-- pagamento --> mediante el comando **Nuevo<!-- Nuovo -->** (caso particular y generalmente no típico, por ejemplo para cerrar una partida<!-- partita --> abierta extra-contablemente), después de la apertura del formulario de ingreso de nuevo pago<!-- pagamento -->, se señala el comando **Crear desde partidas / pagos<!-- Crea da partite / pagamenti -->**, que permite buscar la partida<!-- partita --> a cerrar creando el pago<!-- pagamento --> correspondiente de manera automática.

También es útil el comando **Vinculación de registros<!-- Collegamento registrazioni -->** para reconstruir la relación (faltante) entre el pago<!-- pagamento --> y el registro contable relacionado si fuera necesario.
