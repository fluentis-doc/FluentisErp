La pestaña de Reversión<!-- scheda di Rollback --> permite deshacer la creación: en este caso específico, se obtiene el mismo resultado ingresando a la búsqueda de partidas solicitadas<!-- ricerca partite sollecitate --> y eliminando directamente desde allí.

| Función | Significado |
| --- | --- |
| Buscar<!-- Ricerca --> | Botón para buscar la creación de solicitudes<!-- creazione solleciti -->. |
| Revertir todo<!-- Rollback tutti --> | Botón para eliminar todas las solicitudes<!-- solleciti --> de la creación seleccionada. |
| Revertir selección<!-- Rollback selezione --> | Botón para eliminar solo algunas solicitudes<!-- solleciti --> de la creación seleccionada. |
