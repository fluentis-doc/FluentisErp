:::note NOTA:<!-- NOTA: -->
Las siguientes instrucciones se refieren al procedimiento de generación automatizada de las auto-facturas<!-- auto-fatture --> de venta creadas con el fin de comunicar al SDI, a través del circuito de facturación electrónica, los datos sobre las compras previamente comunicados mediante la obligación derogada conocida como "Esterometro".<!-- Le seguenti istruzioni riguardano la procedura di generazione automatizzata delle auto-fatture di vendita create al fine di comunicare allo SDI, tramite il circuito della fatturazione elettronica, i dati sugli acquisti precedentemente comunicati tramite l'abrogato adempimento noto come "Esterometro". -->
:::

Dentro de las causales<!-- causali --> de contabilidad general<!-- contabilità generale --> que registran las facturas<!-- fatture --> de compra intra/reverse interno, es necesario indicar cuál es el tipo de factura<!-- fattura --> a generar en las ventas<!-- vendite -->. Por ejemplo, tanto para las facturas<!-- fatture --> como para las notas de crédito de las compras intra indicaremos

![](/img/it-it/finance-area/e-invoice/auto-invoice/ledger-templates1.png)

![](/img/it-it/finance-area/e-invoice/auto-invoice/ledger-templates2.png)

La Agencia Tributaria ha indicado utilizar siempre autofacturas<!-- autofatture --> de tipo TD18 (o las que correspondan al caso específico como TD16, etc.) incluso para enviar los posibles datos de las notas de crédito, asegurándose de que los datos en su interior sean negativos para señalar precisamente la reversión.
