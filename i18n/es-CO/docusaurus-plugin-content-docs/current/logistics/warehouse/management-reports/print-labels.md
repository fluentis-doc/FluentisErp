La impresión se encuentra en la ruta **Logística > Almacén<!-- Magazzino -->  >  Impresiones de gestión > Etiquetas** y permite al usuario consultar la base de datos para obtener una impresión de etiquetas para utilizar en el almacén<!-- magazzino -->. La mayoría de las veces, esta impresión muestra, además de los datos del artículo, también su propio código de barras, que luego puede utilizarse con WMSMobile.

En la cuadrícula se pueden ingresar los artículos para los cuales se desea imprimir la etiqueta. Para insertar un artículo en la cuadrícula, es posible escribirlo desde el teclado o utilizar la *Ayuda de artículos<!-- Help articoli -->* que se activa con doble clic en el campo de matrícula. Cabe destacar que, usando la *Ayuda de artículos<!-- Help articoli -->*, es posible hacer una inserción múltiple mediante la selección múltiple.

Una vez ingresados los artículos deseados, es posible especificar la cantidad de etiquetas a imprimir para cada artículo e imprimir reportes estándar o personalizados (con previa mapeo en la gestión de impresiones). En caso de querer imprimir un reporte personalizado, es necesario elegirlo del listado de los parámetros de impresión.

Para todo lo no detallado en este documento sobre el funcionamiento común de los formularios de impresión, consulte el siguiente enlace [Vista previa e impresión<!-- Anteprima e stampa -->](/docs/guide/common/operations-with-data/reports).
