# Ingreso de inventarios físicos<!-- Inserimento inventari fisici -->

## Introducción

El formulario se abre desde la ruta **Logística > Inventarios físicos**, seleccionando el botón **Nuevo**.  
Desde aquí es posible crear un nuevo inventario físico<!-- inventario fisico --> y gestionar las correspondientes listas de inventario<!-- liste inventariali -->.

## Procedimiento

1. Ingrese **Número**, **Fecha**, **Descripción**, eventuales **Notas** y el **Operador**.  
2. Una vez ejecutadas las diferencias de inventario, active el indicador **Cerrado<!-- Chiusa -->** para marcar el inventario<!-- inventario --> como cerrado, que por lo tanto no será considerado para el procedimiento de alineación de existencias<!-- allineamento giacenze -->.  
3. Complete la cuadrícula de **Listas de inventario<!-- Liste inventariali -->**, indicando:
   - **Tipo** de lista  
   - **Operador**  
   - **Almacén<!-- Magazzino -->** de referencia  
4. Opcionalmente, puede configurar una **Fecha de cierre<!-- Data chiusura -->** para excluir la lista del cálculo de las diferencias de inventario<!-- differenze inventariali -->.  
5. Una vez ingresadas las listas de inventario<!-- liste inventariali -->, es necesario completarlas. Para hacerlo, seleccione una lista de inventario<!-- lista inventariale --> y haga doble clic para acceder a la ventana de edición [Modificar inventario de artículos](/docs/logistics/physical-inventory/inventory-management/inventory-lists-management).

## Resumen y ampliación<!-- Riepilogo e approfondimenti -->

El procedimiento de ingreso de inventarios físicos<!-- inserimento inventari fisici --> permite crear un inventario, definir las listas correspondientes y gestionar su llenado hasta el eventual cierre. Las listas de inventario<!-- liste inventariali --> constituyen la base para la gestión de las diferencias y para la actualización de las existencias<!-- giacenze -->.
