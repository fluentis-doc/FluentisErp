# Inventarios físicos<!-- Inventari fisici -->

:::important[¿Para qué sirve?<!-- A cosa serve -->]
Los **inventarios físicos** en Fluentis permiten crear y gestionar listas de inventario, registrar las existencias físicas y compararlas con las existencias lógicas presentes en el sistema.  
Mediante procesos automatizados, Fluentis posibilita la alineación de las existencias, apoyando una gestión eficiente de los recursos y facilitando la identificación y corrección de discrepancias.

El formulario dedicado permite seleccionar inventarios<!-- inventari --> existentes (compuestos por una o más listas de inventario), modificarlos o crear nuevos. Es posible ingresar información como número, fecha, notas y marcar el inventario<!-- inventario --> como *cerrado* al finalizar las operaciones. Estas funcionalidades aseguran que los inventarios<!-- inventari --> se gestionen eficazmente y que cualquier discrepancia pueda ser solucionada rápidamente.

Los artículos inventariados pueden ser ingresados manualmente o mediante sistemas de asistencia como WMS, lo cual facilita el ingreso y registro de las existencias. Tras completar las listas de recogida, se puede proceder con el cálculo de las diferencias de inventario, haciendo el proceso de gestión de almacén<!-- magazzino --> aún más preciso y eficiente.
:::

El formulario se abre desde la ruta **Logística > Inventarios físicos<!-- Logistica > Inventari fisici -->** y permite seleccionar los inventarios<!-- inventari --> presentes en la base de datos.  
Los inventarios<!-- inventari --> representan la entidad que agrupa una o más listas de inventario.

La ventana está compuesta por:

- una **área de filtro**, para la búsqueda de inventarios<!-- inventari --> según los criterios establecidos  
- una **rejilla de resultados**, que muestra los inventarios<!-- inventari --> encontrados

Después de establecer los filtros deseados, haga clic en **Buscar<!-- Ricerca -->** para poblar la rejilla.

Para abrir los detalles de una lista, basta con:

- seleccionarla y hacer doble clic con el ratón  
- hacer clic en el botón **Modificar<!-- Modifica -->** para editar sus detalles  
- hacer clic en el botón **Visualizar<!-- Visualizza -->** para una consulta no editable

También es posible crear una nueva lista usando el botón **Nuevo<!-- Nuovo -->**.

Para información general sobre el uso de los formularios, consulte la guía:  
[Funcionalidades, botones y campos comunes<!-- Funzionalità, pulsanti e campi comuni -->](/docs/guide/common).
