# Planes de carga<!-- Piani di carico -->

## Consulta de planes de carga<!-- Consultazione piani di carico -->

En la pantalla de **Búsqueda de planes de carga<!-- Ricerca piani di carico -->** es posible consultar los planes de carga<!-- piani di carico --> previamente ingresados y crear listas UDC, picking, facturas y DDT a partir del plan de carga<!-- piano di carico --> seleccionado, utilizando los botones disponibles en el encabezado.

## Funcionalidades disponibles<!-- Funzionalità disponibili -->

Los botones específicos para la creación son:

- **Creación Lista UDC**: crea la Lista UDC a partir del plan de carga<!-- piano di carico --> seleccionado.
- **Creación Picking**: crea el picking a partir del plan de carga<!-- piano di carico --> seleccionado.
- **Creación Factura**: crea la factura UDC a partir del plan de carga<!-- piano di carico --> seleccionado.
- **Creación DDT**: crea el DDT a partir del plan de carga<!-- piano di carico --> seleccionado.

:::important
Para poder realizar la *Creación Picking*<!-- Creazione Picking --> desde el plan de carga<!-- Piano di carico --> es necesario haber especificado previamente las dimensiones<!-- dimensioni --> del plan.  
Las dimensiones<!-- dimensioni --> se calculan automáticamente en base a las dimensiones<!-- dimensioni -->, los pesos y el volumen ingresados para cada artículo en la pestaña [Pesos/Dimensiones<!-- Pesi/Dimensioni -->](/docs/erp-home/registers/items/create-new-item) de su ficha.
:::
