Application Resource Manager, denominado a continuación también con el acrónimo **Arm**, permite la configuración de las aplicaciones y de los parámetros de acceso de las aplicaciones a nivel de base de datos<!-- database --> y usuario<!-- utente -->.

Define los menús aplicativos estándar y personalizados para roles/usuarios<!-- utenti -->. Además, establece restricciones y derechos de acceso a cada procedimiento y funcionalidad de la aplicación.

En lo que respecta al desarrollo de aplicaciones, ARM es la herramienta que permite la consulta, gestión y creación de los objetos de negocio<!-- oggetti business --> utilizables y utilizados por las aplicaciones.

Para más detalles, consulte la *documentación completa de la aplicación ARM* disponible [**aquí<!-- qui -->**](https://docs.fluentis.com/Arm/).
