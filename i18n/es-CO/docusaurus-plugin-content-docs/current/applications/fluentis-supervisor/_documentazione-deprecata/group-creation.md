En Supervisor, las actividades<!-- attività -->, tareas<!-- task --> y eventos están asociados a sus respectivos grupos<!-- gruppi -->. Por lo tanto, existen los siguientes grupos<!-- gruppi -->:

* *Grupos de actividades (activity groups) > Gruppi di attività (activity groups)*
* *Grupos de tareas (task groups) > Gruppi di task (task groups)*
* *Grupos de eventos (event groups) > Gruppi di eventi (event groups)*

La creación de los respectivos grupos<!-- gruppi --> es indispensable para la incorporación de los elementos relacionados con supervisor (actividades<!-- attività -->, grupos<!-- gruppi -->, eventos), ya que estarán contenidos en ellos.

Para crear un nuevo grupo de Actividades, Tareas o Eventos, abra el formulario de gestión de Grupos desde **Configuración** > **Tablas** > **Fluentis Supervisor** > **Grupos de Actividades/Tareas/Eventos<!-- Gruppi Attività/Tasks/Eventi -->**.

Algunas de las funcionalidades dentro del formulario se refieren a un uso estándar. Para cualquier detalle sobre el uso de este formulario no explicitado aquí, consulte este [artículo](/docs/guide/common).
