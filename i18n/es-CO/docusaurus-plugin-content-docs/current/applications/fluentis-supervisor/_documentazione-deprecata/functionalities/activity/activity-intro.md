Las **Actividades<!-- Attività -->** son acciones atómicas que pueden ser configuradas para ser ejecutadas y encadenadas en otros contextos.  
El usuario crea una actividad cuando una acción necesita ser aplicada de manera repetitiva en diferentes escenarios de actuación.  
A través de las actividades es posible definir scripts que realizan una determinada función para que luego puedan ser reutilizados dentro de una Tarea<!-- Task -->.  

Finalmente, las Actividades<!-- Attività --> se organizan en grupos (como por ejemplo *Utilidades<!-- Utilità -->*, *Ejemplos<!-- Esempi -->*, *Demo*) según el contexto de uso.  
Para crear un nuevo grupo de Actividades<!-- Attività -->, consulte el artículo [creación de grupos<!-- creazioni gruppi -->](../../group-creation).

El formulario **Actividades<!-- Attività -->** se abre desde el menú complementario **Herramientas<!-- Strumenti -->** > **Fluentis Supervisor** > **Actividades<!-- Attività -->**.
