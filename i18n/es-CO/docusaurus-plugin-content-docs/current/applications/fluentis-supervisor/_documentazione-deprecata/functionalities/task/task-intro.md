Las tareas<!-- I task --> definen un escenario de acción que se puede crear desde cero o contener actividades<!-- attività -->. La tarea<!-- task --> se puede guardar, ejecutar manualmente o programar.

Para utilizar las *Actividades<!-- Attività -->*, es necesario un contenedor, la **Tarea<!-- Task -->**. Cuando se crea una *Tarea<!-- Task -->*, el script vinculado a ella contiene código predefinido, necesario para su funcionamiento; el usuario tiene la posibilidad de agregar código al script vinculado, utilizando el *Script Editor* o la plataforma codeless *Blockly*.

El formulario de la **Tarea<!-- Task -->** se abre mediante la ruta **Herramientas<!-- Strumenti -->** > **Fluentis Supervisor** > **Tareas<!-- Tasks -->**.

Algunas de las funcionalidades dentro del formulario se refieren a un uso estándar. Para cualquier detalle sobre el uso de este formulario que no esté expresamente indicado aquí, consulte este [artículo](/docs/guide/common).

Las *Tareas<!-- Tasks -->* están organizadas en grupos (como por ejemplo *Utilidades<!-- Utilità -->*, *Ejemplos<!-- Esempi -->*, *Demo*) según el contexto de uso. Para crear un nuevo grupo de *Tareas<!-- Tasks -->*, consulte el artículo sobre [creación de grupos<!-- creazioni gruppi -->](../../group-creation).
