El formulario<!-- form --> relativo a los task<!-- task --> está dividido en otras tres pestañas<!-- tab -->.  
En particular:

Pestaña<!-- Tab --> **Script**
En esta pestaña<!-- tab --> es posible:
* elegir el modo de creación del script (editor visual Blockly o editor textual).
* compilar el script a través del botón correspondiente.
  
Pestaña<!-- Tab --> **Parámetros<!-- Parametri -->**
Se pueden definir y gestionar varios parámetros<!-- parametri --> de entrada y salida que el usuario puede configurar de distintas formas.

Pestaña<!-- Tab --> **Logs de ejecución<!-- Logs Esecuzione -->**
En esta pestaña<!-- tab --> se muestran todos los logs relacionados con la ejecución del script.
