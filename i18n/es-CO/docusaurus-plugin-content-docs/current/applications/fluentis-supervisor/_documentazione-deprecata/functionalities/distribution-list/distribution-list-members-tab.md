:::info Nota<!-- Nota -->
Una vez abierta la forma de detalle de la *Lista de Distribución<!-- Lista di Distrubuzione -->*, en la parte superior están presentes algunos parámetros estándar como *Código<!-- Codice -->*, *Nombre<!-- Nome -->*, *Descripción<!-- Descrizione -->*, *Fecha de Creación<!-- Data Crazione -->*, *Fecha de Última Modificación<!-- Data Ultima Modifica -->*, *Usuario de Creación<!-- Utente di Creazione -->*, *Usuario de Modificación<!-- Utente di Modifica -->* y *Lista de Distribución Relacionada<!-- Lista di Distribuzione Collegata -->* (este último, se refiere a las listas CRM).
:::
En esta pestaña<!-- tab --> pueden definirse listas de distribución compuestas por los siguientes parámetros<!-- parametri -->:
- código<!-- codice -->;
- descripción<!-- descrizione -->;
- fecha de creación<!-- data di creazione -->;
- fecha de última modificación<!-- data di ultima modifica -->.

Además, expandiendo mediante el ícono **'+'** es posible definir los contactos que componen la lista de distribución<!-- lista di distribuzione -->.
