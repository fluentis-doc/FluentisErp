Un <mark>Evento</mark> es una funcionalidad de **Supervisor** que permite al usuario ejecutar de manera programada un **Task** según una planificación temporal.  
Se podrá ejecutar un task<!-- task --> de forma automatizada de manera horaria, diaria, semanal o mensual.  

El formulario **Eventos<!-- Eventi -->** se abre desde el menú complementario **Herramientas<!-- Strumenti -->** > **Fluentis Supervisor** > **Eventos<!-- Eventi -->**.

Algunas de las funcionalidades dentro del formulario se refieren a un uso estándar. Para cualquier detalle sobre el uso de este formulario no especificado aquí, consulte este [artículo](/docs/guide/common).

Los *Eventos<!-- Eventi -->* están organizados en grupos (como por ejemplo *Utilidades<!-- Utilità -->*, *Ejemplos<!-- Esempi -->*, *Demo<!-- Demo -->*) según el contexto de uso. Para crear un nuevo grupo de *Eventos<!-- Eventi -->* consulte el artículo [creación de grupos<!-- creazioni gruppi -->](../../group-creation).
