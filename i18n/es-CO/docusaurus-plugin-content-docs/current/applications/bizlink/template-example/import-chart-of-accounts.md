La hoja de Excel (plantilla) para la importación del plan de cuentas<!-- piano dei conti --> en el módulo de Administración<!-- Amministrazione --> permite poblar la sección de detalle del plan de cuentas<!-- piano dei conti -->.

**Por lo tanto, ya debe estar presente la parte de la estructura (grupos) en la cuadrícula superior del formulario del plan de cuentas<!-- piano dei conti -->.**

**Columnas Grupo Nivel (1, 2, 3, 4, 5):** En estas columnas, indique los códigos de los diferentes niveles y subniveles (posibles hasta el quinto) donde colocar las cuentas<!-- conti --> a importar.

ATENCIÓN: se buscan grupos ya existentes, la importación no crea nuevos grupos.

**Cuenta<!-- Conto --> / Subcuenta<!-- Sottoconto --> / Descripción<!-- Descrizione -->:** ingrese el código de la cuenta<!-- conto --> y subcuenta<!-- sottoconto --> y la descripción a importar (en este caso se creará la cuenta<!-- conto --> / subcuenta<!-- sottoconto --> con el código ingresado).

**Descripción 2<!-- Descrizione 2 -->:** también es posible completar la segunda descripción (opcional) para la nueva cuenta<!-- conto -->.

**Tipo de cuenta<!-- Tipo conto -->:** complete con el código de uno de los tipos de cuentas<!-- tipo conti --> ya presentes en Fluentis para asignarlo a la nueva cuenta<!-- conto --> que se crea.
