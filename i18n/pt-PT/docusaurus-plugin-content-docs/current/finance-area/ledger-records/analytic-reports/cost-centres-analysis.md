---
title: Análise centros de custo
sidebar_position: 1
---

O relatório da Análise centros de custo permite imprimir o detalhe dos movimentos contábeis que tem valorizado para cada centro de custo, totalizando as quantias sobre o mesmo centro de custo. Entre os vários filtros presentes na máscara, é interessante explicar aquele denominado 'Imprensa até o detalhe do nível número': dato que a estrutura dos centros de custo é múlti-nível, essa opção (que executa um relatório com estrutura tabelar específica) permite pedir para a imprensa um específico nível, que vai apresentar a soma das quantias de todos os níveis inferiores aquele selecionado (p.ex. se você imprimi o segundo nível significa que vai visualizar só os CdC desse mas com os dados do terceiro, quarto, enésimo nível abaixo).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa da análise dos centros de custo. |
| Imprimir | Botão para executar a imprensa física da análise centros de custo. |
