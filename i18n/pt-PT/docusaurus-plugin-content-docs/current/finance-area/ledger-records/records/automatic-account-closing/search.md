RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para a pesquisa de um encerramento das contas efetuadas precedentemente. |
| Nova | Botão para criar um novo encerramento das contas. |
| Modificar | Botão, ativável depois de ter selecionado um encerramento das contas, que permite a mudança da registração de encerramento. |
| Visualizar | Botão, ativável depois de ter selecionado um encerramento das contas, que permite a visualização da registração de encerramento. |
| Cancelar | Botão, ativável depois de ter selecionado um encerramento das contas, que permite o cancelamento da registração de encerramento. |
