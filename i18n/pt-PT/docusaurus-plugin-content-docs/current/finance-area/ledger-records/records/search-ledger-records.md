---
title: Procurar registrações contábeis
sidebar_position: 7
---

O procedimento Procurar registrações contábeis foi desinhado para permitir ao usuário procurar facilmente as registrações precedentemente inseridas com o objetivo de visualizá-las, modificá-las, eventualmente cancelá-las ou proceder com a inserção de uma nova registração.

É possível a partir da Gestão de pesquisa das registrações contábeis fazer todas as operações necessárias no âmbito das mesmas registrações, após a possibilidade de ter os direitos necessários.

O procedimento Procurar registrações contábeis é formado na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar registração	 | Botão para aplicar os filtros de pesquisa dentro do banco de dados das registrações contábeis inseridas. |
| Nova registração | Botão para inserir uma nova registração contábil. |
| Modificar registração | Botão para modificar uma registração contábil precedentemente inserida. |
| Visualizar registração | Botão para visualizar uma registração contábil precedentemente inserida. |
| Cancelar registração | Botão para cancelar uma registração contábil precedentemente inserida. |
| Documento origem | Botão através da qual chamar de novo o documento de origem pela qual foi originada a registração contábil. |



**Para as informações relativas as imprensas que seguem, se envia aos respectivos elementos de menu.**



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Imprensas analítica	](/docs/finance-area/ledger-records/analytic-reports/cost-centres-analysis)  | Refere-se ao procedimento de contabilidade analítica para imprimir a situação relativa a: - Analises centros de custo;- Analises centros de lucro;- Analises movimentos CdC;- Controle desequilíbrios CdC/CdL/encomendas. |
|  [Imprensas contábeis](/docs/finance-area/ledger-records/accounting-report/trial-balance)  | Refere-se ao procedimento de contabilidade geral para imprimir, em relação os critérios de filtro especificados, a situação relativa a:- Extrato bancário;- Balanço de verifica;- Agenda;- Extrato contrapartidas;- Resumo movimentos para a causal;- Resumo movimentos por tipo documento;- Situação subconta;- Subcontas. |
|  [Imprensas fiscais](/docs/finance-area/ledger-records/fiscal-report/period-balance-sheet)  | Refere-se ao procedimento para imprimir, em relação os critérios de filtro especificados, as situações relativas a:- Balanço de exercício;- Declaração de intenção;- Livro jornal;- Liquidação Imposto anual- Liquidação Imposto periódico;- Liquidação Imposto 74 ter;- Módulos marcados;- Registro declaração de intenção;- Registro Imposto correspondentes;- Registros Imposto;- Resumo por códigos Imposto. |
| Imprensas financeiras | Refere-se ao procedimento para imprimir, em relação com os critérios especificados, as situações relativas a:- Diferença entre o saldo contábil e partidas;- Adiamento médio cobranças e pagamentos;- Extrato bancário com partidas abertas;- Lista partidas;- Agrupamentos partidas;- Resumo vencimentos;- Plano de trabalho clientes/fornecedores. |



**Para as informações relativas aos procedimentos que seguem, se envia para os respectivos elementos de menu.**



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Compensação partidas | Botão para gerir a compensação das partidas ativas/passivas das várias subcontas. |
| Agrupamento partidas | Botão para gerir a criação de um ou mais novos grupos partidas. |
| Saldar a conta | Botão para gerir a relevação das cobranças das partidas ativas. |
| Escrituras de ajustamento | Botão para a gestão das escrituras de ajustamento de final de ano. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das registrações que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das registrações, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das registrações que quiser.
