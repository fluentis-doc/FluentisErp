sidebar_position: 1
---

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para a pesquisa de uma abertura das partidas efetuadas precedentemente. |
| Nova abertura | Botão para criar uma nova abertura contas. |
| Modificar registração | Botão, ativável depois de ter selecionado uma abertura contas, que permite a mudança da registração de abertura. |
| Visualizar | Botão, ativável depois de ter selecionado uma abertura contas, que permite a visualização da registração da abertura. |
| Cancelar | Botão, ativável depois de ter selecionado uma abertura contas, que permite o cancelamento da registração de abertura. |
