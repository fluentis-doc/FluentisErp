title: Criação ajustamento
sidebar_position: 1
---

Com este procedimento se pode fazer calcular ao procedimento os dados dos acréscimos e diferimentos relativos as registrações contábeis presentes no banco de dados: se trata de retificações/integrações (respectivamente diferimentos e acréscimos) relativos a assegurações, locações, cânone de leasing ou assistência, que tem como características própria aquelas de ser relativos a um período.

As condições para usufruir este procedimento são:

1) foram definidos, na tabela Tipos conta, os tipos de conta econômicas 'a retificar' que se identificam para o flag 'Serviço' ativado. O cálculo é efetuado, assim como nos balanços infra-anuais, não para todas as subcontas mas só para aquelas no plano das contas foi atribuída este tipo de conta de custo ou de receita particular.

2) foram definidos nos parâmetros de contabilidade as subcontas padrão de relevação dos movimentos de acréscimos/diferimentos ativo/passivo. Se lembra que no nível do plano das contas, em cada subconta de custo/receita podem ser especificadas subcontas de acréscimos/diferimentos que vão ser utilizados em via prioritária para a relevação da integração ou retificação.

3) Foram utilizados corretamente as gamas de datas econômicas de competência nos movimentos de contabilidade. Se lembra, que para quem quiser utilizar a opção, se aconselha de estabelecer os parâmetros de inserção usuário com a opção 'Nenhuma data por tipo conta serviço' com o objetivo de obrigar o usuário a inserir, vez por vez, a gama de datas econômicas válidas para cada movimento.
