sidebar_position: 1

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar ajustamento | Botão para a pesquisa das escrituras de ajustamento. |
| Criação ajustamento | Botão para a criação das escrituras de ajustamento. |
| Modificar | Botão, ativável depois de ter selecionado uma escritura de ajustamento, que permite a mudança. |
| Visualizar | Botão, ativável depois de ter selecionado uma escritura de ajustamento, que permite a visualização. |
| Cancelar | Botão, ativável depois de ter selecionado um encerramento das contas, que permite o cancelamento. |
