A ficha 'Custo-lucro' está presente só se a causal tem ao menos um dos dois flag de gestão CdC/CdL.

A ficha Custo-Lucro é dedicada a gestão do centro de custo/lucro dos dados econômicos relevados na registração: em relação das impostações da causal vai estar presente uma seção, uma outra, ou ambas (nesse último caso com duas opções: no caso de 'Gestão separada Cdc/CdL' ambas vão ser a valorizar contemporaneamente, caso contrário, o procedimento vai atribuir a quantia só para uma das duas seções, os custos vão nos centros de custo e as receitas nos centros de lucro). Se foi ativada a opção 'Gestão máquina' vai estar presente também uma grelha de valorização custos/receitas aos automóveis das empresas.

O carregamento dos dados nas seções CdC/CdL pode ser automatizado (para as registrações manuais diretamente na contabilidade) em duas maneiras: através da atribuição no plano das contas, sobre as subcontas de custo/receita, da valorização porcentagem relativa, ou nos registros clientes/fornecedores, sempre na porcentagem (nesse caso todas as subcontas econômicas inseridas na registração titular do sujeito vão ser valorizadas na mesma maneira). A prioridade de valorização se estabelece dentro dos parâmetros de contabilidade geral.

São permitidas mudanças anuais com as quantias calculadas, assim como diretamente uma inserção manual das linhas: nesse segundo caso o procedimento vai propor automaticamente as quantias para cobrir, subconta por subconta, o total movimentado na contabilidade, reportando a mesma gama de datas de competência. É importante avisar, de fato, que os procedimentos de retificação e integrações preveem também as retificações dos movimentos atribuídos ao centro de custo/lucro.

No momento do salvamento da registração o aplicativo vai proceder a retornar uma máscara com os detalhes das diferenças de valorização, bloqueando eventualmente o salvamento se assim foi estabelecido na causal em utilização.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar a registração. |
| Nova registração | Botão para criar uma nova registração. |
| Parâmetros de inserção | Refere-se aos parâmetros de inserção da primeira nota. |
| Abrir registro | Botão para chamar de novo a gestão registro da subconta selecionada, ou em ausência, a pesquisa de um outro registro. |
| Cancelar os centros de custo selecionados | Botão para cancelar os centros de custo selecionados. |
| Cancelar os centros de lucro selecionados | Botão para cancelar os centros de lucro selecionados. |
| Expandir/Reduzir | Botão para reduzir a visualização só da parte dos centros de custo ou lucro e vice-versa ou expandir ambas. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document Manager](/docs/guide/common/operations-with-data/document-manager)  | Botão para a ligação com a gestão documental. |
