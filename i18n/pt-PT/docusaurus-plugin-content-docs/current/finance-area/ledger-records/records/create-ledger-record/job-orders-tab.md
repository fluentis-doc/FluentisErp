A ficha 'Encomendas' está presente só se a causal tem o flag 'Encomendas' estabelecido.

Neste tabulador vão ser atribuídos as várias encomendas de venda os dados econômicos relevados na registração de contabilidade: automaticamente vão ser atribuídos todas a encomenda inserida no cabeçalho, mas o usuário vai poder subdividir as quantias em maneira diferente manualmente diretamente nesta seção.

Não estão previstos automatismos na inserção manual das quantias, nem as gamas de datas de competência econômica, nem os controles do balanceamento das quantias com a contabilidade geral.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar a registração. |
| Nova registração | Botão para criar uma nova registração. |
| Parâmetros de inserção | Refere-se aos parâmetros de inserção da primeira nota. |
| Abrir registro | Botão para chamar de novo a gestão registro da subconta selecionada, ou em ausência, a pesquisa de um outro registro. |
| Cancelar encomendas selecionadas | Botão para cancelar as encomendas selecionadas. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document Manager](/docs/guide/operations-with-data/document-manager)  | Botão para a ligação com a gestão documental. |
