A máscara apresenta na parte superior os filtros de pesquisa para individuar a lista das registrações provisórias no sistema. Uma vez selecionada uma das linhas da/s registração/ões visualizada/as, vão-se ativar acima os botões de gestão: com 'Mudança registração' vai ser possível abrir na mudança a mesma registração, para intervir em vários dados e eventualmente modificar diretamente o estado dos seus parâmetros; com o 'Visualizar registração' vai ser possível visualizar a registração sem poder modificar; o último botão 'Geração' invés, vai executar a troca do estado em relação os parâmetros estabelecidos na base da máscara, que abaixo vamos detalhar.

Na base da máscara estão presentes dois campos com a data registração e a data competência a utilizar para registrar em maneira definitiva as registrações provisórias selecionadas, além disso, também a opção 'Data registração igual a última data de registro Imposto'. Se este flag é ativo, a contabilização vai acontecer com a última data utilizada para o registro Imposto (se presente) de cada registração, em modo de colocar na fila as registrações definitivas já presentes.

O procedimento da 'Gestão registrações provisórias' é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Este botão executa a pesquisa das registrações provisórias em relação os filtros estabelecimentos. |
| Modificar registração | Abre na mudança a registração contábil. |
| Visualizar a registração | Abre em visualização a registração selecionada. |
| Geração | Executa a contabilização definitiva das registrações selecionadas em relação os parâmetros estabelecidos. |
