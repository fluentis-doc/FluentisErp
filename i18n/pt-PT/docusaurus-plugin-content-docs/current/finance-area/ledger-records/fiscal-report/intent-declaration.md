---
title: Declaração de intenção
sidebar_position: 2
---

Na abertura da janela se propõe o exercício contábil atual.

É possível imprimir todas as declarações de intenção ou filtrar a imprensa para:

**Tipo declaração**: cliente ou fornecedor;

**Procurar uma subconta de registro cliente ou fornecedor**: em função de quanto selecionado no campo precedente;

**Número de nosso protocolo da/a**.

Entre os elementos a imprimir é possível variar a descrição do campo Escritório Imposto de - e o campo descrição operação.

A data de imprensa proposta é a data atual do sistema.

É possível estabelecer mais números de cópias para a declaração; o valor de default é 1.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa das declarações de intenção. |
| Imprimir | Botão para executar a imprensa física das declarações de intenção. |
