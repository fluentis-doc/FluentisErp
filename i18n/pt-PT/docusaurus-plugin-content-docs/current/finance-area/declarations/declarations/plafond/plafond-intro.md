O procedimento da pesquisa Plafond foi desinhado para permitir ao usuário procurar facilmente os Plafond inseridos precedentemente com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. A partir da pesquisa é possível fazer todas as operações necessárias no âmbito do documento, após a presença dos direitos usuários necessários.

O procedimento da pesquisa Plafond é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar Plafond | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos Plafond inseridos. |
| Novo Plafond | Chama o procedimento para inserir um novo Plafond. |
| Modificar Plafond | Chama o procedimento para modificar um Plafond precedentemente inserido. |
| Visualizção Plafond | Chama o procedimento para visualizar um Plafond precedentemente inserido. |
| Cancelar Plafond | Executa o procedimento para cancelar um Plafond precedentemente inserido. |
