---
title: Gestão Plafond
sidebar_position: 5
---

De mês em mês vai precisar entrar na máscara de gestão do mesmo para operar o cálculo atualizado dos dados através do botão 'Propor valores'. Para anular o cálculo de um mês precisa anular os períodos do último mês presente na lista até aquele interessado. O dado do valor disponível no começo do mês é visível na coluna relativa.

O dado do Plafond é resumido na 'Imprensa da liquidação Imposto periódico', mas só com os relatórios denominados 'CA_LiquidazioneIVAPer(plafond).rpt' ou 'CA_LiquidazioneIVAPer(PlafMese).rpt': no primeiro mês se visualiza a situação do Plafond do segundo mês antecedente a liquidação imprimida, no segundo a situação do mesmo mês.

O procedimento é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Novo Plafond | Estabelece o cursor numa nova linha de criação dos dados. |
| Cancelar Plafond | Cancel a linha de cálculo selecionada. Não é possível cancelar os dados intermédios, só o último da lista. |
| Propor valores | Executa o cálculo para o novo mês, propondo os valores na grelha. |
| Ir aos dados iniciais | Anula o Plafond para redefinir os dados da partida. |
