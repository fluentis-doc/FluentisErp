title: General Overview
sidebar_position: 1
---

O módulo Intrastat é dedicado a criação das declarações estatísticas Intrastat: é prevista seja uma gestão manual dos dados que uma inserção automática em relação os dados presentes dentro das faturas de compra e de venda das relativas áreas de compras e vendas, assim como uma retomada dos dados dos serviços das registrações contábeis. O módulo se encontra na área administrativa dentro do módulo Declarações.

As impostações de base são associadas com os registros e com os tipos de documento em compra e venda: o documento tem que ser de tipo intracomunitário e que o registro titular do mesmo foi definido como sujeito Intrastat. Para obter o máximo dos automatismos vai precisar definir, nos registros artigos, os códigos nomenclatura de cada um assim como os eventuais fatores de conversão para obter a valorização automática da unidade de medida complementar. Claramente necessário, o quanto possível, definir dentro dos vários documentos em maneira completa os dados do tipo de transporte e as condições de retorno.

A utilização do módulo se efetua com a criação de uma nova declaração Intrastat1 (para as vendas) e Intrastat2 (para as compras): se vai poder completar os dados retirando-os das faturas de compra e vendas com o procedimento automático, ou das registrações contábeis para os serviços a reportar nas seções quater/quinquies, ou com a inserção manual nas seções relativas.

Completados os dados dos modelo vai ser possível proceder com a imprensa dos módulos ou com o envio telemático dos mesmos com a criação do arquivo previsto.
