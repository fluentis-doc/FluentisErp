---
title: Procurar a lista de adiantamento
---

O procedimento Procurar a lista de adiantamento foi desinhado para permitir ao usuário procurar facilmente as listas de adiantamento inseridas precedentemente com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. A partir da pesquisa é possível fazer todas as operações necessárias, após a presença dos direitos do usuário necessários.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar listas | Botão para aplicar os filtros de pesquisa dentro do banco de dados das listas de adiantamentos precedentemente inseridos. |
| Nova lista | Refere-se ao procedimento para inserir uma nova lista de adiantamento. |
| Modificar a lista | Refere-se ao procedimento para modificar uma lista de adiantamento precedentemente inserida. |
| Visualizar a lista | Refere-se ao procedimento para visualizar uma lista de adiantamento precedentemente inserida. |
| Cancelar as listas | Refere-se ao procedimento para cancelar uma lista de adiantamento precedentemente inserida. |

 

Para as informações relativas as Imprensas e os Procedimentos que seguem, se envia aos respectivos elementos de menu.



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Imprimir](/docs/treasury/advances-collections/reports)  | Refere-se ao procedimento de impresa relativa a lista das faturas adiantadas/cobradas. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Gestão cobranças | Botão para proceder com a gestão das cobranças. |
|  [Contabilização adiantamentos cobranças](/docs/treasury/advances-collections/accounting/accounting)  | Botão para a criação da registração contábil de adiantamentos cobranças. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma listade adiantamento que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das listas de adiantamento, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista de adiantamento que quiser.
