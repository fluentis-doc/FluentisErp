---
title: Criação automática cobranças
sidebar_position: 2
---

Nesta máscara é possível estabelecer uma lista de clientes a não cobrar, uma lista de partidas excluídas das cobranças e o tipo de cobrança de default a criar.

Uma vez salvas essas impostações se vai ativar o botão de criação automática: com este aplicativo se procede a criar as cobranças para todas as partidas que satisfazem os critériso de seleção estabelecidas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar a geração cobranças. |
| Geração cobranças | Botão para efetuar a geração cobranças das partidas que sadisfazem os critérios de seleção estabelecidos. |
| Procurar cliente a excluir | Botão para a pesquisa dos clientes que se querem excluir da geração automática das cobranças. |
| Cancelar clientes | Botão para cancelar os clientes, da seção lista clientes excluídos da cobrança. |
| Procurar partidas a excluir  | Botão para a pesquisa das partidas que se quer excluir. |
| Cancelar as partidas | Botão para cancelar as partidas a excluir precedentemente selecionadas. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das partidas cobradas que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das partidas cobradas, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das partidas cobradas que quiser.
