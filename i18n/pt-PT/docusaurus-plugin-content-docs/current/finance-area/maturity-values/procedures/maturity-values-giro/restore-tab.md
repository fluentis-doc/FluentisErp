title: Tab Restauração
sidebar_position: 2

É presente uma ficha final 'Restauração' para visualizar em pré-visualização a registração efetuada ou também para anulá-las através do adequado botão.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar a transferência das partidas efetuadas precedentemente. |
| Restaurar a transferência das partidas | Botão para visualizar e/ou restaurar a transferência das partidas. |
