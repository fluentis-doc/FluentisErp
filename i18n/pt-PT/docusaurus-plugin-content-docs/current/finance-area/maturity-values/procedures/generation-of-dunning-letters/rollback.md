A ficha Redução permite anular a criação: no caso específico o mesmo resultado se obtem entrando na pesquisa cobrança e cancelando diretamente daqui.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para a pesquisa da criação cobranças. |
| Redução todos | Botão para cancelar todas as cobranças da criação selecionada. |
| Redução seleção | Botão para cancelar só algums cobranças da criação selecionada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das cobranças criadas que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das cobranças, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das cobranças que quiser.
