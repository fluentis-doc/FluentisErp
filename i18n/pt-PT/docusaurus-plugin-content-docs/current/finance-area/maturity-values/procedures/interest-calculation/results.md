No tabulador Resultados é possível visualizar e cancelar o resultado dos cálculos efetuados, com a possibilidade de poder filtrar a interrogação para a conta/subconta e tipo de pagamento.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar resultados | O botão procura os resultados de cálculo executados. |
| Cancelar resultados | O botão cancela os resultados de cálculo executados. |
| Cancelar os detalhes selecionados | O botão cancela só os detalhes selecionados de um determinado cálculo. |
