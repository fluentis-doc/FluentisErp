---
title: Criação dos vencimentos
---

Com esta ajuda o usuário pode inserir na lista cada vencimento das faturas emitidas: nessa maneira vai ser possível utilizar o procedimento de contabilização adiantamentos/cobranças do módulo, com a qual vão ser relevadas as cobranças cliente e vão atualizar os estados dos vencimentos dentro da lista.

De fundamental importância, nessa máscara, é proceder a estabelecer a porcentagem de adiantamento e a base se aplicação do mesmo, na escolha entre o tributável e o total fatura. Na lista dos documentos vão ser apresentadas só as faturas imprimidas, não pró-forma, que não já não foram adiantadas numa outra lista.

Uma vez reportadas na lista, as quantias dos adiantamentos não vão ser mais modificáveis.

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos vencimentos que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos vencimentos, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos vencimentos que quiser.
