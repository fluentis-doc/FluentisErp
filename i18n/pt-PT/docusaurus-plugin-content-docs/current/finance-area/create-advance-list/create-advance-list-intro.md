---
title: Criar lista de adiantamento
---

O cabeçalho da lista de adiantamento prevê antes de tudo a seleção do tipo de lista a utilizar: o número sucessivo é atribuído automaticamente, assim como o ano e a data atual de sistema (seja como data de apresentação que como data de emissão).

No campo **Banco c/c** se indica a subconta do registro do banco que vão ser apresentadas para o adiantamento das faturas. Uma vez atribuída uma descrição para essa lista se estabelece também o estado da lista: a gestão, só manual, do estado vai ser estabelecida na 'Lista emitida' ou na 'Lista apresentada'. Uma vez confirmada também esse dado essencial, o sistema vai proceder a salvar os dados do cabeçalho e a ativar os botões de seleção sucessivos, com as quais o usuário vai poder chamar de novo as faturas da área de venda ou cada vencimento na fatura: lembra-se que uma lista pode ser gerida só com a Ajuda faturas ou com a Ajuda vencimentos, não pode obter os dados para ambas.

Dentro da máscara está presente um botão de execução da pré-visualização de imprensa e da imprensa direta da mesma.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar adiantamentos | Botão para salvar a lista de adiantamento, ativo depois de ter completado os dados do cabeçalho. |
|  [Aquisição faturas](/docs/treasury/advances-collections/create-advance-list/create-advance-list)  | Refere-se ao procedimento para comprar as faturas da área de venda. Com um duplo click na fatura selecionada se pode entrar ma mudança da mesma fatura. |
| Aquisição com vencimentos | Refere-se ao procedimento para comprar cada vencimento das faturas. |
| Cancelar as faturas | Botão para cancelar da lista uma fatura precedentemente selecionada. |
| Pré-visualização das faturas inseridas | Botão para proceder com a pré-visualização da lista. |
| Imprimir faturas | Botão para proceder com a imprensa da lista. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document manager](/docs/guide/operations-with-data/document-manager)  | Botão para a ligação com a gestão documental. |
