---
title: Simulação planos de amortização
sidebar_position: 3
---

Cada simulação é associada com o usuário que a calculou: na abertura da máscara vai ser disponível o botão para visualizar o último cálculo efetuado pelo usuário (esta elaboração pode ser exigente em termos de tempo e recursos de elaboração).

A gama máxima de anos previstos para a simulação é uma gama de 10 anos. Os flag sucessivos permitem definir quais tipos de propriedade (propriedade, leasing, aluguel) sejam a considerar na simulação.

Na lista abaixo se vai a indicar os diferentes planos de amortização a calcular: de cada um vai ser possível definir quais tipos de amortização vão ser simulados, com a opção também fiscal ou não. O flag Calcular identifica por quais planos se procede com a simulação.

Uma vez salvas as impostações se procede com o cálculo apertando o adequado botão: no final da elaboração vai ser visualizada uma máscara de resumo dos totais de categoria para cada ano/plano ou para cada plano/ano. Pode-se trocar a visualização entre um e outro critério com o adequado botão. A partir dessa máscara é prevista uma exportação Excel.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Salva as impostações de cálculo para o usuário ativo. |
| Visualizar último cálculo | Visualiza o resultado do último cálculo executado. |
| Calcular | Executa o cálculo em relação as impostações salvas. |
