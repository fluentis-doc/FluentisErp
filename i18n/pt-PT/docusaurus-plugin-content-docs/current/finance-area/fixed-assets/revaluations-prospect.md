A máscara de imprensa do Prospecto reavaliações permite definir os filtros de pesquisa na extração dos dados.

O relatório permite resumir os dados de detalhe das reavaliações associadas com os vários ativos fixos, em relação com o código de agrupamento estabelecido na tabela categorias ativos fixos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa física do documento. |
