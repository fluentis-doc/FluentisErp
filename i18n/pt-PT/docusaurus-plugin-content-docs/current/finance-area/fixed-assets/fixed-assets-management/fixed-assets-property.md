---
title: Propriedade ativos fixos
sidebar_position: 5
---

Última ficha padrão é aquela relativa as 'Propriedade do ativo fixo': é possível atribuir uma lista de propriedade diferentes do ativo fixo em utilização, com uma eventual nota explicativa acrescentada.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nova propriedade | Ativa o cursor para a inserção de uma propriedade ativo fixo. |
| Cancelar propriedade | Cancela a propriedade selecionada na grelha. |
