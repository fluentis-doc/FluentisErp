---
title: Procurar ativos fixos
sidebar_position: 6
---

O procedimento Procurar ativos fixos foi desinhado para permitir ao usuário procurar facilmente os ativos fixos precedentemente inseridos com o objetivo de visualizá-los, modificá-los e eventualmente cancelá-los. A partir desse procedimento é possível fazer todas as operações necessárias no âmbito do documento, após a presença dos direitos do usuário necessários.

O procedimento é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar ativos fixos | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos ativos fixos inseridos. |
| Novo ativo fixo | Se refere ao procedimento para inserir um novo ativo fixo. |
| Modificar ativo fixo | Se refere ao procedimento para modificar um ativo fixo precedentemente inserido. |
| Visualizar ativo fixo | Se refere ao procedimento para visualizar um ativo fixo precedentemente inserido. |
| Cancelar ativo fixo | Executa o procedimento para cancelar um ativo fixo precedentemente inserido. |
| Sbdividir o ativo fixo | O botão, ativo só se não foram selecionados mais ativos fixos, permite abrir uma máscara de subdivisão do ativo fixo de esse ainda não foi amortizado. Ver a aplicação do cabeçalho A. |
| Trasferência ativo fixo | Executa o procedimento para criar uma máquino do módulo automóveis. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma procurar sobre o total dos ativos fixos inseridos. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos ativos fixos que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos ativos fixos, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos ativos fixos que quiser.

APLICAÇÕES CABEÇALHO:

A: Subdividir o ativo fixo na máscara, definir o número de fichas a abrir em cada divisão/categoria ativo fixo. Confirmar com SIM para executar a operação ou anular com o NÃO.
