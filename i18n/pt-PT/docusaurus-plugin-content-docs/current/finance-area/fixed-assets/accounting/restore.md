---
title: Restauração
sidebar_position: 3
---

A ficha Restauração permite visualizar a registração contábil através dum duplo click na seção relativa as registrações, assim como a anulação parcial/total das operações selecionadas. A anulação vai ser possível só se o período é ainda modificável (imprensa livro jornal e encerramento contas não executadas).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura a lista das contabilizações executadas. |
| Restaurar contabilização | Executa a restauração das registrações contábeis selecionadas. |
| Visualizar ativo fixo | Abre em visualização o registro do ativo fixo cuja amortização foi selecionada na grelha. |
| Visualizar registração contábil | Visualiza a registração contábil selecionada. |
