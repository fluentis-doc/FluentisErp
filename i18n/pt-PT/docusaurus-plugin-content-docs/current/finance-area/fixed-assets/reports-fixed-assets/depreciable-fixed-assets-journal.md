title: Livro ativos fixos amortizáveis
sidebar_position: 4
---

A máscara de imprensa do livro ativos fixos permite definir os filtros de pesquisa na extração dos dados.

Em particular merece atenção: normalmente os ativos fixos vendidos ou destruídos nos anos precedentes o ano de referência inserido não vai ser mais reportados na imprensa, se não impondo o adequado flag. Também para os ativos fixos 'Cessados' é previsto um botão de inserção no relatório. Na mesma maneira vão ser excluídos da imprensa os ativos fixos que têm uma data de fim atividade no registro.

O flag 'Imprensa definitiva' vai bloquear os dados do cabeçalho dos vários registros ativos fixos imprimidos, vai bloquear também o cancelamento de cada linha imprimida e atualizar o contador das páginas para o ano de referência dentro dos parâmetros de contabilidade geral.

Merece mencionar alguns relatórios padrão que se podem associar com este procedimento: imprensa do livro ativos fixos 'Fiscal' (a utilizar para os clientes que usufruem o cálculo fiscal) e a imprensa do livro ativos fixos 'Com valores iniciais' (que não reportar todo o detalhe histórico de cada ativo fixo mas só uma linha inicial de resumo dos dados de partida do ano pedido e depois o detalhe desse).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa física do documento. |
