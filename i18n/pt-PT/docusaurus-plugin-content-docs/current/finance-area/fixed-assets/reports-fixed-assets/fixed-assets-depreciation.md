Com este relatório é possível extrair os detalhes das amortizações calculadas em relação os parâmetros indicados na máscara.

Importante avisar as particularidades dos relatórios padrões que se podem associar com o procedimento:

**Imprimir de default**: efetua um cálculo hipotético das amortizações aplicáveis aos ativos fixos em relação as impostações da categoria ativo fixo: tal cálculo não leva em consideração o restante a amortizar;

**Imprimir amortizações calculadas** (CE_AmmCespiteCalc.rpt): aconselha-se sempre de associar com o processo, é aquela que extrai os dados assim como calculados pelo programa ou inseridos manualmente dentro dos ativos fixos;

**Imprimir com o dado fiscal**: é de particular interesse para os clientes que utilizam essa característica, vai totais as quotas inseduzíveis das amortizações.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa física do documento. |
