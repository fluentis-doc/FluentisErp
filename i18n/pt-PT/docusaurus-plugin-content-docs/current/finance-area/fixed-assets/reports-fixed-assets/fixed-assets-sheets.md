---
title: Fichas ativos fixos
sidebar_position: 1
---

Os relatórios dessa máscara permitem visualizar os dados ativos fixos ficha por ficha. São permitidos os critérios de ordenação definidos na base da máscara.

Merece avisar que são previstos diferentes relatórios padrão executáveis a partir desta janela de diálogo: uma lista ativos fixos ou uma imprensa das fichas com os dados das amortizações fiscais.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
| Imprensa | Botão para executar a imprensa física do documento. |
