---
title: Gestão cobrança
---

A partir do procedimento o usuário pode visualizar o estado das faturas de venda emitidas e/ou apresentadas, com o objetivo de gerir manualmente o estado de adiantamento.

Na primeira ficha se visualizam os dados dos vencimentos/faturas apresentadas: é possível modificar o estado e atribuir uma quantia de cobrança já recebida pelo cliente, assim como anotar liberalmente no campo notas quanto necessário.

Na segunda ficha se visualizam todas as faturas não apresentadas e se pode gerir manualmente o estado de cobrança assim como inserir notas livres.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar faturas | Botão através da qual procurar as faturas apresentadas ou imprimidas em relação com o tabulador aonde são colocados. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das faturas que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das faturas, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das faturas que quiser.
