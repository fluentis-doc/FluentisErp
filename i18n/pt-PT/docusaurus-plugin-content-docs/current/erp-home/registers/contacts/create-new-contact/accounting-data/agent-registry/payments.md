Esta janela é a correspondente da janela Pagamentos do registro cliente/fornecedor.

Aqui porém, se estabelece só uma tipologia de pagamento e uma solução de pagamento (Home > Tabelas > Gerais) com as referências bancárias padrão arrociadas.

Na grelha abaixo, invés, tem a lista dos bancos de apoio alternativos ao agente.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar agente | Botão para salvar as mudanças feitas no agente. |
| Cancelar o banco | Botão para cancelar o banco de apoio selecionado. |
