---
title: Registros bancos
sidebar_position: 1
---

O registro banco apresenta no começo os dados fundamentais a indicar: os códigos ABI, CAB, o número de C/C (para os bancos italianos precisa indicar 5 caracteres para o ABI e o CAB - portanto também o zero inicial - e todos os 12 caracteres da conta corrente italiana), o código internacional Swift, o CIN, a moeda com a qual é expresso o conto bancário, o código para o DocFinance® (programa de tesouraria externa do sistema), um eventual código banco livre, os primeiros 4 caracteres do Iban no campo ISO (portanto 2 caracteres para o código nação + 2 para o código instituto), o código RBN (para o DocFinance®) e o código SIA para o envio dos arquivos telemáticos italianos (se não indicado se utiliza aquele de default inserido na tabela Sociedade (Home > Tabelas > Gerais).

O campo do banco de apoio permite associar o registro banco com o relativo código de banco de apoio geral. Isso pode servir para canalizar automaticamente os movimentos financeiros dos livros-razões e avaliá-los nas análises de fluxo de caixa.
