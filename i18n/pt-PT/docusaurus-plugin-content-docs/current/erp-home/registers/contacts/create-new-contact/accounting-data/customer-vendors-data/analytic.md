Nesta janela é possível definir qual é a estrutura de default de valorização porcentagem dos custos/receitas nos centros de custo e de lucro (Home > Tabelas > Controle) para os documentos/registrações intituladas pelo sujeito em uso. Tal estrutura vai a interagir com as outras opções de valorização (em relação o artigo, o plano das contas geral, aquele industrial e o depósito) que pode ser definido módulo para o módulo em relação à prioridade.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestão documental. Se envia para o documento específico. |
