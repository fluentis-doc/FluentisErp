Nesta janela é possível definir qual é a estrutura de default de valorização porcentagem dos custos/receita nos centros de custo e de lucro (Home > Tabelas > Controle) para os documentos/registrações intituladas no sujeito em utilização. Tal estrutura vai interagir com as outras opções de valorização (em relação o artigo, o plano das contas geral, aquele industrial e o depósito) que pode ser definido módulo por módulo em relação a prioridade.

Não é presente nenhuma Ribbon Bar para essa ficha.
