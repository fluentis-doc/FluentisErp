O procedimento de pesquisa dos registros foi desinhado para permitir ao usuário pesquisar facilmente dentro da rubrica dos contatos, com o objetivo de visualizá-los, modificá-los, anulá-los e eventualmente cancelá-los ou proceder com a inserção de uma nova referencia, após a possibilidade de ter os direitos necessários para essas operações.

Com a ajuda dos filtros, pode-se visualizar: um só registro (p.ex.: procurar por sobrenome, unívoco), mais documentos (p.ex.: procurar por nações), os registros não mais válidos (filtro para o 'Registro vencidos') ou aqueles em uso contabilmente ou não (filtro para os 'Registros contábeis').

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar os registros no banco dos dados. |
| Nova | Botão para criar um novo registro. |
| Modificar | Botão para modificar um registro precedentemente inserido. |
| Visualização | Botão para visualizar os dados do registro selecionado. |
| Cancelar | Botão para cancelar um registro precedentemente inserido. Não é possível cancelar um registro que foi utilizado em qualquer ponto do sistema. |
| Convertor de moeda | Botão para a conversão das impostações do registro numa outra moeda. |
| Duplica | Botão para duplicar o registro selecionado. Apertando o botão vai se abrir uma máscara dentro da qual definir os elementos de base da cópia: razão social, sobrenome (quer dizer o código do mesmo registro), endereço, CPF, município e província. Uma vez confirmada a cópia, essa vai ser aberta na mudança automaticamente. Na cópia do registro, vão ser recriadas também todas as subcontas do plano das contas associados com o registro da sociedade ativa, atribuindo uma nova subconta em relação o progressivo de cada mastro. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos registros que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos registros, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos registros que quiser.
