---
title: Atributos encomenda
sidebar_position: 2
---

Esse tab foi constituído por uma grelha e por uma pequena barra de ferramentas de 3 botões. Na grelha o usuário pode inserir manualmente os dados que vem das tabelas do Codificados. As colunas presentes na grelha são as seguintes:

**Atributo**: a partir dessa coluna é possível, através o utilizo da caixa de combinação, chamar de novo os nomes das tabelas do Codificador;

**Valor atributo**: nessa coluna é possível inserir manualmente, utilizando a tooltip ou utilizando uma ajuda que se explode com um duplo click na caixa amarela, dados precodificados que vem da tabela que foi selecionada na coluna precedente. Isso é possível se a tabela do Codificador foi criada com o sistema de codifica 'tabelar', portanto com a inserção na tabela dos dados precodificados, caso contrário, o valor atributo vai ser inserido manualmente no outro campo que se detalha no ponto seguinte;

**Valor atributo**: nessa coluna o usuário pode inserir manualmente um valor que quer associá-lo com a tabela selecionada na coluna 'Atributo'. Se a tabela foi criada sem o sistema de codifica 'tabelar', o usuário precisa inserir um dado alfanumérico dentro dessa coluna, pena a impossibilidade de salvar a linha da grelha;

**Modificável**: nessa coluna aparece um flag que se ativado, permite ao usuário decidir que aquele atributo inserido na encomenda de produção pode ser modificado numa passagem sucessiva, por exemplo, dentro da ordem de produção que é gerada diretamente por aquela encomenda de produção, e portanto dentro da ordem de produção pela qual o artigo é o mesmo daquele da Encomenda de produção.

Existe também uma pequena barra de ferramentas, onde são presentes 3 botões, onde o primeiro da direita, identificado com um ícone que contém um 'P', permite inserir automaticamente na grelha os mesmos atributos com os mesmos valores presentes numa Encomenda de produção já eventualmente criada precedentemente para o mesmo artigo e para o mesmo cliente; o segundo, identificado com un ícone que contém a escrita 'ART', permite inserir automaticamente na grelha os mesmos atributos com os mesmos valores presentes no Registro artigo. tab 'Outros dados'; o último botão, na direita do tab, permite, para a linha selecionada na grelha, visualizar as informações acrescentadas relativas ao dado da tabela do codificador, quer dizer o código, o ID e a eventual descrição.
