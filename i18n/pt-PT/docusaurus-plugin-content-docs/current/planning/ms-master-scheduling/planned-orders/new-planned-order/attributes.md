---
title: Atributos
sidebar_position: 9
---

Este tab é constituído pela grelha e por uma pequena barra de ferramentas composta de 3 botões. Na grelha o usuário pode inserir manualmente dados que vem das tabelas do Codificador. As colunas presentes na grelha são as seguintes:

**Atributo**: a partir desta coluna é possível, através o utilizo da adequada caixa de combinação, chamar de novo os nomes das tabelas do Codificador;

**Valor atributo**: nesta coluna é possível inserir manualmente, utilizando a tooltip ou utilizando uma ajuda que se explode com um duplo click na casa amarela, dados pré-codificados que vem da tabela que foi selecionada na coluna precedente. Isto é possível se a tabela do Codificador foi criada com o sistema de codifica 'Tabelar', portanto com a inserção na tabela de dados precodificados, senão o valor atributo vai ser inserido manualmente no outro campo que se detalha no ponto a seguir;

**Valor atributo**: nesta coluna o usuário pode inserir manualmente um valor que quer que seja associado com a tabela selecionada na coluna 'Atributo'. Se for criada uma tabela sem o sistema de codifica 'tabelar', o usuário precisa inserir um dado alfanumérico dentro desta coluna, pena a impossibilidade de salvar a linha da grelha;

**Modificável**: nesta coluna aparece um flag que se ativado, permite ao usuário decidir que aquele atributo inserido na ordem planejada pode ser modificado numa passagem sucessiva, por exemplo dentro da ordem de produção que é gerada diretamente pela ordem planejada, e portanto dentro da ordem de produção pela qual o artigo é o mesmo que aquela ordem planejada.

Existe também uma pequena barra de ferramentas, pela qual são presentes 3 botões: o primeiro a esquerda, é identificado com um ícone que contém um 'P' e permite inserir automaticamente na grelha os mesmos atributos com os mesmos valores presentes numa ordem planejada já eventualmente criada precedentemente para o mesmo artigo e para o mesmo cliente; o segundo, é identificado com um ícone que contém a escrita 'ART' e permite inserir automaticamente na grelha os mesmos atributos com os mesmos valores presentes no Registro artigo, tab 'Outros dados'; o último botão, na direita, permite, para a linha selecionada na grelha, visualizar as informações acrescentadas relativas aos dados da tabela do codificador, e quer dizer o código, o ID e a eventual descrição.
