---
title: Materiais
sidebar_position: 2
---

Este tab é constituído por uma grelha, dentro da qual o usuário pode inserir manualmente os materiais que quer que sejam utilizados para a produção do artigo objeto da ordem planejada (só se de produção). Ele resulta já compilado se a ordem planejada vem da programação geral ou da elaboração PNR, ao contrário, se a ordem foi inserida manualmente, para estabelecer os dados da estrutura de produtos, precisa clicar no botão 'Conclusão dados ordem' presente na barra de ferramentas. As colunas presentes na grelha são as seguintes:

**Prioridade**: aqui se visualiza a prioridade do componente inserida na estrutura de produto. Ela pode ser modificada como todos os outros dados presentes nessa grelha;

**Classe**: através do relativo campo pode-se estabelecer a classe do artigo;

**Código artigo**: com um duplo click se entra dentro da ajuda artigos pela qual se selecionar o código artigo;

**Variante**: neste campo se estabelece a variante do artigo;

**Opção**: neste campo se estabelece a opção do artigo;

**Empreit.**: neste campo se retoma o flag presente nos parâmetros PNR do artigo chamado 'Curso de produção';

**UM**: neste campo se insere a unidade de medida do artigo;

**UMU**: neste campo se insere a eventual unidade de medida alternativa do artigo;

**Data utilizo**: corresponde com a data de início prevista da ordem planejada (modificando a primeira troca automaticamente também a última);

**Quant. utilizo**: é a quantidade unitária necessária para este artigo (prevista pela estrutura de produtos) que pode ser modificada;

**Quant. total**: é a quantidade que se obtém multiplicando a quant. utilizo pela quant. a produzir;

**% sobra**: nesta casa se insere ou se retoma pela estrutura do produto a eventual % de sobra para este artigo;

**Fase/Subfase**: com um duplo click se abre uma Ajuda fases de produção pela qual se pode selecionar a relativa fase e subfase, e portanto atribuir o material selecionado numa determinada fase de produção do artigo;

**Descrição artigo**: neste campo aparece a descrição do artigo;

**Descrição opção**: neste campo aparece a descrição da opção;

**Quant. disp.**: neste campo aparece a quantidade disponível do artigo na data de utilizo.

Precisa sublinhar que, se tiver a conclusão automática desta grelha, todos os dados em cima indicados vão ser atualizados automaticamente, deixando de qualquer maneira a possibilidade de modificá-los e/ou acrescentá-los mais materiais na lista componentes da ordem planejada. Se ordem planejada é de compra o tab Materiais vai resultar não ativo.
