---
title: Equipamentos
sidebar_position: 6
---

Neste tab, constituído principalmente por uma grelha, é possível definir quais e quantos equipamentos vão ter que ser utilizados na produção da fase selecionada no tab 'Fases'. Os dados presentes na grelha são os seguintes:

**Seq.**: neste campo é possível indicar a sequência que deve ser utilizada os equipamentos;

**Equipamentos**: com esta caixa de combinação é possível chamar de novo o código do equipamento, retomado pela adequada tabela 'Equipamentos';

**Plan.**: este flag, se ativo, permite fazer em modo que o equipamento seja planejado como se fosse uma material da estrutura de produtos;

**Classe/Código artigo**: nestes campos é possível indicar a classe e o código do artigo. Se procurar pela Ajuda artigos, a ajuda vai filtrar automaticamente só os artigos que estão identificados como Equipamentos no campo 'Natura artigo' do registro de depósito;

**Variante**: nesta coluna vai ser inserida/visualizada a variante do artigo;

**Quantidade**: nesta coluna se insere a quantidade do artigo;

**Descrição equipamentos/artigo/variante**: nestas colunas são visualizadas as relativas descrições.
