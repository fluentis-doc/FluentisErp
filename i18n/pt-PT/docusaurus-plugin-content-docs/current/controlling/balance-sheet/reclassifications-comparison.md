---
title: Procurar a comparação das reclassificações
sidebar_position: 5
---

O procedimento Procurar a comparação das reclassificações foi desinhada para permitir ao usuário procurar facilmente as comparações criadas precedentemente com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. A partir desta pesquisa é possível fazer todas as operações necessárias no âmbito do documento, após a presença dos direitos do usuário necessários.

O procedimento permite de confrontar os valores dos vários reclassificados de um certo modelo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de procura ao interno do banco de dados das comparações. |
| Nova | Refere-se ao procedimento para inserir uma nova comparação. |
| Modificar | Refere-se ao procedimento para modificar uma comparação precedentemente inserida. |
| Visualizar | Refere-se ao procedimento para visualizar uma comparação precedentemente inserida. |
| Cancelar | Executa o procedimento para cancelar uma comparação precedentemente inserida. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Modelos de agrupamento | Refere-se ao procedimento para imprimir os modelos de reclassificação. |
|  Valorização reclassificação | Refere-se ao procedimento para imprimir as reclassificações de balanço. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das comparações que correspondem os dados de filtro especificadas acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das comparações, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista das comparações.
