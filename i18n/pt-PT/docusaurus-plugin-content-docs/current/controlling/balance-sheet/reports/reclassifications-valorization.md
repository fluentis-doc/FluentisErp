title: Valorização reclassificações
sidebar_position: 2

A partir desta máscara se continua com a imprensa das reclassificações valorizadas.

É possível indicar duas reclassificações a comparar (eliminar neste caso o campo tabulado para executar um relatório específico), uma moeda com o câmbio para converter os valores nesta, a possibilidade de esconder os níveis de quantia igual a zero.
