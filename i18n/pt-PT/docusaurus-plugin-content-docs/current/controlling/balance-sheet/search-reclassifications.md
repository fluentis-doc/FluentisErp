---
title: Procurar reclassificação
sidebar_position: 4
---

O procedimento Procurar reclassificação foi desinhado para permitir ao usuário procurar facilmente as reclassificações de balanço criadas precedentemente com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. A partir do Procurar reclassificações é possível fazer todas as operações necessárias âmbito do documento, após a presença dos direitos do usuário necessários.

O procedimento da pesquisa registros é permitido na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de procura ao interno do banco de dados das reclassificações inseridas. |
| Nova | Se refere ao procedimento para inserir uma nova reclassificação. |
| Modificar | Se refere ao procedimento para modificar uma reclassificação precedentemente inserida. |
| Visualizar | Se refere ao procedimento para visualizar uma reclassificação precedentemente inserida. |
| Cancelar | Executa o procedimento para cancelar uma reclassificação precedentemente inserida. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Imprimir os modelos de reclassificação | Se refere ao procedimento para imprimir os modelos de reclassificação. |
|  Imprimir a valorização de reclassificação  | Se refere ao procedimento para imprimir uma valorização de reclassificações. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Comparação reclassificações  | Se refere ao procedimento para criar uma comparação entre as reclassificações. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das reclassificações que correspondem os dados de filtro especificadas acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das registrações, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista das reclassificações que quiser.
