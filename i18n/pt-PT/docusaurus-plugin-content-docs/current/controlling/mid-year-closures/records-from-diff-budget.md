---
title: Escrituras de dif. orçamento
sidebar_position: 13
---

Estamos dentro do módulo Encerramentos infra-anuais e vamos selecionar o procedimento 'Escrituras da diferença de orçamento': com esse procedimento é possível atribuir ao encerramento infra-anual os custos/receitas já relevados no orçamento de referência do período. Se refere aos custos de relevação adiantados pelas quais o orçamento contém já o valor correto por que previsto contratualmente: na contabilidade geral a registração não está, portanto, ainda presente, mas já está prevista corretamente no orçamento. Nesses casos, ao invés de registrar a linha nas operações manuais, é permitida uma retomada automática pelos dados do orçamento se o encerramento infra-anual foi associado com um orçamento. (Não é permitido retomar uma diferença 'positiva', quer dizer se o valor contábil é superior do valor a orçamento).

Uma vez estabelecido no primeiro campo o encerramento pela qual se quer trabalhar, vai ser necessário estabelecer no campo sucessivo o subobjeto de referência para o cálculo das diferenças: os campos de cabeçalho sucessivos são, ao contrário, de filtro de procura.

Na primeira grelha vão ser visualizadas as diferenças negativas: vai precisar importar o flag 'Selecionar' para definir quais diferenças cobrir com a registração automática, com a indicação da relativa valorização do mesmo a CdC/CdL (proposto pelas impostações predefinidas no plano das contas, se presente).

Na base da máscara vai precisar indicar a causal de registração (que deve ter o flag CdC/CdL para valorizar os dados acima estabelecidos) e a subconta de contrapartidas a utilizar (p.ex. faturas a receber).

A partir dessa máscara se procede com a imprensa dos modelos de reclassificação. Entre os relatórios disponíveis é possível associar um que visualiza todos os detalhes das subcontas/CdC/CdL/Encomendas e um, ao contrário, só com a estrutura do modelo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Executa a procura das escrituras de diferença orçamento a gerir para o encerramento selecionado. |
| Efetuar as escrituras | Executa a relevação extra-contábil das retomadas dos valores pelo orçamento selecionado. |
| Cancelar as escrituras | Cancela as retomadas do orçamento para o encerramento estabelecido. |
