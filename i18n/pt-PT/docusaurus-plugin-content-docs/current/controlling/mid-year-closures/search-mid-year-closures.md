---
title: Procurar os encerramentos das contas infra-anuais
sidebar_position: 4
---

O procedimento Procurar os encerramentos das contas infra-anuais foi desinhado para permitir ao usuário procurar facilmente os encerramentos criados precedentemente com o objetivo de visualizá-los, modificá-los e eventualmente cancelá-los. A partir da procura dos encerramentos é possível fazer todas as operações necessárias no âmbito do documento, após a presença dos direitos do usuário necessários.

O procedimento Procurar os encerramentos das contas infra-anuais é constituída na maneira seguinte:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar os encerramentos | Botão para aplicar os filtros de pesquisa ao interno do banco de dados dos encerramentos infra-anuais inseridos. |
| Criação de um novo encerramento | Refere-se ao procedimento para inserir um novo encerramento infra-anual. |
| Modificar a registração dos encerramentos | Refere-se ao procedimento para modificar a primeira registração de encerrametno infra-anual. |
| Visualização da registração do encerramento | Refere-se ao procedimento para visualizar a primeira registração de encerramento infra-anual. |
| Cancelamento do encerramento selecionado | Executa o procedimento para cancelar um encerramento infra-anual, com todas as registrações associadas. Não é possível cancelar um encerrametno se essa foi utilizada numa reclassificação ou numa análise de variações de orçamento. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos encerramentos que correspondem os dados de filtro especificadas acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos encerramentos, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista dos encerramentos que quiser.
