---
title: Procurar as registrações extracontábeis
sidebar_position: 7
---

Procurar as registrações extra-contábeis permite visualizar todos os movimentos associados com um determinado encerramento infra-anual. A máscara vai ter todos os botões desabilitados até não estabelecer em qual encerramento infra-anual se quer operar.

A partir da máscara vai ser possível: encontrar e modificar as registrações associadas ao encerramento infra-anual; cancelar as registrações associadas no encerramento infra-anual; acrescentar as registrações manuais no encerramento infra-anual (p.ex. relevar os estoques de depósito na data); duplicar uma registração manual (não aquelas automáticas criadas com os procedimentos do módulo) selecionada na grelha num outro encerramento já presente no módulo.

A criação de uma registração extra-contábil é parecida com a criação de uma registração de primeira nota: se vai selecionar sempre uma causal de contabilidade geral e daqui se procede com a registração (a única diferença é que não existe o conceito de protocolo imposto no infra-anual: o módulo, de qualquer maneira, é uma cópia que prevê as mesmas tabelas lógicas da contabilidade, mesmo sendo fisicamente diferentes para não sujar em nenhuma maneira os dados oficiais de contabilidade). São permitidas as registrações extra-contábeis só dos movimentos para o centro de custo/lucro.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar registração | Executa a procura das registrações de acordo com os filtros estabelecidos. |
| Nova registração | Executa o procedimento de criação de uma nova registração extracontábil associada com o encerramento infra-anual selecionado. |
| Mudança da registração selecionada | Apre em mudança a registração extracontábil selecionada. Se o encerramento é 'definitivo' as registrações associadas não vão ser modificáveis. |
| Visualizar a registração selecionada | Abre em visualização a registração selecionada. |
| Cancelar | Cancela a registração extracontábil selecionada. |
| Copiar a registração | Executa a cópia da registração selecionada num outro encerramento a individuar na máscara de duplicação que vai se abrir com o mesmo botão. Não se pode duplicar as registrações criadas pelos procedimentos automáticos (como retomadas de amortizações, registrações de retificação/integração, etc.). |
