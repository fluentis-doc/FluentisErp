---
title: Criar encerramentos infra-anuais
sidebar_position: 1
---

O procedimento de criação 'Encerramentos infra-anuais' foi desinhada para permitir ao usuário criar novas simulações de balanço infra-anual.

A origem dos dados de partida é sempre a situação da Contabilidade geral no momento da criação do mesmo encerramento: o programa vai retomar os saldos das várias subcontas, de acordo com os parâmetros da criação do encerramento, para permitir depois a adição extra contábil de todas as operações de acréscimos, diferimentos, amortizações e registrações manuais para chegar numa simulação de balanço precisa. A operação não tem nenhum efeito contábil de nenhum tipo: o módulo encerramentos infra-anuais é gerido nas tabelas dos bancos de dados completamente separadas da contabilidade geral.

Vamos ver os vários elementos presentes na máscara.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Criação do encerramento extra contábel | Botão para executar a criação das registrações extra contábeis de encerramento infra-anual de acordo com os parâmetros estabelecidos. |
| Mudança da registração | Executa a abertura da primeira registração de encerramento criada com a contabilização. |
| Visualizar a registração | Visualiza a primeira registração de encerramento criada com a contabilização. |
