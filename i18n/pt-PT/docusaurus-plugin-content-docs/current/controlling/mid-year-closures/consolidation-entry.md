---
title: Escrituras de consolidação
sidebar_position: 9
---

A partir da máscara de pesquisa das 'Escrituras de consolidação', como já explicado para as Registrações extracontábeis, vai ser possível completar as operações de consolidação com as registrações completamente manuais (para as participações infra-societárias por exemplo). Essas registrações também preveem uma causal contábil de referência: em respeito as normais registrações, cada linha de contabilidade prevê antes de tudo a indicação da sociedade de referência da mesma linha.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar registrações | Executa a procura das registrações de consolidação de acordo com o filtro estabelecido. |
| Nova registração | Executa o procediemento de criação de uma nova registração extra-contábil associada com a consolidação selecionada. |
| Mudança das registrações selecionadas | Abre em modifica a registração extra-contábil selecionada. |
| Visualizar a registração selecionada | Abre em visualização a registração associada. |
| Cancelar | Cancela a registração extra-contábil selecionada. |
| Copiar a registração | Executa a cópia da registração selecionda num outro encerramento a individuar na máscara de duplicação que vai se abrir com o mesmo botão. Não se pode duplicar as registrações criadas pelos procedimentos automáticos (como as retomadas  das amortizações, registrações de retificação/integração, etc.). |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Exportar Excel | Executa a exportação no Excel dos dados de variações visualizadas. |
