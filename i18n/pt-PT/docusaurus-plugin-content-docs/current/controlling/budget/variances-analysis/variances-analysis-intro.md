---
title: Análise variações
sidebar_position: 1
---

O procedimento é dedicado a validação dos consuntivos de orçamento: na máscara vai ser possível selecionar em cima o orçamento de referência, na direita o detalhe do suborçamento a analisar e embaixo a lista das retomadas dos dados a considerar. Apertando o botão 'Visualizar as variações' vai-se abrir uma máscara com o detalhe dos valores previstos no orçamento, do valor consuntivo e a porcentagem de variação relativa. 

A partir dessa máscara é possível proceder com a visualização de um gráfico do andamento para a linha selecionada, modificar as cores presentes para as várias colunas, ainda mais importante é prevista uma exportação no Excel para proceder com ulteriores elaborações ou na imprensa do resultado.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura os dados do orçamento para poder analizar. |
| Visualizar as variações | Visualiza a análise das variações selecionadas. |
