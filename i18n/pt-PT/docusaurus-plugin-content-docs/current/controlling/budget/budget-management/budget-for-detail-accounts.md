---
title: Orçamento para as subcontas
sidebar_position: 2
---

A metodologia de carregamento dos dados pode ser simplesmente manual, inserindo a subconta e o valor total anual para ver valorizado os detalhes periódicos na seção a direita pelas quais é possível intervir liberalmente, ou através o procedimento guiado pela importação Excel. Precisa levar em consideração que o dado a inserir é um valor absoluto: seja os custos que as receitas, por exemplo, vão ser carregadas, então, um valor positivo.

A importação Excel prevê as seguintes passagens:

1) duplo click no campo 'arquivo Excel' para individuar o arquivo na rede;

2) apertar o botão 'Importação' para ler o arquivo estabelecido e visualizar a ficha de trabalho no campo relativo;

3) estabelecer, se serve, um número linhas e número colunas superior ao default de 100-20 propostos;

4) individuar a folha de Excel pela qual ler os dados: o software vai carregar o detalhe na grelha abaixo;

5) neste ponto precisa trabalhar na grelha para individuar a área a importar e definir os campos essenciais para a importação: se procede através da pressão do botão direito do mouse;

6) para a importação dum orçamento nas subcontas precisa estabelecer a coluna da conta, da subconta e da quantia total. É possível definir também as colunas com os detalhes período por período, mas precisa levar em conta que o total desses deve ser igual ao total geral que resta obrigatório. Uma vez estabelecido tudo, vai ser suficiente apertar na 'Transferência' para executar a importação assim como estabelecida: o software vai reportar uma máscara com os detalhes dos eventuais erros relevados durante o procedimento.

Na máscara está presente também o botão de exportação no Excel de detalhe do orçamento definido.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Modificar a grelha período com os valores de default | Botão para recalcular a grelha dos detalhes de período com os valores de default. |
| Esportação Excel | Executa o procedimento de exportação Excel. |
| Importação dados | Refere-se ao procedimento para importar um orçamento para as subcontas do Excel. |
| Eliminar | Cancela a linha selecionada. |
| Imprimir a valorização de reclassificação | Refere-se ao procedimento para imprimir uma valorização de reclassificações. |
