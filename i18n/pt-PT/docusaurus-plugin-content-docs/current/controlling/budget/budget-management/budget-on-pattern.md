title: Orçamento no modelo
sidebar_position: 6
---

A valorização do orçamento no modelo pode ser efetuada com o nível dos vários agrupamentos definidos na estrutura: a análise das variações vai ser associada com o detalhe da estrutura, sem entrar a valorizar cada valor de origem específica.

O carregamento dos dados pode ser de dois tipos: manual, indo a estabelecer a quantia para pelo menos um elemento de cada nível presente; ou automático, de acordo com os flag de atualização estabelecidos nos vários suborçamentos. Os suborçamentos de tipo subconta, CdC e CdL que têm o flag estabelecido vão ser de origem dados de valorização dos detalhes dos agrupamentos do modelo para o suborçamento que por sua vez tem o flag estabelecido.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Eliminar | Cancela os dados selecionados. |
| Modificar a grelha período com os valores de default | Botão para recalcular a grelha dos detalhes de período com os valores de default. |
| Exportação Excel | Executa o procedimento de exportação Excel. |
| Criação do suborçamento para as subcontas | Executa a criação do suborçamento para as subcontas. |
| Importação dados | Refere-se ao procedimento para importar um orçamento para as subcontas industriais do Excel. |
