Os parâmetros do módulo Risco cliente se encontram no [Home > Parâmetros > Tesouraria].

Nos parâmetros do módulo Risco cliente se podem preestabelecer os elementos de default a propor nos procedimentos do módulo. Em particular:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nº dias além do prazo para os efeitos apresentados ao salvo bom fim | Este campo permite definir por quantos dias considerar, o efeito apresentado sbf, não cobrado depois do seu vencimento. |
| Nº dias além do prazo para os efeitos apresentados ao depois da cobrança | Este campo permite definir por quantos dias considerar, o efeito apresentado depois da cobrança, não cobrado depois do seu vencimento. |
| Data de início elaboração (saldos contábeis, faturas, CT-e) | É a data de partida do cálculo. Documentos com data antecedente não vão ser considerados. |
| Visualizar os não pagos contabilizados | A impostação deste flag vai visualizar no módulo também as quantias dos não pagos contabilizados, independentemente também do estado cobrado.Tem simplesmente um valor informativo. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Saldo contábil | Com tal impostação o cálculo do crédito vai ser efetuado em relação ao estado contábil do cliente. |
| Partidas abertas | Com tal impostação o cálculo do crédito vai ser efetuado em relação as partidas abertas do cliente. |
| Considerar as partidas de registrações provisórias | Se ativo o flag precedente o cálculo considera também os eventuais movimentos que derivam da registrações provisórias. |
| Bloquear a imprensa física | Se o crédito cliente prevê o bloqueio documento, este flag bloqueia a possibilidade de imprimir fisicamente o documento bloqueado. Se não foi estabelecido  o documento se pode imprimir mas o flag 'Imprimido' não vai ser atribuído ao documento. |
