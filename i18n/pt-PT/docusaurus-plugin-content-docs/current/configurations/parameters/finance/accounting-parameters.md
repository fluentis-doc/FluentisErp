Os parâmetros de contabilidade se encontram na [Home > Parâmetros > Administrativa].

A máscara dos parâmetros de contabilidade permite definir os vários parâmetros de gestão de cada ano contábil selecionado na grelha da esquerda.

Importante indicar dois botões de gestão da máscara: o botão de 'Anulação das imprensas definitivas do livro jornal' e aquele da 'Imprensa definitiva do libro ativo fixos', cujo utilização é associada com um específico direito do usuário gerenciáveis com o ApplicationRessourceManager ativável para cada um. Nos ambos casos, uma vez anulada a imprensa definitiva, o usuário tem de redefinir os contadores relativos.

Lembra-se que os parâmetros de contabilidade é a tabela base para individuar o último ano gestional ativo para o procedimento de geração do novo ano contábil que se encontra dentro da Utilidade.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as impostações relativas ao ano de gestão contábil em que foi colocado. |
| Inserção | Botão para inserir um novo ano de gestão contábil. |
| Cancelar | Botão para cancelar o ano de gestão contábil em que foi colocado. |
| Inserção conta | Botão para inserir os tipos de contas registros autorizados para cada conta inserida. A impostação autoriza a substituição automática da conta geral na causal com a subconta titular do documento fiscal, assim como vai bloquear o indicar as contas incompatíveis. |
| Cancelar a conta | Botão para cancelar as contas precedentemente inseridas. |
| Anular as imprensas definitivas | Botão para proceder a anulação das imprensas definitivas do libro jornal, cujo utilizo é associado a um especifico direito usuário gestivel com a ApplicationResourceManager ativável para cada um. Uma vez anulada a imprensa definitiva, o usuário tem de redefinir os contadores relativos. |
| Anular o livro amortização | Botão para proceder a anulação das imprensas definitivas do libro ativos fixos, cujo utilizo é associado a um especifico direito usuário gestivel com a ApplicationResourceManager ativável para cada um. Uma vez anulada a imprensa definitiva, o usuário tem de redefinir os contadores relativos. |

#1.1 Tab geral

| RibbonFunction | RibbonMeaning |
| --- | --- |
| Início/fim período em linha | Campos obrigatórios, definem a gama de datas do documento gerável dentro das registrações do exercício selecionado. |
| Início/fim competência em linha | Campos facultativos aconselhados, definem a gama de datas de competência gerável dentro das registrações do exercício selecionado. |
| Encerramento exercício precedente | Define a data de encerramento do exercício precedente. |
| Data última imprensa livro jornal | Memoriza a última data imprimida em definitivo para o exercício atual. Na abertura de um novo exercício esta data tem de corresponder com a data de encerramento do exercício precedente. |
| Início/fim exercício | Defini a gama de datas de registração do exercício selecionado. Cada registração contábil, para os controles dos campos precedentes, é inserida no exercício da pertenção em relação a essas datas, que não devem sobrecolocar entre os vários exercícios. |
| Última página livro jornal imprimida | Memoriza a última página do livro jornal do exercício selecionado imprimida em definitivo. Na criação de um novo exercício este campo tem de ser estabelecido zero. |
| Última linha livro jornal imprimida | Memoriza a última página do livro jornal do exercício selecionado imprimida em definitivo. Na criação de um novo exercício este campo tem de ser estabelecido zero. |
| Saldo débito/crédito | Memoriza os últimos saldos débito/crédito do livro jornal do exercício selecionado imprimidos em definitivo. Na criação de um novo exercício este campo tem de ser estabelecido zero. |
| Última página livro ativos fixos imprimidos | Memoriza a última página do livro ativos fixo do exercício selecionado imprimida em definitivo. Na criação de um novo exercício este campo tem de ser estabelecido zero. |
| Porcentagem pro-rata | O campo permite de estabelecer o valor da porcentagem do pro-rata válido para o exercício selecionado. Se não for uma opção deixar o campo vazio ou estabelecido a 100% |
| Causal abertura e encerramento contas | Os dois campos, não obrigatórios, permitem préestabelecer as causais a utilizar nos procedimentos de encerramento/abertura das contas. |
| Proposta centros de custo/lucro da | A seção define a ordem lógica de valorização contábil dos centros de custo/lucro. |
| Tipo contabilização imposto | O campo permite definir o tipo de contabilização de default dos impostos nas várias subcontas do plano das contas. |
| Liquidação imposto | Estabelece a periodicidade das liquidações imposto periódicas. |
| Tipo rounding imposto | Estabelece o arredondamento imposto sempre por excesso com a seleção 'Libras' ou de acordo com a lógica do Euro. |
| Registros imposto | Define se imprimir os registros imposto por data registração ou por data de competência imposto dentro do período selecionado. Para que utiliza a possibilidade de  faturação diferida dentro do dia 15 do mês sucessivo o resultado  não troca. |
| Data competência plafond igual a | Definir se a data competência plafond a atribuir aos movimentos imposto com a alíquota associado ao plafond vai ser estabelecida em relação a data registração ou a data documento. |
| Lista contas clientes fornecedores | Nesta seção se estabelecem os tipos de registros autorizados para cada conta inserida. A impostação autoriza a substituição automática da conta geral na causal com a subconta titular do documento fiscal, assim como também vai bloquear ou indicar as contas incompatíveis. |

| --- | --- |

| Subconta imposto vendas | Permite estabelecer a subconta a propor na relevação automático da rotação imposto no momento da imprensa definitiva da liquidação imposto periódica. |
| Estado patrimonial inicial | Permite estabelecer a subconta a propor para as operações de abertura automática das contas. |
| Útil do exercício atual | Permite estabelecer a subconta a propor para as operações de abertura/encerramento automático das contas. |
| Perda de exercício atual | Permite estabelecer a subconta a propor para as operações de abertura/encerramento automático das contas. |
| Útil do exercício precedente | Permite estabelecer a subconta a propor para as operações de abertura/encerramento automático das contas. |
| Perda do exercício precedente | Permite estabelecer a subconta a propor para as operações de abertura/encerramento automático das contas. |
| Conta econômica | Permite estabelecer a subconta a propor para as operações de encerramento automático das contas. |
| Perda de capital | Permite estabelecer a subconta a utilizar nas relevações das diferenças econômicas negativas da cessão dos ativos fixos. |
| Ganho de capital | Permite estabelecer a subconta a utilizar nas relevações das diferenças econômicas positivas da cessão dos ativos fixos. |
| Ocorrências ativas | Permite estabelecer a subconta a utilizar nas relevações das ocorrências ativas. |
| Ocorrências passivas | Permite estabelecer a subconta a utilizar nas relevações das ocorrências passivas. |
| Ausências ativas | Permite estabelecer a subconta a utilizar nas relevações das ausências ativas. |
| Ausências passivas | Permite estabelecer a subconta a utilizar nas relevações das ausências passivas. |
| Débitos v erário | Permite estabelecer a subconta a propor na relevação automática da rotação imposto no momento da imprensa definitiva da liquidação periódica. |
| Créditos v erário | Permite estabelecer a subconta de crédito verso o erário a propor na relevação automática da rotação imposto no momento da imprensa definitiva da liquidação imposto periódica. |
| Contas correspondentes | Permite estabelecer a subconta do cliente fictício de relevação patrimonial dos correspondentes. |

| Diferimentos ativos | O campo permite estabelecer a subconta geral a utilizar no procedimento das rectificações automáticas do módulo 'Encerramentos infra-anuais'. |
| Acréscimos ativos | O campo permite estabelecer a subconta geral a utilizar no procedimento das rectificações automáticas do módulo 'Encerramentos infra-anuais'. |
| Diferimentos passivos | O campo permite estabelecer a subconta geral a utilizar no procedimento das rectificações automáticas do módulo 'Encerramentos infra-anuais'. |
| Acréscimos passivos | O campo permite estabelecer a subconta geral a utilizar no procedimento das rectificações automáticas do módulo 'Encerramentos infra-anuais'. |
| Custos suspensos | Subconta padrão de relevação dos custos suspensos, para a contabilidade simplificada/profissionais. |
| Receitas suspensas | Subconta padrão de relevação das receitas suspensas, para a contabilidade simplificada/profissionais. |
