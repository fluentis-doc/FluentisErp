Os parâmetos do módulo Ativos fixos se encontram no [Home > Parâmetros > Administrativa].

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar os parâmetros | Permite salvar os parâmetors estabelecidos. |
| Anular as impostações | Permite anular as impostações. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Verificar o précódigo | Se o flag é estabelecido, o précódigo tem de ser predefinido na tabela e não pode ser estabelecido liberalmente na criação do ativo fixo. |
| Visualizar a quantidade | Se o flag é estabelecido vai ser visualizada a quantidade. |
| Default UM | Define a UM de default se o flag precedente foi estabelecido. |
| Porcentagem em cada ativo fixo | Com o flag estabelecido vai se ativar, no cabeçalho do ativo fixo, um campo aonde se pode estabelecer uma porcentagem de amortização que vai ser prioritária a porcentagem estabelecida na categoria de pertença do ativo fixo. |
| Gestão divisão | Se o flag é estabelecido vai ser possível atribuir, no registro do ativo fixo, a divisão desejada, senão o campo divisão vai ser proposto de default. |
