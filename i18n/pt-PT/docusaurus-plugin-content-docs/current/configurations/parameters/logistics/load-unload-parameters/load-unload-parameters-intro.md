title: Parâmetros carregamento/descarregamento
sidebar_position: 1
---

O procedimento da gestão dos Parâmetros carregamento/descarregamento é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos parâmetros carregamento/descarregamento inseridos. |
| Inserir | Refere-se ao procedimento para inserir um novo parâmetro de carregamento/descarregamento. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos parâmetros carregamento/descarregamento inseridos. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso os filtros são para o empregado e depósito.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos parâmetros carregamento/descarregamento que correspondem os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa ddos parâmetros carregamento/descarregamento, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista dos parâmetros carregamento/descarregamento que quiser.
