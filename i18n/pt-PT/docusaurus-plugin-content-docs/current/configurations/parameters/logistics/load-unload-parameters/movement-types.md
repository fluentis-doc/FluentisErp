---
title: Tipos movimento
sidebar_position: 2
---

Os tipos movimento determinam qual é a função habilitada. Abaixo se específica para cada tipo de movimento a função associada:

**Carregamento**: função de WMS Mobile que permite criar uma registração de depósito de carregamento;

**Descarregamento**: função WMS Mobile que permite criar uma registração de depósito de descarregamento;

**Versamento produção**: função WMS Mobile que permite declarar a produção de um produto acabado ou semitrabalhado;

**Retirada de produção**: função WMS Mobile que permite descarregar do depósito os materiais necessários para a produção;

**Retorno de produção**: função de WMS Mobile que permite declarar os retornos de produção com uma registração de carregamento no depósito;

**Retificação de carregamento**: função de WMS Mobile que permite efetuar uma retificação de depósito positiva;

**Retificação de descarregamento**: função de WMS Mobile que permite efetuar uma retificação de depósito negativa;

**Carregamento pálete**: função de WMS Mobile e de ERP que permite efetuar o carregamento no depósito da UdC;

**Descarregamento pálete**: função de WMS Mobile e ERP que permite efetuar o descarregamento no depósito da UdC;

**Retornos**: função de WMS Mobile para a gestão do retorno do material;

**Confirmação transferência**: função de WMS Mobile para a confirmação das retiradas de transferência;

**Atribuir localização**: função de WMS Mobile que descarrega o artigo da localização de default e carrega o artigo na localização desejada;

**Deslocação artigos**: função de WMS Mobile que permite mover os artigos de uma localização a outro. A causal de depósito a enganchar vai ter que ser de descarregamento com contrapartida;

**Compra**: função de WMS Mobile para criar o CT-e de compra ao fim de criar uma ordem cliente e o relativo carregamento no depósito dos artigos;

**Mover o lote não bom**: função de WMS Mobile que permite a deslocação dos lotes vencidos;

**Aceitação materiais geridas a lotes**: função de WMS Mobile para criar o CT-e de compra ao fim de criar uma ordem cliente e o relativo carregamento no depósito dos artigos geridos a lotes;

**Tirar da UdC**: função de WMS Mobile que permite tirar os artigos de uma UdC precedentemente criada;

**Vendas**: função de WMS Mobile para criar o CT-e de venda ao fim de criar uma ordem cliente manual e o relativo descarregamento dos artigos;

**Retirada**: função WMS Mobile para criar uma nova retirada ou para confirmar uma criada precedentemente;

**Inventário UdC**: função de WMS Mobile para a criação de um inventário UdC;

**Transferência UdC**: função de WMS Mobile para a criação ou a confirmação das listas de transferência UdC;

**Carregamento UdC da criação UdC**: função de WMS Mobile e ERP para a criação das UdC e o relativo carregamento no depósito.
