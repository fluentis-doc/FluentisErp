title: Tab WMS Mobile
sidebar_position: 2
---

No tab WMS Mobile se definem os parâmetros para o funcionamento das Retiradas no WMS Mobile.

Se define o tipo de conta transportadora que vai ser a transportadora que vai ser inserida nas retiradas criadas pelo WMS Mobile, o tipo retirada para as listas de entrega, para as listas de transferência, para as listas de transferência UdC, para as retiradas de entrega, para as retiradas de transferência e para as retiradas de transferência por empreiteiro.

É também oportuno especificar as causais que vão ser utilizadas para os seguintes movimentos:

- descarregamento UdC para confirmar a transferência;

- causal proposta para o descarregamento retirada;

- descarregamento para o depósito de partida;

- carregamento para o depósito destino;

- causal descarregamento depósito empreiteiro.
