---
title: Diferenças de inventário
sidebar_position: 7
---

Neste tab é possível estabelecer, para cada depósito, as causais que devem ser utilizadas automaticamente pela aplicação no caso de criação automática de registrações de depósito através da utilização do procedimento 'Diferenças de inventário', e que permite ao usuário fazer calcular a diferença entre a quantidade lógica e a quantidade física, quer dizer aquela relevada pelo inventário físico, e fazer gerar automaticamente as registrações de compensação das diferenças entre a quantidade lógica e a quantidade física.

As causais de depósito que podem ser estabelecidas nesse tab são exclusivamente as causais que têm o flag 'Gest. inv.' ativado. É necessário estabelecer, para cada depósito, uma causal de carregamento de retificação de inventário e uma causal de descarregamento de retificação de inventário.
