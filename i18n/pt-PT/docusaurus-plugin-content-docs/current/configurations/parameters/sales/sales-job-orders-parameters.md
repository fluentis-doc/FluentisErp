---
title: Parâmetros encomendas de venda
sidebar_position: 4
---

Os parâmetros encomendas de venda permitem a impostação de base para gerir corretamente e de acordo com os específicos pedidos de cada sociedade.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Restaurar os parâmetros | Permite restaurar os parâmetros nos valores iniciais. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Execução obrigatória | Utilizado nas ordens clientes: avisa que a ordem cliente tem de importar alguns artigos de uma ou mais encomendas de venda. |
| Execução artigos da encomenda obrigatória | Utilizado nas ordens clientes: avisa que a ordem cliente tem de conter só alguns artigos importados da encomenda de venda. |
| Quantidade fixa | Utilizado nas ordens clientes: avisa que a ordem cliente não pode executar pela encomenda de venda uma quantidade maior daquela da linha artigo encomenda. |
| Retomada dos atributos do registro artigos | Não geridos. |
| Visualizar só os artigos relacionados | Utilizado no empreiteiro: avisa que a execução da encomenda de venda se pode fazer só pelos artigos filhos. |
| Execução encomenda no CT-e entrega com conta diferente  | Utilizado no empreiteiro: avisa que a execução da encomenda de venda se pode fazer também por uma conta diferente daquela da nota de entrega de empreiteiro. |
| Gestão dupla unidade de medida  | Avisa o fato que é gerida a unidade de medida alternativa na encomenda de venda. |
| Proposta automática UM alternativa | Avisa o fato que é proposta em automático a unidade de medida alternativa de default. |
