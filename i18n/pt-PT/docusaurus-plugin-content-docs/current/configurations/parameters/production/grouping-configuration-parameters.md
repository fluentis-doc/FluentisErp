---
title: Parâmetros Gantt
---

A janela dos Parâmetros ordens de produção se abre através do percurso Home > Parâmetros > Produção > Parâmetros Gantt. Através desta janela se estabelecem todos os parâmetros que se referem a visualização do Gantt.

O flag **Bloqueio data entrega ordem cliente**, se ativo, permite fazer em modo que o usuário não possa mover para frente as encomendas até fazer em modo que a data de entrega estabelecida na ordem cliente pela qual foi gerada a encomenda seja rejeitada.

Na seção **Cor do documento** é possível estabelecer as cores que se referem as diferentes tipologias dos documentos que se quer visualizar no Gantt.

Na seção **Cor do histograma** é possível estabelecer as cores que se referem ao começo e fim dos centros de trabalho, e começo e fim dos sobrecarregamentos de trabalho.

Na seção **Cor do fundo** é possível estabelecer as cores do fundo que se referem ao Gantt e ao histograma.

Na seção **Cor dos caracteres** é possível estabelecer as cores dos caracteres padrão, dos documentos executados e daqueles taxativos.

Na seção **Cor do marcador** é possível estabelecer as cores da fonte que vai ser utilizada para indicar os nomes dos documentos que vão ser visualizados no Gantt.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as mudanças efetuadas nos parâmetros Gantt. |
