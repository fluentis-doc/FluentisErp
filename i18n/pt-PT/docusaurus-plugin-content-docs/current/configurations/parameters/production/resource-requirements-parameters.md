A janela dos Parâmetros necessidade materiais se abre através do percurso Home > Parâmetros > Produção > Parâmetros necessidade materiais. Através dessa janela se estabelecem todos os parâmetros que se referem algumas impostações de acordo com a área de produção.

Antes de tudo, é possível decidir qual é o dia que tem de ser considerado como primeiro da semana no calendário da Capacidade produtiva, portanto no calendário dos Centros de trabalho. Normalmente se estabelece segunda-feira.

Logo abaixo se visualiza o último número progressivo utilizado no ano em curso para as ordens planejadas.

O flag **Libertação PC autorizadas** permite fazer em modo que os pedidos de compra gerados através da libertação das ordens planejadas de compra sejam criados com o flag  de 'Autorizado' ativado automaticamente. Diversamente, o usuário que tem de gerar ordens fornecedores do PC vai ser constringido a autorizar todas as linhas de pedido de compra antes de poder gerar uma ordem. É possível autorizar também só algumas das linhas de PC e portanto proceder com a geração ordens fornecedores partindo só das linhas autorizadas.

O flag** Gestão simplificada da produção** permite ativar o consequente flag **Libertação ordens de produção executivas**; se ativo, o flag permite fazer em modo que as ordens de produção geradas através da libertação das ordens planejadas de produção sejam criadas já no estado de 'Executivos', e portanto não solicita um ulterior procedimento de Libertação.

O flag **Visualização formado a árvore das ordens** permite visualizar, na parte abaixo da máscara Procurar ordens planejadas, uma árvore que junta os elementos de uma encomenda de produção, em relação as ligações das estruturas de produtos envolvidas na programação daquela encomenda.

A seção das **Cores do fundo** permite estabelecer a cor do fundo para as diferentes tipologias de ordem: de compra, de produção ou de empreiteiro, enquanto a seção das cores texto permite estabelecer a cor do texto para as diferentes tipologias de ordem: em atraso e/ou vencidas.

O flag **Manutenção associação entre as ordens de uma encomenda**, se ativado, permite, utilizando também os flag imediatamente sucessivos, decidir que se é modificada manualmente a data de uma ordem planejada, o procedimento atua a mudança consequente também nas ordens planejadas imediatamente adjacente aquela que teve uma mudança inicial.

A grelha que apresenta os **Depósitos de programação ordens** é a simples visualização da lista de depósito cujo tem de ser controlada a disponibilidade. Essa lista se estabelece na máscara chamada 'Cálculo disponibilidade' que se encontra entre as unidades do gestional.

Na seção chamada **Parâmetros para a libertação ordens planejadas** o usuário pode estabelecer o tipo de pedido de compra e o tipo ordem de empreiteiro que devem ser gerados automaticamente durante a libertação das ordens planejadas rispectivamente de compra e de empreiteiro.

Na seção chamada **Parâmetros para a exportação no Project** o usuário pode estabelecer os parâmetros se tem a necessidade de exportar o Gantt das ordens de produção na aplicação 'Microsoft Project'.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:

| --- | --- |
| Salvar | Botão para salvar as mudanças efetuadas nos Parâmetros de necessidade dos materiais. |
