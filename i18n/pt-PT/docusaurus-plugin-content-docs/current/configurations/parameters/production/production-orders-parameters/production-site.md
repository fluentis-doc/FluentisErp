---
title: Tab Local de produção
sidebar_position: 3
---

Na grelha superior se define o local produtivo, ao contrário na seção de máscara abaixo aparecem os estabelecimentos dos relativos depósitos, os mesmos presentes no precedente tab (geral).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as mudanças efetuadas nos Parâmetros ordens de produção. |
| Pré-visualização de imprensa | Visualiza uma pré-visualização de imprensa dos Parâmetros ordens de produção. |
| Imprimir | Lança a imprensa na impressora predefinida do relatório dos Parâmetros ordens de produção. |
