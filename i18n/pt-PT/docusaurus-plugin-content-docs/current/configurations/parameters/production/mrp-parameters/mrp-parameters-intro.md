A janela dos Parâmetros PNR se abre através do percurso [Home > Parâmetros > Parâmetros PNR]. Refere-se a janela de pesquisa dos parâmetros PNR dos artigos presentes no banco de dados. Para cada artigo, se cria um registro dos parâmetros PNR quando se insere o seu registro 'principal' da adequada janela da seção Registros que permite de codificar um novo artigo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procura | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos artigos inseridos. |
| Modificar os parâmetros | Refere-se ao procedimento para modificar os parâmetros PNR de um artigo precedentemente inserido. |
| Visualizar os parâmetros | Refere-se ao procedimento para visualizar os parâmetros PNR de um artigo precedentemente inserido. |
| Substituição informações | Abre uma janela de pop-up que permite decidir quais informações relativas ao artigo modificar. O usuário tem a possibilidade, para os dados cujo habilitou os flag, de trocar massivamente as informações de mais artigos selecionados preventivamente na grelha. |

ÁREA DE FILTRO: nesta seção é possível utilizar uma série de filtros de pesquisa quais o filtro por classe, código, descrição e variante do artigo, além que o filtro por Tipo aprovisionamento (compra, produção e empreiteiro);

GRELHA DE RESULTADO: com um duplo click na linha do artigo se entra na janela de detalhe dos Parâmetros PNR do artigo. A janela é formada por uma parte de cabeçalho cujo são visualizadas a classe, o código e a descrição do artigo, a relativa eventual variante e a sua unidade de medida, e por uma parte formada por 2 tab.
