---
title: Posição partidas
sidebar_position: 17
---

A definição desta tabela, com impostação livre, vai permitir, através das máscara Partidas, atribuir uma posição numa lista de partidas selecionadas através o botão 'Trocar posição partidas'.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar uma pesquisa da posição das partidas, através os campos código e descrição. |
| Inserção | Botão para efetuar a inserção de uma nova posição partidas. |
| Cancelamento | Botão para efetuar o cancelamento da posição das partidas cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
