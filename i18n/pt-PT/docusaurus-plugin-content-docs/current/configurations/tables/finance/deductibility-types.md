A definição da tabela permite ao usuário poder prever alguns tipos de dedutibilidade fiscal a associar na categoria ativo fixo no tipo de amortização desejado.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do tipo de dedutibilidade, através os campos filtro código e descrição. |
| Novo tipo | Botão para efetuar a inserção do tipo de dedutibilidade. |
| Cancelar o tipo | Botão para efetuar o cancelamento do tipo de dedutibilidade cujo é colocado. |
| Novo detalhe | Botão para efetuar a inserção de um novo detalhe do tipo de dedutibilidade cujo é colocado. |
| Cancelar o detalhe | Botão para efetuar o cancelamento do detalhe do tipo de dedutibilidade cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |

A primeria seção, Tipos dedutibilidade, contém os seguintes campos:

**Código**: Código de identificação do tipo de dedutibilidade.

**Descrição**: Descrição do tipo de dedutibilidade.

A segunda seção, Detalhes, contém os seguintes campos:

**Data início**: Data de início do tipo de dedutibilidade cujo é colocado.

**Data fim**: Data fim do tipo de dedutibilidade cujo é colocado.

**Porc. dedut.**: Porcentagem de dedutibilidade a aplicar ao tipo de dedutibilidade cujo é colocado.
