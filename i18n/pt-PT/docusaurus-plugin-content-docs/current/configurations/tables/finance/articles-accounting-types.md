RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do tipo de contabilização artigos, através dos campos filtro código e descrição. |
| Inserção | Botão para efetuar a inserção do tipo de contabilização artigos. |
| Cancelamento | Botão para efetuar o cancelamento do tipo de contabilização artigos cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |

Campos:

**Código**: código de identificação do tipo de contabilização dos artigos;

**Descrição**: descrição do tipo de contabilização artigos;

**Cliente**: se ativar o campo de contabilização artigos no cliente;

**Fornecedor**: se ativar o campo de contabilização artigos no fornecedor.
