RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa das causais precedentemente inseridas, através os campos de pesquisa da causal, descrição, tipo movimento e flag de causais válidas ou não válidas. |
| Inserção | Botão para efetuar a inserção de uma nova causal de contabilidade geral. |
| Modificar a linha selecionada | Botão para efetuar as mudanças da causal contábil selecionada. |
| Duplicar a causal selecionada | Botão para efetuar a duplicação da causal selecionada, útil para criar uma nova causal partindo de uma com características parecidas.Vai-se abrir um dialog onde se pede de atribuir um código a causal nova que se quer criar e depois de ser confirmado, vai ser pedido de poder entrar na causal criada para modificar algumas características. |
| Cancelamento | Botão para efetuar o cancelamento da causal contábil selecionada. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
