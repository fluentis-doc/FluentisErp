A definição desta tabela permite codificar e estabelecer as várias tipologias de tipos de taxa a gerir na sociedade ativa.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão pra procurar o tipo de taxa através os campos de filtro código e descrição. |
| Novo tipo taxa | Botão para efetuar a inserção de um novo tipo de taxa. |
| Cancelar o tipo taxa | Botão para efetuar o cancelamento do tipo taxa aonde é colocado. |
| Novo detalhe | Botão para inserir o detalhe do tipo taxa aonde é colocado. |
| Cancelar o detalhe | Botão para efetuar o cancelamento do detalhe do tipo taxa aonde é colocado. |

**Código**: código de identificação da taxa de interesse.

**Descrição**: descrição da taxa de interesse.

**Da data documento**: data documento a partir da qual se gere o cálculo dos interesses (a ref. é DECRETO LEGISLATIVO 9 outubro 2002, nº 231).

**Da data vencimento**: data vencimento a partir da qual se gere o cálculo dos interesses (a ref. é DECRETO LEGISLATIVO 9 outubro 2002, nº 231).

**Interesse min.**: quantia mínima dos interesses abaixo da qual não valorizar nenhuma quantia.

**Dias de franquia**: número de dias de franquia que vão ser detraídos pela conta dos dias de atraso.

**Cálculo também do desconto**: flag de gestão também dos descontos no caso de pagamento adiantados.

**Código**: código de identificação da taxa.

**Da data**: data a partir da qual a taixa vai ser gerenciada.

**A data**: data até quando a taixa vai ser gerenciada.

**Porcentagem**: porcentagem da taxa de interesse aplicável.
