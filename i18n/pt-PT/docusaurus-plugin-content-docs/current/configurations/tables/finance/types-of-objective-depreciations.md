title: Tipos amortizações objetivas
sidebar_position: 22
---

A definição da tabela permite ao usuário poder operar das amortizações objetivas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do tipo de amortização objetiva, através os campos filtro tipo e descrição. |
| Inserção | Botão para efetuar a inserção do tipo de amortização objetiva. |
| Cancelamento | Botão para efetuar o cancelamento do tipo de amortização objetiva cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |

**Tipo**: código de identificação do tipo de amortização objetiva;

**Descrição**: descrição do tipo de amortização objetiva;

**Oper. Incremental**: flag de gestão como 'incrementais' também para os dados objetivos (se devolve ao homônimo flag da categoria ativo fixo);

**Cálculo dias de uso**: o flag de cálculo para os dias de uso que vai calcular efetivamente os dias de utilizo do ativo fixo no ano invés que utilizar todo o exercício como para o cálculo normal;

**Porc.**: a porcentagem padrão a aplicar;

**Porc. Fixa**: com tal flag ativo a porcentagem vai ser fixa sem a possibilidade de personalizá-la ativo fixo por ativo fixo.
