title: Tipos conta

---

A definição da tabela Tipos conta é uma 'qualificação' que vai indicar ao procedimento como gerir o código de subconta cujo é atribuído:

- antes de tudo se indica como valorizar o detalhe de gestão das 4 tipologias de registros previstas;

- em segundo lugar, indica as seções que pertencem ao balanço (ativo/passivo/custos/receitas);

- pode, enfim, servir como critério de filtro, tipicamente dentro da ajuda contas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do tipo conta, através os campos filtro código e descrição. |
| Inserção | Botão para efetuar a inserção do tipo conta. |
| Cancelamento | Botão para efetuar o cancelamento do tipo conta cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |

**Código**: código tipo conta;

**Descrição**: descrição tipo conta;

**P. ativo**: com o flag no ativo;

**P. passivo**: com o flag no passivo;

**E. custos**: com o flag nos custos;

**E. receitas**: com o flag nas receitas;

**C. ordem**: com o flag nas contas ordem;

**Cliente**: com o flag no clientes + ativo;

**Fornecedor**: com o flag nos fornecedores + passivo;

**Banco**: com o flag no ativo ou passivo + banco;

**Agente**: com o flag no agente + passivo;

**Serviço**: com o flag no serviço - as contas com o tal flag ativo são as subcontas pela quais o procedimento gere em automático as escrituras de retificação e integração a nível de balanços infra-anuais, mas também as escrituras de ajustamento de contabilidade geral.
