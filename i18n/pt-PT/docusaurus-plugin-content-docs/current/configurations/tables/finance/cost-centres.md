title: Centros de custo

A tabela permite a definição da estrutura, também em diferentes níveis, e as características dos centros de custo. A seção geral apresenta a estrutura ramificada dos Centros de custo definidos para a sociedade ativa: em vermelho são visualizados os CdC que não são valorizáveis na contabilidade, em negrito aqueles utilizáveis na produção.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Centro di custo | Código alfanumérico do CdC. |
| Descrição | Descrição do CdC. |
| Segunda descrição | Segunda descrição do CdC. |
| Flag PNR | O flag permite utilizar o centro de custo no módulo da produção. |
| Flag valorizável | Quando este flag é estabelecido o centro de custo pode ser valorizado com os movimentos de contabilidade. Tirar o flag dos níveis de estrutura que não se quer valorizar erroneamente e deixá-lo só, portanto, no nível mais baixo. |
| Tipo centro de custo | Define se o centro de custo é direto ou indireto. |
| Centro de responsabilidade | Associa o centro de custo ao relativo centro de responsabilidade. |
| Cost driver | Atribui ao centro de custo um cost driver de reatribuição da porcentagem nos encerramentos infra-anuais. |
| Data início | Define a data de início do CdC. |
| Data fim | Define a data fim de utilizo dos CdC. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar o próximo CdC | Botão para efetuar a pesquisa do CdC sucessivo áquele que é colocado. |
| Procurar o precedente CdC | Botão para efetuar a pesquisa do CdC precedente  áquele que é colocado. |
| Novo CdC | Botão para efetuar a inserção de um novo centro de custo. |
| Nova ligação CdC | Botão para efetuar a inserção de uma ligação/nível do CdC áquele que é colocado. |
| Cancelar a ligação CdC | Botão para efetuar o cancelamento da ligação/nível do CdC. |
| Explodir o CdC | Botão para explodir a estrutura a níveis do CdC. |
| Implodir o CdC | Botão para implodir a estrutura a níveis do CdC. |
| Diminuição tabulação | Botão para diminuir a tabulação em termos de visualização. |
| Aumento tabulação | Botão para aumentar a tabulação em termos de visualização. |
| Pré-visualização | Botão para efetuar a pré-visualização do CdC pela qual se colocou. |
| Imprimir | Botão para efetuar a imprensa. |
