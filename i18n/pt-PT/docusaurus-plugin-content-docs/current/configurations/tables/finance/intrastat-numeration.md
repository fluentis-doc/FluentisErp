---
title: Numeração Intrastat
sidebar_position: 15
---

A tabela permite definir, para todas as várias sociedade carregadas no banco de dados, o contador de início e a data de referência.

**Sociedade**: código da sociedade.

**Descrição sociedade**: descrição da sociedade.

**Tipo**: tipo de numeração.

**Descrição**: descrição numeração.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de uma numeração Intrastat. |
| Cancelar | Botão para efetuar o cancelamento de uma numeração Intrastat. |
