---
title: Precódigo ativo fixo
sidebar_position: 18
---

Com a definição desta tabela se codificam os precódigos válidos para todas as sociedades do banco de dados, que vão ser utilizáveis se está presente o flag de gestão precódigos nos parâmetros do módulo.

Os campos presentes são:

Código: Código alfanumérico do precódigo.

Descrição: Descrição do precódigo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do precódigo, através os campos filtro código e descrição. |
| Inserção | Botão para efetuar a inserção do precódigo. |
| Cancelamento | Botão para efetuar o cancelamento do precódigo cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
