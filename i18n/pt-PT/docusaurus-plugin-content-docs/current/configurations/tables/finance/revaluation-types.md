---
title: Tipos reavaliação
sidebar_position: 29
---

A definição desta tabela permite codificar as referências nas leis de reavaliação a utilizar para gerir os detalhes das reavaliações dentro de cada ficha Ativo fixo. Em particular, a gestão dos detalhes se habilita só para a categoria ativos fixo que apresentam o flag 'Reavaliações'.

**Código**: código de identificação do tipo de reavaliação.

**Descrição**: descrição do tipo de reavaliação.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo de reavaliação através os campos de filtro código e descrição. |
| Novo | Botão para efetuar a inserção de um novo tipo de reavaliação. |
| Cancelar | Botão para efetuar o cancelamento do tipo de reavaliação aonde é colocado. |
