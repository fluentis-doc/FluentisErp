A tabela permite definir uma lista de códigos de retribuição dos movimentos dos centros de custo/lucro.

Nesta primeira seção se insere o código alfanumérico e a descrição do Cost Driver.

**Código**: código Cost Driver alfanumérico de 5 caracteres;

**Descrição**: descrição Cost Driver.

Nesta segunda grelha é possível definir o critério porcentagem de repartição associada ao Cost Driver selecionado acima: a soma das porcentagens deste detalhe, liberalmente atribuíveis seja no CdC que no CdL para o mesmo Cost Driver, tem de dar o 100% do movimento.

**CdC**: centro de custo cujo atribuir o critério de repartição associado com o Cost Driver seguinte àquele em que é colocado;

**Descrição**: descrição CdC;

**CdL**: centro de lucro cujo atribuir o critério de repartição associado com o Cost Driver seguinte àquele em que é colocado;

**Descrição**: descrição CdL;

**Porcentagem**: porcentagem de atribuição do critério;

**Divisão**: a coluna Divisão, opcional, permite de forçar a atribuição do movimento numa particular divisão de impresa;

**Descrição**: descrição divisão;

**Exercício**: o campo 'Exercício', opcional, permite de associar a validade do critério de repartição num particular ano contábil.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um Cost Driver, através os campos código, descrição, CdC e CdL. |
| Inserção | Botão para efetuar a inserção de um novo Cost Driver. |
| Cancelamento | Botão para efetuar o cancelamento do Cost Driver cujo se colocou. |
| Cancelar os detalhes | Botão para efetuar o cancelamento do detalhe do Cost Driver cujo se colocou. |
| Pré-visualização de imprensa | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
