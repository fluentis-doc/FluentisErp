---
title: Câmbio moedas
sidebar_position: 4
---

A tabela dos câmbios moedas, comum em todas as sociedades do mesmo banco de dados, permite estabelecer dia por dia os câmbios uficiais a utilizar nos vários procedimentos do gestional: em particular se avisa que o programa vai recuperar pela tabela o último câmbio válido ao inverso da data do documento/registração em gestão.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Câmbio | Normalmente se insere a moeda da sociedade (euro). |
| Descrição | Descrição da moeda. |
| Em | Moeda de referência. |
| Descrição | Descrição da moeda de referência. |
| Data moeda | Data de referênica do câmbio. |
| Valor câmbio | Refere-se ao câmbio 'incerto por certo', utilizado pelo programa nos cálculos dos controvalores. No caso de euro, é o valor inverso ao câmbio uficial do dia. |
| Valor direto | Valor direto. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um câmbio com a possibilidade de filtrar pelos campos a 'Moeda' e 'Da data a data'. |
| Inserção | Botão para inserir um novo câmbio. |
| Atualização câmbios | Botão para atualizar os câmbios, apresenta uma dialog com a lista dos últimos câmbios estabelecidos na tabela. |
| Importar do arquivo | Botão para proceder com a importação do arquivo, no caso de arquivo da Home banking estruturado coerentemente. |
| Cancelamento | Botão para efetuar o cancelamento dos câmbios. |
| Pré-visualização de imprensa | Botão para efetuar a pré-visualização dos câmbios. |
| Imprimir | Botão para efetuar a imprensa dos câmbios. |
