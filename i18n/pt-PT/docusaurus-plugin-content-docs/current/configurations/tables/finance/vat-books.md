Tabela de impostação dos registros Imposto a gerir na sociedade ativa.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Inserção | Botão para efetuar a inserção de um novo registro Imposto. |
| Inserção ano | Botão para efetuar a inserção de um novo ano para o registro Imposto cujo é colocado. |
| Cancelar o registro Imposto | Botão para efetuar o cancelamento do registro Imposto cujo é colocado. |
| Cancelar o ano | Botão para efetuar o cancelamento do ano do registro Imposto cujo é colocado. |
| Pré-visualização de imprensa | Botão para efetuar a imprensa. |
| Imprimir | Botão para efetuar a imprensa. |
| Anular as imprensas definitivas | Botão para efetuar a anulação das imprensas definiticas dos registros. Se abre um Dialog pela qual selecionar o registro e o período que se quer anular. |

Os campos da primeira seção: 

**Código**: código alfanumérico identificativo do registro;

**Descrição**: descrição do registro imprimido normalmente nos relatórios uficiais;

**Tipo registro**: tipologia precodificada do registro;

**Descrição**: descrição do tipo registro;

**Controle protocolos**: a impostação deste flag bloqueia a possibilidade de imprimir em definitivo um registro que tem buracos nos protocoloes ou incoerências na classificação desses;

**A ventilar**: este flag impõe a gestão do registro relativo ao procedimento de ventilação dos correspetivos;

**Na liquidação Imposto**: permite determinar quais registros Imposto são a gerir na imprensa da liquidação Imposto periódica;

**Ref. reg. compra**: o campo é ativo só para os registros vendas/correspetivos, com o objetivo de indicar o registro Imposto compras de origem da ventilação dos correspetivos;

**Descrição**: descrição ref. reg. compra.

A segunda seção permite estabelecer o contador dos protocolos para cada ano contábil no campo 'Prot. Imposto inicial'.

**Ano**: ano de referência;

**Prot. Imposto inicial**: procotolo Imposto inicial do ano de referência;

**Último prot. Imposto**: o campo 'Último prot. Imposto' permite de visualizar qual é o último protocolo Imposto que foi imprimido em definitivo no registro;

**Data última imprensa**: data de execução da última imprensa definitiva;

**Última pag. imprimida**: último número de página imprimida em definitivo.
