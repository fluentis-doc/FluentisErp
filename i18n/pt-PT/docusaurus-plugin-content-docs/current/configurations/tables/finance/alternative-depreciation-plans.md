---
title: Planos de amortização alternativos
sidebar_position: 16
---

A definição desta tabela permite definir os planos de amortização a valorizar na simulação plurianual das amortizações.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Código | Código do plano de amortização. |
| Descrição | Descrição do plano de amortização. |
| Divisão | Flag de gestão numa só divisão ativa. |

Para simplificar a inserção dos detalhes das categorias é presente um botão porcentagem a aplicar em todas as categorias ainda a inserir no plano de amortização selecionado.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Categoria | Categoria cujo aplicar o plano de amortização. |
| Descrição | Descrição da categoria cujo aplicar o plano de amortização. |
| Porcentagem de aplicação  | Porcentagem a aplicar. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa do plano de amortização, através os campos filtro código e descrição. |
| Novo plano | Botão para efetuar a inserção de um novo plano de amortização. |
| Novo detalhe | Botão para efetuar a inserção do detalhe do plano de amortização cujo é colocado. |
| Cancelar o plano | Botão para efetuar o cancelamento do plano de amortização cujo é colocado. |
| Cancelar o detalhe | Botão para efetuar o cancelamento do detalhe do plano de amortização cujo é colocado. |
| Acrescentar as categorias que faltam | Botão para efetuar a atribuição em automático da porcentagem para todas as categorias ainda em falta. |
