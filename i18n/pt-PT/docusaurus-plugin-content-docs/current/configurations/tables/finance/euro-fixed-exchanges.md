title: Câmbios fixos para o euro
sidebar_position: 3

A tabela presenta a lista dos câmbios fixos de conversão do euro: estabelecendo o valor do euro o programa vai proceder automaticamente ao cálculo do valor inverso que vai ser utilizado nos cálculos dos controvalores do gestional.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Moeda | Código moeda. |
| Descrição moeda | Descrição moeda. |
| Data moeda | Data moeda. |
| 1 EURO = | Valore de 1 EURO = moeda de referência. |
| Câmbio inverso | Cálculo do câmbio inverso. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa, se um câmbio, com a possibilidade de fitlrar para os campos a moeda e da data a data. |
| Inserção | Botão para inserir um novo câmbio fixo para o euro. |
| Cancelamento | Botão para cancelar um câmbio. |
| Pré-visualização de imprensa | Botão para efetuar a pré-visualização de imprensa dos câmbios fixos para o euro. |
| Imprimir | Botão para efetuar a imprensa dos câmbios fixos para o euro. |
