---
title: Natura transação Intrastat
sidebar_position: 14
---

A tabela, precarregada, define a natura das transações para as declarações Intrastat.

Foi acrescentada uma tipologia fictícia 'S - Serviços' que tem o objetivo de poder individuar quais documentos (se inseridos como documento da área compras/vendas) reportar nas seções quater/quinquies previstas para as declarações a partir do ano 2010.

**Código**: código natura transitória.

**Descrição**: descrição natura transitória.

**Triangulação**: o flag 'Operações triangulares' permite de carregar a lista da última coluna da tabela.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um código natura de transação, através os campos código e descrição. |
| Novo | Botão para efetuar a inserção de um código natura de transação. |
| Cancelamento | Botão para efetuar o cancelamento de um código natura de transação cujo se colocou. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
