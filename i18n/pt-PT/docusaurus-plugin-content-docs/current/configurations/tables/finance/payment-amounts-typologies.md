---
title: Tipologias quantias de pagamento
sidebar_position: 30
---

A definição da tabela permite codificar as tipologias de pagamento a utilizar no módulo Gestão cobranças.

**Código**: código de identificação da tipologia de pagamento.

**Descrição**: descrição da tipologia de pagamento.

**Conta**: campo opcional para indicar a conta de relevação dos descrontos e brindes para a tipologia de pagamento.

**Subconta**: campo opcional para indicar a subconta de relevação dos descontos e brindes para a tipologia de pagamento.

**Descrição**: descrição da subconta.

**Quantia gravada**: o flag que individua em quais tipologias se vai calcular a efetiva quantia gravada.

**Notas**: nota a associar ao movimento contábil que releva o tipo de quantia.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar a tipologia de pagamento através os campos de filtro código e descrição. |
| Nova tipologia quantias | Botão para efetuar a inserção da tipologia de pagamento. |
| Cancelar a tipologia quantias | Botão para efetuar o cancelamento da tipologia de pagamento aonde é colocado. |
| Pré-visualização de imprensa | Botão para efetuar a pré-visualização de imprensa. |
| Imprimir | Botão para efetuar a imprensa. |
