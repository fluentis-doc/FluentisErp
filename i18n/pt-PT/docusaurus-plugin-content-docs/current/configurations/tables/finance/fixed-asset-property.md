---
title: Propriedade ativo fixo
sidebar_position: 19
---

A definição da tabela prevê um simples código e descrição da característica para poder atribuir aos vários ativos fixo: vai ser utilizável como filtro de pesquisa/imprensa para obter uma lista de ativos que tem um determinada propriedade.

Os campos são:

**Código**: código alfanumérico da propriedade ativo fixo.

**Descrição**: descrição da propriedade ativo fixo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:

 



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa da propriedade ativo fixo, através os campos filtro código e descrição. |
| Inserção | Botão para efetuar a inserção da propriedade ativo fixo. |
| Cancelamento | Botão para efetuar o cancelamento da propriedade ativo fixo cujo é colocado. |
| Pré-visualização | Botão para efetuar a pré-visualização. |
| Imprimir | Botão para efetuar a imprensa. |
