---
title: Categoria ativo fixo
sidebar_position: 5
---

Dentro desta tabela se codificam as categorias dos ativos fixos geridas na sociedade.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Código | Código alfanumérico. |
| Descrição | Descrição da categoria. |
| Conta | A conta patrimonial da imobilização. |
| Subconta | A subconta pratrimonial da imobilização. |
| Descrição | Descrição da subconta patrimonial da imobilização. |
| Mat./Imat. | Indicação da tipologia material ou imaterial do bem. |
| Máximo amort. | Quantia máxima da amortização acessível para cada ativo fixo da categoria. |
| Porc. amort. | A porcentagem máxima do valor do ativo fixo que pode ser amortizada. |
| Oper. aumento | Se o flag não é estabelecido, assim como de default, cada linha de aumento do valor do ativo fixo vai ser gerida com um detalhe de amortização separada e autonôma.Se o flag está estabelecido, ao contrário, cada aumento vai ser atribuído a linha de compra inicial, indo a aumentar sempre o valor inicial da sua data de origem. |
| Limite amort. total | Quota limite para a aplicação da amortização total no 100% no ano de compra. |
| Reavaliações | Flag de gestão dos detalhes das reavaliações. |
| Cód. balanço | Código de balanço para a imprensa do prospecto das reavaliações. |
| Amort. objetivo | Ativa a gestão dos dados objetivos do ativo fixo: ela prevê uma duplicação dos valores patrimoniais contábeis ao fim de aplicar uma lógica de amortização extracontábil separada. |

A segunda seção é dedicada a impostação dos detalhes de amortização aplicáveis na categoria superior selecionada.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo amort. | Selecionar a tipologia de amortização entre a amortização ordinária, amortização adiantada e amortização desacelerada, esta última normalmente não é utilizada. |
| Custo conta | Conta de custo aonde relevar a amortização periódica. |
| Custo subconta | Subconta de custo aonde relevar a amortização periódica. |
| Descrição conta | Descrição subconta de custo. |
| Fundo conta | Conta de fundo aonde pôr de lado a amortização periódica. |
| Fundo subconta | Subconta de fundo aonde pôr de lado a amortização periódica que poderia ser também o mesmmo valor patrimonial ativo para estornar diretamente deste o valor. |
| Descrição conta | Descrição subconta de fundo. |
| Excluir | O flag permite excluir do cálculo automático das amortizações o tipo de amortização selecionada. |
| Porcentagem | A porcentagem de amortização a aplicar para o tipo de amortização estabelecido. |
| Nº ano | Se ativa só em correspondência das amortizações adiantadas, pela qual vão ser inseridos os 3 anos de aplicação. Em referência a este valor, se lembra que para os ativos fixos utilizados o número de anos de aplicação é estabelecido automaticamente só no ano de compra. |
| Amort. primeiro ano | Se propõe automaticamente para todas as categorias materiais, e vai dividir a porcentagem de amortização no ano de início atividade do ativo fixo. |
| Fiscal | Pode ser estabelecido para as linhas de amortização adiantada para geri-los como amortização fiscal não contabilizável. |
| Tipo dedutibilidade | Permite atribuir ao tipo de amortização um código de dedutibilidade aos fins fiscais. |
| Descr. dedutibilidade | Descrição do tipo de dedutibilidade. |

Na terceira seção é possível atribuir, em referência a cada linha de amortização, a valorização porcentagem de default no centro de custo para os ativos fixos associados com a categoria. Tal impostação é secundária em relação à atribuição dos centros de custo dentro de cada registro ativo fixo.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| CdC | Centro de custo cujo atribuir a valorização dos ativos fixos que pertencem a categoria. |
| Descrição | Descrição centro de custo. |
| Porcentagem | Impostação da porcentagem de valorização no centro de custo. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar na tabela superior | Botão para efetuar a pesquisa na tabela superior através dos campos de filtro categoria e descrição. |
| Inserir na tabela superior | Botão para inserir uma categoria ativo fixo. |
| Inserir na tabela detalhe | Botão para inserir o detalhe da tabela superior selecionada. |
| Inserir na tabela CdC | Botão para inserir os CdC da tabela superior selecionada. |
| Cancelar as linhas selecionadas | Botão para efetuar o cancelamento das linhas selecionadas (categoria ativo fixo). |
| Cancelar as linhas selecionadas pela tabela inferior | Botão para efetuar o cancelamento das linhas selecionadas da tabela inferior (detalhe do tipo de amortização). |
| Cancelar as linhas selecionadas da tabela CdC | Botão para efetuar o cancelamento das linhas selecionadas de tabela centros de custo. |
| Pré-visualização imprensa | Botão para efetuar a pré-visualização de imprensa. |
| Imprimir | Botão para efetuar a imprensa. |
