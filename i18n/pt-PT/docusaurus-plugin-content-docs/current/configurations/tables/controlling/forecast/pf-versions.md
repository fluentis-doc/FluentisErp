Esta Form permite: inserir mais versões para as previsões, estabelecer uma ou mais versões ativas (aquelas não ativas vão poder ser utilizadas para criar novas previsões mas só para filtrar as previsões existentes) e estabelecer a versão de default entre aquelas ativas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Inserir uma nova versão | Botão para inserir uma nova versão. |
| Cancelar as versões selecionadas | Botão para cancelar as versões selecionadas. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação so filtros estabelecidos e a configuração da impressora. |
