title: Versões
sidebar_position: 1
---

Esta Form permite inserir mais versões para as previsões, estabelecer uma ou mais versões ativas (aquelas não ativas não vão poder ser utilizadas para criar novas previsões mas podem ser utilizadas para filtrar as previsões existentes), estabelecer a versão de default entre aquelas ativas, aceitar a ligação com a previsão semanal e modificar as previsões históricas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Inserir versão | Botão para inserir uma nova versão. |
| Cancelar as versões | Botão para cancelar as versões selecionadas. |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
