---
title: Períodos
sidebar_position: 3
---

A Form permite inserir alguns períodos para cada ano. Para cada período se podem estabelecer as semanas necessárias. A Form contém três áreas: a primeira com os filtros de pesquisa, a segunda com os períodos e a terceira com os detalhes da semana.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar os períodos de um certo ano. |
| Inserir | Botão para criar um novo período.  |
| Cancelar | Botão para cancelar a linha selecionada pela tabela dos períodos. |
| Cancelar detalhe | Botão para cancelar um detalhe/semana do período atual. |
