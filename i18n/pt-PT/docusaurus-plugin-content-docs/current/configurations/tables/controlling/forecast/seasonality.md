---
title: Sazonalidade
sidebar_position: 2
---

Esta Form permite gerir as variações de sazonalidade misuradas nas vendas dos períodos precedentes, por artigo e período. Esta operação, pode-se fazer manualmente ou em automático com o procedimento de retomada dos dados da previsão selecionada pela atividade de venda anterior; o procedimento oferece a possibilidade de gerar a sazonalidade. A Form contém três áreas: a primeira com filtros de pesquisa, a segunda com os artigos já registrados na sazonalidade e a terceira, sazonalidade, visualiza, para cada mês do ano, a quantidade comprada para o artigo selecionado na área dos artigos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar artigos | Botão para procurar os artigos no banco de dados. |
| Inserir ano | Para o artigo selecionado se cria uma nova linha para um novo ano na tabela de sazonalidade, permitindo ao usuário de específicar as vendas para cada período do novo ano. |
| Cancelar artigos | Botão para cancelar o artigo selecionada e a sua sazonalidade. |
| Duplicar | Este procedimento duplica a sazonalidade do precedente ano pelo artigo selecionado. |
| Pré-visualização | Permite de executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação aos filtros estabelecidos e a configuração da impressora. |
