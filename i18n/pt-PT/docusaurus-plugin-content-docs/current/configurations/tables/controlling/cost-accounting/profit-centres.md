A tabela permite a definição da estrutura, também em níveis diferentes, e as características dos centros de lucro.

A seção geral apresenta a estrutura ramificada dos Centros de lucro definidos para a sociedade ativa: em vermelho são visualizados os CdL que não são valorizáveis na contabilidade, em negrito aqueles utilizáveis na produção.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Centro de lucro | Código alfanumérico do CdL. |
| Descrição | Descrição do CdL. |
| Segunda descrição | Segunda descrição do CdL. |
| Flag PNR | O flag permite utilizar o centro de lucro no módulo da produção. |
| Flag valorizável | Quando este flag é estabelecido o centro de lucro pode ser valorizado com movimentos de contabilidade.Tirar o flag dos níveis de estrutura que não se quer valorizar erroneamente e deixá-lo só, portanto, no nível mais baixo. |
| Tipo centro de lucro | Define se o centro de lucro é direto ou indireto. |
| Centro de responsabilidade | Associa o centro de lucro ao relativo centro de responsabilidade. |
| Data início | Define a data início do CdL. |
| Data fim | Define a data fim de utilizo dos CdL. |

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar o próximo CdL | Botão para efetuar a pesquisa do CdL sucessivo áquele que é colocado. |
| Procurar o precedente CdL | Botão para efetuar a pesquisa do CdL precedente áquele que é colocado. |
| Novo CdL | Botão para efetuar a inserção de um novo centro de lucro. |
| Nova ligação CdL | Botão para efetuar a inserção de uma ligação/nível do CdL aonde se é colocado. |
| Cancelar a ligação CdL | Botão para efetuar o cancelamento da ligação/nível do CdL. |
| Explodir o CdL | Botão para explodir a estrutura a níveis do CdL. |
| Implodir o CdL | Botão para implodir a estrutura a níveis do CdL. |
| Diminuição tabulação | Botão para diminuir a tabulação em termos de visualização. |
| Aumento tabulação | Botão para aumentar a tabulação em termos de visualização. |
| Pré-visualização | Botão para efetuar a pré-visualização do CdL pela qual se colocou. |
| Imprimir | Botão para efetuar a imprensa. |
