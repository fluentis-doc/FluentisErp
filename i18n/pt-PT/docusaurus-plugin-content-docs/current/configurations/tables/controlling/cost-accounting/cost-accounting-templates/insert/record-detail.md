---
title: Detalhe registração
sidebar_position: 2
---

Na seção 'Detalhe registração' é possível indicar uma lista de subcontas industrial a propor automaticamente nas registrações industriais criadas com a causal em uso. Essas subcontas vão ser propostas (com a relativa nota inserida no campo dedicado: não é prevista a gestão dos códigos paramétricos da causal) só se a causal for 'A valor' e a conta industrial têm associado no plano das contas uma conta de contabilidade geral.

Na base é permitido depois definir uma descrição paramétrica a utilizar para descrever o movimento.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nova causal | Com este botão se executa a abertura de uma nova causal industrial. |
| Cancelar a causal | Cancela a causal em uso. |
| Salvar a registração | Salva a causal em uso. |
| Cancelar os detalhes selecionados | Cancela os detalhes selecionados. |
