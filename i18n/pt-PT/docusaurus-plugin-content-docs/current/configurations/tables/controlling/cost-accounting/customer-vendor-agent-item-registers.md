---
title: Registros clientes/fornecedores/agentes/artigos
sidebar_position: 2
---

No tab 'Administração' do registro clientes/fornecedores é possível indicar a subconta industrial de default a utilizar para a contabilização industrial se a linha artigo não tem associado um código de faturado compras/vendas.

No tab 'Detalhe' do registro agentes, invés, está presente a subconta industrial de default a propor nos detalhes remunerações para os tipos retenção que preveem uma causal de contabilidade industrial.

No registro artigo é possível indicar a subconta de contabilidade industrial a propor nas registrações a quantidade com artigo vendas/compras.
