---
title: Tipos de reclassificação
sidebar_position: 6
---

Os campos previstos são aqueles do código alfanumérico de 5 caracteres, da relativa descrição e de dois flag: o flag 'Filtro encomendas' vai render disponível, para as reclassificações baseadas no modelos deste tipo, um filtro dos dados de acordo com a encomenda de venda, em modo de obter um reclassificado só dos dados relativos ao filtro; no mesmo modo, o 'Filtro CdC/CdL' vai ativar a possibilidade de filtrar os dados do reclassificado por um determinado centro de custo/lucro.

Existem 3 tipologias padrão: uma geral '01 - econômico', um 'BEP - Break Even Point' e um 'IV - Imposto diretiva' para o modelo de reclassificação estabelecido de acordo quanto previsto na IV diretiva Cee.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com este botão se executa a pesquisa dos tipos de reclassificação de acordo com os filtros estabelecidos. |
| Inserção | Posiciona o cursor na inserção de uma nova linha. |
| Cancelamento | Cancela o tipo de reclassificação selecionado. |
| Pré-visualização imprensa | Executa a pré-visualização de imprensa da tabela em uso. |
| Imprimir | Executa a imprensa da tabela em uso. |
