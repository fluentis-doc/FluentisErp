title: Numeração CT-e/Notas de entrega
sidebar_position: 7
---

A tabela de Numeração CT-e/Notas de entrega permite definir alguns progressivos de numeração diferentes dentro do mesmo ano a atribuir um ou mais tipos CT-e/Notas de entrega.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um tipo numeração. |
| Inserção  | Botão para inserir um novo tipo de numeração. |
| Cancelamento | Botão para cancelar o tipo de numeração. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Ano | Ano de referência da numeração. |
| Tipo | Tipo numeração. |
| Descrição | Descrição do tipo numeração. |
| Nº prog. | Número progressivo de partida para a numeração dos CT-e. |
| Retorno cliente | Se estabelecido, indica que o tipo de numeração é utilizado para numerar os retornos, portanto não controla mais o número\data progressivos e se acrescenta a sua referência na linha nota criada na valorização CT-e. |
| Séries | Se inserida, é utilizada na contabilização para reportar junto com o número documento também a série. |
| Nº fixo caracteres | Se presente, é utilizado para compor a série. |
