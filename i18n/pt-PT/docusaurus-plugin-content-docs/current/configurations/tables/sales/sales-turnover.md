Os dados contidos na tabela do tipo faturada são necessários no momento da contabilização dos documentos.

O código faturado pode ser codificado só se foi inserida também a conta subconta contábil, o tipo Imposto e o tipo quantia. Se o tipo faturado foi inserido no registro artigos, esse código vai ser proposto nas linhas documento e é propositivo para a contabilização dos documentos, se no artigo não estiver inserido o tipo faturado, vai ser retomada a conta de custo presente no registro do fornecedor. A conta-subconta presente nesta tabela é propositiva também para os centros de custo nas linhas artigo dos documentos.

**Detalhe para as nações**: esses dados são utilizados só para a troca dos dados com os programas externos. No futuro vai ser possível definir as contas de custo diferentes por nação e tipo de contabilização artigo, associado com o registro cliente.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo | Código de identificação do faturado das vendas. |
| Descrição | Descrição do faturado de vendas. |
| Conta/Subconta descrição subconta | Conta de custo utilizado na contabilização dos documentos de venda. |
| Tipo Imposto | Utilizado na registração da contabilização do documento. Normalmente é estabelecido a geral, para outros detalhes verificar a utilização das causais contábeis e nas registrações contábeis na área administrativa. |
| Conta industrial/subconta industrial e descrição  | Conta utilizada para contabilizar o documento na contabilidade analítica. |
| Tipo/Descrição | Utilizado na registração da contabilização do documento. Normalmente é estabelecido no 11 Tributável subconta + taxa indedutível, para outros detalhes verificar a utilização nas causais contábeis e nas registrações contábeis na área administrativa. |
