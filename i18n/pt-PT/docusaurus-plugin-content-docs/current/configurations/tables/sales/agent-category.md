A tabela das categorias agente, única para as sociedades presentes no banco de dados, é a tabela de base para os cálculos Enasarco/FIRR dos agentes.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de uma categoria agente através os campos filtro. |
| Inserção | Botão para inserir uma nova categoria agente. |
| Cancelamento | Botão para cancelar uma categoria agente, só se essa já não foi utilizada. |
| Pré-visualização de imprensa  | Botão para efetuar a pré-visualização de imprensa das categorias agentes. |
| Stampa | Pulsante per effettuare la stampa delle categorie agenti. |

A primeira seção contém os seguintes campos:

**Código**: código, alfanumérico, de identificação da categoria agente.

**Descrição**: sescrição da categoria agente.

**Mín. Enasarco**: é o valor mínimo do Enasarco.

**Max. Enasarco**: é o valor tributável máximo anuo do Enasarco.

**Tipo retenção**: é o tipo de retenção: se refere a seção Home, Tabelas, Administrativas para os detalhes. Se aconselha de identificar o tipo de retenção no registro agente.  

**Porc. Enasarco**: é a porcentagem de cálculo Enasarco da quota agente.  

**Porc. Enasarco empresa**: é a porcentagem de cálculo Enasarco da quota empresa. 

**Trib. Enasarco**: é a porcentagem (100%) de aplicação das porcentagens Enasarco no total de comissão.

**Monomandatário**: flag que identifica as categorias monomandatárias. 

**Plurimandatário**: flag que identifica as categorias plurimandatárias.

**Geral**: flag que identifica as categorias gerais.

A segunda seção contém, para a categoria agente selecionada, os escalões de cálculo do FIRR.

**Linha**: flag que identifica o número de linha. Tem de ser unívovo em todas as categorias agentes.

**Porcentagem**: porcentagem do tributo.

**Da comissões**: limite inferior do escalão.

**A comissões**: limite superior ou igual do escalão.
