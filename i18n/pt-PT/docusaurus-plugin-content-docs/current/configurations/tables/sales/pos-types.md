---
title: Tipos POS
sidebar_position: 20
---

O Tipos POS é utilizado para determinar as propriedades de uma fatura.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação aos filtros estabelecidos e a configuração da impressora. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo  | Tipo POS. |
| Descrição | Descrição do tipo POS. |
| Tipo Num. /Descrição | Tipo numeração associada com o tipo POS. |
| Default | Se estabelecido, indica o tipo POS de default que é proposto na criação de um novo POS. |
| Notas | Notas descritivas para o tipo POS, a utilizar nas imprensas. |
| Preço com Imposto | Se estabelecido, os preços presentes no POS são com Imposto. |
| Adiantamento | Se estabelecido, o POS é de tipo adiantamento e vai poder ser estornado num POS normal. |
| Server POS | Server onde ler e/ou escrever os dados dos POS. |
| Banco de dados POS | Banco de dados onde ler e/ou escrever os dados dos POS. |
| Tabela POS | Tabela onde ler e/ou escrever os dados dos POS. |
| Tabela assoc. documentos | Tabela onde ler e/ou escrever os dados dos documentos associados com o POS. |
| UM cheg. | UM utilizada na criação de CT-e e faturas de POS. |
| IVA cheg./Descrição Imposto | Código Imposto utilizado na criação do CT-e e fatudas do POS. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Ponto de venda | Ponto de venda cujo são criados os POS. |
| Caixa | Código da caixa a associar com o POS. |
| Cat.produto | Categoria produto. |
| Imposto | Código Imposto a utilizar nos POS. |
| Depósito | Depósito de utilização da mercadoria contida no POS desse tipo. |
| Causal | Causal de movimentação da mercadoria utilizada contida no POS desse tipo. |
| Tipo pag. | Tipo pagamento proposto nos POS desse tipo. |
| Sol. pag. | Solução de pagamento proposta nos POS desse tipo. |
| Catálogo | Código catálogo geral pela qual são retomados os preços dos artigos a inserir no POS. |
| Data Validade | Data validade do catálogo geral. |
| Descarregamento automático no encerramento do correspetivo | Se estabelecido, quando o POS é imprimido é automaticamente também descarregado pelo depósito. |
| É necessário a caixa registradora | Se estabelecido, para utilizar os POS é necessário a ligação com uma caixa registradora para a imprensa da nota fiscal. |
| Visualizar as variantes | Se estabelecido, podem ser utilizadas também as variantes artigo. |
| Visualizar os atributos | Se estabelecido, podem ser utilizados também os atributos artigo. |
