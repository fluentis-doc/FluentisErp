title: Tipo faturas

O Tipo faturas é utilizado para determinar as propriedades de uma fatura.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |

**Tipo**: tipo fatura.

**Descrição**: descrição do tipo fatura. 

**Natura fatura**: natura associada com o tipo fatura.

**Tipo Num./Descrição**: tipo numeração associada com o tipo fatura. Numerazione fatture.

**Da Ordem**: se estabelecido significa que a fatura pode ser criada executando uma ordem cliente. 

**Fatura imediata**: se estabelcido, significa que a fatura é de tipo imediato e vão ter que ser compilados, na fatura, também os dados para o transporte.

**Trans. código**: código de transação utilizados para a declaração Intrastat.

**Na estatística**: se estabelecido, significa que a fatura desse tipo vai ser considerada para as estatísticas.

**Próforma**: se estabelecido, a fatura é de tipo próforma e não pode ser contabilizada.

**Causal/Descrição causal**: causal contábil utilizada para a contabilização da fatura. Causali di contabilità generale - Inserimento.

**Causal analítica/Descrição causal analítica**: causal analítica utilizada para a contabilização da fatura na contabilidade analítica. 

**Notas**: podem ser inseridas algumas notas descritivas do tipo fatura. 

**Controle cliente**: na inserção da subconta de título da ordem é verificada que o tipo conta é obrigatoriamente cliente. 

**Bloqueio doc. impr.**: se estabelecido, não é possível modificar uma fatura já imprimida.

**Intracomunitária**: se estabelecido, se verifica se a Nação do cliente é CEE, se o tipo fatura e o cliente não for do mesmo tipo de lança uma mensagem de aviso. 

**Exec. fat. próforma**: se estabelecido, e se a fatura é próforma a execução executa a ordem, se não a ordem permanece não executada. 

**Causal de transporte**: descrição da causal do transporte.

**Crédito**: se estabelecido, a fatura com este tipo retorna no controle do crédito. 

**Excluir o bloqueio**: se estabelecido, significa que mesmo se supera o crédito, e o documento vai ser bloqueado, deixa mesmo inseri-lo e imprimi-lo. 

**Causal depósito/Descrição causal depósito**: causal de movimentação da mercadoria utilizada contida nas faturas desse tipo. 

**Dep. /Descrição depósito**: depósito de utilização da meradoria contida nas faturas desse tipo. 

**Agrup. partidas**: se estabelecido, na contabilização das faturas vão ser agrupadas as partidas.

**Sem somar a quantidade**: se estabelecido, na execução das ordens deste tipo as quantidades executadas vão ser reportadas no documento de execução para cada linha de execução. 

**Agr. Venda Imposto**: se estabelecido, é procurada pelo artigo se existe o código Imposto agrícola, se não existe é pego o código Imposto normal. 

**Controlar a retirada descarr.**: se estabelecido, quando se cria uma nota de entrega da retirada descarregada, se ativa em automático o flag descarregado para a nota de entrega.

**Bloquear a quant. retir.**: bloqueia a quantidade e a quantidade alternativa na nota de entrega criada pela lista UdC. 

**Calcular as comissões**: se estabelecido, para as faturas com esse tipo vão ser calculadas as comissões dos agentes. 

**Desconectar o CT-e**: se estabelecido, é possível descarregar também as faturas diferidas.

**Gest. EDI**: se estabelecido, para as faturas com este tipo é possível criar os arquivos EDI. 

**Não executa a retirada**: se estabelecido, criando a fatura da retirada, essa não resulta executada. 

**Preço com Imposto**: se estabelecido os preços nas faturas desse tipo vão ser compreendidos de Imposto. 

**Comp. econ.**: se estabelecido, o tipo fatura tem competência econômica. 

**Causal comp. econ.**: causal de competência econômica utilizada na contabilização das faturas.
