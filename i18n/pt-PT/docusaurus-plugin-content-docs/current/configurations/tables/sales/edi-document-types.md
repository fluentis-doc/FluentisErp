title: Tipos documentos EDI
sidebar_position: 15
---

O Tipos documentos EDI é utilizado para determinar as propriedades de um documento EDI.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa de um tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo  | Código tipo EDI. |
| Descrição | Descrição tipo EDI. |
| Tipo num.   | Tipo numeração EDI associado com o tipo EDI. |
| Tipo ordem | Tipo ordem cliente para a criação do documento EDI. |
| Tipo previsão de venda | Foi estabelecido em automático a previsão semanal para a criação do documento EDI. |
| Default | Se estabelecido, assinala o tipo EDI de default para a criação automática de documentos EDI. |
| Notas | Notas relativas ao tipo EDI. |
