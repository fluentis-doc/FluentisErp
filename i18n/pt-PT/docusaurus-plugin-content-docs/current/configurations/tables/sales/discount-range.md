title: Gama desconto
sidebar_position: 12
---

A tabela Gama desconto permite definir qual é o elemento pela qual definir a variabilidade dos escalões de aplicação dos descontos, cujos definir as comissões agentes.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para efetuar a pesquisa da gama de desconto através os campos filtro. |
| Inserção | Botão para inserir uma nova gama de desconto. |
| Cancelamento | Botão para cancelar uma gama de desconto, se essa já não foi utilizada. |
| Pré-visualização de imprensa | Botão para efetuar a pré-visualização de imprensa das gamas de desconto. |
| Imprimir | Botão para efetuar a imprensa das gamas de desconto. |

**Gama**: código, alfanumérico, de identificação da gama de desconto.

**Descrição**: descrição da gama de desconto.

**Único**: com este flag as faixas de comissão de desconto vão ser aplicadas em todos os artigos indiferentemente.

**Classes art.**: com este flag se vai poder definir tantas faixas de comissões de desconto quantas são as classes artigo.

**Catálogos**: com este flag se vai poder definir tantas faixas de comissões de desconto quantas são os catálogos de venda (não personalizados) definidos.

**Tipos art.**: com este flag pode-se definir tantas faixas de comissões de desconto quantas são os tipos artigo.

**Característica**: com este flag se vai poder definir tantas faixas de comissões de desconto quantas são as caracteríticas artigo.

**Cat. produto**: com este flag se pode definir tantas faixas de comissões de desconto quantas são as categorias produto dos artigos.
