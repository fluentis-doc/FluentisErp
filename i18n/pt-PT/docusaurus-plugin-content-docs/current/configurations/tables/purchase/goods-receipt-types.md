title: Tipos recebimento mercadorias
sidebar_position: 13
---

O Tipo recebimento mercadoria é utilizado para determinar as propriedades de um recebimento de mercadorias.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo documento. |
| Inserção  | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa  | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação so filtros estabelecidos e a configuração da impressora. |

**Código**: código do tipo recebimento mercadorias.

**Descrição**: descrição do tipo recebimento mercadorias.

**Sem somar a quantidade**: se estabelecido, na execução das ordens deste tipo, as quantidades executadas vão ser reportadas no documento de execução para cada linha de execução.
