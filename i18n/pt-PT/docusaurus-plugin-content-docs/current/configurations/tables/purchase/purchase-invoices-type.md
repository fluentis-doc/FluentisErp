O Tipo faturas de compra é utilizado para determinar as propriedades de uma fatura de compra.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |

**Tipo**: tipo fatura

**Descrição**: descrição do tipo fatura.

**Natura fatura**: indica a natura da fatura: fatura, recibo fiscal, rec.fisc.remp, nota de crédito do fornecedor, nota de débito do fornecedor.

**Causal/Descrição causal**: causal contábil utilizada para contabilizaar a fatura. 

Causal analítica/Descrição causal analítica: causal analítica utilizada para contabilizar a fatura.

**Tran. código**: código de agrupamento para a declaração Intrastat.

**Dep./Descrição depósito**: depósito de chegada da mercadoria contida nas faturas deste tipo de fatura.

**Causal depósito/Descrição causal depósito**: causal de movimentação da mercadoria em chegada contida nas faturas deste tipo de fatura.

**Da ordem**: a fatura com este tipo pode executar as ordens fornecedores. 

**Fatura imediata**: a fatura com este tipo é de tipo imediato e vão ter que ser compilados, na fatura, também os dados para o transporte. 

**Intracomunitária**: se é estabelecido, se verifica se a Nação do fornecedor é CEE; se o tipo fatura e o fornecedor não for do mesmo tipo vem fora a mensagem de aviso.

**Em estatística**: a fatura com este tipo é reportada nas estatísticas. 

**Controle fornecedor**: se é estabelecido, é possível criar as faturas de compra só com conta e subconta com o tipo conta fornecedor; se não for estabelecido se podem utilizar também os clientes.

**Agrup. partidas**: se estabelecido, na contabilização da fatura as partidas vão ser agrupadas.

**Sem agrupar a quantidade**: se estabelecido, na execução das ordens deste tipo as quantidades executadas são reportadas no documento de execução para cada linha de execução.

**Agr. Compra Imposto**: se estabelecido, se procura para o artigo se existe o código de Imposto agrícola, se não existe se pega o código Imposto normal. 

**Fat. a rec.**: se estabelecido, a fatura com este tipo é uma fatura a receber. 

**Estorno fat. a rec.**: se estabelecido, a fatura com este tipo é um estorno de uma fatura a receber.

**Causal comp. econ.**: causal contábil para a competência econômica da fatura.
