---
title: Tipo ordens fornecedores
sidebar_position: 10
---

O Tipo ordem fornecedor é utilizado para determinar as propriedades de uma ordem.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação com os filtros estabelecidos e a configuração da impressora. |

**Tipo**: tipo ordem cliente.

**Descrição**: descrição do tipo ordem.

**Tipo Num./Descrição**: tipo numeração associada com o tipo ordem.

**Autoexecução**: se estabelecida, a ordem pode ser executada pelos documentos notas de entrega de compra, recebimento mercadorias e faturas de compra.

**Fluxo de caixa**: se estabelecido, o tipo ordem calcula o cálculo do fluxo de caixa.

**Dep./Descrição depósito**: depósito em chegada da mercadoria contida nas ordens deste tipo ordem.

**Causal/Descrição causal**: causal de movimentação da mercadoria em chegada contida nas ordens destes tipo ordem.

**Controle disponibilidade**: se estabelecido, as ordens deste tipo concorrem ao cálculo da disponibilidade

**Agr.Comp.Imposto**: se estabelecido, se procura para o artigo se existe o código de Imposto agrícola, se não existe se pega o código Imposto normal.
