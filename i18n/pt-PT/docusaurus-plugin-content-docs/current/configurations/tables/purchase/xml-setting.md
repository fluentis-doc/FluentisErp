Esta tabela é necessária para estabelecer os dados a importar um catálogo fornecedor descarregando-o de um site internet no formato XML.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo de documento. |
| Aplicar |  |
| Duplicar | Botão para duplicar uma impostação XML. Permite a criação de uma cópia de um documento, reportando todas as condições do documento de origem, mas com os dados autais. |
| Cancelamento | Botão para cancelar o tipo documento. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Conta-Subconta/Descrição subconta | Conta e subconta do cliente cujo estabelecer o catálogo. |
| Endereço | Endereço cujo descarregar o arquivio XML:http://zoppelletto@gruppomade.com:srvr5m9p@ws.madelogo.net. |
| Usuário | Usuário para se conectar com o endereço. |
| Password | Password para conectar com o endereço. |
| Webservice | Caminho aonde encontrar os dados a importar e o arquivo: /listino/MPI.xml |
| Informação inicial | Informação inicial aonde partir para a criação do arquivo XML: offset='1'. |
| Tag inicial | Define o tag inicial para reconhecer o início do arquivo XML: catálogo. |
| Linha inicial | Define a linha pela qual iniciar com a importação: artigos. |
| Página inicial | Define a página inicial: página. |
| Dimensão página | Define a dimensão das páginas: offset='500'. |
| Classe  | Classe a utilizar para importar os artigos. |
| UM | UM a utilizar para importar os artigos. |
| Marca | Marca a utilizar para importar os artigos. |
