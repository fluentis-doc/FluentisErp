title: Tipo pedidos
sidebar_position: 14
---

O Tipo pedido é utilizado para determinar as propriedades de um pedido.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar o tipo documento. |
| Inserção | Botão para inserir um novo tipo documento. |
| Cancelamento | Botão para cancelar o tipo documento. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo  | Tipo pedido de compra. |
| Descrição | Descrição do tipo de pedido. |
| Tipo Nº/Descrição numeração | Tipo numeração associada com o tipo pedido. |
| CdC | Centro de custo a propor na criação do pedido de compra. |
| Dep./Descrição dep. | Depósito a utilizar na criação do pedido de compra. |
| Controlar a disponibilidade | Se estabelecido, os pedidos de compra criados com este tipo são utilizados para o cálculo da disponibilidade. |
