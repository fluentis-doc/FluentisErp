title: Tipos apresentação listas efeitos
sidebar_position: 4
---

A tabela é compilada através a impostação de um código e de uma descrição usuário, pelas quais se atribui depois um numerador de referência.

As opções ulteriores previstas na máscara oferecem a possibilidade de definir:

- se o tipo lista é a gerir como lista de variação efeitos ao fornecedor;

- se o tipo lista é a gerir como lista RID 'utilitários' na criação do arquivo telemático no módulo RIBA.

As subcontas de movimentação contábil a propor no ato da criação de uma nova lista: se refere a subconta de registro banco pela qual vai ter o crédito final efeitos e do transitório de apresentação para a contabilização das listas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com esta botão se executa a pesquisa dos tipos lista em relação os filtros estabelecidos. |
| Inserção | Posiciona o cursor na inserção de uma nova linha. |
| Cancelamento | Cancela o tipo de adiantamento selecionado. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa da tabela em uso. |
| Imprimir | Executa a imprensa da tabela em uso. |
