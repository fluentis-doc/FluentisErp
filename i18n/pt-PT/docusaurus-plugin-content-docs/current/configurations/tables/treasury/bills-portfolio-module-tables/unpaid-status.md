title: Estados não pagos
sidebar_position: 6
---

A tabela é compilada através a imposição de um código e de uma descrição usuário para os estados necessários na gestão dos não pagos da sociedade, estabelecendo um dos estados predefinidos entre:

**Não pago**;

**Contabilizado**;

**Emitida notas de débito (não utilizado)**;

**Contabilizada notas de débito (não utilizado)**;

**Cobrado**;

**Autorizado (não utilizado)**.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com este botão se executa a pesquisa dos estados não pagos em relação os filtros estabelecidos. |
| Inserção | Posiciona o cursor na inserção de uma nova linha. |
| Cancelamento | Cancela o tipo de adiantamento selecionado. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa da tabela em uso. |
| Imprimir | Executa a imprensa da tabela em uso. |
