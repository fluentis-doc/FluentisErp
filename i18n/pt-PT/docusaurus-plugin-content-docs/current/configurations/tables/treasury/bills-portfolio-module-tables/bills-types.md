title: Tipos efeitos
sidebar_position: 3
---

A tabela é compilada através a imposição de um código e de uma descrição usuário, pelas quais se atribui um numerador de referência, uma tipologia de efeito entre aquelas predefinidas e enfim uma subconta de movimentação: tal subconta vai ser utilizada na contrapartida no encerramento do cliente dentro do procedimento de contabilização efeitos. O tipo efeito pode ser associado com o tipo de pagamento na tabela homônima. Precisa avisar, enfim, que só os efeitos de tipo cambiário vão poder ser geridos no procedimento de contabilização não pagos na cessão do módulo Não pagos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com este botão se executa a pesquisa dos tipos efeitos em relação com os filtros estabelecidos. |
| Inserção | Posiciona o cursor na inserção de uma nova linha. |
| Cancelamento | Cancela o tipo de adiantamento selecionado. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa da tabela em uso. |
| Imprimir | Executa a imprensa da tabela em uso. |
