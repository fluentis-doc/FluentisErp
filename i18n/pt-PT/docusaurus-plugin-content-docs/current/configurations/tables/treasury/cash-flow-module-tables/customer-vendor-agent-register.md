---
title: Registro cliente/fornecedor/agente
sidebar_position: 5
---

No cabeçalho dos registros é possível estabelecer a subconta de pagamento: essa conta vai ser associada com as partidas contábeis intitulada pelo registro e a partir daqui utilizadas no fluxo de caixa para gerir a canalização dos fluxos.

Além disso, o campo 'Dias em atraso' é utilizado para calcular, a partir da data de vencimento, a data de adiantamento/pagamento prevista com esse número de dias em atraso padrão do registro.
