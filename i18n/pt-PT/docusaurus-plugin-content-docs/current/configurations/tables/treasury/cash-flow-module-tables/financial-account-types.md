Nesta tabela, vai-se definir a lista das subcontas contábeis pelas quais calcular o saldo contábil que vai ser considerado como disponibilidade inicial do fluxo de caixa.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com este botão se executa a pesquisa dos tipos conta financeira em relação aos filtros estabelecidos. |
| Novo tipo | Posiciona o cursor na inserção de uma nova linha. |
| Cancelar os tipos | Cancela o tipo conta financeira selecionado. |
