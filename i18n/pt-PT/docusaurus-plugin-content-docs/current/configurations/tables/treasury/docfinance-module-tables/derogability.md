A tabela 'Derrogação' permite definir os códigos de derrogação dos fluxos financeiros previstos, assim como previstos no DocFinance (o código do gestional tem de ser o mesmo do DocFinance).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Com este botão se executa a pesquisa em todas as grelhas da máscara, em relação aos filtros estabelecidos. |
| Inserção | Posiciona o cursor na inserção de uma nova linha. |
| Cancelamento | Cancela a derrogação selecionada. |
