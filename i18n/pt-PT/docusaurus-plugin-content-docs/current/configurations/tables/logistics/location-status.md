---
title: Estado localização
---

Esta tabela gere a codificação dos diferentes estados de localização que precisa para poder ser disponíveis nos módulos competentes.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos estados de localização inseridos. |
| Inserção | Permite inserir um novo estado de localização. |
| Cancelamento | Cancela o estado de localização selecionado. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é código e descrição estado de localização.

GRELHA DE RESULTADO: representa a lista dos estados de localização que correspondem com os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos estados de localização, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos estados de localização.

**Inserção estado localização**

Para poder inserir novas codificações de estado localização é necessário clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na Ribbon Bar). Se indica portanto o código alafanumérico para o tipo e a sua descrição.
