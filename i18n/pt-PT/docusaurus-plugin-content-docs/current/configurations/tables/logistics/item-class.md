Esta tabela gere a codificação das diferentes classes artigo e é necessária para poder efetuar um agrupamento dos artigos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das classes inseridas. |
| Inserção | Permite inserir uma nova classe. |
| Cancelamento | Cancela a classe selecionada. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é código e descrição classe.

GRELHA DE RESULTADO: representa a lista das categorias que correspondem com os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das codificações existentes, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos códigos procurados. 

**Inserção classes artigos**

Além de procurar as classes presentes no sistema é possível acrescentar uma ou mais classes. Para fazer isso é necessário clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na Ribbon Bar). Se indica portanto o código alafanumérico para a classe e a sua descrição.
