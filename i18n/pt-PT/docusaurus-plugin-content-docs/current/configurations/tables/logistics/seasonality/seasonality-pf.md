Esta Form permite gerir as variações de sazonalidade de aprovisionamento nos períodos precedentes, por artigo e período. Esta operação, pode-se fazer manualmente ou em automático com o procedimento de retomada dos dados da previsão atual do aprovisionamento precedente; o procedimento oferece a possibilidade de gerar a sazonalidade. A Form contém três áreas: a primeira com filtros de pesquisa, a segunda com os artigos já registrados na sazonalidade e a terceira, sazonalidade, visualiza, para cada mês do ano, a quantidade comprada para o artigo selecionado na área dos artigos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar artigos | Botão para procurar os artigos no banco de dados. |
| Novo | Botão para inserir uma nova linha na área de sazonalidade para o artigo selecionado. |
| Cancelar os artigos selecionados | Botão para cancelar o artigo selecionado e a sua sazonalidade. |
| Duplicar | Este procedimento permite de duplicar num outro ano a sazonalidade desejada para o artigo selecionado. |
| Prè-visualização de imprensa | Botão para executar a pré-visualizaçao de imprensa. |
| Imprimir | Botão para executar a imprensa em relação os filtros estabelecidos e a configuração da impressora. |
