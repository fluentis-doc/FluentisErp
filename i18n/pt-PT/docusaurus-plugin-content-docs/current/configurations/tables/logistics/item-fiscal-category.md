Esta tabela gere a codificação das diferentes categorias em relação a lógica fiscal que pode ser por sua vez associada com o produto e utilizada como agrupamento, como estatístico ou como funcionalidade nos módulos competentes.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das categorias fiscais inseridas. |
| Inserção | Permite inserir uma nova categoria. |
| Cancelamento | Cancela a categoria selecionada. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa. |
| Imprimir | Executa a imprensa física do relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é código e descrição categoria.

GRELHA DE RESULTADO: a grelha de resultado é uma lista de categorias que correspondem os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das codificações existentes, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos códigos procurados. 

**Inserção categoria fiscal artigo**

Para poder inserir nova codificações de categoria fiscal é necessário clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na Ribbon Bar].

Se indica portanto o código alfanumérico para a categoria fiscal e a sua descrição.
