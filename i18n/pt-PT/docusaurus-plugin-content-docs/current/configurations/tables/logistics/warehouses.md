A tabela de inserção depósitos é necessária para poder utilizar em maneira correta todas as funções do sistema que têm a ver com a movimentação dos materiais.

A tabela dos depósitos é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos depósitos inseridos. |
| Inserção | Permite inserir um novo depósito. |
| Cancelamento | Cancela o depósito selecionado. |
| Cancelar os centros de custo | Cancela o centro de custo selecionado. |
| Cancelar os centros de lucro | Cancela o centro de lucro selecionado. |
| Prè-visualização de imprensa | Executa a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é descrição depósito.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos depósitos que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos depósitos, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos depósitos que quiser. 

**Inserção depósito**

Além de procurar os depósitos presentes no sistema é possível acrescentar um ou mais depósitos. Para fazer isso é necessário cliclar no primeiro relatório vazio (ou apertar o botão [Inserção] na barra de ferramentas). Se indica portanto no código o depósito e a sua descrição. Eventualmente é possível indicar o local de produção, a área gestional, uma data fim validade do depósito, o destino (retomado pelo registro contatos) e/ou o ponto de venda.

Para cada depósito é possível inserir os relativos centros de custo e/ou de lucro.
