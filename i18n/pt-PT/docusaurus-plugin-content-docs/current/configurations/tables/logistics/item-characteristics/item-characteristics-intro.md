A tabela das características artigo é formada na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das características inseridas. |
| Inserção | Permite inserir uma nova característica. |
| Cancelamento | Cancela a característica selecionada. |
| Pré-visualização de imprensa | Executa a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é código e descrição característica.

GRELHA DE RESULTADO: representa a lista dos estados de características que correspondem com os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das características, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das características desejadas.

**Inserção características artigos **

Além de procurara as características presentes no sistema é possível acrescentar uma ou mais características. Para fazer isso é necessário clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na Ribbon Bar). Se indica portanto o código alafanumérico para a característica e a sua descrição.
