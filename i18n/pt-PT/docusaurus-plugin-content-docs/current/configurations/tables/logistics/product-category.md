Esta tabela gere a codificação das diferentes categorias em relação a lógica do produto que pode ser associada com o produto e pode utilizar -la ou como agrupamento, como estatístico ou como funcionalidade nos módulos competentes.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das categorias inseridas. |
| Inserção | Permite inserir uma nova categoria de produto. |
| Cancelamento | Cancela a categoria selecionada. |
| Pré-visualização | Executa a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é código e descrição característica.

GRELHA DE RESULTADO: a grelha de resultado é uma lista de categorias que correspondem os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das categorias, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das categorias que quiser. Além disso, existe a opção de selecionar na lista, um ou mais CT-e /Notas de entrega a descarregar.

Inserção categoria produto

Além de procurar as categorias presentes no sistema é possível acrescentar uma ou mais categorias. Para fazer isso precisa clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na bara de ferramentas). Se indica portanto o código alfanumérico para a categoria e a sua descrição.
