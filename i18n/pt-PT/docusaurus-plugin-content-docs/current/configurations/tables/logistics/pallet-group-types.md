title: Tipos grupo UC
sidebar_position: 11
---

A tabela de inserção dos Tipos grupo UC é uma tabela de base para a utilização das Unidades de carregamento. No específico são definidos os grupos UC para fazer em modo de identificar a UC em relação um número progressivo, o número da encomenda de produção, o número da ordem fornecedor e o número da ordem de produção.

A tabela de tipos grupo UdC é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para eplicar os filtros de pesquisa dentro do banco de dados dos tipos grupo UdC presentes no sistema. |
| Inserção | Permite inserir um novo tipo grupo UdC. |
| Cancelamento | Cancela o tipo grupo UdC selecionado. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Neste caso o filtro de pesquisa é tipo e descrição grupo UC.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos tipos grupo UdC que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos tipos grupo UC, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos tipos grupo UC 

Além de procurar os tipos grupo UC presentes no sistema é possível acrescentar um ou mais tipos grupo UC. Para fazer isso é necessário clicar no primeiro relatório vazio (ou apertar o botão [Inserção] na caixa de combinação). Se indicam portanto o tipo grupo UdC (código alfanumérico), descrição grupo UC e grupo escolhendo entre:

- número progressivo;

- encomenda de produção;

- ordem fornecedor.
