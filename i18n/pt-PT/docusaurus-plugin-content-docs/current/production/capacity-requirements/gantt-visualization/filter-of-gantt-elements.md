---
title: Filtro elementos de Gantt
sidebar_position: 2
---

Neste tab é possível estabelecer os critérios de pesquisa dos elementos que se quer visualizar graficamente no Gantt que se encontra no segundo tab.

Os filtros a disposição para o usuário são os seguintes:

**Cliente**: nestes 3 campos são especificados os relativos dados do cliente: 'Conta, 'Subconta' e 'Nome do cliente', com um duplo click no primeiro campo é possível abrir uma 'Ajuda contas' pela qual se pode selecionar o cliente a filtrar;

**Artigo**: o primeiro campo permite estabelecer a classe do artigo, com um duplo click no segundo campo se abre uma 'Ajuda artigos' pela qual se pode selecionar cada artigo (código e descrição);

**Local produção**: com este campo é possível filtrar as encomendas interessadas por um determinado local produtivo;

**Centro de trabalho**: esta caixa de combinação visualiza o código do centro de trabalho pela qual se quer filtrar, e portanto vai encontrar todos os dias as ordens planejadas e/ou de produção que tem dentro ao menos uma fase com aquele CdT;

**Grupo**: nestes campos é possível filtrar por grupo, e portanto visualizar todas as ordens planejadas e/ou de produção que fazem parte daquele grupo;

**Da data...A data...**: nestes campos é possível estabelecer uma gama de datas aonde encontrar as encomendas de produção;

**Gantt para a OP**: com este flag o usuário pode decidir se visualizar na lista as ordens de produção. Ativando-o, na lista não vai ser apresentado o detalhe da encomenda de produção;

**Gantt para a OP plan.**: com este flag o usuário pode decidir se visualizar na lista as ordens planejadas. Ativando-o, na lista não vai ser apresentado o detalhe da encomenda de produção;

**Considerar os documentos executados**: com este flag o usuário pode decidir se visualizar na lista também os documentos já executados;

**Ordens de manutenção**: com este flag o usuário pode decidir se visualizar na lista as ordens de manutenção;

**Encomendas de produção**: com esta grelha o usuário tem a possibilidade de filtrar, exercitando um duplo click na linha livre da grelha, as encomendas de produção;

Depois de ter estabelecido os filtros acima, o usuário pode clicar no botão 'Procurar' colocado na Ribbon Bar e portanto obter, na seção abaixo deste tab, a lista de documento procurados, onde para cada um o usuário pode anexar uma flag em correspondência da caixa de seleção, e depois, lançando o botão 'Criação Gantt', pode decidir de visualizar todos os documentos selecionados com a caixa de seleção no tab sucessivo chamado 'Visualizar Gantt'.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos documentos associados com a produção. |
| Criação Gantt | Refere-se ao procedimento Gerar o Gantt a visualizar no tab sucessivo, de acordo com os botões de opção ativados nos documentos visualizados na seção abaixo do tab 'Filtro elementos de Gantt'. |
| Selecionar tudo | Anexa o botão de opções em todos os documentos filtrados e portanto presentes na seção abaixo do tab 'Filtro elementos de Gantt'. |
