---
title: Plano de trabalho retornos
sidebar_position: 1
---

Esta janela permite imprimir a lista dos artigos a produzir com as relativas datas de prevista retorno e se encontra no percurso Empreiteiro > Retornos > Imprensas > Plano de trabalho retornos. É composta de 5 diferentes seções:

**Filtros**: nesta seção é possível filtrar por empreiteiro, por classe, código, descrição e variante do artigo a entregar, por tipo ordem de empreiteiro, do número ordem ao número ordem de empreiteiro, por data confirmação ordem da...a..., por data prevista entrega da...a...Enfim é possível decidir se visualizar ou não as ordens de manutenção, utilizando o adequado flag;

**Tipo imprensa**: nesta seção é possível decidir de obter uma imprensa sintética ou analítica. A imprensa analítica oferece também um detalhe das eventuais notas de entrega já criadas para as entregas parciais;

**Classificação**: nesta seção é possível decidir que tipo de ordenamento dar ao relatório de imprensa;

**Retornos**: nesta seção é possível decidir se visualizar só as linhas de ordens de empreiteiro em atraso a data de prevista entrega ou todas indistintamente;

**Parâmetros** **de** **imprensa**: nesta seção se podem estabelecer o número de cópias a imprimir e o tabulado de imprensa a utilizar.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Visualiza a pré-visualização de imprensa de acordo com os filtros estabelecidos. |
| Imprensa | Lança a imprensa do relatório de acordo com os filtros estabelecidos na impressora prédefinida. |
