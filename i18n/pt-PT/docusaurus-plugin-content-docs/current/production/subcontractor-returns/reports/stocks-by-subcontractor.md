---
title: Estoques para o empreiteiro
sidebar_position: 3
---

Esta janela permite imprimir os estoques presentes nos vários empreiteiros, e se encontra no percurso Empreiteiro > Retornos > Imprensas > Estoques para o empreiteiro. Nesta caixa de diálogo de imprensa existe a obrigação de filtrar para o empreiteiro; o detalhe que é fornecido se refere ao estoque inicial, as entradas, saídas e ao estoque final para cada artigo e para cada depósito. Se compõe de 7 seções diferentes:

**Filtros**: nesta seção é possível filtrar por depósito e causal de depósito, por classe, código, descrição e variante do artigo, por tipo artigo, característica, categoria produto e opção, por localização e lote do artigo, por empreiteiro e por encomenda de venda. É possível também estabelecer uma data no campo 'Inventário ao', aonde vai ser calculado o estoque;

**Agrupamento por**: nesta seção é possível decidir se visualizar os dados agrupados para o artigo ou pelo depósito;

**Detalhe por**: nesta seção é possível decidir se obter eventualmente também o detalhe por Variante, atualmente o único disponível nesse relatório de imprensa;

**Tipo imprensa**: nesta seção é possível decidir de obter uma imprensa sintética ou analítica. A imprensa analítica oferece também um detalhe  das eventuais notas de entrega de retorno já efetuadas para as entregas parciais;

**Classificação**: nesta seção é possível decidir que tipo de classificação dar ao relatório de imprensa;

**Artigos com estoque**: nesta seção é possível decidir se visualizar os artigos com estoque positivo, negativo ou nulo. Os flag são de default todos habilitados;

**Parâmetros de imprensa**: nesta seção é possível estabelecer o número de copias a imprimir e o tabulado de imprensa a utilizar.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Visualiza a pré-visualização de imprensa de acordo com os filtros estabelecidos. |
| Imprimir | Lança a imprensa do relatório de acordo com os filtros estabelecidos na impressora prédefinida. |
