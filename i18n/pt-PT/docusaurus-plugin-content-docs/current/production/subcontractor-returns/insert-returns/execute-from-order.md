---
title: Execução da ordem
sidebar_position: 3
---

Nesta janela, que se abre utilizando o botão 'Retornos previstos' colocado na Ribbon Bar, é possível filtrar as ordens de empreiteiro de acordo com alguns critérios:

- classe, código, descrição e variante do artigo a fazer retornar;

- depósito cujo é previsto que tenha que retornar o artigo;

- tipo ordem de empreiteiro;

- data previsto retorno (até um certa data);

- encomenda de venda associada a linha da ordem de empreiteiro.

Na grelha abaixo aparece uma linha para cada ordem de empreiteiro ainda não executada ou parcialmente executada; o usuário, com um duplo click na linha da ordem, tem a possibilidade de explodir as linhas que ainda tem de ser feitas retornar pelo empreiteiro, pode selecioná-las ou inserir a quantidade em cada uma delas, enfim apertar o botão 'Transferir' que se encontra na Ribbon Bar. O procedimento executa nesse modo a transferência das quantidades dos artigos selecionados dentro da grelha do tab 'Artigos retornados' do retorno empreiteiro. 

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar ordens | Botão para aplicar os filtros de pesquisa dentro do banco de dados das ordens de empreiteiro inseridas e ainda não executadas ou só parcialmente executadas. |
| Transferir | Refere-se ao procedimento que transfere na grelha do tab Artigos retornados do retorno de empreiteiro os artigos selecionados na janela abaixo. |
| Execução ordem forçada | Refere-se ao procedimento que permite forçar a execução da linha da ordem de empreiteiro que o usuário está transferindo. |
| Individuar o objeto selecionado dentro da lista | Permite visualizar, na seção da direita da janela abaixo, o artigo selecionado na seção da direita. |
| Para cima/Para abaixo | Permite se mover no nível superior/inferior dentro da seção da direita. |
| Selecionar | Permite selecionar todas as linhas presentes na seção da direita da janela cujo aparecem os artigos a transferir. |
| Bloquear as ordens para a execução | Permite bloquear a ordem de empreiteiro selecionado: neste modo não vai ser possível receber os artigos que fazem parte desta ordem. |
| Libertar as ordens bloqueadas | Permite desbloquear para o recebimento a ordem de empreiteiro selecionado. |
