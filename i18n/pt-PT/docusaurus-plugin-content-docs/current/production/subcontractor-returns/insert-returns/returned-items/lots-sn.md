title: Lotes/NS
sidebar_position: 3
---

Neste tab se dá a possibilidade de estabelecer, para cada linha selecionada na grelha, os lotes ou os números de série a carregar no depósito com a quantidade de artigo retornada, se refere-se a mais lotes diferentes retornados com a mesma linha de retorno. Se o artigo é gerido a lotes o a números de série resulta ativada a correspondente grelha deste tab.

A grelha dos lotes permite ao usuário definir o tipo código lote, código lote, lote fornecedor, tipo lote, data início e data vencimento, quantidade do lote, encomenda de venda (normalmente é a mesma da linha artigo), se o lote é vendível ou não (flag. 'L.vend.') e notas de livre digitação. Obviamente a soma da quantidades dos vários lotes que vai ter que corresponder exatamente com a quantidade retornada da linha artigo.

A grelha dos números de série ao contrário, permite ao usuário estabelecer o tipo NS e o número de série. Obviamente neste caso o número das linhas inseridas vai ter que corresponder exatamente com a quantidade retornada da linha artigo. É possível também utilizar o campo do 'Cálculo automático dos números de série', que permite ao usuário estabelecer o primeiro número de série, clicar no adequado botão e ver completada a grelha de acordo com a quantidade a encher com os números de série em sequência a partir do primeiro estabelecido.
