---
title: Procurar os retornos
sidebar_position: 2
---

A janela Procurar os retornos se abre através do percurso Empreiteiro > Retornos > Procurar retornos > Procurar retornos e apresenta a lista dos retornos de empreiteiro já inseridos, permitindo modificá-los, visualizá-los e inserir novos.

A janela se compõe de uma área de filtre e de uma grelha de resultado.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos retornos de empreiteiros inseridos. |
| Inserir | Refere-se ao procedimento para inserir um novo retorno de empreiteiro. |
| Modificar | Refere-se ao procedimento para abrir na modifica um retorno de empreiteiro. |
| Visualizar | Refere-se ao procedimento apra abrir em visualização um retorno de empreiteiro. |
| Cancelar | Refere-se ao procedimento para cancelar um retorno de empreiteiro precedentemente inserido. |
| Modificar o filtro atributos | Abre uma janela que permite decidir se ativar ou não o filtro Atributos, que permite filtrar os retornos de empreiteiro de acordo com os atributos inseridos através do codificador nos artigos objeto do retorno. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos retornos de empreiteiro inseridos. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

O usuário pode utilizar os seguintes filtros:

**Empreiteiro**: nestes 2 campos são especificados os relativos dados do empreiteiro titular do retorno: 'conta' e 'subconta', enquanto a razão social do empreiteiro aparece em automático; com um duplo click no primeiro campo é possível abrir uma 'ajuda contas' onde se pode selecionar o empreiteiro cujo filtrar. O empreiteiro normalmente é inserido no plano das contas entre os fornecedores;

**Da data nota de entrega...A data nota de entrega...**: nestes campos, pode-se inserir (numa gama) a data da nota de entrega de retorno do empreiteiro (da...a...);

**Da data retorno....A data retorno...**: nestes campos, pode-se inserir (numa gama) a data do efetivo retorno da mercadoria pelo empreiteiro (da...a...), que normalmente é diferente da data da nota de entrega; se refere a data cujo vai ser proposta a registração a depósito do retorno;

**Tipo retorno**: neste campo é possível inserir o tipo retorno e filtrar de acordo com este critério, visualizando portanto todos os retornos do tipo selecionado;

**Nº nota de entrega retorno**: neste campo é possível inserir o número da nota de entrega de retorno e filtrar de acordo com este critério. O campo do número nota de entrega é um campo de texto, não numérico;

**Encomenda**: neste campo, pode-se inserir o número da encomenda de venda, através da adequada ajuda encomendas;

**Valorizados**: este flag permite filtrar os retornos já valorizados, quer dizer os retornos cujos foi efetuada a operação de registração retorno;

**Todos**: este campo permite filtrar todos os retornos de empreiteiro, independentemente se foram carregados e/ou valorizados;

**Procurar artigos**: este botão permite ativar uma seção que permite filtrar utilizando uma série de outros parâmetros, detalhadas abaixo:

**Classe/Cód.art./Descrição**: o primeiro campo permite estabelecer a classe do artigo, com um duplo click no segundo campo se abre uma 'Ajuda artigos' pela qual selecionar cada artigo (código e descrição);

**Tipo enc.**: neste campo é possível inserir o tipo encomenda de produção e filtrar de acordo com este critério, visualizando portanto todos os retornos associados com as encomendas do tipo selecionado;

**Enc. prod.**: nestes 2 campos são estabelecidos o número e o ano da encomenda de produção.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos retornos de empreiteiro que correspondem os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos retornos de empreiteiro, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista dos retornos de empreiteiro que quiser. 

As colunas visualizadas na grelha de resultado são as seguintes:

**Controlado**: o flag permite entender se o retorno foi controlado ou não. Se o retorno não foi controlado não se pode efetuar nem a operação de valorização nem aquela de registração;

**Valorizado**: o flag permite entender se o retorno foi valorizado ou não;

**Carregado**: permite entender se o retorno foi registrado ou não;

**Nota de entrega**: a coluna visualiza o número da nota de entrega de retorno;

**Data nota de entrega**: a coluna visualiza a data do efetivo retorno da mercadoria pelo empreiteiro;

**Empreiteiro**: a coluna visualiza a conta e a subconta do empreiteiro titular do retorno;

**Descrição empreiteiro**: a coluna visualiza a razão social do empreiteiro titular do retorno;

**Total**: a coluna visualiza o total do fornecimento bruto do imposto.
