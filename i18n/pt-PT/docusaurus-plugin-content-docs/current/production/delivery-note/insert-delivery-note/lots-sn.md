Neste tab se dá a possibilidade de estabelecer, para cada linha selecionada na grelha, os lotes a descarregar pelo depósito com a quantidade de artigo retirada para o envio ao empreiteiro, só se refere a mais lotes diferentes retirados com a mesma linha de nota de entrega.

A grelha dos lotes permite ao usuário retirar o lote atingindo a Ajuda lotes, que filtrar entre os lotes disponíveis do artigo selecionado na grelha superior. As informações que são reportadas na grelha lotes são tipo código lote, código lote, lote fornecedor, tipo lote, data início e data prazo, quantidade do lote, localização, dados do fornecedor do lote (conta, subconta e razão social). Obviamente a soma das quantidades dos vários lotes vai ter que corresponder exatamente a quantidade entregada da linha artigo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Inserir | Refere-se ao procedimento para inserir uma nova nota de entrega: a aplicação salva automaticamente a nota de entrega justo inserida e limpa todos os campos fazendo a janela disponível para a inserção de uma nova nota de entrega. |
| Salvar | Botão para salvar as mudançar efetuadas na nota de entrega. |
| Entregas previstas | Refere-se ao procedimento que explode a janela de Execução da ordem e que permmite de transferir das ordens empreiteiro os artigos na nota de entrega. |
| Execução encomendas | Refere-se ao procedimento para transferir os artigos das encomenda de venda. |
| Inserir o pagamento | Refere-se ao procedimento para inserir uma nova linha de pagamento na adequada grelha 'Pagamentos'. |
| Cancelar os pagamentos | Refere-se ao procedimento para cancelar uma linha de pagamento da adequada grelha 'Pagamentos'. |
| Inserir material | Refere-se ao procedimento para inserir um novo material na grelha dos artigos do tab 'Entregas'. |
| Cancelar material | Refere-se ao procedimento para cancelar um material da grelha dos artigos do tab 'Entregas'. |
| Pré-visualização | Visualiza a pré-visualização de imprensa da nota de entrega. |
| Imprimir | Lança a imprensa do relatório da nota de entrega na impressora prédefinida. |
| Documentos | Permite aceder a Gestão documental e de visualizar os eventuais documentos associados com a nota de entrega. |
