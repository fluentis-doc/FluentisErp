---
title: Procurar notas de entrega
sidebar_position: 2
---

A janela Notas de entrega se abre através do percurso Empreiteiro > Notas de entrega > Notas de entrega > Notas de entrega e apresenta a lista das notas de entrega de empreiteiro já inseridas, permitindo modificá-las, visualizá-las e inserir novas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das notas de entrega de empreiteiro inseridos. |
| Inserir | Refere-se ao procedimento para inserir uma nova nota de entrega. |
| Modificar | Refere-se ao procedimento para abrir na modifica uma nota de entrega. |
| Visualizar | Refere-se ao procedimento para abrir na visualização uma nota de entrega. |
| Cancelar | Refere-se ao procedimento para cancelar uma nota de entrega precedentemente inserida. |
| Modificar o filtro atributos | Abre uma janela que permite de decidir se ativar ou não o filtro atributos, que permite de filtrar as notas de entrega de empreiteiro de acordo com os atributos inseridos através do codificador nos artigos objeto da nota de entrega. |

A janela se compõe por uma área de filtro e por uma grelha de resultado:

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das notas de entrefa inseridas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

O usuário pode utilizar os seguintes filtros:

**Empreiteiro**: nestes 2 campos são especificados os relativos dados do empreiteiro cujo titular da nota de entrega: 'conta' e 'subconta', enquanto a razão social do empreiteiro aparece em automático, com um duplo click no primeiro campo é possível abrir uma Ajuda contas pela qual poder selecionar o empreiteiro cujo filtrar. O empreiteiro normalmente é inserido no plano das contas entre os fornecedores;

**Tipo regresso**: neste campo é possível inserir o tipo nota de entrega e filtrar segundo este critério, visualizando portanto todas as notas de entrega do tipo selecionado. A caixa de combinação apresenta uma lista de todos os tipos de notas de entrega, sendo as notas de entregas assimiladas aquelas de venda pelo ponto de vista da gestão, sendo notas de entrega de saída mercadorias;

**Do número nota de entrega... Ao número nota de entrega...**: nestes campos se pode inserir (numa gama) o número da nota de entrega (da...a...);

**Da data nota de entrega... A data nota de entrega...**: nestes campos se pode inserir (numa gama) a data de inserção da nota de entrega (da...a...);

**Encomenda**: neste campo se pode inserir o número da encomenda de venda, através da adequada ajuda encomendas;

**Descarregadas**: este flag permite filtrar as notas de entrega já descarregadas pelo depósito; quer dizer as notas de entrega pela qual foi efetuada a operação de 'Descarregamento notas de entrega';

**Imprimidas**: este flag permite filtrar as notas de entrega já imprimidas. A janela de pesquisa das notas de entrega abre com estes últimos 2 flag desativados de default, portanto para visualizar todas as notas de entrega precisa ativá-los;

Procurar artigos: este botão permite ativar uma seção que permite filtrar utilizando uma série de outros parâmetros, detalhados aqui abaixo;

**Classe/Cód. art./Descrição**: o primeiro campo permite estabelecer a classe do artigo, com um duplo click no segundo campo se abre uma 'Ajuda artigo' pela qual se seleciona cada artigo (código e descrição);

**Encomenda/Ano**: neste campo se pode inserir o número da encomenda de venda, através da ajuda encomendas. Se refere neste casa a encomenda de venda enganchada com cada linha artigo da nota de entrega;

**Tipo enc.**: neste campo é possível inserir o tipo encomenda de produção e filtrar de acordo com este critério, visualizando todos os retornos associados com as encomendas do tipo selecionado;

**Enc. prod.**: nestes 2 campos são estabelecidos o número e o ano da encomenda de produção.

GRELHA DE RESULTADO: a grelha de resultado é uma lista de notas de entrega que correspondem os dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das notas de entrega, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha da lista de notas de entrega que quiser.

As colunas visualizadas na grelha de resultado são as seguintes:

**Imprimida**: o flag permite entender se a grelha foi imprimida ou não. Se a nota de entrega não foi imprimida não pode ser efetuada a operação de descarregamento dela;

**Descarregado**: o flag permite entender se a nota de entrega foi descarregada ou não;

**Número**: a coluna visualiza o número da nota de entrega;

**Data**: a coluna visualiza a data da nota de entrega;

**Empreiteiro**: a coluna visualiza a conta e a subconta do empreiteiro titular da nota de entrega;

**Descrição empreiteiro**: a coluna visualiza a razão social do empreiteiro titular da nota de entrega;

**Total**: a coluna visualiza o total da nota de entrega no bruto do imposto.
