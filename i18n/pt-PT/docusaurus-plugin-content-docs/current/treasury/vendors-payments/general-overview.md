---
title: General Overview
sidebar_position: 1
---

O módulo Pagamentos fornecedores se utiliza principalmente para criar as listas de pagamento, tipicamente para a transferência bancária: o módulo, de fato, prevê a possibilidade do envio telemático do traçado ABI dos pagamentos para a transferência bancária dentro da Itália.

Procede-se com a criação de uma nova lista de pagamento indo a retirar os dados das partidas fornecedores presentes: vai ser possível proceder na contabilização dos movimentos relativos, com o encerramento das partidas relativas.

Na criação da lista vai ser possível proceder com a imprensa da lista ou com o envio telemático da mesma.

O módulo se encontra dentro da área Tesouraria no módulo Pagamento fornecedores.
