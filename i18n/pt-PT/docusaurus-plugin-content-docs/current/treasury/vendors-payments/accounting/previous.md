A ficha de restauração permite visualizar a registração contábil através dum duplo click na seção relativa as registrações, assim como a anulação parcial/total das operações de pagamento selecionadas. A anulação vai ser possível só se o período é ainda modificável (imprensa livro jornal e encerramento contas não executadas).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura a lista das registrações contábeis executadas. |
| Redução contabilização | O botão executa a restauração da inteira contabilização selecionada. |
| Redução registrações contábeis | O botão executa a restauração da cada registração contábeis selecionadas. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das registrações contábeis que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das registrações contábeis, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das registrações contábeis que quiser.
