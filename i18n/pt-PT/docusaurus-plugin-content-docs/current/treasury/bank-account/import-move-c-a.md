---
title: Importação mov. C/C
sidebar_position: 2
---

Esta máscara permite executar a importação dos movimentos de conta corrente pelo arquivo telemático (padrão ABI) que pode ser extraída através da Home Banking.

Para proceder com a importação precisa antes de tudo estabelecer na base da máscara a subconta pela qual quer-se importar o arquivo, depois de apertar o botão 'Executar importação' vai se abrir a máscara de abertura do arquivo telemático. Na conclusão da operação vai aparecer na grelha uma nova linha de importação: com um duplo click na mesma ou através do botão 'Visualizar arquivo' é possível visualizar o arquivo de texto de origem dos dados importados.

Os movimentos assim importados vão ser utilizáveis no procedimento de Reconciliação C/C.

O procedimento 'Importação mov. C/C' é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar importações | Botão para aplicar os filtros de pesquisa dentro do banco de dados das importações executadas. |
| Executar importação | Com este botão se executa a seleção do arquivo telemático e a sua importação. |
| Visualizar o arquivo | Com este botão é possível visualizar o arquivo telemático associado com a importação selecionada. |
| Cancelar importação | O botão cancela a importação selecionada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das importações executadas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das importações que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das importações, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das importações que quiser.
