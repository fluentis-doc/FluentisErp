O módulo DocFinance é dedicado a interface com o programa de tesouraria homônimo, para uma gestão de tesouraria empurrada. Antes de proceder com a introdução ou a realização da carteira efeitos precisa assegurar-se de ter corretamente estabelecido todos os dados tabelares como também os parâmetros do módulo.
