---
title: Procurar fluxo de caixa
sidebar_position: 1
---

O procedimento Procurar fluxo de caixa foi desinhado para permitir ao usuário procurar facilmente as simulações inseridas precedentemente com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. A partir desse procedimento é possível fazer todas as operações necessárias no âmbito do documento, após a presença dos direitos usuários necessários. Se aconselha de criar o fluxo de caixa pelo procedimentos 'Criação automática fluxo de caixa' invés que 'Criar fluxo de caixa'.

O procedimento Procurar fluxo de caixa é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados dos fluxos de caixa inseridos. |
| Novo | Refere-se ao procedimento para inserir um novo fluxo de caixa. |
| Modificar | Refere-se ao procedimento para modificar um fluxo de caixa precedentemente inserido. |
| Visualização | Refere-se ao procedimento para visualizar um fluxo de caixa precedentemente inserido. |
| Cancelar | Executa o procedimento para cancelar um fluxo de caixa precedentemente inserido. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Análises fluxo de caixa | Refere-se ao procedimento para imprimir um fluxo de caixa. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Geração automática fluxo de caixa | Refere-se ao procedimento para gerir a criação automática do fluxo de caixa. |
| Análises fluxo de caixa global | Refere-se ao procedimento para visualizar um gráfico do fluxo de caixa. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos fluxos de caixa inseridos. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos fluxos de caixa que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa dos fluxos de caixa, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos fluxos de caixa que quiser.
