title: Prazos extra-contábeis
sidebar_position: 3
---

Com essa máscara é possível gerir os prazos extra-contábeis que tem influência nos fluxos de caixa, como por exemplo prever as saídas para os tributos, os salários/salário, empréstimo, etc.

Se vai simplesmente a acrescentar/modificar na grelha os novos prazos previstos, estabelecendo o flag 'Fechada' para não considerar mais a linha nas próximas simulações.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa na lista dos prazos extra-contábeis. |
| Novo | Estabelece o cursor na inserção de um novo prazo extra-contábil. |
| Cancelar | Cancela o prazo extra-contábil selecionada. |
