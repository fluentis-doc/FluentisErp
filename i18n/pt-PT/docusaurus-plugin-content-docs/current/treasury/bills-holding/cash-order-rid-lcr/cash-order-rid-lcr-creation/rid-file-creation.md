---
title: Geração arquivo DDA
sidebar_position: 2
---

A partir desta máscara se geram os traçados telemáticos para as listas de apresentação DDA.

A parte superior da máscara é dedicada a definição dos filtros de pesquisa das listas emitidas mas pelas quais o arquivo não foi ainda criado.

Uma vez individuada a lista a gerir, na base da máscara se podem visualizar e gerir os dados de default assim como predefinidos nos parâmetros: a data de criação do arquivo, o seu nome (estabelecido como [DataDistinta]_[NumeroDistinta].RID), o percurso de criação do arquivo, o nome da sociedade e a descrição das referências fatura a inserir. Na base da máscara estão presentes os flag de gestão do retorno final do arquivo e a exclusão da gestão das referências para os não pagos.

Para executar a criação do arquivo é suficiente apertar o botão de geração: o programa vai proceder com um controle dos dados obrigatórios a inserir no traçado (mas não da honestidade do dado obrigatório inserido), retornando no caso uma máscara com os detalhes dos erros individuados.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar as listas | Procura o elenco das listas pelas quais criar o arquivo telemático. |
| Criação | Executa a criação do traçado DDA para a lista selecionada. |
