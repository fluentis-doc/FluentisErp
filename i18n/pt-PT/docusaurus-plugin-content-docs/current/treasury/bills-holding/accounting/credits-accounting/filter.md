A partir desta máscara se procede com a última fase da carteira efeitos: o procedimento vai fechar a conta de apresentação para acreditar a conta corrente efetiva do registro banco na lista.

A parte superior da máscara é dedicada a **Impostação dos filtros de pesquisa**: a contabilização créditos pode ser efetuada só para os efeitos contabilizados. A seleção é efetuada sobre o detalhe dos efeitos para fazer em maneira de individuar para a data de vencimento as quantias acreditadas.

A seção **Impostação da contabilização **pode ser predefinida dentro dos parâmetros do módulo Carteira efeitos. Trata-se de atribuir a causal de contabilidade, os dias de referência para a registração e as eventuais subcontas de contabilização. Em particular, as comissões podem ser propostas para cada efeito em relação as comissões estabelecidas no registro banco. As taxas passivas são calculadas para as listas no desconto.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura a lista dos efeitos a acreditar. |
| Contabilização | Executa a contabilização para as linhas selecionadas. |
