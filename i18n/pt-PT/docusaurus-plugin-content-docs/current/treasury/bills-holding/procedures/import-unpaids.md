---
title: Importação não pago
sidebar_position: 3
---

A partir desta máscara se procede com a importação da lista não pagos pelo arquivo telemático padrão de comunicação bancária. É possível normalmente, ativar o recebimento de um traçado dessa lista, que retoma o mesmo arquivo enviado com a apresentação, dentro da qual foi inserido o identificativo do efeito: através a leitura desse dado, portanto, se pode criar a lista dos não pagos automaticamente, com a relevação direta também das despesas bancárias de cada um.

O procedimento começa com o botão 'Executar a importação', que vai abrir a máscara de abertura arquivo do sistema: uma vez individuado o arquivo no catálogo correto precisa simplesmente abrir o arquivo para proceder com a importação da lista. A eventual lista de efeitos não encontrados vai ser reportada numa máscara de erro: para esses vai precisar, portanto, proceder com a criação manual do não pago.

A partir da máscara é possível cancelar a importação executada: essa vai anular a lista dos não pagos criados com a mesma importação.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar importações | Procura a lista das importações dos não pagos já executadas. |
| Executar a importação | Com este botão se abre a máscara de seleção do arquivo telemático pela qual executar o procedimento de importação. |
| Visualizar o arquivo não pago | Visualiza o arquivo telemático do não pago importado. |
| Cancelar a importação | Cancela a importação executada. |
