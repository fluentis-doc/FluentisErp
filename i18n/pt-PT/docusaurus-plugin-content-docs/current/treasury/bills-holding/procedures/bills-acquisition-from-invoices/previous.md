Executada a criação, as linhas vão desaparecer da grelha e o resultado vai ser visível na ficha 'Restauração' dentro da qual é possível anular a operação. Não é possível anular a criação de efeitos se esses são contabilizados ou apresentados na lista.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura a lista das criações executadas. |
| Redução aquisição | Com este botão vai ser cancelada toda a aquisição efetuada, com todos os efeitos associados. |
| Redução efeito | Com este botão vai ser cancelada a criação de cada efeito selecionado. |
