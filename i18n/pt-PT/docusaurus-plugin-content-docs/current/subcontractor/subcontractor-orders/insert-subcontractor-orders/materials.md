title: Materiais
sidebar_position: 4
---

O tab Materiais é constituído por duas grelhas que reportam os materiais a enviar ao empreiteiro para a produção e os materiais que vão ser utilizados no retorno do produto trabalhado.

Os materiais enviados e utilizados podem ser geridos pelo operados e neste caso precisa emitir a quantidade, o eventual preço da produção, a data de previsto retorno, a data de utilização do material. Esses dados já são valorizados se a ordem vem da programação geral ou da elaboração PNR.

**Grelha materiais a entregar**

**Código artigo**: indica o material a entregar para a produção, com um duplo click se entra dentro de uma ajuda artigos através da qual se seleciona o código do artigo;

**Variante**: nesta caixa de combinação se estabelece a variante do artigo;

**Opção**: nesta caixa de combinação se estabelece a opção do artigo;

**UM utilização**: unidade de medida que se refere a quantidade a entregar;

**Quantidade a entregar**: total a enviar ao fornecedor;

**Quantidade entregada**: total já entregado;

**Depósito retirada**: representa o depósito pela qual retirar para o envio ao fornecedor.

**Grelha materiais a entregar**

**Código artigo**: indica o material a entregar no retorno do trabalhado, com um duplo click se entra dentro de uma ajuda artigos através da qual se seleciona o código do artigo;

**Variante**: nesta caixa de combinação se estabelece a variante do artigo;

**Opção**: nesta caixa de combinação se estabelece a opção do artigo;

**UM utilização**: unidade de medida que se refere a quantidade a entregar;

**Utilização unitária**: quantidade a utilizar para cada unidade de retorno do trabalhado, no caso de gestão automática da programação de listas; este dado é valorizado pelo sistema;

**Utilização total**: quantidade total a utilizar para a completa produção no caso de gestão automático da programação de listas; este dado é valorizado pelo sistema;

**% sobra**: quantidade a mais a utilizar que resulta ser uma sobra;

**Data prev. utilização**: de acordo com esta data o sistema planeja a disponibilidade da necessidade.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Inserir entrega | Permite inserir novo record na grelha Materiais a entregar. |
| Cancelar entrega | Permite cancelar um record da grelha Materiais a entregar. |
| Inserir utilizações | Permite inserir novo record numa grelha Materiais a utilizar. |
| Cancelar utilizações | Permite cancelar um record da grelha Materiais a utilizar. |
