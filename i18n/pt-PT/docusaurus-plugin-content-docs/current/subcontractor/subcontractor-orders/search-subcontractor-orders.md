---
title: Procurar ordens de empreiteiro
sidebar_position: 3
---

Neste formulário é possível: procurar, visualizar e modificar as ordens de empreiteiro geradas automaticamente pela libertação das ordens planejadas de produção cujo estão presentes as fases externas e pela libertação das ordens planejadas de empreiteiro; procurar, visualizar e modificar as ordens de empreiteiro geradas manualmente; inserir novas ordens de empreiteiro.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão através da qual procurar as ordens já inseridas no banco de dados. |
| Inserir | Botão através da qual se pode inserir uma nova ordem empreiteiro. |
| Modificar | Botão para modificar uma ordem precedentemente inserida. |
| Visualizar | Botão para visualizar as ordens precedentemente inseridas. Não permite a mudança. |
| Cancelar | Botão para cancelar uma ordem precedentemente inserida. |
| Modificar filtro atributos | Possibilidade de ativar ou desativar a pesquisa por filtro atributos artigo. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das ordens inseridas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos artigos que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das ordens.
