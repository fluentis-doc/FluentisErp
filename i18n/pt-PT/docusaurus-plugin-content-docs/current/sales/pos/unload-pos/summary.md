Neste tab, o usuário tem a possibilidade de procurar, visualizar e anular as operações de descarregamento executadas, que satisfaçam as condições específicas das registrações de depósito.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar POS descarregados | Botão para procurar os POS descarregados. O resultado deste procedimento vão ser visualizado na grelha de resultado. |
| Cancelar descarregamento | Botões para cancelar por inteiro as operações de descarregamento selecionadas na grelha de resultados se satisfazem as condições específicas de cancelamento das registrações de depósito. |
| Restaurar POS | Botão para cancelar as operações de descarregamento selecionadas na grelha de resultado, para o POS em uso na grelha dos detalhes. |
| Restaurar o POS e a registração | Botão para cancelar a registração de depósito do documento selecionado na operação de descarregamento em uso, se satisfazem as condições específicas de cancelamento das registrações de depósito. |
| Propriedade | Permite visualizar o modo em que foi executada a operação de descarregamento selecionada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das operações de descarregamento que correspondem os dados de filtro especificadas acima. A máscara contém duas áreas: uma com a operação e a outra com os detalhes, os POS e as registrações geradas pela operação de descarregamento selecionada no tab das operações. A partir da grelha principal desta área, que é populada no procedimento de pesquisa, podem ser selecionadas as operações de descarregamento na qual aplicar as funções da Ribbon.

Além disso, nesta máscara, tem a possibilidade de visualizar o POS e a sua registração de depósito (duplo click na linha interessada).
