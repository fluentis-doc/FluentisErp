sidebar_position: 1

A inserção de um POS pede: o tipo POS predefinido na [Home > Tabelas > Venda]. Para cada documento vai ser atribuído um número de acordo com a numeração especificada pelo usuário na tabela 'Numeração POS' e no tipo de documento que contém a numeração; o ano e a data atual que foi compilada em automático. No campo 'Sobrenome/Nome' pode-se inserir manualmente o nome do operador que inseriu o POS, e no campo 'Distrito' insere-se o distrito aonde encontra-se o POS. O flag 'Imprimido' torna-se ativo quando é impressa a nota fiscal. (Para a versão RO, o botão 'Imprimir' gera a nota fiscal).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as operações efetuadas. O mesmo resultado se obtém movendo o cursor numa linha diferente daquela que se quer salvar. |
| Salvar novo | Salva o POS atual e cria um novo. |
| Descarregamento POS | Permite o descarregamento POS, se para cada artigo foram inseridas o depósito e a causal. |
| Pré-visualização | Botão para executar a pré-visualização de imprensa do documento. |
| Imprimir | Botão para executar a imprensa física do documento. |
