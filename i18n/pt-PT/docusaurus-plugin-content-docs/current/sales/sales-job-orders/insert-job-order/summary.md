Nos Resumos são apresentadas as informações principais do inteiro documento. 

**Descontos finais**

Pode ser inserido só os descontos expressos em porcentagem. Os descontos inseridos nesta seção são calculados em cada artigo do documento;

**Resumos imposto**

É proposto o resumo imposto do documento para cada código imposto;

**Despesas**

São propostas as despesas inseridas no registro cliente, no tab 'Despesas/Descontos'. Além disso há a possibilidade de inserir despesas que fazem referência só ao documento em utilizo;

**Notas finais**

É um campo livre que pode ser compilado pelo usuário com o auxílio da Ajuda notas codificadas;

**Resumo comissões agentes**

São os agentes inseridos a nível de cliente com as relativas comissões.

**Total documento**

**Quantia bruta artigos**: representa a soma dos valores de todos os artigos;

**Total descontos aplicados**: representa o valor total dos descontos aplicados sem os descontos finais;

**Quantia líquida artigos**: quantia bruta artigos - total descontos aplicados;

**Total descontos finais**: representa o valor dos descontos finais;

**Despesas não documentadas**: representa o valor das despesas inseridas como artigos de tipo 'Despesas';

**Despesas documentadas**: representa o valor das despesas inseridas na grelha Despesas;

**Tributável**: quantia líquida artigos - descontos finais + total despesas não documentadas + total despesas documentadas;

**Imposto**: representa a soma dos valores contidos nos resumos imposto;

**Total encomenda**: tributável + imposto.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as operações efetuadas. |
| Cancelar os descontos finais | Permite cancelar os descontos finais da grelha correspondente. |
| Cancelar as despesas | Permite cancelar as despesas da grelha correspondente. |
| Pré-visualização | Botão para executar a pré-visualização de imprensa do documento. |
| Imprimir | Botão para executar a imprensa física do documento. |



| RibbonFunction | RibbonMeaning |

| Document manager | Gestão do armazenamento documental associado a máscara em uso. |
