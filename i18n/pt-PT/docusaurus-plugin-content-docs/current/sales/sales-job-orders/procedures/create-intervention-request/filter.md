sidebar_position: 2

Segundo os critérios de filtro especificados (cliente, artigo, número, data e período previsto para a criação da intervenção), no tab Filtro são visualizados todos os artigos das encomendas que contêm os dados para o pedido de intervenção (ver Encomendas-Artigos-Tab pedido de intervenção). Na parte inferior da janela, nos 'Parâmetros', o usuário pode inserir o estado do pedido e a data de emissão da mesma (essa data poderia ser diferente da data proposta). Para acabar o procedimento, o usuário tem de apertar o botão 'Transferência'.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Permite criar os artigos que contêm os dados para o pedido de intervenção. |
| Transferência | Permite executar a transferência dos artigos selecionados. |
