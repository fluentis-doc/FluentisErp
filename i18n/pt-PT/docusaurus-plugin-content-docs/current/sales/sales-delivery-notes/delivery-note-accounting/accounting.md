---
title: Contabilização
sidebar_position: 2
---

Uma nota de entrega, para ser contabilizada, tem de satisfazer uma série de condições: no seu cabeçalho precisa ter estabelecido o flag ‘Imprimida' e o tipo nota de entrega tem de conter a causal geral para a contabilização. O usuário pode usar os filtros de pesquisa para encontrar uma certa nota de entrega. Há mais opções para a registração contábil que se quer criar: data registração igual a data documento; data competência imposto igual a data documento; uso da causal industrial inserida para a registração da fatura na contabilidade industrial (quando necessário).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar notas de entrega | Botão para procurar as faturas; são propostas todas as notas de entrega imprimidas e não contabilizadas. |
| Contabilização notas de entrega | Executa a contabilização das notas de entrega selecionadas. |
