---
title: Criação nota de entrega da lista de embalagem
sidebar_position: 3
---

Permite de retomar os artigos da lista de embalagem. A lista de embalagem, depois deste procedimento, vai ter o estado ‘Descarregado'. As operações deste procedimento são: pesquisa da lista de embalagem com a ajuda dos filtros; inserção dos parâmetros e restauração das operações.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar lista de embalagem | Botão para procurar lista de embalagem. |
| Selecionar tudo | Botão para selecionar todos os artigos da lista de emabalagem. |
| Transferência artigos | Botão para transferir os artigos selecionados pela lista de embalagem. |
