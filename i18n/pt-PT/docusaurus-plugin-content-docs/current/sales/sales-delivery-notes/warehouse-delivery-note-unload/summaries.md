Neste tab, o usuário tem a possibilidade de procurar, visualizar e anular as operações de descarregamento executadas, que satisfazem as condições específicas de cancelamento das registrações de depósito.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar as notas de entrega descarregadas. O resultado deste procedimento vai ser visualizado na grelha de resultado. |
| Cancelar | Botões para cancelar por inteiro as operações de descarregamento selecionadas na grelha de resultado se satisfazem as condições especificadas de cancelamento das registrações de depósito. |
| Restaurar notas de entrega | Botão para cancelar as operações de descarregamento selecionadas na grelha de resultado, para a nota de entrega em uso na grelha dos detalhes. |
| Restaurar todos | Botão para cancelar a registração de depósito do documento selecionado na operação de descarregamento em uso. |
| Propriedade | Permite de visualizar a maneira cuja foi executada a operação de descarregamento selecionada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das operações de descarregamento que correspondem os dados de filtro especificados acima. A máscara contém duas zonas: uma com a operação e a outro com os detalhes, as notas de entrega e as registrações geridas pela operação de descarregamento selecionada no tab das operações. A partir da grelha principal desta zona, que é povoada no procedimento de pesquisa, podem ser selecionadas as operações de descarregamento nas quais aplicar as funções da ribbon.

Além disso, nesta máscara, há a possibilidade de visualizar as registrações de depósito (duplo click na linha).
