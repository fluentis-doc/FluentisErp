---
title: Lista faturas de venda
sidebar_position: 2
---

Este procedimento serve para imprimir uma lista de faturas de venda, de acordo com os critérios de filtro especificados (tipo fatura, número, data, cliente); além disso, tem a possibilidade de ordená-las para o cliente, o número ou tipo fatura.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Botão para executar a pré-visualização de imprensa. |
| Imprimir | Botão para executar a imprensa de acordo com os filtros estabelecidos e a configuração da impressora. |
