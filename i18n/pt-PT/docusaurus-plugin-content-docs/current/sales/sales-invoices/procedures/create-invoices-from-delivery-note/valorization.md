title: Valorização
sidebar_position: 2
---

Na janela da valorização são propostas todas as notas de entrega com o estado 'Imprimida'. Podem ser selecionados os CT-e/Notas de entrega a faturar com o auxílio dos filtros (data, número, cliente). 'Data criação' é a data da fatura que vai ser criada. O tipo fatura vai ser retomado pelo 'Tipo fatura' (tabela tipos CT-e/N). O usuário precisa inserir a data da fatura, senão o número vai ser igual ao primeiro número disponível. Se você quiser criar uma fatura com mais CT-e/NE o usuário precisa ter estabelecido o flag Faturação resumida no registro contatos.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efeituar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar CT-e/NE | Botão para procurar as notas de entrega não valorizadas. |
| Criação faturas do CT-e/NE | Botão para executar o procedimento de valorização das notas de entrega. |
| Abertura nota de entrega | Permite visualizar e modificar a nota de entrega selecionada. |
