title: Imprimir detalhe ordens na lista
sidebar_position: 2
---

O procedimento Imprimir detalhe ordens na lista foi desinhado para permitir ao usuário interrogar facilmente o sistema e extrapolar rapidamente informações que se referem as listas de retirada e as relativas ordens associadas.

O procedimento Imprimir detalhe ordens na lista é constituido na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório do detalhe ordens na lista. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das listas de retirada presentes no sistema. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso é possível filtrar por tipo lista, por gama de número lista, por gama de data inserção documento. É também possível imprimir um report personalizado (se escolhe através a caixa de combinação tabulada) precedentemente configurado na gestão imprensas.
