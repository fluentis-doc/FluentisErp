---
title: Inserção nova UC
sidebar_position: 3
---

Para criar uma nova UC é suficiente inserir uma nova linha na grelha principal especificando ano, grupo, número e tipo. Esses campos definem a chave unívoca para cada UC. Vão ser completados em automático os campos de registro relativos a dimensões e pesos que podem ser trocados. É obrigatório inserir o operador (se lembra que o operador tem que ter os direitos para a criação da UC) pela qual foi estabelecido o depósito de carregamento da UC. Os campos referenciados com documentos vão ser populados só se a UC foi criada a partir dos procedimentos de Retirada e lista de retirada.

Uma vez criada a UC é possível inserir dentro o artigo ou os artigos que as formam. Para fazer isso, precisa selecionar a partir da grelha principal a UC e portanto inserir uma nova linha na grelha secundária (aquela que se refere aos artigos). A inserção prevê de especificar obrigatoriamente classe, código (variante se o artigo foi gerido com variante) e quantidade. Também nesse caso os campos referidos com documentos vão ser populados só se a criação foi efetuado por procedimentos automáticos.
