title: Procurar UC

O procedimento da gestão UC foi desinhado para permitir ao usuário procurar facilmente as UC precedentemente inseridas com o objetivo de visaulizá-las, modificá-las e eventualmente cancelá-las. É possível a partir da gestão UC fazer todas as operações necessárias no âmbito, através da possibilidade de ter os direitos necessários.

O procedimento da gestão UC é constituida na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das UC presente no sistema. |
| Nova UC | Refere-se ao procedimento para inserir uma nova UC. |
| Cancelar UC | Refere-se ao procedimento para eliminar uma UC. |
| Novo artigo UC | Refere-se ao procedimento para inserir dentro da UC selecionada um ou mais artigos. |
| Cancelar artigo UC | Refere-se ao procedimento para eliminar de dentro da UC selecionada um ou mais artigos. |
| Inserção múltiplo UC | Executa o procedimento para inserir mais artigos dentro de uma UC (se trata de uma janela que permite efetuar a inserção artigo em maneira rápida). |
| Inserção ordens na UC | Permite inserir os artigos de uma ordem cliente dentro de uma UC. Este procedimento cria uma associação entre a UC e a ordem cliente. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Retorno cliente | Executa o procedimento para criar o documento nota de entrega partindo de uma retirada. |
| Carregamento UC | Executa o procedimento para criar o documento fatura partinda de uma retirada. |
| Scarico UDC | Esegue la procedura per creare il documento POS partendo da un picking. |
| Visualizar registração de carregamento | Abre na registração de depósito do carregamento UC. |
| Visualizar registração de descarregamento | Abre a registração de depósito do descarregamento UC. |

| Redução descarregamento Uc | Se a UC foi carregada e descarregada executa a redução da registração de depósito de descarregamento. |

| Confirmar transferência |  |
| Mover UC | Permite mover a UC de uma localização a uma outra. |
| Pré-visualização | Visualiza a pré-visualização de imprensa. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das UC inseridas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das UC que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das UC, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das UC que quiser. As informações que se podem ler a partir desta grelha são a identificação da UC (através dos campos ano, grupo, número e tipo), as dimensões da UC, o estado da UC (se foi movimentada no depósito), o eventual número de lista com a qual foi criada, o depósito de pertença e a relativa localização de carregamento e outro.

GRELHA DO RESULTADO: a grelha de detalhe mostra o conteúdo da UC selecionada na grelha do resultado. As informações que se podem ler a partir desta grelha são o identificativo do artigo ou dos artigos contidos na UC selecionada, a quantidade e eventualmente as referências ao documento Ordem cliente de partida.
