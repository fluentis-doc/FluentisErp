---
title: Zerar o estoque
sidebar_position: 3
---

É possível, em caso de necessidade, zerar em maneira máxima o estoque de depósito. Normalmente esta função se utiliza se estão presentes no sistema alguns dados não coerente de estoques negativos que se quer zerar.

Para utilizar esta função precisa efetuar uma pesquisa selecionando nos filtros o depósito e estabelecendo o flag de visualização estoque positivos e negativos. A partir do tab detalhe estoque artigo selecionar todos os artigos a zerar e apertar o botão 'Geração registrações'. A partir da janela que aparece escolher a causal e sem especificar a quantidade apertar o botão OK. Nessa maneira se vê zerar estoques dos artigos desejados.
