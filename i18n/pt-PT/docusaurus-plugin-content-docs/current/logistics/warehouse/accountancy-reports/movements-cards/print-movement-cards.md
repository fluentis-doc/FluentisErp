O procedimento Imprimir fichas de movimentação é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório de fichas de movimentação. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos dados presentes no sistema. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso é possível filtrar por depósito, causal, artigo (classe, código, descrição), eventual variante, categoria fiscal, lote, lote fornecedor, cliente/fornecedor, documento de referência, de mês a mês. É também possível ativar o agrupamento por causal. De default a imprensa não extrai os movimentos não de interesse fiscal. É portanto possível extrapolá-los ativando o flag relativo.

A janela permite imprimir o relatório padrão ou imprimir um relatório personalizado (após o mapeamento na gestão imprensas). Se quiser imprimir um relatório personalizado precisa escolher o mesmo do tab tabulado nos parâmetros de imprensa.
