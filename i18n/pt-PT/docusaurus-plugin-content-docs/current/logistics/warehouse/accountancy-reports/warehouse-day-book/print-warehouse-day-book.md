O procedimento Imprimir agenda de depósito é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório da agenda de depósito. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos dados presentes no sistema. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso é possível filtrar por depósito, artigo (classe, código, descrição), eventual variante, categoria fiscal, de mês a mês e encomenda de venda. É também possível habilitar o resumo por causal.

A janela permite imprimir o relatório padrão ou imprimir um relatório personalizado (após o mapeamento na gestão imprensas). Se quiser imprimir um relatório personalizado precisa escolher o mesmo do tab tabulado nos parâmetros de imprensa.
