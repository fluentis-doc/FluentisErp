---
title: Imprimir artigos não movimentados
sidebar_position: 2
---

O procedimento Imprimir artigos não movimentados foi desinhado para permitir ao usuário interrogar facilmente o sistema e extrapolar rapidamente informações que se referem aos artigos não movimentados no depósito. Tal imprensa permite criar portanto as estatísticas de imprensa logística sobre os artigos.

O procedimento Imprimir artigos não movimentados é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório dos artigos não movimentados. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total dos artigos presentes no sistema. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso é possível filtrar por depósito, artigo (classe, código artigo e descrição). Precisa portanto inserir a data desejada entesa como data de partida para o controle dos artigos não movimentados.

A janela permite imprimir o relatório padrão ou imprimir um relatório personalizado (após o mapeamento na gestão imprensas). No caso se quer imprimir um relatório personalizado precisa escolher o mesmo pelo tab tabulado nos parâmetros de imprensa.
