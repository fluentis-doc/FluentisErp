---
title: Imprimir etiquetas
sidebar_position: 2
---

O procedimento Imprimir etiquestas foi desinhado para permitir ao usuário interrogar facilmente o sistema e extrapolar rapidamente algumas etiquetas a utilizar no depósito.

O procedimento Imprimir etiquetas é constituídos na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório das etiquetas. |

GRELHA DE INSERÇÃO: a grelha de inserção é aquela dedicada a inserção dos artigos pelas quais se quer imprimir a etiqueta. Para inserir um artigo na grelha é possível digitá-lo ou utilizar uma ajuda artigos ativável com duplo click no campo matrícula. Se avisa que, utilizando a ajuda artigos, é possível fazer uma inserção máxima através o multiselect. Uma vez inseridos os artigos desejados é possível especificar o número de etiquetas a imprimir para cada artigo.

A janela permite imprimir o relatório padrão ou imprimir um relatório personalizado (após o mapeamento na gestão imprensas). Se quiser imprimir um relatório personalizado precisa escolher o mesmo do tab tabulado nos parâmetros de imprensa.
