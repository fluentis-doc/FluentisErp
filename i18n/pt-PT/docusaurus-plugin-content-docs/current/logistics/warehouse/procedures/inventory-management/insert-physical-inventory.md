---
title: Inserção inventários físicos
sidebar_position: 3
---

Para inserir um novo inventário físico precisa apertar o botão (Inserir) da Ribbon Bar portanto inserir o número, a data, a descrição, eventuais notas e o operador. Está presente também o flag (Fechado) que permite, uma vez executada as diferenças de inventários, tornar fechado um inventário que não vai ser portanto mais pego em consideração para o procedimento de alinhamento estoques.

Uma vez inserido o inventário precisa completar a lista das listas de inventários que forma o inventário. Na grelha se insere um novo relatório especificando o tipo lista, o operador e o depósito de referência. É também possível inserir uma data fechada para fazer em modo que a lista não seja pega em consideração para os cálculos das diferenças de inventários. No detalhe das listas de inventário na Ribbon Bar principal se acrescentam outros botões:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Tipo lista | Botão para inserir uma nova lista escolhendo o tipo lista. |
| Modificar | Refere-se ao procedimento para modificar as listas de inventário. |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa do inventário atual. |
| Imprimir | Refere-se ao procedimento para imprimir fisicamente o relatório do inventário atual. |

Uma vez inseridas as listas de inventários precisa compilá-las. Para fazer isso se entra na lista de inventário desejada com um duplo click e se tem o acesso a janela  [Modificar inventários artigos](/docs/logistics/physical-inventory/inventory-management/inventory-lists-management).
