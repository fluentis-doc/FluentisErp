---
title: Modificar listas inventariais
sidebar_position: 4
---

Esta janela permite ao operador completar as listas inventariais depois de tê-las inseridas na janela  [Inserção inventários físicos](/docs/logistics/physical-inventory/inventory-management/insert-physical-inventory).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa artigo sobre a inteira lista inventarial. |
| Inserir artigo | Refere-se ao procedimento para inserir um novo artigo dentro da lista. |
| Cancelar artigos | Refere-se ao procedimento para cancelar um novo artigo dentros da lista. |
| Importar | Refere-se ao procedimento para importar os artigos inventariais de uma procedimento externo. |

Precisa portanto inserir os artigos inventariais. Para fazer isso precisa se colocar no último record (ou apertar o botão [Inserir artigo] da Ribbon Bar) e especificar os seguintes campos:

**Artigo** (classe, código artigo, descrição). A inserção é possível também através da ajuda;

**Eventual variante**;

**Opção**;

**Depósito**;

**Localização**;

**Encomenda de venda**;

**Cliente/Fornecedor**;

**Eventual lote**;

**Quantidade relevada**;

**Eventual unidade de medida alternativa e quantidade alternativa**;

**Notas**;

**Operador**.

Em automático se completam os campos de Unidade de medida, Data inserção e Estoque a data de inserção da lista de retirada.

Uma vez completadas todas as listas de retiradas que compõem um inventário é possível passar para a fase  [Diferenças inventariais](/docs/configurations/parameters/logistics/warehouse-initial-parameters/stock-difference).
