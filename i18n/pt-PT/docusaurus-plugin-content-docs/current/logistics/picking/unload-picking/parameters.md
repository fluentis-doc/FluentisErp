---
title: Parâmetros
sidebar_position: 3
---

A partir do tab Parâmetros é possível especificar os parâmetros necessários para a criação dos movimentos no depósito. Em particular, para as linhas retiradas sem depósito e causal é possível escolher de Não efetuar o descarregamento para o inteiro documento, omitir o artigo incriminado, visualizar os artigos que não foram descarregados, estabelecer de default o depósito e a causal de default.
