title: Resumo
sidebar_position: 4
---

O tab Resumo permite visualizar as operações efetuadas e, eventualmente, executar a redução das operações.

É possível efetuar uma série de filtros para fazer em modo de visualizar só as operações desejadas. Selecionando na grelha de resultados uma linha vai ser possível ver o detalhe (e quer dizer as retiradas de referência e relativas registrações) sobre o tab na parte baixa da janela.

Selecionando uma ou mais linhas sobre a grelha de pesquisa e apertando o botão Cancelar os descarregamentos selecionados vão ser cancelados as registrações de depósito efetuadas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Efetua a pesquisa de acordo com os filtros estabelecidos. |
| Cancelar os descarregamentos selecionados | Cancela as registrações que derivam das linhas selecionadas sobre a grelha dos resultados. |
| Propriedade |  |
| Restaurar retirada | Cancela as registrações limitadamente as linhas retiradas selecionadas. |
