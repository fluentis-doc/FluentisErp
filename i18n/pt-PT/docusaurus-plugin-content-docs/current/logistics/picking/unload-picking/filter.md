title: Filtro
sidebar_position: 2
---

A partir do tab filtro é possível procurar as retiradas desejadas de acordo com os filtros inseridos. É também possível especificar uma data de registração de depósito.

A partir da grelha dos resultados precisa selecionar as retiradas desejadas e apertar o botão Descarregamento retirada. O procedimento vai efetuar, para as retiradas selecionadas, a movimentação de depósito.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Efetua a pesquisa de acordo com os filtros estabelecidos. |
| Descarregamento retirada | Cria as registrações de depósito para as retiradas selecionadas. |
