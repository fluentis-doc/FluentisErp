---
title: Gestão retirada
sidebar_position: 1
---

Na máscara se definem o tipo retirada e o cliente/fornecedor de referência. Além disso é possível substituir o número retirada e a data proposta e modificar o estado do documento (que pode ser Acabado, Verificado ou Suspendido).

Na grelha de inserção se insere os artigos especificando a classe, o código e a quantidade de retirada. É também possível inserir uma unidade de medida alternativa com a relativa quantidade. No campo depósito se define o depósito de movimentação e no campo Causal a relativa causal. É também possível especificar a localização pela qual retirar o artigo inserido.

Uma vez inseridos todos os artigos se passa a definir alguns parâmetros necessários para a movimentação do depósito. Na Data retirada depósito é possível especificar a data cujo tem que ser feita a registração do depósito. No Usuário é possível especificar o código usuário de referência para a retirada atual. Pelo que se concerne as retiradas de transferência precisa especificar o depósito de partida e o depósito de destino; eventualmente é possível especificar a localização aonde carregar os artigos, inserir algumas notas livres e especificar o cliente/fornecedor pela qual efetuar o movimento no depósito.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Salvar a retirada. |
| Novo artigo | Permite inserir um novo artigo dentro da retirada. |
| Cancelar o artigo selecionado | Permite cancelar o artigo selecionado. |
| Expandir grelha | Permite expandir a grelha artigos. |
| Propriedade do artigo | Inserendo um artigo a execução ordem é possível ver as suas propriedades. |
| Abrir disponibilidade | Abre a máscara de disponibilidade filtrada para o artigo selecionado. |
| Pré-visualização | Permite visualizar a pré-visualização de imprensa. |
| Imprimir | Permite imprimir o documento. |
| Cancelamento do lote selecionado  | Permite eliminar os lotes selecionados. |
| Cancelamento dos números de série | Permite eliminar os números de série selecionados. |
