---
title: Imprimir lista de retirada
sidebar_position: 2
---

O procedimento Imprimir lista de retirada foi desinhado para permitir ao usuário interrogar facilmente o sistema e extrapolar rapidamente informações que se referem as listas de retirada.

O procedimento Imprimir lista de retirada é constituído na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Pré-visualização | Refere-se ao procedimento para visualizar a pré-visualização de imprensa. |
| Imprimir | Imprime fisicamente o relatório da lista de retirada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das listas de retirada presentes no sistema. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Neste caso é possível filtrar por depósito, tipo retirada, gama número de retirada, gama data inserção retirada, cliente/fornecedor, estado documento (acabado, a controlar, suspendido). É também possível procurar as retiradas descarregadas pelo depósito e/ou executadas com a nota de entrega e/ou executadas com a fatura e/ou executadas com a lista de retirada.

A janela permite imprimir o relatório padrão ou imprimir um relatório personalizado (após o mapeamento na gestão imprensas). Se quiser imprimir um relatório personalizado precisa escolher o mesmo do tab tabulado nos parâmetros de imprensa.
