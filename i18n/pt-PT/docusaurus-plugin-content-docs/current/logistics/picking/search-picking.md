---
title: Procurar retirada
sidebar_position: 1
---

O procedimento Gestão retirada foi desinhado para permitir ao usuário procurar facilmente as retiradas precedentemente inseridas com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. É possível a partir da gestão Retirada fazer todas as operações necessárias no âmbito do documento, após a possibilidade de ter os direitos necessários.

O procedimento da Gestão retirada é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar retirada | Botão para aplicar os filtros de pesquisa dentro do banco de dados das retiradas inseridas. |
| Nova retirada | Refere-se ao procedimento para inserir uma nova retirada. |
| Modificar a retirada selecionada | Refere-se ao procedimento para modificar uma retirada precedentemente inserida. |
| Visualizar a retirada selecionada | Refere-se ao procedimento para visualizar uma retirada precedentemente inserida. |
| Cancelar a retirada selecionada | Executa o procedimento para cancelar uma retirada precedentemente inserida. |
| Descarregamento retirada | Executa o procedimento para efetuar o descarregamento do depósito de uma retirada. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nota de entrega | Executa o procedimento para criar o documento nota de entrega partindo de uma retirada. |
| Fatura | Executa o procedimento para criar o documento fatura partindo de uma retirada. |
| POS | Executa o procedimento para criar o documento POS partindo de uma retirada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das retiradas inseridas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das retiradas que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das retiradas, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das retiradas que quiser.
