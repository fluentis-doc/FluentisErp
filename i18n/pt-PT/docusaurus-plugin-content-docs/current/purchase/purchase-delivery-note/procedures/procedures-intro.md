RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Contabilização notas de entrega | O procedimento permite contabilizar as notas de entrega. |
|  [Carregamento notas de entrega no depósito](/docs/purchase/purchase-delivery-note/procedures/warehouse-purchase-delivery-note-record/load-delivery-notes-on-warehouse)  | O procedimento permite carregar na gestão uma nota de entrega já inserida. Com a ajuda dos filtros, procuram-se as notas de entrega não carregadas. No tab dos parâmetros se estabelecem o depósito e a causal e depois se aperta o botão 'Carregamento notas de entrega'. |
|  [Criação notas de entrega do recebimento de mercadorias](/docs/purchase/purchase-delivery-note/procedures/create-delivery-note-from-goods-receipt/create-delivery-note-from-goods-receipt)  | O procedimento permite criar as notas de entrega de compra pelo recebimento das mercadorias. Com a ajuda dos filtros, procuram-se os documentos de carregamento sem a nota de entrega. No tab dos parâmetros se estabelecem o tipo nota de entrega, Imposto, faturado de compras e se aperta o botão 'Criar nota de entrega do recebimento mercadorias'. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Notas de entrega de compra | Refere-se ao procedimento para imprimir as notas de entrega de compra em relação os critérios de filtro especificados (tipo nota de entrega, número, data, fornecedor, etc.). |
| Lista notas de entrega | Refere-se ao procedimento para imprimir uma lista de notas de entrega em relação os critérios de filtro especificados (tipo nota de entrega, número, data, fornecedor), oferecendo a possibilidade de ordenar em relação o fornecedor, o número nota de entrega, tipo nota de entrega ou data de inserção. |
