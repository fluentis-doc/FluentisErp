Nos Resumos são presentadas as informações principais do inteiro documento.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Cancelar os descontos finais | Permite cancelar os descontos pela grelha correspondente. |
| Cancelar as despesas | Permite cancelar as despesas pela grelha correspondente. |
| Pré-visualização de imprensa | Botão para executar a pré-visualização de imprensa do documento. |
| Imprimir | Botão para executar a imprensa física do documento. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document manager](/docs/guide/common/operations-with-data/document-manager)  | Gestão do armazenamento documental associado a máscara em uso. |

**Descontos finais**

Podem ser inseridos só os descontos exprimidos em porcentagem. Os descontos inseridos nessa seção vão ser calculados em cada artigo do documento.

**Resumos Imposto**

É proposto o resumo Imposto do documento, para cada código Imposto.

**Despesas**

São propostas as despesas inseridas no registro cliente, no tab 'Despesas/Descontos'. Tem a possibilidade também de inserir despesas que se referem só ao documento em utilização.

**Notas finais**

É um campo livre que pode ser preenchido pelo usuário com a ajuda do auxílio da 'Ajuda notas codificadas'.

**Totais documento**

**Quantia bruto artigos**: representa a soma dos valores de todos os artigos;

**Total descontos aplicados**: representa o valor total dos descontos aplicados sem os descontos finais;

**Quantia líquida artigos**: quantia bruta artigos - total descontos aplicados;

**Total descontos finais**: representa o valor dos descontos finais;

**Despesas não documentadas**: representa o valor das despesas inseridas como artigos de tipo 'Despesas';

**Despesas documentadas**: representa o valor das despesas inseridas na grelha Despesas;

**Tributável**: quantia líquida artigos - descontos finais + total despesas não documentadas + total despesas documentadas;

**Imposto**: representa a soma dos valores contidos nos resumos Imposto;

**Total**: tributável + imposto.
