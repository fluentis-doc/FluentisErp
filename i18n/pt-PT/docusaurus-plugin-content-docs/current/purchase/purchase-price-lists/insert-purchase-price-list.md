A inserção do catálogo fornecedor pede o código (conta - subconta) do fornecedor, a escolha do fornecedor pode acontecer através da ajuda de campo, que tem de ser especificada a 'Data validade da' e a 'moeda'. A 'Data validade a' não é obrigatória e é utilizada se quer definir uma campanha de promoção ou dar uma data de vencimento ao catálogo. Podem-se também inserir a encomenda de venda, as notas e o desconto do registro.

Na grelha artigos é possível inserir artigos através da ajuda artigos ou manualmente especificando classe e código artigo. É possível especificar na linha artigo catálogo:

**Variante artigo**, se além do código artigo, a variante também marca um artigo por um outro;

**Artigo fornecedor**, se o código artigo do fornecedor é diferente do código artigo da sociedade;

**Código variante artigo do fornecedor**, se além do código artigo fornecedor, a variante também marca um artigo fornecedor por um outro código artigo fornecedor;

**Unidade de medida**, que pode ser aquela gestional do artigo ou uma das suas unidade de medida alternativas;

**Marca do artigo**, se especificada;

**Quantidade**, proposta igual a 1, é a quantidade de referência do preço, se diferente de 1, quando tem de ser calculado o preço é dividido pela quantidade do catálogo;

**Preço**, refere-se a moeda do catálogo.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar os artigos | Permite procurar os artigos em relação o filtro de pesquisa especificados na procura dos artigos. |
| Novo artigo | Permite inserir um novo artigo no catálogo. |
| Salvar os artigos | Permite salvar os artigos inseridos no catálogo. |
| Cancelar os artigos | Permite cancelar os artigos inseridos no catálogo. |
| Atualizar os preços | Permite atualizar os preços de um catálogo aumentando de uma porcentagem. |
| Cancelar os descontos | Permite cancelar os descontos inseridos na linha artigo selecionada. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document manager](/docs/guide/common/operations-with-data/document-manager)  | Gestão do armazenamento documental associado a máscara em uso. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos artigos presentes no catálogo que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos catálogos que quiser.
