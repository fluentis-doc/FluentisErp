O módulo Catálogos fornecedores permite ao operador criar e atualizar um catálogo preços para o código fornecedor. O módulo se encontra na área compras e representa o passo inicial do ciclo passivo para poder redigir documentos que procuram os preços nos catálogos. O aplicativo permite: criar manualmente um catálogo fornecedor; a criação automática do catálogo, a importação dos catálogos do arquivo; imprimir o catálogo para confrontar os preços para o artigo.

Antes de proceder com a introdução ou realização de uma catálogo precisa assegurar-se de ter corretamente estabelecido todos os parâmetros de módulo. Não tem tabelas preestabelecidas que interessam só a codificação dos catálogos fornecedor.
