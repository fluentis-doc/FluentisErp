---
title: Importação catálogos fornecedores
sidebar_position: 2
---

Com a ajuda deste procedimento é possível criar novos catálogos fornecedores, através da importação dos documentos de tipo Excel, texto (ASCII) ou XML. O procedimento permite inserir os artigos, que se encontram nos documentos pelas quais se executa a importação, diretamente no registro artigos.

Esse procedimento pede de:

1) selecionar 'Novo catálogo';

2) apertar o botão 'Importar da' para abrir a janela de importação;

3) selecionar o tipo importação: do arquivo Excel, do arquivo Texto (ASCII) e do arquivo XML;

4) apertar o botão 'Abrir arquivo' para selecionar o documento desejado;

5) no campo 'Fonte' aparece o percurso do documento selecionado;

6) selecionar a opção 'A primeira linha contém o nome das colunas';

7) apertar o botão 'Adiante';

Nesta nova janela o usuário vai ter que:

1) executar click direito do mouse no cabeçalho da tabela 'Visualização de 25 linhas dos campos formatados' para visualizar os novos nomes que vão ser atribuídos nas colunas;

2) eliminar os eventuais espaços;

3) escolher a unidade de medida;

4) selecionar 'Criar artigos no registro artigos' se quer inserir os artigos do documento diretamente no registro artigos;

5) apertar o botão 'Transferência' para importar os artigos do documento num catálogo.

A aplicação oferece a possibilidade de ter a imagem da importação: 'Pré-visualização importação'. Se os artigos não foram transferidos, o usuário vai ter a possibilidade de selecionar, nessa janela, os dados atuais e de terminar a operação através do botão 'Fim'.

Com a ajuda do flag 'Catálogo selecionado' pode-se escolher a oferta cujo importar o novo catálogo; apertar o botão 'Importação' e escolher os dados 'Catálogo cujo aplicar'. O catálogo assim criado pode ser visualizado no 'Procurar catálogos fornecedores'. Além disso, tem a possibilidade de visualizar e completar os dados dos artigos criados através do procedimento de importação.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para procurar os artigos. |
| Descarregamento XML | Botão para o descarregamentos dos documentos XML. |
| Importação | O procedimento permite executar a importação. |
