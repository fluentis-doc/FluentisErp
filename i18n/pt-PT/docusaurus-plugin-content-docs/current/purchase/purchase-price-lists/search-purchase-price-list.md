title: Procurar os catálogos fornecedores
sidebar_position: 2
---

O procedimento Procurar os catálogos fornecedores foi desinhado para permitir ao usuário procurar facilmente os catálogos, precedentemente inseridos, com o objetivo de visualizá-los, modificá-los, eventualmente cancelá-los ou proceder com a inserção de um novo catálogo, após a possibilidade de ter os direitos necessários para essas operações.

Com a ajuda dos filtros pdoe-se visualizar: um só catálogo (p.ex.: procurar por fornecedor), mais catálogos (p.ex.: procurar por data), os catálogos inseridos por um certo fornecedor (p.ex.: procurar por conta/subconta).

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar os catálogos | Botão para procurar os catálogos no banco de dados. |
| Novo catálogo | Botão para criar um novo catálogo. |
| Modificar o catálogo | Botão para modificar um catálogo precedentemente inserido. |
| Visualizar o catálogo | Botão para visualizar os catálogos precedentemente inseridos. Não permite a mudança. |
| Cancelar o catálogo | Botão para cancelar um catálogo precedentemente inserido. |
| Conversor de moeda | Botão para a conversão dos preços de um catálogo numa outra moeda. |
| Duplicar os catálogos | Botão para criar um novo catálogo partindo de um catálogo de referência. Aparece uma Form aonde é possível trocar o fornecedor, a data validade da, a data validade a e a moeda. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos catálogos fornecedores que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos catálogos que quiser.
