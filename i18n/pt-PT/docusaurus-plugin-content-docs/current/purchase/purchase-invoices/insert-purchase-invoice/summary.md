Nos Resumos são apresentadas as informações principais do inteiro documento.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Salvar | Botão para salvar as operações efetuadas. |
| Cancelar as despesas | Permite cancelar as despesas pela grelha correspondente. |
| Cancelar os descontos finais | Permite cancelar os descontos pela grelha correspondente. |
| Visualizar a registração contábil | Permite visualizar a registração contábil. |
| Contabilizar e fechar | Permite contabilizar a fatura e de fechar a janela. |
| Restauração contabilização | Permite restaurar a contabilização da fatura. |
| Agrupar prazos | Permite agrupar os prazos. |
| Recalcular os prazos relativos a fatura | Permite recalcular os prazos. |
| Pré-visualização | Botão para executar a pré-visualização de imprensa do documento. |
| Imprimir | Botão para executar a imprensa física do documento. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [Document manager](/docs/guide/operations-with-data/document-manager)  | Gestão do armazenamento documental associado a máscara em uso. |

**Descontos finais**

Podem ser inseridos só os descontos expressos em porcentagem. Os descontos inseridos nessa seção são calculados em cada artigo do documento.

**Resumo Imposto**

É proposto o resumo Imposto do documento, para cada código Imposto.

**Despesas**

São propostas as despesas inseridas no registro fornecedor. Além disso, tem a possibilidade de inserir despesas que se referem só ao documento em utilização.

**Notas fiscais**

É um campo livre que pode ser preenchido pelo usuário com o auxílio da 'Ajuda notas codificadas'.

**Fatura de acompanhamento**

Representa uma tabela pela qual pode-se inserir a relativa fatura de acompanhamento. Se aconselha de inserir precedentemente as faturas de acompanhamento em maneira de pode-las associar com a ajuda da 'Ajuda faturas'.

**Total documento**

**Quantia bruta artigos**: representa a soma dos valores de todos os artigos;

**Total descontos aplicados**: representa o valor total dos descontos aplicados sem os descontos finais;

**Quantia líquida artigos**: quantia bruta artigos - total descontos aplicados;

**Total descontos finais**: representa o valor dos descontos finais;

**Despesas não documentadas**: representa o valor das despesas inseridas como artigos de tipo 'Despesas';

**Despesas documentadas**: representa o valor das despesas inseridas na grelha Despesas;

**Tributável**: quantia líquida artigos - descontos finais + total despesas não documentadas + total despesas documentadas;

**Imposto**: representa a soma dos valores contidos nos resumos Imposto;

**Total**: tributável + Imposto.
