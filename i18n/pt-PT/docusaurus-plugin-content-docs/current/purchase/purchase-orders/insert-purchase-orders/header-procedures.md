---
title: Retomada dos artigos pela encomenda de compra
sidebar_position: 5
---

O procedimento de Retomada dos artigos pela encomenda de compra permite executar também parcialmente a encomenda de compra. São propostas todas as encomendas não executadas, não forçamente executadas e parcialmente executadas.

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar ordens | Procurar as encomendas não executadas e não forçamente executadas. |
| Bloquear a encomenda | Bloqueia a encomenda para a execução. |
| Desbloquear a encomenda | Desbloqueia a encomenda para a execução. |
| Transferência artigos | Transfere na ordem as linhas de encomenda selecionadas. |
| Subir de um nível na lista | Subir de um nível na lista. |
| Descer de um nível na lista | Descer de um nível na lista. |
| Execução forçada encomenda | Permite executar forçamente a encomenda durante a transferência na ordem. |
| Abrir a encomenda na visualização | Abre a encomenda na visualização. |
| Visualizar o estado da encomenda selecionada | Abre a visualização do estado da execução da encomenda selecionada. |
| Legenda | Despois de ter especificado a data pedido entrega e/ou a data de prevista entrega no cabeçalho ordem é possível substuituir em quantidade tal datas nas linhas artigo já inseridas. |
| Posição do objeto na lista | Depois de ter especificado a data pedido de entrega e/ou a data de prevista entrega no cabeçalho ordem é possível substituir em quantidade tal datas nas linhas artigo já inseridas. |
| Mostrar/Esconder o pai objeto | Mostra/Esconde o pai do objeto selecionado. |
| Mostrar/Esconder o painel da encomenda | Mostra/Esconde o painal da encomenda. |
| Selecionar tudo | Seleciona todos os objetos no painel de visão da lista. |
| Cancelar as impostações de registro | Botão para cancelar as impostações de registro. |
