---
title: Procurar ordens fornecedores
sidebar_position: 2
---

O procedimento Procurar ordens fornecedores foi desinhado para permitir ao usuário procurar facilmente as ordens precedentementes inseridas com o objetivo de visualizá-las, modificá-las e eventualmente cancelá-las. É possível, a partir de 'Procurar ordens fornecedores', fazer todas as operações necessárias no âmbito do documento, após a necessidade de ter os direitos necessários.

O procedimento Procurar ordens fornecedores é constituído na maneira seguinte:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Botão para aplicar os filtros de pesquisa dentro do banco de dados das ordens fornecedores inseridos. |
| Novo | Refere-se ao procedimento para inserir uma nova ordem fornecedor. |
| Modificar | Refere-se ao procedimento para modificar uma ordem fornecedor precedentemente inserida. |
| Visualizar | Refere-se ao procedimento para visualizar uma ordem fornecedor precedentemente inserida. |
| Cancelar | Executa o procedimento para cancelar uma ordem fornecedor precedentemente inserida. |
| Propriedade execução | Executa o procedimento para criar o documento de estorno relativo a ordem fornecedor selecionada. |
| Duplicar a ordem fornecedor | Executa o procedimento para criar um documento parecido com a ordem fornecedor selecionada na grelha abaixo. |
| Conversão moeda | Executa o procedimento para transformar o documento selecionado numa Moeda oportunamente especificada. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos. Os dados de filtro dão a possibilidade, através do botão [Procurar artigos >>], de procurar também os dados de cada artigo inserido na ordem fornecedor.

GRELHA DE RESULTADO: a grelha de resultado é uma lista das ordens fornecedores que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter uma pesquisa das ordens, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista das ordens que quiser.
