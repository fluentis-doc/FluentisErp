O procedimento de criação automática ordens é constituída na seguinte maneira:

RIBBON BAR: representa o menu do módulo em questão, ou seja, a área na qual efetuar algumas ações. A lista de possíveis funcionalidades é sucessivamente representada:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Procurar | Procura os pedidos de compra ou as previsões de compra em relação aos filtros inseridos. |
| Selecionar tudo | Seleciona todas as linhas obtidas aplicando o filtro para a criação do protótipo ordem. |
| Marcar tudo | Desseleciona todas as linhas obtidas aplicando o filtro que eram selecionados para a criação do protótipo ordem. |
| Geração protótipos ordem | Gera o protótipo de ordem para todas as linhas selecionadas. |

ÁREA DE FILTRO: a área de filtro é dedicada à hospedagem na lista de possíveis tipos de dados através do qual é possível fazer uma seleção e uma pesquisa sobre o total das ordens inseridas. Os filtros são em condição 'E' e é possível especificar mais critérios de filtro contemporâneos.

GRELHA DE RESULTADO: a grelha de resultado é uma lista dos pedidos de compra ou das previsões de compra que correspondem aos dados de filtro especificados acima. O usuário, depois de especificar os valores pelos quais deseja obter os dados, pressionando o botão [Pesquisa] presente na relativa Ribbon, resultará na grelha a lista dos dados que quiser.
