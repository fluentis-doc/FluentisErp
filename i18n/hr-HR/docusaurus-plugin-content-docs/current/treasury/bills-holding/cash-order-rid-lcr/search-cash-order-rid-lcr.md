---
title: Ricerca Riba/Rid/Lcr
sidebar_position: 1
---

La ricerca Riba/Rid/Lcr è stata disegnata per permettere all'utente di ricercare facilmente i file telematici creati nel modulo portafoglio effetti, allo scopo di visualizzare il file creato oppure cancellare e riabilitare la creazione di un nuovo file.

RIBBON BAR: rappresenta il menu della Form in oggetto, ossia l'area nella quale è possibile effettuare delle azioni collegate alla procedura in uso. La lista delle funzionalità possibili è di seguito rappresentata:



| Funzione | Significato |
| --- | --- |
| Ricerca Riba | Ricerca la lista dei file telematici creati. |
| Visualizza Riba | Esegue la ricerca dei file creati secondo i filtri impostati. |
| Cancella Riba | Cancella il flag di stato ‘file emesso' e chiede conferma della cancellazione dei file fisico dal percorso di rete memorizzato. |
