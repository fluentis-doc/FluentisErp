**Trgovački predstavnik**: trgovački predstavnik (agent) povezan s odgovarajućom provizijom za odabrani artikl; podatak odgovara onome što je prisutno u *Zaglavlje > kartica Trgovački predstavnik*. Postotak se može promijeniti.
