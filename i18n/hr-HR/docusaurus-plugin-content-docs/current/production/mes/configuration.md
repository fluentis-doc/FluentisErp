# Aktivacija Fluentis prikaza za MES

Za korištenje načina rada **Fluentis**, osmišljenog za optimizaciju prikaza Fluentis MES-a na **proizvodnom radnom mjestu**, potrebno je pravilno konfigurirati postavke korisničkog sučelja.

## Postupak aktivacije

1. Kliknite na ikonu **zupčanika** koja se nalazi na lijevoj strani ekrana.
2. Otvorite odjeljak **Postavke**.
3. Odaberite opciju **Fluentis prikaz**.

Nakon dovršetka postupka Fluentis će se otvoriti u Fluentis prikazu.
