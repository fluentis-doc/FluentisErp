---
title: Récapitulation
sidebar_position: 4
---

Le tab Récapitulation permet de visualiser les opérations effectuées et, éventuellement, exécuter la réduction des opérations.

Il est possible d'effectuer une série de filtres pour faire en façon de visualiser seule les opérations désirées. En sélectionnant sur la grille des résultats une ligne sera possible de voir le détail (et c'est-à-dire les prélèvements de référence et les relatifs enregistrements) sur les tab dans la partie basse de la fenêtre.

En sélectionnant une ou plusieurs lignes sur la grille de recherche et en appuyant le bouton Supprimer les décharges sélectionnées seront supprimées les enregistrements d'entrepôt effectués.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Il effectue la recherche selon les filtres établis. |
| Supprimer les décharges sélectionnés | Il supprime les enregistrements dérivés par les lignes sélectionnées sur la grille des résultats. |
| Propriété |  |
| Restaurer prélèvement | Il suprime les enregistrements d'une manière limitée aux lignes prélèvement sélectionnées. |
