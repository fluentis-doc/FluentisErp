---
title: Gestion prélèvement
sidebar_position: 1
---

Dans le masque si défine le type prélèvement et le client/fournisseur de référence. Entre autres est possible de remplacer le numéro prélèvement et la date proposée et de modifier l'état du document (qui peut-être Terminé, Vérifié ou Suspendu).

Dans la grille d'insertion on va à insérer les articles en spécifiant la classe, le code article et la quantité de prélèvement. Il est en outre possible d'insérer une unité de mesure alternative avec la relative quantité. Dans le champ entrepôt se définit l'entrepôt de mouvementation et dans le champ cause la relative cause. Il est en autre possible de spécifier l'emplacement par lequel retirer l'article inséré.

Une fois insérés tous les articles on passe à définir certains paramètres nécessaires pour la mouvementation de l'entrepôt. En Date prélèvement entrepôt est possible de spécifier la date où doit être fait l'enregistrement d'entrepôt, en Utilisateur est possible de spécifier le code utilisateur de référence pour le prélèvement courant. Pour ce qui concerne les prélèvements de transfert est nécessaire spécifier l'entrepôt de départ et l'entrepôt de destination; éventuellement est possible de spécifier l'emplacement sur lequel charger les articles, insérer des notes libres et spécifier le client/fournisseur pour lequel effectuer le mouvement à entrepôt.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer | Il enregistre le prélèvement. |
| Nouvel article | Il permet d'insérer un nouvel article à l'intérieur du prélèvement. |
| Supprimer l'article sélectionné | Il permet de supprimer l'article sélectionné. |
| Expandre la grille | Il permet d'expandre la grille article. |
| Propriété de l'article | En insérant un article par exécution ordre est possible de voir ses propriété. |
| Ouvrir la disponibilité | Il ouvre le masque de disponibilité filtrée pour l'article sélectionné. |
| Aperçu avant impression | Il permet de visualiser l'aperçu avant impression. |
| Impression | Il permet d'imprimer le document. |
| Supprimation du lot sélectionné | Il permet d'éliminer les lots sélectionnés. |
| Supprimation des numéros de série | Il permet d'éliminer les numéros de série sélectionnés. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestion de classement documentaire associée au masque en utilisation. |
