Au moment que la création a été exécutée, les lignes disparaissant par la grille et le résultat sera visible dans le dossier 'Restaurer' à l'intérieur duquel est possible d'annuler l'opération. Il n'est pas possible d'annuler la création des effets si ces sont comptabilisés ou présentés dans le registre.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Rechercher la liste des créations exécutées. |
| Rollback acquisition | Avec ce bouton sera supprimée toute l'acquisition effectuée, avec tous les effets associés. |
| Rollback effet | Avec ce bouton sera supprimée la création de chanque effet sélectionné. |
