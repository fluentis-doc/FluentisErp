---
title: Exécution par commande fournisseur
sidebar_position: 2
---

La procédure offre la possibilité d'exécuter une facture par une commande fournisseur.

Conditions: les fournisseurs de l'ordre et de la facture doivent être les mêmes; l'ordre doit avoir établi le flag 'Imprimé' et la 'Date confirmation ordre'.

Cette procédure reprendre toutes les données présentes dans l'ordre et par conséquent sera appliquée les conditions présentes dans l'ordre, aussi si ces actuellement sont changée. Dans le masque sont proposés, pour le fournisseur sélectionné, tous les ordres imprimés, confirmés et n'exécute pas ou exécuté partiellement. En ce moment, l'utilisateur a la possibilité de: sélectionner l'ordre pour entier (en établissant le flag au début de la ligne ordre); importer l'ordre dans la facture avec l'aide du bouton ou de sélectionner seul une partie des articles de l'ordre, éventuellement avec quantité différente. Pour ce, après avoir exécuté double click sur la ligne ordre, l'utilisateur peut sélectionner l'entière quantité de la ligne relative (en établissant le flag au débit de la ligne), ou peut changer le champ 'Quantité' pour exécuter seul partiellement l'ordre. Pour finir la procédure, l'utilisateur doit appuyer le bouton 'Transfert articles'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher les ordres | Il permet de chercher les commandes fournisseurs. |
| Visualisation état exécution ordres | Il permet de visualiser l'état de l'ordre. |
| Visualisation ordre | Il permet de visualiser la commande fournisseur. |
| Légende | Légende de tous les symboles utilisés pour l'exécution. |
| Supprimer les paramétrages des registres | Il permet de supprimer les paramétrages des registres. |
| Exécuter le transfert | Il permet de transferer les articles sélectionnés. |
| Exécution forcée des ordres sélectionnés | Il permet l'exécution forcée de l'ordre. |
| Salir d'un niveau dans la liste | Il permet de salir d'un niveau dans la liste. |
| Descendre d'un niveau dans la liste | Il permet de descendre d'un niveau dans la liste. |
| Individuer l'objet dans la liste | Il permet de posizioner les objets de la liste. |
| Sélectionnes tous les objets dans la liste | Il permet de sélectionner les articles dans la liste. |
| Montrer/Cacher le parent de l'objet sélectionné | Il permet de montrer/cacher le parent de l'objet sélectionné. |
| Montrer/Cacher le panneau des ordres | Il permet de montrer/cacher le panneau des ordres. |
| Bloque les ordres pour l'exécution | Il permet de bloquer les ordres pour l'exécution. |
| Relâcher les ordres bloqués | Il permet de débloquer les ordres bloqués pour l'exécution. |
