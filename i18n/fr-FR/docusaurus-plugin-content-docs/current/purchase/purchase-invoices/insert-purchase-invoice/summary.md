Dans les Récapitulations ils sont présentées les informations principales de l'entier document.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer | Bouton pour enregistrer les opérations effectuées. |
| Supprimer les frais | Il permet de supprimer les frais par la grille correspondante. |
| Supprimer remises finales | Il permet de supprimer les remises par la grille correspondante. |
| Visualiser l'enregistrement comptable | Il permet de visualiser l'enregistrement comptable. |
| Comptabiliser et sortir | Il permet de comptabiliser la facture et de fermer la fenêtre. |
| Restauration comptabilisation | Il permet de restaurer la comptabilisation de la facture. |
| Regrouper les échéances | Il permet de regrouper les échéances. |
| Recalculer les échéances relatives à la facture | Il permet de recalculer les échéances. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression du document. |
| Impression | Bouton pour exécuter l'impression physique du document. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestion de classement documentaire associée à la masque en utilisation. |

**Remises finales**

Ils peuvent être insérés seul les remises expresses en pourcentage. Les remises insérées en cette section viennent calculées sur chaque article du document;

**Récapitulations TVA**

Il vient proposée la récapitulation TVA du document, pour chaque code TVA;

**Frais**

Ils sont proposées les frais insérés dans le registre fournisseur, dans le tab 'Frais/Remises'. En outre il y a la possibilité d'insérer frais qui font référence seuls au document en utilisation;

**Notes finales**

Il est un champ libre qui peut être compilé par l'utilisateur avec l'aire de 'Aide notes codées';

**Facture de transport**

Il représente un tableau où est possible d'insérer la relative facture de transport. On conseille d'insérer précédemment les factures de transport en façon de la pouvoir joindre avec l'aide d'Aide factures'.

**Total document**

**Montant brut articles**: il représente la somme des valeurs de tous les articles;

**Total remises appliquées**: il représente la valeur totale des remises appliquées sans les remises finales;

**Montant net articles**: montant brut articles - totale remise appliquée;

**Total remises finales**: il représente la valeur des remises finales;

**Frais ne documentés pas**: il représente la valeur des frais insérés comme articles du type 'Frais';

**Frais documentés**: il représente la valeur des frais insérés dans la grille Frais;

**Imposable**: montant net articles - remises finales totale frais nedocumentés pas total frais documenté;

**TVA**: il représente la somme des valeurs contenues dans les récapitulations TVA;

**Total**: imposable + TVA.
