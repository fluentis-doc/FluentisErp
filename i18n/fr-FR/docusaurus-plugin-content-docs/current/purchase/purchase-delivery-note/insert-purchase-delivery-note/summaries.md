Dans les Résumés sont présentées les informations principales de l'entier document.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Supprimer remises finales | Il permet de supprimer les remises par la grille correspondante. |
| Supprimer frais | Il permet de supprimer les frais par la grille correspondante. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant du document. |
| Impression | Bouton pour exécuter l'impression physique du document. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestion de classement documentaire associée à la masque en utilisation. |

   **Remises finales**Il peuvent être insérés seul les remises exprimées en pourcentage. Les remises insérées en cette section viennent calculés sur chaque article du document.

**Récapitulation TVA**

Il vient proposé la récapitulation TVA du document, pour chaque code TVA.

**Frais**

Il sont proposés les frais insérés dans le registre client, dans le tab 'Frais/Remises'. En outre il y a la possibilité d'insérer frais qui font référence seuls au document en utilisation.

**Notes finales**

Il est un champ libre qui peut être compilé par l'utilisateur avec l'aide de l''Aide notes codées'.

**Totals documents**

**Montant brut articles**: il représente la somme des valeurs de tous les articles;

**Totale remises appliquées**: il représente la valeur totale des remises appliquées sans les remises finales;

**Montant net articles**: montant net articles - totale remise appliquée;

**Total remises finales**: il représente les valeurs des remises finales;

**Frais ne prouvés ** **pas**: il représente la valeur des frais insérés comme articles du type "Frais';

**Frais prouvés**: il représente les valeurs des frais insérés dans la grille Frais;

**Imposable**: montant net articles - remis finals totaux frais ne prouvés pas total frais prouvés;

**TVA**: il représente la somme des valeurs contenis dans les résumés TVA;

**Total**: Imposable TVA.
