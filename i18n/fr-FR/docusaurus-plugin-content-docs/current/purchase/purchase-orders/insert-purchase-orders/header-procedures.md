---
title: Reprise des articles par la commande d'achat
sidebar_position: 5
---

La procédure de Reprise des articles par la commande d'achat permet d'exécuter aussi partiellement la commande d'achat. Ils sont proposées toutes les commandes n'exécutées pas, ne forcément exécutées pas et partiellement exécutées.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher les ordres | Il recherche les commandes n'exécutées pas et ne forcément exécutées. |
| Bloquer la commande | Il bloque la commande pour l'exécution. |
| Débloquer la commande | Débloquer la commande pour l'exécution. |
| Transfert des articles | Il transfère dans l'ordre les lignes de commande sélectionnées. |
| Salir d'un niveau dans la liste | Salir d'un niveau dans la liste. |
| Descendre d'un niveau dans la liste | Descendre d'un niveau dans la liste. |
| Exécution forcée de la commande | Il permet d'exécuter forcément la commande pendant le transfert en ordre. |
| Ouvrir la commande en visualisation | Il ouvre la commande en visualisation. |
| Visualiser l'état de la commande sélectionnée | Il ouvre la visualisation de l'état exécution de la commande sélectionnée. |
| Légende | Légende des symboles utilisés. |
| Position de l'objet dans la liste | Après avoir spécifié la date de livraison demandée et/ou la date prévue de livraison en en-tête ordre est possible remplacer en masse telles dates dans les lignes article déjà insérées. |
| Montrer/Cacher le parent objet | Montrer/Cacher le parent de l'objet sélectionné. |
| Montrer/Cacher le panneau commande | Montrer/Cacher le panneau de la commande. |
| Sélectionner tout | Il sélectionne tous les objets dans le panneaux de vision de la liste. |
| Supprimer les paramétrages de registre | Bouton pour supprimer les paramétrages de registre. |
