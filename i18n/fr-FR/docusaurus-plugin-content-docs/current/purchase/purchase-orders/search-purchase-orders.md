---
title: Rechercher les commandes fournisseurs
sidebar_position: 2
---

La procédure Rechercher commandes fournisseurs a été désignée pour permettre à l'utilisateur de rechercher facilement les ordres précédemment insérés avec le bût de les visualiser, les modifier et éventuellement les supprimer. Il est possible, par Rechercher commandes fournisseurs, de faire tous les opérations nécessaires dans le milieu du document, après la nécessité d'avoir les droits nécessaires.

La procédure de Rechercher commandes fournisseurs est formée par la façon suivante:

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des commandes fournisseurs insérées. |
| Nouveau | Il se réfère à la procédure pour insérer un nouvelle commande fournisseur. |
| Modifier | Il se rédère à la procédure pour modifier une commande fournisseur précédemment inséré. |
| Visualiser | Il se réfère à la procédure pour visualiser une commande fournisseur précédemment inséré. |
| Supprimer | Il exécute la procédure pour supprimer une commande fournisseur précédemment inséré. |
| Propriété exécution | Il exécute la procédure pour créer le document de virement relatif à une commande fournisseur sélectionnée. |
| Doubler la commande fournisseur | Il exécute la procédure pour créer un document semblable à une commande fournisseur sélectionnée dans la grille en-dessous. |
| Conversion devise | Il exécute la procédure pour trasformer le document sélectionné en une 'Devise' spécifiée. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps. Les données de filtre donnent la possibilité, à travers le bouton [Rechercher articles>>] sont possibles de rechercher aussi les données de chaque article inséré dans la commande fournisseur.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des commandes fournisseurs qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des ordres, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des ordres désirés.
