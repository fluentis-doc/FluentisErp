---
title: Importation catalogues fournisseurs
sidebar_position: 2
---

Avec l'aide de cette procédure il est possible de créer nouveaux catalogues fournisseurs, à travers l'importation des documents du type Excel, texte (ASCII) ou XML. La procédure permet d'insérer les articles, qui se trouvent dans les documents par lesquels s'exécute l'importation, directement dans le registre articles.

Cette procédure demande de:

1) sélectionner 'Nouveau catalogue';

2) appuyer le bouton 'Importer de' pour ouvrir la fenêtre d'importation;

3) sélectionner le type importation: par le fichier Excel, par le fichier texte (ASCII) et par le fichier XML;

4) appuyer le bouton 'Ouvrir fichier' pour sélectionner le document désiré;

5) dans le champ 'Source' parait le parcours du document sélectionné;

6) sélectionner l'option 'La première ligne contient le nom des colonnes';

7) appuyer le bouton 'Avancer'.

En cette nouvelle fenêtre l'utilisateur devra:

1) exécuter le click droit du souris dans l'en-tête du tableau 'Visualisation de 25 lignes des champs formatés' pour visualiser les nouveaux noms qui seront attribués aux colonnes;

2) éliminer les éventuels espaces;

3) choisir l'unité de mesure;

4) sélectionner 'Créer articles dans le registre articles' s'on veut insérer les articles du document directement dans le registre articles;

5) appuyer le bouton 'Transfert' pour importer les articles par le document en un catalogue.

L'application offre la possibilité d'avoir l'imagine de l'importation: 'Aperçu avant impression importation'. Si les articles n'ont pas été trasférés, l'utilisateur aura la possibilité de sélectionner, en cette fenêtre, les données correctes et de terminer l'opération à travers le bouton 'Fin'.

Avec l'aide du flag 'Catalogue sélectionné' est possible de choisir l'offre sur laquelle importer le nouveau catalogue; appuyer le bouton 'Importation' et choisir les données 'Catalogue sur lequel importer'. Le catalogue ainsi créé peut-être visualisé en 'Chercher les catalogues fournisseurs'. En plus, dans le registre articles, il y a la possibilité de visualiser et compléter les données des articles créés à travers la procédure d'importation.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher les articles. |
| Déchargé XML | Bouton pour le déchargé des documents XML. |
| Importation | La procédure permet d'exécuter l'importation. |
