---
title: Rechercher les catalogues fournisseurs
sidebar_position: 2
---

La procédure Rechercher les catalogues fournisseurs a été désignée pour permettre à l'utilisateur de rechercher facilement les catalogues, précédemment insérés, avec le bût de les visualiser, les modifier, éventuellement les supprimer ou avancer à l'insertion d'un nouveau catalogue, après la possibilité d'avoir les droits nécessaires pour ces opérations.

Avec l'aide des filtres est possible de visualiser: un seul catalogue (p.ex. rechercher pour fournisseur), plusieurs catalogues (p.ex. rechercher pour date), les catalogues insérés pour un certain fournisseur (p.ex. rechercher pour compte/sous-compte).

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher les catalogues | Bouton pour rechercher les catalogues dans la base de données. |
| Nouveau catalogue | Bouton pour créer un nouveau catalogue. |
| Modifier le catalogue | Bouton pour modifier un catalogue précédemment inséré. |
| Visualiser le catalogue | Bouton pour visualiser les catalogues précédemment insérés. Il ne permet pas la modification. |
| Supprimer le catalogue | Bouton pour supprimer un catalogue précédemment inséré. |
| Currency Convert | Bouton pour la conversion des prix d'un catalogue en un autre devise. |
| Dupliquer les catalogues | Bouton pour créer un nouveau catalogue en partant par un catalogue de référence. Parait une form où est possible changer le fournisseur, la date validité pas, la date validité à et la devise. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des catalogues fournisseurs qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des catalogues désirés.
