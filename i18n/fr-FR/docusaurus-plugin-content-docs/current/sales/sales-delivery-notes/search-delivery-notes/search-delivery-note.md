---
title: Rechercher BL
---

La procédure Rechercher DDT/Bons de livraison de vente a été élaboré pour permettre à l'utilisateur de rechercher facilement les DDT/BL, insérées d'abord, avec le but de les visualiser, les modifier, les annuler et éventuellement les supprimer, avec la possibilité d'avoir des droits nécessaires pour ces opérations.

Avec l'aide des filtres est possible de visualiser: un seul document (p. ex. recherche pour numéro document), plusieurs documents (p. ex. recherche par date), les documents insérés en une année (p. ex. recherche par année), les documents insérés par un déterminé client (p. ex. recherche par comptes/sous-compte), les documents insérés par un déterminé agent ou relatif à une commande de vente (seul si la commande a été inséré dans l'en-tête). De plus, il y a la possibilité de chercher les documents qui contiennent un déterminé article ou une déterminée classe articles en utilisant 'Rechercher articles'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Recherche | Bouton pour rechercher les DDT/BL dans la base de données. |
| Nouveau bon de livraison | Bouton pour créer une nouvelle DDT/Bon de livraison. |
| Modifier bon de livraison | Bouton pour modifier une DDT/Bon de livraison précédemment insérée. |
| Visualiser bon de livraison | Bouton pour visualiser les bons de livraison précédemment insérées. Il ne permet pas la modification. |
| Supprimer bon de livraison | Bouton pour supprimer une bon de livraison précédemment insérée. |
| Dupliquer | Bouton pour doubler une bon de livraison. Permettre la création d'une copie d'un document, en reportant tous les conditions du document d'origine, mais avec les données actuelles. |
| Conversion | Bouton pour la conversion des prix d'un document en un autre devise. |
| Ristourne DDT/Bon de livraison | Bouton pour créer le document de ristourne de la bon de livraison d'origine insérée. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des DDT/BL qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des DDT/BL, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des DDT/BL désirées.
