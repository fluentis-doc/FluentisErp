---
title: Insérer PDV
sidebar_position: 1
---

L'insertion d'un PDV demande: le type PDV prédéfini en [Home  >  Tableaux  >  Vente]. À chaque document vient assigné un numéro selon la numérotation spécifiée par l'utilisateur dans la tableau 'Numérotation PDV' et au type de document qui contient la numérotation; l'année et la date courante vient compilée en automatique. Dans le champ 'Surnom/Nom' est possible d'insérer manuellement le surnom de l'opérateur qui a inséré le PDV, et dans le champ 'Commune' on insère où se trouve le PDV. Le flag 'Imprimé' devient actif quand vient imprimé le ticket de caisse. (Pour la version RO, le bouton 'Imprimer' engendre le ticket de caisse).

RIBBON BAR: représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer | Bouton pour enregistrer les opérations effectuées. Le même résultat si obtient en déplaçant le curseur sur une ligne divers de celle qu'on désire enregistrer. |
| Enregistrer nouveau | Enregistrer le PDV courant et créer un nouveau. |
| Déchargé PDV | Il permet de décharger PDV si par chaque article ont été insérés l'entrepôt et la cause. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression du document. |
| Imprimer | Bouton pour exécuter l'impression physique du document. |
| Aide compte PDV |  |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestion de classement documentaire associée à la masque en utilisation. |
