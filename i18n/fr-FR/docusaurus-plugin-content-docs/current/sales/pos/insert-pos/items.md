Le **tipe ligne** offre la possibilité de sélectionner les articles avec caractéristiques divers:

**Article codé**: ils sont les articles codés dans le registre et peuvent être comptabilisés dans la comptabilité analytique et enregistrés dans l'entrepôt;

**Article note**: il est une simple note et elle ne grave pas dans la comptabilité et dans l'entrepôt;

Ces données peuventêtre insérés automatiquement combien on insére directement la classe et le matricule parce que le type ligne **Article codé** vient sélectionné en automatique.

**Code**: il fait référence au code inséré dans les articles, tab 'Code BarCode'. Combien cet est inséré, le programme reprendre automatiquement l'article associé, avec touts les données de son register.

L'insertion de la classe ou du matricule peut avenir manuellement ou avec le secours du 'Aide articles' (double click dans la section jeune) lequelle proposera tous les données rélatives insérées dans le registre article. Après l'insertion de l'article, sa description sera reprise automatiquement dans le registre.

**Unité de mesure**: il vient proposée l'UM principale de l'article;

**Quantité**: il représente la quantité de la UM principal; peut être insérée manuellement;

**Prix avec TVA**: il représente le prix avec TVA et doit être inséré précédemment dans le Registre articles > Coûts et peut être divers par les entrepôts;

Les champs **TVA** et **Imposable** viennent calculés en automatique, en prélevant les valeurs par le prix avec TVA;

**Chiffre d'affaire vente**: il vient proposé la donnée insérée dans le tab 'Généralité' du registre articles. Si ce n'est pas présent, il faut insérer manuellement.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Noveau article | Bouton pour insérer un nouveau article. |
| Supprimer article | Bouton pour supprimer l'article inséré. |
| Étendre/Réduire la grille des articles | Il permet de visualiser ou cacher les tab  'Récapitulation document' et 'Détail article'. |
