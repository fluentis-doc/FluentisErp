title: Factures
sidebar_position: 6
---

À l'intérieur de la grille supérieure, une fois exécutées la recherche, sont visualisées les factures qui ont été comptées pour le calcul du règlement agent en utilisation. À travers les deux boutons de gestion, "Liaison documents avec le réglement' et ' Supprimer les liaisons avec le règlement', est possible d'ajouter manuellement des nouveaux documents dans le règlement ou les supprimer par les insérer en une successive.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Exécuter la recherche des factures insérées dans le réglement en utilisation. |
| Liaison des documents avec le réglement | Ouvrir une masque de recherche et d'insertion manuel des documents dans le réglement en utilisation. |
| Supprimer les liaisons avec le réglement | Enlever le document sélectionné par le réglement en utilisation. |
