---
title: Rechercher les règlements des agents
sidebar_position: 3
---

La procédure Rechercher des règlements agents a été désignée par permettre à l'utilisateur de rechercher facilement les règlements, précédemment insérés, avec le but de les visualiser, les modifier, les annuler et éventuellement les supprimer ou avancer à l'insertion d'un nouveau règlement, après la possibilité d'avoir les droits nécessaires par ces opérations.

Avec l'aide des filtres est possible visualiser: un seule règlement (p.ex. recherche par numéro règlement), plus règlements (p.ex recherche par date), les règlements insérés en une année (p.ex. rechercher par année), les documents insérés par un certain agent (p.ex. recherche par code agent), les règlements qui contiennent des commissions d'un déterminé document (ordre, bon de livraison ou facture selon que le règlement soit ordonné, expédié, facturé/échu/encaissé).

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher les règlements dans la base de données. |
| Insérer règlements | Bouton pour créer un nouveau règlement. |
| Modifier règlements | Bouton pour modifier un règlement précédemment inséré. |
| Visualiser règlements | Bouton pour visualiser un règlement précédennent inséré. Ne permet pas la modification. |
| Supprimer les règlements | Bouton pour supprimer un règlement précédemment inséré. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des règlements qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des règlements, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des règlements désirés.
