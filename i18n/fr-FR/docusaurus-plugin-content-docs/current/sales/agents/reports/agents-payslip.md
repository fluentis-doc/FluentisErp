title: Bordereaux de paie agents
sidebar_position: 1
---

Cette impression permet d'extraire par les documents de vente les commissions insérées sur divers agents, indépendamment du critère de maturation du règlement de l'agent.

Dans le bouton radio central on effectue la sélection du type de document à considérer: l'ordre client (ordonné), le DDT de livraison (livré) ou la facture (vendu). Selon cette opération, le filtre du type document changera.

Il faut signaler l'option d'impression avec le détail par article: cette version du rapport peut être utile pour contrôler la correction des commissions insérées par le système et pour pouvoir individuer les erreurs des paramétrages du registre agent.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Imprimer | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |
