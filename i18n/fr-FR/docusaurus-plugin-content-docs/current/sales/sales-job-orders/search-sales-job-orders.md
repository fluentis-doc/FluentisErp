---
title: Rechercher commandes de vente
sidebar_position: 3
---

La procédure Rechercher commandes de vente a été désignée par permettre à l'utilisateur de rechercher facilement les commandes précédemment insérées, avec le but de les visualiser, les modifier, et éventuellement les supprimer ou par insérer une nouvelle commande, après la possibilité d'avoir les droits nécessaires pour ces opérations.

Avec l'aide des filtres est possible de visualiser: un seul document (p. ex. rechercher par numéro document), plus documents (p. ex.rechercher par date), les documents insérés en une année (p. ex. rechercher par année), les documents insérés par un certain fournisseur (p. ex. rechercher par compte/sous-compte) et les documents sur la base à la date de livraison prévue ou date de confirmation.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher commande | Bouton pour rechercher les commandes dans la base de données. |
| Nouvelle commande | Bouton pour créer une nouvelle commande. |
| Modifier commande | Bouton pour modifier une commande précédemment insérée.  |
| Visualiser commande | Bouton pour visualiser les commandes précédemment insérées. Ne permet pas la modification. |
| Supprimer commande | Bouton pour supprimer une commande précédemment insérée. |
| Exécution ordres | La procédure automatique permet d'exécuter les commandes client par une commande. |
| Doubler la commande | Bouton pour doubler une commande. Permet la création d'une copie d'un document, en  rapportant toutes les conditions d'origine, mais avec les données actuels. |
| Propriété | Il permet de visualiser l'exécution d'une commande, avec les données des articles commande et les données des documents associés. |
| Conversion | Bouton pour la conversion des prix d'un document en un autre devise. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression du document, selon les critères de filtre spécifiés (type commande, numéro, date, client). |
| Impression | Bouton pour exécuter l'impression physique du document, selon les critères de filtre spécifiés (type commande, numéro, date, client). |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: La grille de résultat représente la liste des commandes qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des enregistrements, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des DDT/BL désirés.
