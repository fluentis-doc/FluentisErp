title: Récapitulations
sidebar_position: 4
---

Dans les récapitulations sont présentées les informations principaux de l'entier document.

**Remises finales**: Il peut être insérés seul les remises exprimées en pourcentage. Les remises insérées en cette séction viennent calculées en chaque article du document;

**Récapitulation TVA**: il vient proposée la récapitulation TVA du document, pour chaque code TVA;

**Dépenses**: ils sont proposées les frais insérées dans le registre client, tab 'Frais/Remises'. En autre il y a la possibilité d'insérer les frais qui font référence seul au document en utilisation;

**Notes finals**: il est un champ libre qui peut-être rempli par l'utilisateur avec le secours de l'Aide codées.

**Récapitulation commissions agents**: ils sont les agents insérés au niveau du client avec les relatives commissions.

**Totals document**

**Montant brut articles**: il représente la somme des valeurs de tous les articles;

**Total remises appliquées**: il représente la valeur total des remises appliquées sans les remises finales;

**Montant net articles**: montant brut articles - total remises appliquées;

**Total remises finales**: il représent la valeur des remises finales;

**Dépenses ne documentés pas**: il représent la valeur des frais insérés comme articles de type 'Frais';

**Dépenses documentés**: il représente la valeur des frais insérés dans la grille 'Frais';

**Imposable**: montant net articles - remises finales + total dépenses ne documentés pas + total dépenses documentés;

**TVA**: il représente la somme des valeurs contenus dans les récapitulations TVA;

**Total commande**: imposable + TVA.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer | Bouton pour enregistrer les opérations effectuées. |
| Supprimer les remises finales | Il permet de supprimer les remises finales par la grille correspondant. |
| Supprimer dépenses | Il permet de supprimer les frais par la grille correspondant. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression du document. |
| Imprimer | Bouton pour exécuter l'impression physique du document. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document manager | Gestion de classement documentaire associée à la masque en utilisation. |
