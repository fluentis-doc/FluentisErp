Il indique le code agent et sa commission; vient proposé le code et le pourcentage défini dans le registre client. Si la commission n'est pas reliée au client dans le registre client, doit être de toute façon insérée l'agent mais avec commission NULLE parce que si fût avec commission 0, signifierait que l'agent est relié à l'agent mais ne percevois pas commission.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Supprimer l'agent | Il permet de supprimer les agents insérés dans l'en-tête; les agents sont reprise par le registre. |
