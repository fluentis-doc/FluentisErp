---
title: Rechercher factures de vente
sidebar_position: 2
---

La procédure Rechercher factures de vente a été désignée par permettre à l'utilisateur de rechercher facilement les factures, précédemment insérées, avec le but de les visualiser, les modifier, les annuler et éventuellement les supprimer ou avancer à l'insertion d'une nouvelle facture, après la possibilité d'avoir les droits nécessaires par ces opérations.

Avec l'aide des filtres est possible visualiser: un seul document (p.ex. recherche par numéro document), plus documents (p.ex. recherche par date), les documents insérés en une année (p.ex. recherche par année), les documents insérés par un certains clients (p.ex. recherche par compte/sous-compte), les documents insérés par un certain agent ou relatif à une commande de vente (seul si la commande a été insérée dans l'en-tête). En plus il y a la possibilité de chercher les documents qui contiennent un certain article o une certaine classe articles en utilisant 'Rechercher articles'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher factures | Bouton pour rechercher les factures dans la base de données. |
| Nouvelle facture | Bouton pour créer une nouvelle facture. |
| Modifier facture | Bouton pour modifier une facture précédemment insérée. |
| Visualiser facture | Bouton pour visualiser les factures prcédemment insérées. Ne permet pas la modification. |
| Supprimer facture | Bouton pour supprimer une facture précédemment insérée. |
| Ristourne facture | Bouton pour créer le document de ristourne par la facture d'origine insérée. Quand l'utilisateur exécute le ristourné sort un formulaire avec la liste de tous les articles de la facture d'origine et l'utilisateur doit sélectionner la cause d'animation de l'article et établir les flag ristourne et enfin le type de facture à créer. La liaison entre la facture de ristourne et la facture d'origine doit être insérée manuellement par l'utilisateur dans le champ de référence Réf. FT e du Type. |
| Doubler facture | Bouton pour doubler une facture. Permet la création d'une copie d'un document, en rapportant toutes les conditions du document d'origine, mais avec les données actuels. |
| Conversion | Bouton pour la conversion des prix d'un document en un autre devise. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Factures de vente  | Il se réfère à la procédure par imprimer les factures de vente selon les critères de filtre spécifiés (type facture, numéro, date, client, etc.) |
|  Liste des factures de vente | Il se réfère à la procédure par imprimer une liste de factures selon les critères spécifiés (type facture, numéro, date, client), en offrant la possibilité d'ordonner sur la base du client, numéro, facture, type facture ou date d'insertion. |
|  Chiffre d'affaire  | Il se réfère à la procédure par imprimer les factures de vente, selon les critères de filtre spécifiés (type facture, numéro, date, client), en offrant la possibilité d'ordonner sur la base de l'articles, classe  articles, pays et client. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Comptabilisation factures de vente  | La procédure permet di comptabiliser les factures de vente. |
|  Création par le DDT  | La procédure permet de créer une facture de vente par un ou plusierus DDT. |
|  Décharger factures immédiats par l'entrepôt  | La procédure permet de décharger une facture de vente par l'entrepôt. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des factures qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des enregistrements, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des factures désirées.
