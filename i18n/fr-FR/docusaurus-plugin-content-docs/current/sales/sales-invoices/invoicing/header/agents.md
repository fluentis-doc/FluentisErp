Il indique le code agent et sa commission; vient proposé le code et la pourcentage définis dans le registre client. Si la commission n'est pas reliées au client dans le registre client, de toute façon doit être insérée l'agent mais avec commission NULLE parce que si était avec commission 0; signifierait que l'agent est relié mais ne recevois pasde commission.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Supprimer agents | Il permet de supprimer les agents sélectionnés. |
