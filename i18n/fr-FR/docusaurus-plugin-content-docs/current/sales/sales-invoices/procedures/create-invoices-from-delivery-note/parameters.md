---
title: Paramètres
sidebar_position: 3
---

Ils sont définis une série de paramètres: Regroupement des bons de livraison pour le même client en une seule facture - les bons de livraison peuvent être: du types divers, des années diverses, d'échanges divers, peuvent être regroupées par type paiement ou par destination. Les types paiement peuvent être repris par le registre du client ou maintenus ceux des DDT. S'il n'y a pas le regroupement pour type paiement la facture peut contenir aussi plusieurs lignes de paiements divers. Regroupe échéances en facture, offre la possibilité de regrouper, en une unique solution, les échéances avec la date égale, mais provenant par les bons de livraison divers. Références bon de livraison - si ce flag est établi, dans la facture sera présent une note qui reprendra les références de du bon de livraison par laquelle a été créée la facture. La composition de cette note doit être insérée dans le champ 'Description' selon les paramètres décrit dans l'étiquette. Après avoir sélectionné les bons de livraison (tab 'Valorisation') et établi les paramètres, la facture peut-être créée avec l'aide du bouton 'Création facture par le DDT/BL'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer les paramètres de valorisation | Il permet d'enregistrer les paramètres insérés. |
