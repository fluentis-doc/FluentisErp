Il étant donné qu'est la dernière fiche, est présent une fiche de restauration: est possible d'utiliser par filtrer, visualiser et annuler les opérations.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher factures créées par DDT/BL | Il permet de chercher les bons de livraison valorisés. |
| Restaurer la création | Il permet de restaurer l'entière opération, c'est-à-dire, dans la création de plus factures en même temps, supprime toutes les factures et restaure les DDT. |
| Aperçu avant impression du rapport création |  |
| Restaurer facture | Il permet de restaurer la facture sélectionnée dans le tab Factures. |
| Enlever le flag imprimé |  |
