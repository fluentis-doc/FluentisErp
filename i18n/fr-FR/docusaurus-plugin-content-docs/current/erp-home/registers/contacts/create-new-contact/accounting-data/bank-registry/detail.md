title: Détail
---

Dans le premier tab est possible d'indiquer certaines données générales appliquées par les banques: 'commissions pour opération' (utilisé dans le crédit effets), pour 'maximum à découvert' (n'utilisé automatiquement), 'frais diverses' (n'utilisé pas automatiquement), 'pour impayé' (n'est pas utilisé automatiquement).

Dans la section en bas il y a des échelons des taux d'intérêts actifs/passifs à appliquer.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Supprimer actifs | Bouton pour supprimer les taux actifs sélectionnés. |
| Supprimer passifs | Bouton pour supprimer les taux passifs sélectionnés. |
