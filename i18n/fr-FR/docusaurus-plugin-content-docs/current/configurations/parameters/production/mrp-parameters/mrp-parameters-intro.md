title: Paramètres MRP

La fenêtre des Paramètres MRP s'ouvre à travers le parcours [Home > Paramètres > Production > Paramètres MRP]. Il se réfère à la fenêtre de recherche des paramètres MRP des articles présents dans la base de données. Pour chaque article, en effet, il vient créée un registre des paramètres MRP dès que vient insérer son registre 'principal' dans la fenêtre de la section Registres qui permet de coder un nouvel article.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Chercher | Bouton pour appliquer les filter de recherche à l'intérieur de la base de données des articles insérés. |
| Modifier paramètres | Il se réfère à la procédure pour modifier les paramètres MRP d'un article précédemment inséré. |
| Visualiser paramètres | Il se réfère à la procédure pour visualiser les paramètres MRP d'un article précédemment inséré. |
| Remplacement informations | Il ouvre une fenêtre animée qui permet de décider quelles informations relatives à l'article modifier. L'utilisateur a la possibilité, pour les données pour qui a habilité les flag, de changer massivement les informations de plusieurs articles sélectionnés préventivement dans la grille. |

ZONE DE FILTRE: en cette section est possible d'utiliser non seulement une série de filtres de recherche quel le filtre pour classe, code, description et variante de l'article, mais le filtre typeapprovisionnement (achat, production, sous-traitant);

GRILLE DE RÉSULTAT: avec un double click sur la ligne de l'article on entre dans la fenêtre de détail des paramètres MRP du mêmearticle. La fenêtre est formée par une partie d'en-tête où sont visualisés la classe, le code et la description de l'article, la relative éventuelle variante et son unité de mesure, et par une partie formée par 2 tabs.
