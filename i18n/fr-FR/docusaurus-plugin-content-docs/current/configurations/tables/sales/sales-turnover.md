---
title: Chiffre d'affaires de vente
sidebar_position: 6
---

Les données contenues dans le tableau du type chiffre d'affaire se rend nécessaires au moment de la comptabilisation des documents.

Le code chiffre d'affaires peut-être codé seul s'il vient inséré aussi le compte sous-compte comptable, le type TVA et le type montant. Si le type chiffre d'affaire vient inséré dans le registre articles ce code vient proposé dans les lignes document et est constructif pour la comptabilisation des documents, si dans l'article n'est pas inséré le type chiffre d'affaires alors il vient repris le compte de coût présent dans le registre du fournisseur. Le compte sous-compte présent en ce tableau est constructif aussi pour les centres de coût dans les lignes article des documents.

**Détail pour nations**: ces données sont utilisés seuls pour l'échange des données avec programmes extérieurs. En future il sera possible de définir comptes de coûts différents pour nation et type comptabilisation article, associé dans le registre client.
