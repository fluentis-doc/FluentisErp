---
title: Numérotation DDT/Bons de livraison
sidebar_position: 7
---

Le tableau de numérotation DDT/Bons de livraison permet de définir des progressifs de numérotation différents à l'intérieur de la même année à attribuer à un ou plusieurs types DDT/Bons de livraison.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un type numérotation. |
| Insertion | Bouton pour insérer un nouveau type de numérotation. |
| Supprimation | Bouton pour supprimer le type numérotation. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Année | Année de référence de la numérotation. |
| Type | Type numérotation. |
| Description | Description du type numérotation. |
| N. prog. | Numéro progressif de départ pour la numérotation des DDT. |
| Retour client | S'il est établi, il indique que le type numérotation est utilisé pour numéroter les retours c'est pourquoi ne vérifie plus le numéro/date progressifs et vient ajouté la vôtre référence dans la ligne créée dans la valorisation DDT. |
| Séries | S'il est établi, il vient utilisée en comptabilisation pour ramener ensembles au numéro document aussi la série. |
| N. fix caractères | S'il est présent, il vient utilisé pour composer la série. |
