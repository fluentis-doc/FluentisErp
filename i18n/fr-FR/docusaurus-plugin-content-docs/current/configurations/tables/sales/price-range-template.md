---
title: Définitions tranches de commissions
sidebar_position: 21
---

Le tableau Définition tranches de commissions gère les tranches de commissions standard, avec lesquels relier le pourcentage de commission à celle de la remise appliquée à la ligne article, selon les caractéristiques de définition de la gamme de remise. On rappelle que le tranche de remise se réfère au pourcentage théorique final entre le prix net et le prix brut (p.ex. une remise à cascade 50% 10% corresponde à une remise effective du 55%).

Pour commancer l'activité il est nécessaire indiquer, dans le premier champ, sur quelle gamme remise on veut travailler.

Section de gauche: cette section contient le détail des records du tableau associé à la gamme de remise établie. S'on parle de gamme remise unique cette section est vide, autrement montre le code/description du tableau de référence.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche du record de paramétrage de la tranche de commission selon le paramétrage des champs filtre. |
| Insertion | Bouton pour insérer une nouvelle tranche de remise. |
| Supprimation | Bouton pour supprimer une tranche de remise, sauf que cette n'a pas été déjà utilisée. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression des tranches de commissions. |
| Impression | Bouton pour effectuer l'impression des tranches de commissions. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code d'identification du record du tableau de base. |
| Description | Description du record du tableau de base. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code de la tranche. |
| Description | Description de la tranche. |
| De remise | Niveau pourcentage inférieur de la tranche. |
| À remise | Niveau pourcentage supérieur de la tranche. |
| Valeur | Valeur de la commission. |
| Pourcentage | Il identifie si la valeur est un pourcentage ou une valeur absolue. |
