title: Numérotation PDV
sidebar_position: 11
---

Le tableau de numérotation PDV permet de définir des progressifs de numérotation différents à l'intérieur de la même année à attribuer à un ou plusieurs types PDV.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un type numérotation. |
| Insertion | Bouton pour insérer un nouveau type numérotation. |
| Supprimation | Bouton pour supprimer le type numérotation. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Année | Année de référence de la numérotation. |
| Type | Type numérotation. |
| Description | Description du type numérotation. |
| N.prog. | Numéro progressif de départ pour la numérotation des PDV. |
