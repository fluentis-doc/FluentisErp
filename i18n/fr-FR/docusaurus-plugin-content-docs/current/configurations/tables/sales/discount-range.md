---
title: Gamme remise
sidebar_position: 12
---

Le tableau Gamme remise permet de définir quel soit l'élément sur lequel définir la variabilité des échelons d'application des remises, sur lesquels définir les commissions agents.

RIBBON BAR: représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la rechercher de gamme remise à travers les champs filtre. |
| Insertion | Bouton pour insérer une nouvelle gamme remise. |
| Supprimation | Bouton pour supprimer une gamme remise, sauf que cette n'a pas été utilisée. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression de la gamme remise. |
| Impression | Bouton pour effectuer l'impression de la gamme remise. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Gamme | Code, numérique, d'identification de la gamme de remise. |
| Description | Description de la gamme remise. |
| Unique | Avec ce flag les tranches de commissions remise seront appliquées sur toutes les articles indifféremments. |
| Classes art. | Avec ce flag se pourront définir les tranches de commissions de remise aussi comme les classes article. |
| Catalogues | Avec ce flag se pourront définir les tranches de commissions de remise aussi comme les catalogues de vente (ne personalisés pas) définis. |
| Types art. | Avec ce flag se pourront définir les tranches de commissions de remise aussi comme les types article. |
| Caractéristique | Avec ce flag se pourront définir les tranches de commissions de remise aussi comme les caractéristiques article. |
| Cat. produit | Avec ce flag se pourront définir les tranches de commissions de remise aussi comme les catégories de produits des articles. |
