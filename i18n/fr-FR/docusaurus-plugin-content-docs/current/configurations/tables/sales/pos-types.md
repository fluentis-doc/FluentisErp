---
title: Types PDV
sidebar_position: 20
---

Le type PDV est utilisé pour déterminer les propriétés d'une facture.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un type document. |
| Insertion | Bouton pour insérer un nouveau type document. |
| Supprimation | Bouton pour supprimer le type document. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type | Type PDV. |
| Description | Description du type PDV. |
| Type numér./Description | Type numérotation associée au type PDV. |
| Défault | S'il est établi, il indique le type PDV par défault qui vient proposé à la création d'un nouveau PDV. |
| Notes | Notes descriptives pour le type PDV, à utiliser dans les impressions. |
| Prix avec TVA | S'il est établi, les prix présents dans le PDV sont compréhensifs de TVA. |
| Acompte | S'il est établi, le PDV est de type acompte et pourra être ristourné en un PDV normal. |
| Serveur PDV | Serveur où lire et/ou écrir les données des PDV. |
| Base de données PDV | Base de données où lire et/ou écrir les données des PDV. |
| Tableau PDV | Base de données où lire et/ou écrir les données des PDV. |
| Tableau reli.documents | Base de données où lire et/ou écrir les données des documents reliés aux PDV. |
| UM Arr. | UM utilisée dans la création de DDT et factures de PDV. |
| TVA Arr./Description TVA | Code TVA utilisé dans la création de DDT et factures de PDV. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Point de vente | Point de vente où viennent créés les PDV. |
| Caisse | Code de la caisse à rélier aux PDV. |
| Cat.prod. | Catégorie produit. |
| TVA | Code TVA à utiliser dans les PDV. |
| Entrepôt | Entrepôt d'engagement de la marchandise contenue dans le PDV de ce type. |
| Cause | Cause d'entrentien de la marchandise engagée contenue dans le PDV de ce type. |
| Type paim. | Type paiement proposé dans les PDV de ce type. |
| Soluc. paim. | Soluction de paiement proposée dans les PDV de ce type. |
| Catalogue | Code catalogue général dunquel viennent reprises les prix des articles à insérer dans le PDV. |
| Date validité | Date validité du catalogue général. |
| Déchargé automatique dans la clôture de l'équivalent | S'il est établi, au moment que le PDV vient imprimé vient automatiquement aussi déchargé par l'entrepôt. |
| Il est nécessaire les caises enregistreuses | S'il est établi, pour utiliser les PDV il faut la liaison avec un caisse enregistreuse pour l'impression du ticket de caisse. |
| Visualiser variantes | S'il est établi, ils peuvent être utilisées aussi les variantes article. |
| Visualiser attributs | S'il est établi, ils peuvent être utilisés aussi les attributs article. |
