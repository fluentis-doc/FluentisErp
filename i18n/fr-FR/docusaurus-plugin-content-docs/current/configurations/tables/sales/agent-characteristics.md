---
title: Caractéristiques agent
sidebar_position: 2
---

Il est un tableau descriptif pour l'agent et peut-être utilisé dans son registre. Il est commun à toutes les sociétés présentes dans la base de données et est formée par un simple tableau code/description, aux fins de typiser les agents en rapport/statistiques personnalisées.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'une caractéristique agent à travers les champs filtre. |
| Insertion | Bouton pour insérer une nouvelle caractéristique agent. |
| Supprimation | Bouton pour supprimer un caractéristique agent, sauf que ce n'a pas été déjà utilisée. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression des caractéristiques agents. |
| Impression | Bouton pour effectuer l'impression des caractéristiques agents. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code, alphanumérique, d'identification de la caractéristique agent. |
| Description | Description de la caractéristique agent. |
