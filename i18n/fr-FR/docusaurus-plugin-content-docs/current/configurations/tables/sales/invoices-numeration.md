title: Numérotation factures
sidebar_position: 8
---

Le tableau de numérotation factures permet de définir des progressifs de numérotation différente à l'intérieur de la même année à attribuer à un ou plusieurs types factures.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un type document. |
| Insertion | Bouton pour insérer un nouveau type document. |
| Supprimation | Bouton pour supprimer le type document. |
| Aperçu avant impression | Bouton pour excuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Année | Année de référence de la numérotation. |
| Type | Type numérotation. |
| Description | Description du type numérotation. |
| N.Prog. | Numéro progressif de départ pour la numérotation des factures. |
| Séries | S'il est insérée, il vient utilisée en comptabilisation pour remener ensemble au numéro document aussi la série. |
| N. fix caractères | S'il est présent, il vient utilisé pour composer la série. |
