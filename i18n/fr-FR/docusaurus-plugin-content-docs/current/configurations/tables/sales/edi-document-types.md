---
title: Types document EDI
sidebar_position: 15
---

Le type EDI est utilisé pour déterminer les propriétés d'un document EDI.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un type document. |
| Insertion | Bouton pour insérer un nouveau type document. |
| Supprimation | Bouton pour supprimer le type document. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type | Code type EDI. |
| Description | Description type EDI. |
| Type n. | Type numérotation EDI associé au type EDI. |
| Type ordre | Type commande client pour la création du document EDI. |
| Type prévision de vente | Il est établi en automatique la prévision hebdomadaire pour la création du document EDI. |
| Défaut | S'il est établi, il marque le type EDI par défaut pour la création automatique des documents EDI. |
| Notes | Notes relatives au type EDI. |
