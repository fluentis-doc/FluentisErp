---
title: Changes devises
sidebar_position: 4
---

Le tableau des Changes devise, communs à toutes les sociétés de la même base de données, permet d'établir jour pour jour les changes officiaux à utiliser dans les diverses procédures du gestionnel: en particulier si signale que le programme ira à récupérer par ce tableau le dernier change valable à reculons par la date du document/enregistrement en gestion.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Change | Normalement va insérée la devise de la société (euro). |
| Description | Description de la devise. |
| En  | Devise de référence. |
| Description | Description de la devise de référence. |
| Date devise | Date de référence du change. |
| Valeur change | Il se réfère au change 'incertain pour certain', effectivement utilisé par le programme dans les calculs des contrevaleurs. En cas d'euro, donc, est la valeur contraire au change officiel du jour. |
| Valore diretto | Valore diretto. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un change avec possibilité de filtrer pour les champs 'Devise' et 'De date à date'. |
| Insertion | Bouton pour insérer un noveau change. |
| Actualisation changes | Bouton pour actualiser les changes, présente une dialogue avec la liste des derniers changes établis dans le tableau. |
| Importer par le fichier | Bouton pour avancer au paramétrage de fichier, en cas de fichier de Home Banking structuré cohérentement. |
| Supprimation | Bouton pour effectuer la supprimation des changes. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression des changes. |
| Impression | Bouton pour effectuerl'impression des changes. |
