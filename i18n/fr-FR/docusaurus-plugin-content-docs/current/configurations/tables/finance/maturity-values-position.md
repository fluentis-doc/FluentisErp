La définition de ce tableau, à paramétrages libres, permettra par le masque Comptes d'assigner une position à une liste de comptes sélectionnés à travers le bouton 'Changer position des comptes'.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code de la position des comptes. |
| Description | Description de la position des comptes. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche de la position des comptes, à travers les champs code et description. |
| Insertion | Bouton pour effectuer l'insertion d'une nouvelle posititon des comptes. |
| Supprimation | Bouton pour effectuer la supprimation de la position des comptes sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
