title: Typologies montants paiement
sidebar_position: 30
---

La définition du tableau permet de coder les typologies de paiement à utiliser dans le formulaire Gestion encaissement.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code coordonné de la typologie de paiement. |
| Description | Description de la typologie de paiement. |
| Compte | Champ optionnel pour indiquer le compte de relevé des remises et réductions pour la typologie de paiement. |
| Sous-compte | Champ optionnel pour indiquer les sous-compte de relevé des remises et réductions pour la typologie de paiement. |
| Description | Description du sous-compte. |
| Montant recouvré | Flag qui individue sur quelles typologies va calculé l'effectif montant recouvré. |
| Notes | Note à associer au mouvement comptable qui relève le type montant. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher la typologie de paiement à travers les champs de filtre Code et Description. |
| Nouvelle typologie montants | Bouton pour effectuer l'insertion de la typologie de paiement. |
| Supprimer la typologie montants | Bouton pour effectuer la supprimation de la typologie de paiement sur lequel s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avan impression. |
| Impression | Bouton pour effectuer l'impression. |
