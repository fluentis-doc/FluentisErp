La définition du tableau Types comptes n'est pas autre qu'une 'qualification' qui va à indiquer à la procédure comme gérer le code de sous-compte qui est assigné:

- d'abord de toute va à indiquer comme valoriser le détail de gestion des 4 typologies de registre prévues;

- en deuxième lieu va à inentifier les sections d'appartenance à bilan (actif/passif/coûts/produits);

- peut enfin servir comme critère de filtre, typiquement à l'intérieur de l'aide comptes.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code type compte. |
| Description | Description type compte. |
| P. actif | Avec le flag sur l'actif. |
| P. passif | Avec le flag sur le passif. |
| E. coûts | Avec le flag sur les coûts. |
| E. produits | Avec le flag sur les produits. |
| C. ordre | Avec le flag sur les comptes d'ordre. |
| Client | Avec le flag sur les clients + actif. |
| Fournisseurs | Avec le flag sur les fournisseurs + passif. |
| Banque | Avec le flag sur l'actif ou passif + banque. |
| Agent | Avec le flag sur l'agent + passif. |
| Service | Avec le flag sur le service - Les comptes avec tel flag actif sont les sous-comptes pour lesquels la procédure gère en automatique les écritures de rectification et d'intégration à niveau de bilans intermédiaire, mais aussi les écritures d'ajustement de comptabilité générale. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche du type compte, à travers les champs filtre code et description. |
| Insertion | Bouton pour effectuer l'insertion du type compte. |
| Supprimation | Bouton pour effectuer la supprimation du type compte sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
