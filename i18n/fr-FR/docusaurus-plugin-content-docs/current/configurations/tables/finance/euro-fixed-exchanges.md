Le tableau présent la liste des changes fixes de conversion à l'euro: en établissant la valeur de l'euro le programme avancera automatiquement au calcul des valeurs inverses qui sera utilisé dans les calculs des contrevaleurs du gestionnel.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Devise | Code devise. |
| Description devise | Description devise. |
| Date devise | Date devise. |
| 1 EURO = | Valeur de 1 euro = devise de référence. |
| Change contraire | Calcul di change contraire. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche, d'un change, avec la possibilité de filtrer pour les champs Devise et de date à date. |
| Insertion | Bouton pour insérer un nouveau change fixe pour l'euro. |
| Supprimation | Bouton pour supprimer un change. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression des changes fixes pour l'euro. |
| Impression | Bouton pour effectuer l'impression des changes fixes pour l'euro. |
