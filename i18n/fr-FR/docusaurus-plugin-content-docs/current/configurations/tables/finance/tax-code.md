Le tableau 'Code impôt', préétabli, enregistre tous les codes contribution prévus pour la déclaration F24 gérée à l'intérieur du formulaire receveurs: il se réfère aux codes officiaux ministériel.

Les champs 'Utilisation', 'Type', 'Versement', 'Année' et 'Acte' sont importants pour la gestion du modèle F24.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code contribution ministérielle. |
| Description | Description code contribution. |
| Utilisation | Code utilisation. |
| Type | Type contribution. |
| Versement | Versement contribution. |
| Année | Année contribution. |
| Bureau | Bureau. |
| Acte | Acte. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un code de contribution, à travers les champs code et description. |
| Insertion | Bouton pour effectuer l'insertion d'un nouveau code contribution. |
| Supprimation | Bouton pour effectuer la supprimation d'un code contribution. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
