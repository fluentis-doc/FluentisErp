La définition du tableau prévoit un simple code et description de la caractéristique à pouvoir assigner aux diverses immobilisations: il sera utilisable comme filtre de recherche/impression pour obtenir une liste d'immobilisations qui ont une déterminée propriété.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code alphanumérique de la propriété immobilisation. |
| Description | Description de la propriété immobilisation. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la rechercher de la propriété immobilisation, à travers les champs filtre code et description. |
| Insertion | Bouton pour effectuer l'insertion de la propriété immobilisation. |
| Supprimation | Bouton pour effectuer la supprimation de la propriété immobilisation sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
