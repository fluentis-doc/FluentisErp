---
title: Rechercher
---

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour la recherche du type retenue à traveres les champs de filtre Type et Description. |
| Nouveau | Bouton pour l'insertion d'un nouveau type de retenue. |
| Modifier | Bouton pour la modification du type tribut sur lequel s'est positionné. |
| Visualiser | Bouton pour visualiser le type de retenue sur lequel s'est positionné. |
| Supprimer | Bouton pour supprimer le type de retenue sur lequel s'est positionné. |
