title: Plans d'amortissement alternatifs
sidebar_position: 16
---

La définition de ce tableau permet de définir les plans d'amortissement à valoriser dans la simulation pluriannuelle des amortissements.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code du plan d'amortissement. |
| Description | Description du plan d'amortissement. |
| Division | Flag de gestion sur la seule division active. |

Pour simplifier l'insertion des détails des catégories il est présente un bouton pourcentage à appliquer sur toutes les catégories encore à insérer dans le plan d'amortissement sélectionné.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Catégorie | Catégorie à qui appliquer le plan d'amortissement. |
| Description | Description catégorie à qui appliquer le plan d'amortissement. |
| Pourcentage d'application | Pourcentage à appliquer. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche du plan d'amortissement, à travers les champs filtre code et description. |
| Nouveau plan | Bouton pour effectuer l'insertion d'un nouveau plan d'amortissement. |
| Nouveau détail | Bouton pour effectuer l'insertion du détail du plan d'amortissement sur qui s'est positionné. |
| Supprimer plan | Bouton pour effectuer la supprimation du plan d'amortissement sur qui s'est positionné. |
| Supprimer détail | Bouton pour effectuer la supprimation du détail du plan d'amortissement sur qui s'est positionné. |
| Ajouter catégories manquantes | Bouton pour effectuer l'attribution en automatique au pourcentage pour toutes les catégories encore manquantes. |
