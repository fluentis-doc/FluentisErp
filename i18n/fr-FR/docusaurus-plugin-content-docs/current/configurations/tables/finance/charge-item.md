---
title: Voix de frais
sidebar_position: 31
---

La définition du tableau des Voix de frais permet de coder des voix à associer au plan des comptes pour fins statistiques.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code coordonné de la voix de frais. |
| Description | Description de la voix de frais. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour la recherche de la voix de frais à travers les champs de filtre Code et Description. |
| Nouveau | Bouton pour l'insertion de la voix de frais. |
| Modifier | Bouton pour la modification de la voix de frais sur lequel s'est positionné. |
| Visualiser | Bouton pour visualiser la voix de frais sur lequel s'est positionné. |
| Supprimer | Bouton pour supprimer la voix de frais sur lequel s'est positionné. |
