title: Types amortissements objectifs
sidebar_position: 22
---

La définition du tableau permet à l'utilisateur de pouvoir opérer des amortissements objectifs.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type | Code coordonné du type d'amortissement objectif. |
| Description | Description du type d'amortissement objectif. |
| Opér. d'augmentations | Flag de gestion comme 'augmentations' aussi pour les données objectives (si réfère à l'homonyme flag de la catégorie immobilisation). |
| Calcul des jours d'utilisation | Flag de calcul pour les jours d'utilisation qui ira à calculer effectivement les jours d'utilisation de l'immobilisation dans l'année au contraire qui utiliser tout l'exercice comme pour le calcul normal. |
| Pourc. | Le pourcentage Standard à appliquer. |
| Pourc. fixe | Avec tel flag actif le pourcentage sera fixe sans la possibilité de la personnaliser immobilisation pour immobilisation. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche du type d'amortissement objectif, à travers les champs filtre code et description. |
| Insertion | Bouton pour effectuer l'insertion du type d'amortissemetn objectif. |
| Supprimation | Bouton pour effectuer la supprimation du type d'amortissement objectif sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
