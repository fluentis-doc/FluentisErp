Le tableau permet de définir et gérer la liste des codes nomenclature Intra en utilisation par l'entreprise. Il est donc possible de supprimer, d'ajouter et de modifier la liste existante.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code nomenclature | Code nomenclature. |
| Description | Description. |
| UM supp. | Unité de mesure supplémentaire. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un code nomenclature, à travers les champs code, description et UM. |
| Nouveau | Bouton pour effectuer l'insertion d'un nouveau code nomenclature. |
| Supprimer | Bouton pour effectuer la supprimation du code nomenclature sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
