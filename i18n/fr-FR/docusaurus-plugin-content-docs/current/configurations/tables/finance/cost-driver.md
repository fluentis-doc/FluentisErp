Le tableau permet de définir une liste des codes de réattribution des mouvements des centres de coût/profit.

Dans la première section va inséré le code alphanumérique et la description du Cost Driver.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code Cost Driver alphanumérique de 5 caractères. |
| Description | Description Cost Driver. |

En cette deuxième grille il est possible de définir le critère de pourcentage de répartition liée au Cost Driver sélectionner en dessus: la somme des pourcentages de ce détail, librement attribuables soit à CDC que CDP pour le même Cost Driver, il doit donner le 100% du mouvement.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| CDC | Centre de coût à qui attribuer le critère de répartition liée au Cost Driver sur lequel s'est positionné. |
| Description | Description CDC. |
| CDP | Centre de profit à qui attribuer le critère de répartition liée au Cost Driver sur lequel s'est positionné. |
| Description | Description CDP. |
| Pourcentage | Pourcentage d'attribution du critère. |
| Division | La colonne Division, optionnelle, permet de forcer l'attribution du mouvement à une particulière division d'entreprise. |
| Description | Description division. |
| Exercice | Le champ Exercice, optionnel, permet de lier la validité du critère de répartition à une particulière année de comptabilité. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'un Cost Driver, à travers les champs code, description, CDC et CDP. |
| Insertion | Bouton pour effectuer l'insertion d'un nouveau Cost Driver. |
| Supprimation | Bouton pour effectuer la supprimation du Cost Driver sur qui s'est positionné. |
| Supprimer les détails | Bouton pour effectuer la supprimation du détail du Cost Driver sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
