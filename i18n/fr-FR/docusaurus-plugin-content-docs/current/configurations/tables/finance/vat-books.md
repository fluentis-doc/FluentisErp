Tableau de paramétrages des registres TVA à gérer dans la société active.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code alphanumérique coordoné du registre. |
| Description | Description du registre imprimé normalement dans les rapports officiaux. |
| Type registre | Typologie précodée du registre. |
| Description | Description du type registre. |
| Contrôle protocoles | Le paramétrage de ce flag bloque la possibilité d'imprimer en définitive un registre qui ait trous dans les protocoles ou incohérences dans le système des mêmes. |
| À proposer | Ce flag impose la gestion du registre relatif dans la procédure de ventilation des équivalents. |
| En règlement TVA | Il permet de déterminer quels registres TVA soient à gérer dans l'impression du Règlement TVA périodique. |
| Réf. reg. achat | Le champ est actif seul pour les registres ventes/équivalents, aux fins d'indiquer le registre TVA achats d'origine de la ventilation des équivalents. |
| Description | Description réf. reg. achat. |

La section permet d'établir le contacteur des protocoles pour chaque année comptable dans le champ 'Pro. TVA initial'.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Année | Année de référence. |
| Prot. TVA initial | Protocole TVA initial de l'année de référence. |
| Dernier prot. TVA | Le champ 'Dernier prot TVA' permet de visualiser quel est le dernier protocole TVA qui a été imprimé en définitive dans le même registre. |
| Date dernière impression | Date d'exécution de la dernière impression définitive. |
| Dernière page imprimée | Dernier numéro de page imprimée en définitive. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Insertion | Bouton pour effectuer l'insertion d'un nouveau registre TVA. |
| Insertion année | Bouton pour effectuer l'insertion d'un nouveau année pour le registre TVA sur qui s'est positionné. |
| Supprimer le registre TVA | Bouton pour effectuer la supprimation du registre TVA sur qui s'est positionné. |
| Supprimer année | Bouton pour effectuer la supprimation de l'année du registre TVA sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'impression. |
| Impression | Bouton pour effectuer l'impression. |
| Annuler les impressions définitives | Bouton pour effectuer l'annulation des impressions définitives des registres. S'ouvre un dialogue dans lequel sélectionner le registre et la période qu'on veut annuler. |
