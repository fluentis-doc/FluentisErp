Le tableau permet la définition de la structure, aussi sur différents niveaux, et les caractéristiques des centres de coût. La section générale présente la structure ramifiée des centres de coût définis pour la société active: en rouge sont visualisés les CDC qui ne sont pas valorisables en comptabilité, en gras ces utilisables dans la production.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Centre de coût | Code alphanumérique du CDC. |
| Description | Description du CDC. |
| Deuxième description | Deuxième description du CDC. |
| Flag MRP | Le flag permet d'utiliser le centre de coût dans le formulaire de la production. |
| Flag valorisable | Au moment que ce flag est établi le centre de coût peut être valorisé avec mouvements de comptabilité.Enlever le flag par les niveaux de structure qu'on ne veut pas valoriser de façon erronée et le laisser seul, donc, au niveau plus en bas. |
| Type centre de coût | Il définit si le centre de coût est direct ou indirect. |
| Centre de responsabilité | Il vient reliée le centre de coût au relatif centre de responsabilité. |
| Cost driver | Il assigne au centre de coût un Cost Driver de réattribution du pourcentage dans les Clôtures intermédiaires. |
| Date début | Il définit la date début du CDC. |
| Date fin | Il définit la date de fin utilisation des CDC. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher le prochaine CDC | Bouton pour effectuer la recherche des CDC successifs en respect à ce sur qui s'est positionné. |
| Rechercher le précédent CDC | Bouton pour effectuer la recherche des CDC précédents en respect à ce sur qui s'est positionné. |
| Nouveau CDC | Bouton pour effectuer l'insertion d'un nouveau centre de coût. |
| Nouveau lien CDC | Bouton pour effectuer l'insertion d'un lien/niveau du CDC sur qui s'est positionné. |
| Supprimer le lien CDC | Bouton pour effectuer la supprimation du lien/niveau CDC. |
| Exploser CDC | Bouton pour exploser la structure à niveaux du CDC. |
| Imploser CDC | Bouton pour imploser la structure à niveaux du CDC. |
| Diminution tabulation | Bouton pour diminuer la tabulation en terms de visualisation. |
| Augmentation tabulation | Bouton pour augmenter la tabulation en terms de visualisation. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression du CDC sur lequel s'est positionné.Pulsante per effettuare l'anteprima del CDC sul quale si è posizionati. |
| Impression | Bouton pour effectuer l'impression. |
