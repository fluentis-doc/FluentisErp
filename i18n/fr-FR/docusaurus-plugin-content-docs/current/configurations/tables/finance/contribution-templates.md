title: Causes de contribution
sidebar_position: 8

Le tableau 'Causes de contribution' permet à l'utilisateur de définir les causes de cotisation INPS à relier aux types de retenus: comme dit dans la documentation des types retenue, la gestion de telle section n'est plus nécessaire.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la recherche d'une cause de cotisation. |
| Nouveau | Bouton pour effectuer l'insertion d'une nouvelle cause de cotisation. |
| Supprimer | Bouton pour effectuer la supprimation d'une cause de cotisation. |
