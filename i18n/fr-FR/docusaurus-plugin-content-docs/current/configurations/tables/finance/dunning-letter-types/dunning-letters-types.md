RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le type rappel à travers les champs de filtre Code et Description. |
| Nouveau type | Bouton pour effectuer un nouveau type de rappel. |
| Modifier le type | Bouton pour modifier le type de rappel sur lequel s'est positionné. |
| Visualiser le type | Bouton pour visualiser le type de rappel sur lequel s'est positionné. |
| Supprimer le type | Bouton pour effectuer la supprimation du type de rappel sur lequel s'est positionné. |
