title: Types réévaluation
sidebar_position: 29
---

La définition de ce tableau permet de coder les références aux lois de réévaluation à utiliser pour gérer les détails des réévaluations à l'intérieur des chaque fiches immobilisations. En particulier, la gestion des détails vient habilitée seule pour les catégories qui présentent le flag 'Réévaluations'.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code coordonné du type de réévaluation. |
| Description | Description du type de réévaluation. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le type de réévaluation à travers les champs de filtre Code et Description. |
| Nouveau | Bouton pour effectuer l'insertion d'un nouveau type de réévaluation. |
| Supprimer | Bouton pour effectuer la supprimation du type de réévaluation sur lequel s'est positionné. |
