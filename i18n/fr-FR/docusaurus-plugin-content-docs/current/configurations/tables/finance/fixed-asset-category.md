title: Catégorie immobilisations

À l'intérieur de ce tableau si codent les catégories des immobilisations dans la société.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code alphanumérique. |
| Description | Description de la catégorie. |
| Compte | Le compte patrimoine de l'immobilisation. |
| Sous-compte | Le sous-compte patrimoine de l'immobilisation. |
| Description | Description du sous-compte patrimoine de l'immobilisation. |
| Mat/Imm | Indication de la typologie matérielle ou immatérielle du bien. |
| Maximal amort. | Montant maximal de l'amortissement accessible pour chaque immobilisation de la catégorie. |
| Pourc. amort. | Le pourcentage maximal de la valeur de l'immobilisation que pourra être amortie. |
| Opér. d'augmentation | Si le flag n'est pas établi, ainsi comme par défaut, chaque ligne d'augmentation de la valeur de l'immobilisation sera gérée avec un detail d'amortissement séparé et autonome.Si le flag est établi, au contraire, chaque réception sera attibué à la ligne d'achat initial, en allant à augmenter toujours la valeur initiale à sa date d'origine. |
| Limite d'amort. total | Quote-partie limite pour l'application de l'amortissement total au 100% dans l'année d'achat. |
| Réévaluation | Flag de gestion des détails des réévaluation. |
| Code bilan | Code de bilan pour l'impression du prospect des réévaluations. |
| Amort. objectif | Activer la gestion des données objectives de l'immobilisation: cette prévoit une duplication des valeurs patrimoines comptables  aux fins d'appliquer une logique d'amortissement extracomptable séparée. |

Dans la deuxième section est dédiée au paramétrage des détails d'amortissement applicables à la catégorie supérieure sélectionnée.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type d'amort. | Sélectionner la typologie d'amortissement entre l'amortissement ordinaire, amortissement avancé, et amortissement ralenti, ce dernier normalement ne vient pas utilisé. |
| Coût compte | Compte de coût sur lequel relever l'amortissemetn périodique. |
| Coût sous-compte | Sous-compte de coût sur lequel relever l'amortissement périodique. |
| Description compte | Description sous-compte de coût. |
| Fond compte | Compte de fond sur lequel abandonner l'amortissement périodique. |
| Fond sous-compte | Sous-compte de fond sur lequel abandonner l'amortissement périodique que pourrait être ainsi la même valeur patrimoine active pour ristourner  directement la valeur.  |
| Description compte | Description sous-compte de fond. |
| Excluire | Le flag permet d'excluire par le calcul automatique des amortissements le type amortissement sélectionné. |
| Pourcentage | Le pourcentage d'amortissement à appliquer pour le type d'amortissement établi. |
| N. année | S'active seul en correspondance des amortissements avancés, dans lequel vont insérés les 3 années d'application. En référence à cette valeur, va ramené que pour les immobilisations utilisées le numéro des années d'application est établi automatiquement sur seule année d'achat. |
| Amort. première année | Il vient proposé automatiquement pour toutes les catégories matériaux, ira à réduire le pourcentage d'amortissement  dans l'année de début  activité de l'immobilisation. |
| Fiscal | Il pourra être établi pour les lignes d'amortissement avancé pour le gérer comme amortissement fiscal ne comptabilisable pas. |
| Tyep déductibilité | Il permet d'assigner au type d'amortissement un code de déductibilité aux fins fiscales. |
| Descr. déducibilité | Description du type de déducibilité. |

Dans la troisième section est possible d'assigner, en référence à chaque ligne d'amortissement, la valorisation pourcentage par défautà centre de coût pour les immobilisations reliées à la catégorie. Tel paramétrage est secondaire respect à l'assignation des centres de coût à l'intérieur de chaque registre immobilisation.



| RibbonFunction | RibbonMeaning |
| --- | --- |
| CDC | Centre de coût à qui assigner la valorisation des immobilisations qui appartient à la catégorie. |
| Description | Description centre de coût. |
| Pourcentage | Paramétrage du porcentage de valorisation au centre de coût. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Chercher dans le tableau supérieur | Bouton pour effectuer la rechercher dans le tableau supérieur à travers les champs de filtre catégorie et description. |
| Insérer dans le tableau supérieur | Bouton pour insérer une nouvelle catégorie immobilisation. |
| Imsérer dans le tableau détail | Bouton pour insérer le détail du tableau supérieur sélectionné. |
| Insérer dans le tableau CDC | Bouton pour insérer les CDC du tableau supérieur sélectionné. |
| Supprimer les lignes sélectionnées | Bouton pour effectuer la supprimation des lignes sélectionnées (catégorie immobilisation). |
| Supprimer les lignes sélectionnées par le tableau inférieur | Bouton pour effectuer la supprimation des lignes sélectionnées de tableau inférieur (détail du type amortissement). |
| Supprimer les lignes sélectionnées par le tableau CDC | Bouton pour effectuer la supprimation des lignes sélectionnées de tableau centres de coût. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
