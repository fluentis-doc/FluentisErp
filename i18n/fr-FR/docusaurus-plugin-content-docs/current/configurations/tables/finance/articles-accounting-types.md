| RibbonFunction | RibbonMeaning |
| --- | --- |
| Code | Code coordoné du type de comptabilisation articles. |
| Description | Description du type de comptabilisation articles. |
| Client | S'activer le champ de comptabilisation articles sur le client. |
| Fournisseur | S'activer le champ de comptabilisation articles sur le fournisseur. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour effectuer la rechercher du type de comptabilisation articles, à travers les champs filtre code et description. |
| Insertion | Bouton pour effectuer l'insertion du type de comptabilisation articles. |
| Supprimation | Bouton pour effectuer la supprimation du type de comptabilisation articles sur qui s'est positionné. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression. |
| Impression | Bouton pour effectuer l'impression. |
