En ce tableau s'ira à définir la liste des sous-comptes comptables par lesquels calculer le solde comptable qui sera considérée disponibilité initiale de flux de trésorerie.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des types compte financières selon les filtres établis. |
| Nouveau type | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimer les types | Supprimer le type compte financier sélectionné. |
