title: Types échéance
sidebar_position: 3
---

En ce tableau si définent les typologies d'échéances extracomptables qui seront gérées manuellement à l'intérieur du flux de trésorerie. À chaque typologie peut être associé un sous-compte bancaire de canalisation du flux.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des types échéance selon les filtres établis. |
| Nouveau | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimer | Supprimer le type échéance sélectionné. |
