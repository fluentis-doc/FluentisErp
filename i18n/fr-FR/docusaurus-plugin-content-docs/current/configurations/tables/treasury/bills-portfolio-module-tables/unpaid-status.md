---
title: États impayés
sidebar_position: 6
---

Le tableau vient compilé à travers le paramétrage d'un code et d'une description utilisateur pour les états nécessaires à la gestion impayée de la société, en établissant un des états prédéfinis entre:

**Impayés**;

**Comptabilisé**;

**Émis notes de débit (n'utilisé pas)**;

**Comptabilisé notes de débit (n'utilisé pas)**;

**Encaissé**;

**Autorisé (n'utilisé pas)**.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des états impayés selon les filtres établis. |
| Insertion | Positionner le curseur dans l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type avance sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
