title: Numérotation effets
sidebar_position: 1
---

Pour définir une numérotation est suffissant établir l'année de référence, le code de la numérotation, sa description et le numéro de départ.

Le tableau est entre ces gérés automatiquement dans la procédure de génération des contacteurs du nouvel exercice comptable, qu'on trouve à l'intérieur des 'Struments' - 'Utilité'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des numérotations effets selon les filtres établis. |
| Insertion | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type avancé sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
