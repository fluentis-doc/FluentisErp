title: Types effets
sidebar_position: 3
---

Le tableau vient compilé à travers le paramétrage d'un code et d'une description utilisateur, auquel va assigné un numérateur de référence, une typologie d'effet entre ces prédéfinies et enfin un sous-compte de traitement: tel sous-compte sera utilisée en contrepartie à la clôture client à l'intérieur de la procédure de comptabilisation effets. Le type effet peut être associé au type de paiement dans le tableau homonyme. Il faut signaler, enfin, que seuls les effets de type de change pourrent être gérés dans la procédure de comptabilisation impayées sur la cession du formulaire Impayés.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des types effets selon les filtres établis. |
| Insertion | Positionner le curseur dans l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type avancé sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
