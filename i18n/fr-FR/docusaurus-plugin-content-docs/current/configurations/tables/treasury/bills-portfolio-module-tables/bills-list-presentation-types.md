Le tableau vient compilé à travers le paramétrage d'un code et d'une description utilisateur, auquel va assigné un numéroteur de référence.

Les options ultérieures prévues dans le masque offrent la possibilité de définir:

- si le type registre soit à gérer comme registres de virement effets à fournisseur;

- le type registre soit à gérer comme registre Prélèvement bancaire 'utilisations' dans la création du fichier télématique du formulaire Reçu Bancaire.

Les sous-compte de traitement comptable à proposer à l'acte de la création d'un nouveau registre: il se réfère au sous-compte de registre bancaire sur lequel viendra le crédit final effets et du transitoire de présentation pour la comptabilisation registres.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche types registre selon les filtres établis. |
| Insertion | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type avancé sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
