---
title: États effets
sidebar_position: 5
---

Le tableau vient compilé à travers le paramétrage d'un code et d'une description utilisateur pour les états nécessaires à la gestion effets de la société, en établissant un des états prédéfinis entre:

**Effet émis**;

**Effet présenté dans la remise**;

**Effet présenté sauf bonne fine**;

**Effet présenté depuis l'encaissement**;

**Effet revenu crédité**;

**Effet revenu impayé**;

**Effet circulé**;

**Effet en état générique**.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche états effets selon les filtres établis. |
| Insertion | Positionner le curseur dans l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type avancé sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
