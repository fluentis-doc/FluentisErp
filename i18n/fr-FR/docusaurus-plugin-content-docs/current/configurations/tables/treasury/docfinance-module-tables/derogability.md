Le tableau 'Dérogabilité' permet de définir les codes de dérogabilité des flux financiers prévu, ainsi comme prévus en DocFinance (le code du gestionnel doit être le même de DocFinance).

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche et toutes les grilles du masque, selon les filtres établis. |
| Insertion | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer la dérogabilité sélectionnée. |
