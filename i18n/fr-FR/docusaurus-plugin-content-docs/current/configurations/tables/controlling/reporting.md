title: Types reclassement

Les champs prévus sont ces du code alphanumérique de 5 caractères, la relative description et deux flag: le flag 'Filtre commandes' rendra disponible, pour les reclassements basés sur modèles de ce type, un filtre des données selon la commande de vente, en façon d'obtenir un reclassé des seules données relatives au filtre; aux même façons, le 'Filtre CDC/CDP' activera la possibilité de filtrer les données du reclassé pour un déterminé centre de coût/profit.

Ils existent 3 typologies standard: un générique '01 - Économique', un 'BEP - Break Even Point' et un 'IV - TVA directive' pour le modèle de reclassement établi selon ce prévu dans la IV directive Cee.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des types de reclassement selon les filtres établis. |
| Insertion | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type de reclassement sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécuter l'impression du tableau en utilisation. |
