Le tableau permet la définition de la structure, aussi sur différents niveaux, et les caractéristiques des centres de profit.

La section générale présente la structure ramifiée des Centres de profit définis pour la société active: en rouge sont visualisés les CDP qui ne sont pas valorisables en comptabilité, en gras ces utilisables dans la production.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Centre de profit | Code alphanumérique du CDP. |
| Description | Description du CDP. |
| Deuxième description | Deuxième description du CDP. |
| Flag MRP | Le flag permet d'utiliser le centre de profit dans le formulaire de la production. |
| Flag valorisable | Au moment que ce flag est établi le centre de profit peut être valorisé avec mouvements de comptabilité.Enlever le flag par les niveaux de structure qu'on ne veut pas valoriser en façon erronée et le laisser seul, donc, au niveau plusieurs basses. |
| Type centre de profit | Il définit si le centre de profit est direct ou indirect. |
| Centre de responsabilité | Il vient relié le centre de profit au relatif centre de responsabilité. |
| Date début | Il définit la date de début du CDP. |
| Date fin | Il définit la date de fin utilisation des CDP. |

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher le prochaine CDP | Bouton pour effectuer la recherche du CDP successif en respect à ce sur qui s'est positionné. |
| Rechercher le précédent CDP | Bouton pour effectuer la recherche du CDP précédent en respect à ce sur qui s'est positionné. |
| Nouveau CDP | Bouton pour effectuer l'insertion d'un nouveau centre de profit. |
| Nouveau lien CDP | Bouton pour effectuer l'insertion d'un lien/niveau du CDP sur qui s'est positionné. |
| Supprimer le lien CDP | Bouton pour effectuer la supprimation du lien/niveau du CDP. |
| Exploser CDP | Bouton pour exploser la structure à niveaux du CDP. |
| Imploser CDP | Bouton pour imploser la structure à niveaux du CDP. |
| Diminution tabulation | Bouton pour diminuer la tabulation en terms de visualisation. |
| Augmentation tabulation | Bouton pour augmenter la tabulation en terms de visualisation. |
| Aperçu avant impression | Bouton pour effectuer l'aperçu avant impression du CDP sur lequel s'est positionné. |
| Impression | Bouton pour effectuer l'impression. |
