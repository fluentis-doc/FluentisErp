Le tableau se compose par un code numérique du type enregistrement, une description et enfin deux flag qui caractérisent la typologie. Les possibles options sont:

- **aucun flag établi**: les causes industrielles qui ont ce paramétrage permettent d'exécuter les enregistrements industriels et (si défini à valeur) de comptabilité analytique (cdc/cdp/commandes);

- **compt. générale**: les causes industrielles qui ont ce paramétrage permettent d'associer une Cause de comptabilité générale, pour travailler au même temps sur les deux niveaux (plus cesd'analytiques si définis dans la cause industrielle ou générale);

- **seul analytique**: les causes industrielles qui ont ce paramétrage s'entendent être causes seules à valeur, avec traitement de la seule section de comptabilité analytique (cdc/cdp/commandes) sans avoir aucun mouvement de comptabilité industrielle et générale associée.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Avec ce bouton s'exécute la recherche des types enregistrements industriel selon les filtres établis. |
| Insertion | Positionner le curseur à l'insertion d'une nouvelle ligne. |
| Supprimation | Supprimer le type enregistrement sélectionné. |
| Aperçu avant impression | Exécuter l'aperçu avant impression du tableau en utilisation. |
| Impression | Exécute l'impression du tableau en utilisation. |
