---
title: Détail enregistrement
sidebar_position: 2
---

Dans la section 'Détail enregistrement' est possible d'indiquer une liste de sous-comptes industriels à proposer automatiquement dans les enregistrements industriels créés avec la cause en utilisation. Ces sous-compte seront proposés (avec la relative note insérée dans le champ dédié: il n'est pas prévue ici la gestion des codes paramétriques de la cause) seul si la causa soit 'À valeur' et le compte industriel ait associé dans le plan des comptes un compte de comptabilité générale.

Sur la base est permis de définir une description paramétrique à utiliser pour décrire le mouvement.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nouvelle cause | Avec ce bouton s'exécute l'ouverture d'une nouvelle cause industrielle. |
| Supprimer la cause | Il supprime la cause en utilisation. |
| Enregistrer la cause | Il enregistrer la cause en utilisation. |
| Supprimer les détails sélectionnés | Il supprime les détails sélectionnés. |
