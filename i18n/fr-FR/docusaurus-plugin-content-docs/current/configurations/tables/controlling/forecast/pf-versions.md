---
title: Versions PA
sidebar_position: 4
---

Cette form permet: d'insérer plusieurs versions pour les prévisions, d'établir une ou plusieurs versions actives (ces n'actives pas ne pourront pas être utilisé pour créer nouvelles prévisions mais seul pour filtrer les prévisions existents) et d'établir la version par défaut entre ces actives.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Insérer une nouvelle version | Bouton pour insérer une nouvelle version. |
| Supprimer les versions sélectionnées | Bouton pour supprimer les versions sélectionnées. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |
