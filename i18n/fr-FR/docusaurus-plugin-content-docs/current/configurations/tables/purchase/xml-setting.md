Ce tableau est nécessaire pour établir les données pour importer un catalogue fournisseur en déchargent par un site internet en format XML.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le type document. |
| Appliquer |  |
| Doubler | Bouton pour doubler un paramétrage XML. Il permet la création d'une copie d'un document, en ramènent toutes les conditions du document d'origine, mais avec les données actuelles. |
| Supprimation | Bouton pour supprimer le type document. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Compte - Sous-compte/Description sous-compte | Compte et sous-compte du client pour laquelle importer le catalogue. |
| Adresse | Adresse dunquel décharger le fichier XML: http://zoppelletto@gruppomade.com:srvr5m9p@ws.madelogo.net. |
| Utilisateur | Utilisateur pour se relier à l'adresse. |
| Mot de passe | Mot de passe pour se relier à l'adresse. |
| Webservice | Chemin où trouver les données à importer et le fichier: /catalogue/MPI.xml |
| Information initial | Information initial par où partir pour la création du fichier XML: offset='1'. |
| Tag initial | Il définit le tag initial pour reconnaitre le début du fichier XML: catalogue. |
| Ligne initial | Il définit la ligne dunquel commencer avec l'importation: articles. |
| Page initial | Il définit la page initial: page. |
| Dimension page | Il définit la dimension des pages: offset='500'. |
| Classe  | Classe à utiliser pour importer les articles. |
| UM | UM à utiliser pour importer les articles. |
| Marque | Marque à utiliser pour importer les articles. |
