---
title: Type commandes fournisseurs
sidebar_position: 10
---

Le type commande fournisseurs est utilisé pour déterminer les propriétés d'un ordre.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le type document. |
| Insertion | Bouton pour insérer un nouveau type document. |
| Supprimation | Bouton pour supprimer le type document. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établit et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type | Type commande client. |
| Description | Description du type ordre. |
| Type N./Description | Type numérotation associé au type ordre. |
| Autoévasion | S'il est établi, l'ordre peut être exécuté par les documents bons de livraison d'achat, réception marchandises et factures d'achat. |
| Flux de trésorerie | S'il est établi, le type ordre concourt au calcul du flux de trésorerie. |
| Entr./Description entrepôt | Entrepôt d'arrive de la merchandise contenue dans les ordres de ce type ordre. |
| Causa/Description cause | Cause d'entretien de la marchandise en arrivages contenue dans les ordres de ce type ordre. |
| Contrôle disponibilité | S'il est établi, les ordres de ce type concurent au calcul de la disponibilité. |
| Agr.Ach.TVA | S'il est établi, il vient cherchée pour l'article s'il existe le code TVA agricole, s'il n'existe pas il vient pris le code TVA normal. |
