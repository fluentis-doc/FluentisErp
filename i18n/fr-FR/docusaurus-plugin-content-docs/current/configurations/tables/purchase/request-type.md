---
title: Type demandes
sidebar_position: 14
---

Le type Demandes est utilisé pour déterminer les propriétés d'une demande.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le type document. |
| Insertion | Bouton pour insérer un nouveau type document. |
| Supprimation | Bouton pour supprimer le type document. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Type | Type demande d'achat. |
| Description | Description du type demande. |
| Type n./Description numérotation | Type numérotation associée au type demande. |
| Cdc | Centre de coût à proposer dans la création de la demande d'achat. |
| Entr./Description entr. | Entrepôt à utiliser dans la creation de la demande d'achat. |
| Contrôler la disponibilité | S'il est établit, les demandes d'achat créées avec ce type viennent utilisées pour le calcul de la disponibilité. |
