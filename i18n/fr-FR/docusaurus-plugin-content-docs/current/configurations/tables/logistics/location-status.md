---
title: Lieu emplacement
---

Ce tableau gère la codification des différentes états emplacements à devoir gérer pour après être disponibles dans les formulaires compétentes.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des lieus emplacement insérés. |
| Insertion | Permettre d'insérer un nouveau lieu emplacement. |
| Supprimation | Il supprime le lieu emplacement sélectionné. |
| Aperçu avant impression | Il exécuté l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est le code et la description du lieu emplacement.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des lieus emplacement qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des lieus emplacement, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des lieus emplacement.

**Insertion lieu emplacement**

Pour pouvoir insérer nouvelles codifications de Lieu emplacement il faut cliquer sur le prémier record vide (ou appuyer le bouton [Insertion] sur la Ribbon bar). On indique donc le code alphanumérique pour le type et sa description.
