Ce tableau gère la codification des différentes classes article et il est nécessaire pour pouvoir effectuer un regroupement des articles.

RIBBON BAR: il  représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour apploquer les filtres de recherche à l'intérieur de la base de données des classes insérées. |
| Insertion | Permettre d'insérer une nouvelle classe. |
| Supprimation | Il supprime la classe sélectionnée. |
| Aperçu avant impression | Il exécute l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est le code et la description de la classe.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des catégories qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des codifications existantes, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des codes désirés.

**Insertion classes articles**

Au-delà de rechercher les classes présents à système il est possible d'ajouter une ou plusieurs classes. Pour faire ça il faut cliquer sur le prémier record vide (ou appuyer le bouton [Insertion] sur la Ribbon bar). On indique donc le code alphanumérique pour la classe et sa description.
