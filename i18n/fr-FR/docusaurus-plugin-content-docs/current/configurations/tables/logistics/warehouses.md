Le tableau d'insertion entrepôts est nécessaire pour pouvoir utiliser en façon correcte toutes les fonctions du système que si référent au traitement des matériaux.

Le tableau des entrepôts est formé dans la façon suivante:

RIBBON BAR: ilreprésente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des entrepôts insérés. |
| Insertion | Il permet d'insérer un nouveau entrepôt. |
| Supprimation | Il supprime l'entrepôt sélectionné. |
| Supprimer les centres de coût | Il supprime le centre de coût sélectionné. |
| Supprimer les centres de profit | Il supprime le centre de profit sélectionné. |
| Aperçu avant impression | Il exécute l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est pour description entrepôt.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des entrepôts qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des entrepôts, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des entrepôts désirés.

**Insertion entrepôt**

En plus de rechercher les entrepôts présents à système est possible ajouter un ou plusieurs entrepôts. Pour faire cela il faut cliquer sur le premier record vide (ou appuyer le bouton [insertion] sur la barre d'outils). On indique donc un code entrepôt et sa description. Éventuellement est possible d'indiquer le lieu de production, la zone de gestion, une date de fine validité de l'entrepôt, la destination (reprise par le registre contacts) et/ou le point de vente.
