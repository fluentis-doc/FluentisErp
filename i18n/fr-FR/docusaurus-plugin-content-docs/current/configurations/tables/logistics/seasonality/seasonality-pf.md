---
title: Saisonnalité PA
sidebar_position: 1
---

Cette form permet de gérer les variations de saisonnalité de l'approvisionnement dans les périodes précédentes, pour article et période. Il est possible de faire cette opération manuellement ou en automatique avec la procédure de reprise des données dans la prévision courante par l'approvisionnement précédent; la procédure offre la possibilité d'engendrer la saisonnalité. La Form contient trois zones: la première avec les filtres de recherche, la deuxième avec les articles déjà enregistrés dans la saisonnalité et la troisième, saisonnalité, visualise, pour chaque mois de l'année, la quantité achetée pour l'article sélectionné dans la zone des articles.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher articles | Bouton pour rechercher les articles dans la base de données. |
| Nouveau | Bouton pour insérer une nouvelle ligne dans la zone de saisonnalité pour l'article sélectionné. |
| Supprimer les articles sélectionnés | Bouton pour supprimer l'article sélectionné et sa saisonnalité. |
| Doubler | Cette procédure permet de doubler en un autre année la saisonnalité désirée pour l'article sélectionné. |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression selon les filtres établis et la configuration de l'imprimante. |
