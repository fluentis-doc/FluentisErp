Ce tableau gère la codification des différentes catégories selon la logique produite qui peut être associée au produit et la pouvoir utiliser ou comme regroupement ou comme statistique ou comme fonctionnalité dans les formulaires compétents.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des catégories insérées. |
| Insertion | Il permet d'insérer une nouvelle catégorie de produit. |
| Supprimation | Il supprime la catégorie sélectionnée. |
| Aperçu avant impression | Il exécute l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est code et description caractéristique.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des catégories qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des catégories, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des catégories désirées.

**Insertion catégorie produit**

En plus de rechercher les catégories présentes dans le système est possible d'ajouter une ou plusieurs catégories. Pour faire ici il faut cliquer sur le premier record vide (ou appuyer le bouton [Insertion] sur la barre d'outils). On indique donc en code alphanumérique pour la catégorie et sa description.
