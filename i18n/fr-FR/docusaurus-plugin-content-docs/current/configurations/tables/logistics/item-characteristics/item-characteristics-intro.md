Le tableau des caractéristiques articles est formé dans la façon suivante;

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des caractéristiques insérées. |
| Insertion | Il permet d'insérer une nouvelle caractéristique. |
| Supprimation | Il supprime la caractéristique sélectionnée. |
| Aperçu avant impression | Il exécute l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: est dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est code et description caractéristique.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des caractéristiques qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des caractéristiques, en appuyant le touche [Recherche] présent dans la relative Ribbon, obtiendra dans la grille de résultat la liste des caractéristiques désirés.

**Insertion caractéristiques article**

Au-delà de la recherche des caractéristiques présents au système est possible d'ajouter une ou plusieurs caractéristiques. Pour faire ça il faut cliquer sur le premier record vide (ou appuyer le bouton [Insertion] sur la barre d'outils). On indique donc le code alphanumérique pour la caractéristique et sa description.
