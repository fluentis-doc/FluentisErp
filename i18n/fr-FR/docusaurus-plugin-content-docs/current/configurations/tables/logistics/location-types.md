---
title: Types d'emplacements
---

Ce tableau gère la codification des différents types d'emplacements à devoir gérer pour après être disponibles dans les formulaires compétents.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des types emplacements insérés. |
| Insertion | Permettre d'insérer un nouveau type emplacement. |
| Supprimation | Il supprime le type emplacement sélectionné et ses détails. |
| Aperçu avant impression | Il exécute l'aperçu avant impression. |
| Impression | Il imprime physiquement le rapport. |

ZONE DE FILTRE: il est dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est code et description type emplacement.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des types emplacements qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des codifications existeants, en appuyant le touche [Recherche] présent dans la relative Ribbon, obtiendra dans la grille de résultat la liste des codes recherchés.

**Insertion type emplacement**

Pour pouvoir insérer nouvelles codifications de type emplacement il faut cliquer sur le premier record vide (ou appuyer le bouton [Insertion] sur la Ribbon bar). On indique donc le code alphanumérique pour le type et sa description.
