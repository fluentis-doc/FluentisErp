---
title: Types groupe UC
sidebar_position: 11
---

Le tableau d'insertion des types groupe UC est un tableau de base pour l'utilisation des Unité de Charge. Dans le spécifique, ils viennent définis les groupes UC pour faire en façon d'identifier l'UC selon un numéro progressif, le numéro de la commande de production, le numéro de la commande fournisseur, le numéro de l'ordre de production.

Le tableau de types groupe UC est formée dans la façon suivante:

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des types groupe UC présents à système. |
| Insertion | Il permet d'insérer un nouveau type groupe UC. |
| Supprimation | Il supprime le type groupe UC sélectionné. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. En ce cas le filtre de recherche est type et description groupe UC.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des types groupe UC qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des types groupe UC, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des types groupe UC.

En plus de rechercher les types groupe UC présents à système est possible d'ajouter un ou plusieurs types groupe UC. Pour faire cela il faut cliquer sur le premier record vide (ou appuyer le bouton [insertion] sur la barre d'outils). On indique donc le type groupe UC (code alphanumérique), description groupe UC et groupe en choisissant entre:

- numéro progressif;

- commande de production;

- commande client;

- ordre de production.
