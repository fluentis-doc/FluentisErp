title: Type rentrées

Le tableau du Type rentrées s'ouvre à travers le parcours Tableaux  > Logistique  >  Type rentrées, et il sert pour établir les types de rentrées de sous-traitant, en façon telle que selon le type rentré est possible de véhiculer le matériel de et pour déterminés entrepôts.

La fenêtre, qui permet aussi de filtrer pour code et description du type rentré inséré, est formé principalement par une grille, à l'intérieur de laquelle l'utilisateur peut insérer les types rentrées à gérer.

Les colonnes visualisées dans la grille sont les suivants:

**Code**: en cette colonne l'utilisateur insère le code du type rentrée;

**Description**: en cette colonne l'utilisateur insère la description du type rentrée;

**Entr.**: en cette colonne l'utilisateur peut choisir, à travers le menu déroulant, l'entrepôt sur lequel devra être chargé le matériel produit par le sous-traitant et expédie par le même à travers une rentrée de sous-traitant créé avec le type à peine inséré;

**Description entrepôt**: en cette colonne si visualise la description de l'entrepôt inséré dans la colonne précédente;

**Cause**: en cette colonne l'utilisateur, peut choisir, à travers le menu déroulant, avec quelle cause doit être chargé le matériel produit par le sous-traitant et expédié par le même à travers une rentrée de sous-traitant créé avec le type à peine inséré;

**Description cause**: en cette colonne si visualise la description de la cause insérée dans la colonne précédente;

**Déch. entr.**: en cette colonne l'utilisateur peut choisir, à travers le menu déroulant, l'entrepôt par lequelle devra être déchargé le matériel utilisé par le sous-traitant et inséré dans le tab Matériaux de la rentrée de sous-traitant créé avec le type à peine inséré;

**Déch. entr. description**: en cette colonne se visualise la description de l'entrepôt inséré dans la colonne précédente;

**Déch. Cause**: en cette colonne l'utilisateur peut choisir, à travers le menu déroulant, avec quelle cause doit être déchargé le matériel utilisé par le sous-traitant et inséré dans le tab Matériaux de la rentrée de sous-traitant de sous-traitant créé avec le type à peine inséré;

**Déch. cause description**: en cette colonne si visualise la description de la cause insérée dans la colonne précédente.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des Types rentrées de sous-traitant insérés. |
| Insertion | Il se réfère à la procédure pour insérer un nouveau Type rentrée de sous-traitant. |
| Supprimation | Il se réfère la procédure pour supprimer un Type rentrée de sous-traitant précédemment inséré. |
| Aperçu avant impression | Il visualise un aperçu avant impression des Types rentrées de sous-traitant. |
| Impression | Il lance l'impression sur l'imprimante prédéfinie du rapport des Types rentrée de sous-traitant. |
|  |  |
