Dans le masque, il est présent un dossier de Réduction à disposition pour restaurer la situation précédent.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Rechercher les reprises de l'amortissements comptabilisées. |
| Restaurer l'entier comptabilisation | Supprimer l'entière reprise sélectionnée. |
| Restaurer | Restaurer chaque enregistrement sélectionnée. |
| Visualiser l'enregistrement comptable | Visualiser l'enregistrement extra-comptable sélectionnée. |
