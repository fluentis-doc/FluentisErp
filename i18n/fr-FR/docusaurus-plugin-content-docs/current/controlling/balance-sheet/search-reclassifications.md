---
title: Rechercher le reclassement
sidebar_position: 4
---

La procédure Rechercher le reclassement a été désigné par permettre à l'utilisateur de rechercher facilement les reclassements du bilan crée précédemment avec le but de les visualiser, les modifier et éventuellement les supprimer. Par la Rechercher le reclassement est possible de faire toutes les opérations nécessaires dans le document, après la présence des droits de l'utilisateur nécessaires.

La procédure de recherche devises est constitué dans le suivant façon;

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des reclassements insérés. |
| Nouveau | Il se réfère à la procédure pour insérer un nouveau reclassement. |
| Modifier | Il se réfère à la procédure pourmodifier un reclassement précédemment insérés. |
| Visualiser | Il se réfère à la procédure pour visualiser un reclassement précédemment inséré. |
| Supprimer | Il se réfère à la procédure pour supprimer un reclassement précédemment inséré. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Imprimer les modéles de reclassement | Il se réfère à la procédure pourimprimer les modèles de reclassement. |
|  Imprimer valorisation reclassements | Il se réfère à la procédure pour imprimer une valorisation reclassements. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Comparaison reclassements | Il se réfère à la procédure pour créer une comparaison entre les reclassements. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des reclassements qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des reclassements, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des reclassements désirés.
