---
title: Rechercher les immobilisations
sidebar_position: 6
---

La procédure de Rechercher les immobilisations a été désignée pour permettre à l'utilisateur de rechercher facilement les immobilisations précédemment insérées avec le but de les visualiser, les modifier et éventuellement les supprimer. À partir de Rechercher les immobilisations sont possibles de faire toutes les opérations nécessaires dans le milieu du document, après la présence des droits de l'utilisateur nécessaires.

La procédure de Rechercher les immobilisations est formée dans la façon suivante:

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher les immobilisations | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des immobilisations insérés. |
| Nouvelle immobilisation | Il se réfère à la procédure pour insérer un nouvelle immobilisation. |
| Modifier l'immobilisation | Il se réfère à la procédure pour modifier une immobilisation précédemment insérée. |
| Visualiser l'immobilisation | Il se réfère à la procédure pour visualiser une immobilisation précédemment insérée. |
| Supprimer l'immobilisation | Il exécute la procédure pour supprimer une immobilisation précédemment insérée. |
| Subdiviser l'immobilisation | Le bouton, actif seul si ne sont pas sélectionnées plus immobilisations, permet d'ouvrir un masque de subdivision de l'immobilisation si ce n'a pas déjà été amorti. Voir l'application en-tête A. |
| Transmettre l'immobilisation | Il exécute la procédure pour créer une machine du formulaire véhicules. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection ou une recherche sur le total des immobilisations insérées. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des immobilisations qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des immobilisations, en appuyant le touche [Rechercher immobilisations] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des immobilisations désirées.

APPLICATION EN-TÊTE:

A: Subdiviser l'immobilisation. Dans le masque, définir le numéro de dossier à ouvrir pour chaque division/catégorie immobilisation, confirmer avec 'Oui' pour exécuter l'opération ou annuler avec 'Non'.
