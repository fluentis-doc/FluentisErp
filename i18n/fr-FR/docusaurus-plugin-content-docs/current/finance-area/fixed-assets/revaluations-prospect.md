sidebar_position: 4

Le masque d'impression de l'Aperçu de réévaluation permet dedéfinir les habituels filtres de recherche à l'extraction des données.

Le rapport permet de résumer les données de détail des réévaluations associées aux diverses immobilisations, selon le code de regroupement établi dans le tableau catégories immobilisations.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression physique du document. |
