title: Intrastat1
sidebar_position: 2
---

La procédure Rechercher déclarations Intrastat1 est formée dans la façon suivante:

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des déclarations Intrastat1 précédemment insérées. Il y a la possibilité de pouvoir rechercher pour type résumé, période, année et  numéro de présentation. |
| Création nouveau | Il se réfère à la procédure pour insérer une nouvelle déclaration Intrastat1. |
| Modifier l'Intrastat sélectionné | Il se réfère à la procédure pour modifier une nouvelle déclaration Intrastat1 précédemment insérée. |
| Supprimer | Il se réfère à la procédure pour supprimer une nouvelle déclaration Intrastat1 précédemment insérée. |
| Exporter sur le fichier | Bouton pour la création et contextuel exportation du fichier télématique, avec l'enregistrement du fichier en une Directory choisi par l'utilisateur. |
| Importer par le fichier | Bouton pour l'importation, par le fichier télématique, d'une déclaration Intrastat1. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document Manager | Bouton pour la liaison à la gestion de document. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des déclarations Intrastat1 qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des déclarations, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des déclarations désirées.
