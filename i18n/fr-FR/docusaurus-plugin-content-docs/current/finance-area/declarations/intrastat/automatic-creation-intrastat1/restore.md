---
title: Restauration
sidebar_position: 4
---

Comme dernier dossier il est présent un dossier de restauration: il est possible d'utiliser ce pour annuler les insertions dans les sections bis/ter/quater, mais aussi la supprimation directe qui à partir de ces ira à rouvrir le relatif document à insérer dans le modèle.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour supprimer les factures du type intracommunautaire sélectionnées. |
| Restaurer Intrastat | Bouton pour restaurer l'entière déclaration Intrastat. |
| Restaurer la facture | Bouton pour restaurer une seule facture présente dans la déclaration Intrastat. |

ZONE DE FILTRE: la zone de filtre est celle dédiée à recevoir la liste des possibles types de données grâce à lesquelles est possible d'effectuer une sélection. Les filtres sont typiquement en condition 'AND' et est possible de spécifier plusieurs critères des filtres en même temps.

GRILLE DE RÉSULTAT: la grille de résultat représente la liste des déclarations Intrastat qui correspondent aux données de filtre surspécifie. L'utilisateur, après avoir spécifié les valeurs par lesquelles désire obtenir une recherche des déclarations Intrastat, en appuyant le touche [Recherche] présent dans la rélative Ribbon, obtiendra dans la grille de résultat la liste des déclarations Intrastat désirées.
