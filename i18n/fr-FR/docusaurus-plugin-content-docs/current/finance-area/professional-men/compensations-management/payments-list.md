---
title: Liste des paiements
sidebar_position: 3
---

Le tabulateur 'Liste des paiements' présente, dans la grille supérieure, le paramétrage par défaut des paiements ainsi comme insérés dans le registre agent ou fournisseur de référence: dans la section en dessous, au contraire, est présent la liste des échéances à payer. Pour le formulaire receveurs est obligatoire qui ait été chargée au moins une échéances, autrement ne sera pas possiblede payer la rémunération (et relever la retenue donc) ni comptabiliser la rémunération.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Nouveau paiement | Il insère le curseur dans l'insertion d'une nouvelle ligne de paiement dans la première grille de ce tab. |
| Supprimer les paiements | Il supprime les paiements sélectionnés dans la première grille de ce tab. |
| Nouvelles échéances | Il insère le curseur dans l'insertion d'une nouvelle ligne échéances dans la deuxième grille de ce tab. |
| Supprimer les échéances | Il supprime les échéances sélectionnées dans la deuxième grille de ce tab. |
