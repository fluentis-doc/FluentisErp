---
title: Modèle F24
sidebar_position: 1
---

La procédure est dédiée à la création du tracé ministériel de la déclaration F24, pour son envoi télématique. Ce modèle est dédié à la communication des diverses compensations versées mensuellement par l'entreprise.

Le paramétrage de base est de coder, à l'intérieur du formulaire employé de la zone administration, le sujet signataire de la déclaration. Il se réfère en particulier à indiquer le code charge et les références de résidence, domicile, code fiscal, prénom et nom.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des modèles insérés. |
| Nouveau | Il se réfère à la procédure pour insérer un nouveau modèle. |
| Modifier | Il se réfère à la procédure pour modifier un modèle précédemment inséré. |
| Visualisation | Il se réfère à la procédure pour visualiser un modèle précédemment inséré. |
| Supprimer | Il exécute la procédure pour supprimer un modèle précédemment inséré. |
