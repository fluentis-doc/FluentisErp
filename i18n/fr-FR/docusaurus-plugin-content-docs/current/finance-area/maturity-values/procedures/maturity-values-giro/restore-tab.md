sidebar_position: 2

Il est présent un dossier final 'Restauration' pour visualiser en aperçu avant impression l'enregistrement effectué ou aussi pour les annuler à travers le bouton.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour rechercher le virement des comptes effectués précédemment. |
| Restaurer le virement des comptes | Bouton pour visualiser et/ou restaurer le virement des comptes. |
