---
title: Résultats
sidebar_position: 4
---

Dans le tabulateur 'Résultats' il est possible de visualiser et supprimer le résultat des calculs effectués, avec la possibilité de pouvoir filtrer l'interrogation pour le compte/sous-compte et type de paiement.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher les résultats | Le bouton recherche les résultats de calcul exécutés. |
| Supprimer les résultats | Le bouton supprime les résultats de calcul exécutés. |
| Supprimer les détails sélectionnés | Le bouton supprimer seul les détails sélectionnés d'un déterminé calcul. |
