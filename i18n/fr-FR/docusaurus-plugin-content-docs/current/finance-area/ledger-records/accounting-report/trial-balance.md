---
title: Bilan de vérification
sidebar_position: 1
---

L'impression du Bilan de vérification visualise une liste des sous-comptes avec les relatifs totals débit/crédit et les soldes: il est possible de redemander tous les sous-comptes, seul ces animés, ou les animés avec le solde différent de zéro, avec la description du sous-compte ou avec la deuxième description.

Il est disponible aussi un filtre pour code et les flag d'exclusion des détails des registres.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Aperçu avant impression | Bouton pour exécuter l'aperçu avant impression. |
| Impression | Bouton pour exécuter l'impression physique du document. |
