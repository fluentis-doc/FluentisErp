Le dossier 'Commandes' est présent seul si la cause a le flag 'Commandes' établi.

En ce tabulateur seront assignés aux diverses commandes de vente les donnés économiques relevés dans l'enregistrement de comptabilité: automatiquement ils seront assignés tous à la commande insérée dans l'en-tête, mais l'utilisateur pourra subdiviser les montants en façon différente manuellement directement en cette section.



Ils ne sont pas prévus automatismes dans l'insertion manuelle des montants, ni la gamme de dates d'exercice économique, ni les contrôles du rapprochement des montants avec la comptabilité générale.



RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Enregistrer | Bouton pour enregistrer l'enregistrement. |
| Nouvel enregistrement | Bouton pour créer un nouvel enregistrement. |
| Parmètres d'insertion | Il se réfère aux paramètres d'insertion du livre-journal. |
| Ouvrir le registre | Bouton pour rappeler la gestion du registre du sous-compte sélectionné, ou en absence, la recherche d'un autre registre. |
| Supprimer les commandes sélectionnées. | Bouton pour supprimer lees commandes sélectionnées. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Document Manager | Bouton pour la liaison à la gestion documentale. |
