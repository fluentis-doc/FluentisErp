---
title: Filtre
sidebar_position: 2
---

En ce tab il est possible d'établir les critères de recherche des ordres planifiés et de production qu'on veut visualiser graphiquement dans l'Histogramme qui se trouve dans le deuxième tab. Il existe 3 modalités différentes de paramétrages des filtres; en utilisant en effet le bouton radio que se trouve dans la partie supérieure de la fenêtre, il est possible de décider d'utiliser les Filtres (et en ce cas s'active le carré posé dans la partie inférieure de la fenêtre), le bouton radio 'Centres de travail' (en ce cas s'active seul la grille où l'utilisateur peut insérer les Centres de travail à visualiser dans le graphique), le radio bouton 'Tous les centres de travail', qui désactive les deux sections en permettantde visualiser en automatique les graphiques pour tous les CDT présents dans la base de données pour la société où on est en train de travailler.

L'utilisateur peut utiliser les suivants filtres:

**Type demandée**: en cette section il est possible de décider de visualiser dans le graphique les ordres de planifiés et de production où sont utilisés tous les centres de travail filtrés, où, entre ces filtrés, seul ces avec excès d'occupation (c'est-à-dire ces qui presentant certians journées de travail où la quantité de travail disponible est inférieur à la quantité de travail demandée);

**De date...À date...**: en ces 2 champs il est possible d'établir pour quelle date à quelle date on veut visualiser la situation des ordres planifiés et de production. Le graphique viendra dimensionné selon la gamme des dates insérées.

**Année/Ordre plan.**: en ces 2 champs ils viennent établis l'année et le numéro de l'ordre planifié qu'on veut visualiser dans la graphique. En activant ce bouton radio, l'utilisateur a à disposition aussi le filtre pour type ordre planifié, au contraire si désactivent automatiquement tous les autres filtres contenus dans la même section;

**Année/Ordre prod.**: en ces 2 champs ils viennent établis l'année et le numéro de l'ordre de production qu'on veut visualiser dans le graphique. En activant ce bouton radio, l'utilisateur a à disposition aussi le filtre pour le Type ordre de production, au contraire si désactivent eutomatiquement tous les autres filtres contenus dans la même section;

**Année/Comm.prod.**: en ces 2 champs ils viennent établis l'année et le numéro de la commande de production qu'on veut visualiser dans le graphique. En activant ce bouton radio, l'utilisateur a à disposition tous les filtres contenus dans la même section, aussi les filtres pour article dont est détaillée en suite:

**Classe/Code/Descrip.**: le premier menu déroulant permet d'établir la classe de l'article, avec double click dans le deuxième champ s'ouvre un 'Aide articles' à travers lequelle est possible de sélectionner chaque article (code, description);

**Ordres prod.**: les 3 flag ici présents permettent de décider de visualiser dans le graphique les ordres de production lancés, exécutifs et/ou exécutés;

**Ordre de maintenance**: avec ce flag il est possible de décider de visualiser dans le graphique aussi les ordres planifié de maintenance.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Il se réfère à la procédure pour aller directement au tab 'Histogramme' en visualisant les données et les graphiques basés dans les filtres établis dans le tab 'Filtre'. |
