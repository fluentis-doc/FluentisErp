---
title: Filtre éléments de Gantt
sidebar_position: 2
---

En ce tab est possible d'établir les critères de recherche des éléments qu'on veut visualiser graphiquement dans le Gantt qui si trouve dans le deuxième tab.

Les filtres à disposition par l'utilisateur sont les suivants:

**Client**: en ces 3 champs ils viennent spécifier les relatives données du client: 'Compte', 'Sous-compte", "Nom du client', avec un double click dans le premier champ est possible d'ouvrir un 'Aide comptes' par lequel pouvoir sélectionner le client pour lequel filtrer;

**Article**: le premier menu déroulant permet d'établir la classe de l'article, avec un double click dans le deuxième champ s'ouvre un 'Aide articles' à travers lequel est possible de sélectionner chaque article (code es description);

**Lieu production**: avec ce menu déroulant il est possible de filtrer les commandes concernées par un déterminé lieu productif;

**Centre de travail**: ce menu déroulant il visualise le code du Centre de travail qu'on veut filtrer, et donc trouvera tous les ordres planifiés et/ou de production qui ont à leur intérieur au moins une phase avec ce CDT;

**Groupe**: en ces champs il est possible de filtrer pour Groupe, et donc visualiser tous les ordres planifiés et/ou de production qui font partie de ce groupe;

**De date...à date...**: en ces champs il est possible d'établir une gamme de dates à l'intérieur des quelles trouver les commandes de production;

**Gantt pour CP**: avec ce flag l'utilisateur peut décider si visualiser dans la liste les commandes de production. En l'activant, dans la liste ne viendra pas présenté le détail de la commande de production;

**Gantt pour CP plan.**: avec ce flag l'utilisateur peut décider si visualiser dans la liste les ordres planifiés. En l'activant, dans la liste ne viendra pas présenté le détail de la commande de production;

**Considérer les documents exécutés**: avec ce flag l'utilisateur peut décider si visualiser dans la liste aussi les documents déjà exécutés;

**Ordres de maintenance**: avec ce flag l'utilisateur peut décider si visualiser dans la liste les ordres de maintenance;

**Commande de production**: avec ce grille l'utilisateur a la possibilité de filtrer, en exerçant un double click dans la ligne libre de la grille, les commandes de production.

Après avoir établi les filtres sur, l'utilisateur peut cliquer dans le touche 'Rechercher' posé dans la Ribbon bar et donc obtenir, dans la section en bas de ce tab, la liste des documents recherchés, dans chaque des quelles l'utilisateur peut aposer un flag en correspondance de la case à cocher, après quoi, en laçant le bouton 'Création Gantt', il peurra décider de visualiser tous les documents sélectionnés avec la case à cocher dans le tab suivant appelé 'Visualiser Gantt'.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Rechercher | Bouton pour appliquer les filtres de recherche à l'intérieur de la base de données des documents liés à la production. |
| Création Gantt | Il se réfère à la procédure pour engendrer le Gantt à visualiser dans le tab suivant, sur la base aux menus déroulant activés dans les documents visualisés dans la section en bas du tab 'Filtre éléments de Gantt'. |
| Sélectionner tout | Apposer le menu déroulant dans tous les documents filtrés et donc présents dans la section en bas du tab 'Filtre éléments de Gantt'. |
