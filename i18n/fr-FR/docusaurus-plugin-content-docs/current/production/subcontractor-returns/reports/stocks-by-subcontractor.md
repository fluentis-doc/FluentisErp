---
title: Stocks pour le sous-traitant
sidebar_position: 3
---

Cette fenêtre permet d'imprimer les stocks présents auprès des différents sous-traitants, et di trouve dans le parcours Sous-traitant  > Rentrées > Impressions > Stocks pour les sous-traitants. En cette boîte de dialogue d'impression existe l'obligation de filtrer pour les sous-traitants; le détail qui vient donné concerne le stock initial, les recettes, dépenses et le stock final pour chaque article et pour chaque entrepôt. Elle est formée par 7 différentes sections:

**Filtres**: en cette section il est possible de filtrer pour l'entrepôt et la cause d'entrepôt, pour la classe, code, description et variante de l'article, pour type article, caractéristique, catégorie produite et option, pour la localisation et lot de l'article, pour le sous-traitant, pour la commande de vente. En autre il est possible établir une date dans le champ 'Inventaire au', à laquelle viendra calculée le stock;

**Regroupement pour**: en cette section il est possible de décider si visualiser les données regroupées pour l'article ou pour l'entrepôt;

**Détail pour**: en cette section il est possible de décider si obtenir éventuellement aussi le détail pour Variante, actuellement est l'unique disponible en ce rapport d'impression;

**Type impression**: en cette section il est possible décider d'obtenir une impression synthétique et analytique. L'impression analytique offre aussi un détail des éventuels bons de livraison de rentrées déjà effectuées pour les livraisons partielles;

**Ordonnance**: en cette section il est possible de décider que type d'ordonnance donner au rapport d'impression;

**Articles avec stock**: en cette section il est possible de décider si visualiser les autres articles avec stock positif, négatif ou nul. Les flag sont par défaut tout habilités;

**Paramètres d'impression**: en cette section il est possible d'établir le numéro des copies à imprimer et le listing d'impression à utiliser.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Aperçu avant impression | Il visualise l'aperçu avant impression sur la base des filtres établis. |
| Impression | Il lance l'impression du rapport sur la base des filtres établi dans l'imprimante prédéfinie. |
