En ce tab il vient donnée la possibilité d'établir pour la ligne sélectionnée dans la grille, les lots à décharger par l'entrepôt avec la quantité de l'article prélevée par l'envoi au sous-traitant, si se réfère à plusieurs lots différents prélevés avec la même ligne de bon de livraison.

La grille des lots permet à l'utilisateur de prélèver le lot en puisant par l''Aide lots', qui filtre entre les lots disponibles de l'article sélectionné dans la grille supérieur. Les informations qui viennent rapportées dans la grille lots sont le type code lot, code lot, lot fournisseur, type lot, date début et date écheance, quantité du lot, localisation et les données du fournisseur du lot (compte, sous-compte et raison sociale). La somme des quantité des divers lots devra correspondre exactement à la quantité livrée par la ligne article.

RIBBON BAR: il représente le menu de la forme en objet, à savoir la zone dans laquelle est possible d'effectuer des actions. La liste des fonctionnalités possibles est représentée de suite:



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Insérer | Il se réfère à la procédure pour insérer un nouveau Bon de livraison: l'application enregistre automatiquement la bon de livraison à peine insérée et enleve tous les champs en rendant la fenêtre disponible par l'insertion d'un nouveau bon de livraison. |
| Enregistrer | Bouton pour enregistrer les modifications effectuées dans lu bon de livraison. |
| Livraisons prévues | Il se réfère à la procédure qui explose la fenêtre d'Exécution par l'ordre et chi permet de transférer par les commandes de sous-traitant les articles dans le bon de livraison. |
| Exécution commandes | Il se réfère à la procédure pour transférer les articles par les commandes de vente. |
| Insérer paiement | Il se réfère à la procédure pour insérer une nouvelle ligne de paiement dans la grille 'Paiements'. |
| Supprimer les paiements | Il se réfère à la procédure pour supprimer une ligne de paiement dans la grille 'Paiements'. |
| Insérer matériel | Il se réfère à la procédure pour insérer un nouveau matérieul dans la grille des articles du tab 'Livraisons'. |
| Supprimer le matériel | Il se réfère à la procédure par supprimer un matériel par la grille des articles du tab 'Livraison'. |
| Aperçu avant impression | Il visualise l'aperçu avant impression du bon de livraison. |
| Impression | Il lance l'impresison du rapport du bon de livraison de livraison dans l'imprimante prédéfinie. |
| Documents | Il permet d'accéder à la Gestion documentel et de visualiser les éventuels documents jooindrés au bon de livraison. |
