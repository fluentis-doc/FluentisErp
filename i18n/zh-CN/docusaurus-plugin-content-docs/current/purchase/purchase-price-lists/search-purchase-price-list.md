---
title: 搜索供应商价目表
sidebar_position: 2
---

搜索供应商价目表的过程为容易地搜索已插入的价目表而设计的。通过本过程用户可以 查看、编辑以及删除价目表或者插入新的价目表。相当的权利是必须的。

通过筛选器用户可以查看：单一的价目表（例如按照供应商搜索）、多个价目表（例如按照日期搜索）、关于某个供应商已插入的价目表（例如按账户/明细账目搜索）。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索价目表 | 用于在数据库中搜索价目表 |
| 新的价目表 | 用于生成新的价目表 |
| 编辑价目表 | 用于编辑已插入的价目表 |
| 查看价目表 | 用于查看已插入的价目表。不能编辑价目表 |
| 删除价目表 | 用于删除已插入的价目表 |
| 转换货币 | 用于将价目表的价格转换到其他货币。 |
| 复制价目表 | 从参考的价目表生成新的价目表。在出现的表格中，用户可以改变供应商、有效日期以及货币 |

筛选区域：筛选区包含使用户选择的数据清单。典型地，筛选器的状态是“AND”，同时用户可以选择多种筛选的标准。

结果网格：结果网格表示相当于已确定的筛选器数据供应商价目表的清单。搜索参数确定了以后，通过功能导航栏的搜索按钮，用户能够获得需要的供应商价目表清单。
