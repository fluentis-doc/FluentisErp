插入供应商价目表需要供应商的代码（账户/明细账目）。通过字段的帮助用户可以选择供应商。用户得制定‘开始有效日期'以及货币。结束有效日期不是必须的。如果用户要制定某个推广活动或者价目表的到期，本字段可以是有用的。用户也可以插入销售作业单，票据以及登记的折扣。

在项目网格中用户可以直接地插入项目（指定分类和代码）或者通过‘帮助项目'插入项目。

 

**项目变量**：如果除了项目代码以外变量也是某个项目的特性；

**供应商的项目**：如果供应商项目代码与公司项目代码不同；

**供应商项目的变体代码**：如果除了供应商项目代码变量也是供应商的特性；

**计量单位**：可以是项目的管理计量单位或者替代的计量单位；

**项目的品牌；**

**数量**（表示的数量等于1）是价格的参考数量。如果数量与 1不同的话，计算价格的时候系统按照价目表的数量；

**价格**：按照价目表货币。



| RibbonFunction | RibbonMeaning |
| --- | --- |

| 新的项目 | 用于插入新的项目 |
| 保存项目 | 用于保存在价目表中的项目 |
| 删除项目 | 用于删除在价目表中的项目 |
| 更新价格 | 用于更新价目表的价格，提高价格的百分比1% |
| 删除折扣 | 用于删除插入到项目性的折扣 |



| RibbonFunction | RibbonMeaning |

| 文档经理 | 管理与在使用中掩码有关的文档档案 |

项目筛选器的区域：筛选区包含使用户选择的数据清单。典型地，筛选器的状态是“AND”，同时用户可以选择多种筛选的标准。

项目网格：结果网格表示相当于已确定的筛选器数据的项目清单。搜索参数确定了以后，通过功能导航栏的搜索按钮，或者“搜索项目”按钮，用户能够获得需要的价目表清单。
