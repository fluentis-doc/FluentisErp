功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 货单记账 | 货单记账 |
|  [货单加装到仓库中](/docs/purchase/purchase-delivery-note/procedures/warehouse-purchase-delivery-note-record/load-delivery-notes-on-warehouse)  | 本过程将已插入的货单装到管理中。通过筛选器，用户可以寻找还没装的货单。在参数表格中，用户设置仓库和模板，然后按“加装货单”按钮 |
|  [从货物接收生成货单](/docs/purchase/purchase-delivery-note/procedures/create-delivery-note-from-goods-receipt/create-delivery-note-from-goods-receipt)  | 本程序从货物接收生成采购货单。通过筛选器，用户可以寻找未货单的加装文档。在参数表格中，用户设置货单类型、增值税、采购营业额然后按“从货物接收生成货单”按钮。 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 采购货单 | 按照已制定的筛选器标准（货单类型、编号、日期、供应商等）打印采购文档 |
| 货单清单 | 按照已制定的筛选器标准（货单类型、编号、日期、供应商等）），打印某个货单清单 |
