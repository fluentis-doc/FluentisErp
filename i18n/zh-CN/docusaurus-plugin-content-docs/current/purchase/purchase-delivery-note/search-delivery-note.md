---
title: 搜索采购货单
sidebar_position: 2
---

搜索采购货单的程序为了容易地搜索货单而设计的。通过本程序用户可以查看、编辑、删除以及插入新的货单。用户当然需要相当的权利。

通过筛选器，用户可以查看：单一的文档（例如按文档号码搜索），多个文档（例如按日期搜索）、在一个年度内插入的文档（例如按年份搜索）、对某个供应商插入的文档（例如按账户/明细账目搜索）。而且，通过“搜索项目”按钮，用户可以搜索包括某个项目或者某个项目类型的文档。

 功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索货单 | 用于将货单搜索于数据库 |
| 新的送货单 | 用于生成新的货单 |
| 编辑货单 | 用于编辑已插入的货单 |
| 查看货单 | 用于查看已插入的货单。通过本按钮，用户不能编辑货单 |
| 删除送货单 | 用于删除已删除的货单 |
| 转换 | 将某个文档的价格转换到别的货币 |
| 冲销送货单 | 用于从已插入的本来货单创建注销稳当。用户执行注销之后，一个包括本来货单的所有项目的表格出现。用户得选择项目的转动模板，选择注销标记以及得生成的货单类型。 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

**结果网格**：结果网格表示相当于已确定的筛选器数据的采购货单的清单。搜索参数确定了以后，通过功能导航栏的搜索按钮，用户能够获得需要的货单清单。
