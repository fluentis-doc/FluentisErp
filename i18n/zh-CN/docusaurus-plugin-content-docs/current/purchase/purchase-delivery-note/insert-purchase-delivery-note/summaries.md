摘要表示完全文档的主要数据

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 删除最终折扣 | 在相当的网格中，删除折扣 |
| 删除费用 | 在相当的网格中，删除费用 |
| 预览 | 用于预览报告 |
| 打印 | 用于打印文档 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 管理与在使用中的掩码联系的文档档案 |

**最终折扣**：用户可以插入以百分比表示的折扣。在本部分中的折扣按照文档的每个项目计算的。 

**增值税摘要**：为每个增值税编码，系统表示文档的增值税摘要。 

**费用**：.表示属于供应商登记的“费用/折扣”表格中的费用。用户也可以插入与在使用中文档有关的费用。 

**最终票据**：是自由的字段。用户可以直接地填写本字段或者通过“帮助编码票据”。 

**文档总计** 

**项目毛额**：表示所有项目价值的总计。**应用的折扣总计**：表示折扣的总计，不包括最终折扣。**项目净额**：项目毛额–应用的折扣总计

 **最终折扣的总计**：表示最终折扣的价值

**未记录费用**：表示已插入费用价值，作为“费用”类型的项目。

**记录的费用**：表示在费用网格中的费用价值。

**应纳税**：项目净额–最终折扣+未记录的费用总计+记录费用总计。

**增值税**：表示在增值税摘要中的价值总计总计：应纳税+增值税
