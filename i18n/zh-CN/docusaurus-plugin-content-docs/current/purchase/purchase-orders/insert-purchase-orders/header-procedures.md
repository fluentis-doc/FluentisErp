---
title: 从采购作业单恢复项目
sidebar_position: 5
---

从采购作业单恢复项目的过程使作业单的履行。系统表示未履行的作业单，不强制履行的作业单以及部分地履行的。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索作业单 | 搜索未履行作业单和不强制履行的作业单 |
| 阻止作业单 | 用于阻止作业单 |
| 解冻作业单 | 用于解冻为履行的作业单 |
| 项目转运 | 用于将选中作业单转运到订单 |
| 转向列表上层 | 转向列表上层 |
| 在列表中向下一级 | 在列表中向下一级 |
| 强制履行作业单 | 转运到订单的时候，强制履行作业单 |
| 在视图中打开作业单 | 在视图中打开作业单 |
| 查看选中作业但的状态 | 用于查看选中作业单的履行状态 |
| 图列 | 在订单标头内指定交货申请日期和/或者估计交货日期之后，用户可以将这些日期替代于已插入的项目行 |
| 在列表中的对象位置 | 在订单标头内指定交货申请日期和/或者估计交货日期之后，用户可以将这些日期替代于已插入的项目行 |
| 显示/隐藏对象父级 | 显示/隐藏选中对象的父级 |
| 显示/隐藏作业单的面板 | 显示/隐藏作业单的面板 |
| 选择所有 | 在列表的查看面板中选择所有的对象 |
| 删除登记设置 | 用于删除登记设置 |
