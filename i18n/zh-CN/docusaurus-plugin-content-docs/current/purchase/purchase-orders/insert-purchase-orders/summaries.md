摘要表示完全文档的主要数据：

最终折扣用户可以插入以百分比表示的折扣。插入到本部分的折扣应用于每个文档的项目。

增值税汇总表示文档的增值税汇总，对每个增值税代码

费用表示供应商登记中的费用，在“费用/折扣”表格中。而且用户也可以插入关于在使用中文档的费用。

 

最终票据是自由的字段。通过“帮助编码的票据”用户可以填写本字段

订单状态表示等于订单标头的订单状态，包括履行日期以及确认订单的日期。

文档总额**项目毛额**：表示项目价值的总额；

**应用的折扣总额**：表示折扣的总额，不包括最终折扣；

**项目净额**：项目毛额–应用的折扣总额；

**最终折扣的总额**：表示最终折扣的价值；

**未记录的费用**：表示以“费用”项目插入的费用价值；

**记录的费用**：表示插入到“费用”网格的费用价值；

**应纳税**：项目净额– 最终折扣+为记录的费用总额+记录的费用总额；

**税**：是增值税摘要中的价值总额；

**总额**：应纳税+增值税。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存已完成的操作 |
| 删除最终折扣 | 用于删除在相当行中的折扣 |
| 删除费用 | 用于删除在相当网格中的费用 |
| 预览 | 用于预览报告 |
| 打印 | 用于打印文档 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 管理与在使用中的掩码联系的文档档案 |
