---
title: 搜索采购订单
sidebar_position: 2
---

搜索采购订单的程序为容易地搜索已插入的订单而设计的。通过本过程用户可以查看、编辑以及删除采购订单。在‘搜索采购订单‘中用户可以完成关于某个文档的所有必须操作。相当的权利是必须的。

搜索采购订单的程序由下列的特点组成的。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于采购订单的数据库 |
| 新的 | 用于插入新的采购订单 |
| 编辑 | 用于编辑已插入的采购订单 |
| 查看 | 用于查看已插入的采购订单 |
| 删除 | 用于删除已插入的采购订单 |
| 履行属性 | 用于生成关于某个选中的采购订单的注销文档 |
| 复制采购订单 | 用于生成与在下面网格中选中的某个采购订单很象的文档 |
| 货币转换 | 用于将选中的文档转换到合适地制定的“货币” |

筛选区域：筛选区包含使用户选择的数据清单。典型地，筛选器的状态是“AND”，同时用户可以选择多种筛选的标准。通过[搜索项目>>]按钮，筛选器的数据给用户搜索单一项目数据的可能，在采购订单中插入的。

结果网格：表示相当于已确定的筛选器数据采购订单的清单。搜索参数确定了以后，通过功能导航栏的搜索按钮，用户能够获得需要的采购订单清单。
