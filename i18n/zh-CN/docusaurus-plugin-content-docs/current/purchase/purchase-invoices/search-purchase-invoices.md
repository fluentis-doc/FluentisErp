---
title: 搜索采购发票
sidebar_position: 2
---

搜索采购发票的程序为容易地搜索已插入的发票。通过本方式用户可以查看，编辑，删除或者插入新的发票。相当的权利是必须的。

通过筛选器用户可以查看：单一的文档（例如按文档编号搜索），更多文档（例如按日期搜索），在一年之内被插入的文档（例如按年份搜索），关于某个供应商被插入的文档（例如按账户/明细账目搜索）。而且通过“搜索项目”用户可以搜索包括某个项目/项目类型的文档。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索发票 | 用于将发票搜索于数据库中 |
| 新的发票 | 用于生成新的发票 |
| 编辑发票 | 用于编辑已插入的发票 |
| 查看发票 | 用于查看已插入的发票。不能编辑发票 |
| 删除发票 | 用于删除已插入的发票 |
| 复制发票 | 用于复制某个发票。复制某个文档，表示本来文档的所有条件但是包括当前的数据 |
| 注销发票 | 从本原发票生成注销文档。用户注销发票的时候，一个包括原发票所有项目的表格出现。用户得选择项目的转移模板，选择“注销”标记以及得生成的发票类型。用户得将注销发票与原发票的联系直接地插入到参考的“发票参考”和“类型”字段。 |
| 转换 | 用于将某个文档的价格转换其他的货币 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 采购发票 | 用于按照筛选器标准（发票类型，编号，日期，供应商等）打印采购发票 |
| 采购发票的列表 | 用于按照筛选器标准（发票类型，编号，日期，供应商等）打印发票的列表。订单按照供应商，编号，发票号码，发票类型，插入日期 |
| 采购营业额 | 用于按照筛选器标准打印采购发票（发票类型，编号，日期，供应商）。订单按照项目，项目类型，国家，供应商 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [采购发票的参数](/docs/configurations/parameters/purchase/purchase-invoices-parameters)  | 通过本程序用户可以配置同时查看采购发票的参数 |
|  [采购发票的会计记录](/docs/purchase/purchase-invoices/accounting/purchase-invoices-accounting)  | 用于记录采购发票 |

**筛选区域**：筛选区包含使用户选择的数据清单。典型地，筛选器的状态是“AND”，同时用户可以选择多种筛选的标准。

**结果网格**：表示相当于已指定的筛选条件的发票清单。首先用户指定对记录进行搜索的条件，然后在功能导航栏中单击于“搜索”，最后在网格中用户可以查看需要的发票清单。
