**行类型**使用户选择有不同特点的项目。

**编码的项目**：包括在登记中编码的项目。在分析会计和仓库中可被记录；

**未编码的项目**：包括明说的项目。用户可以将这些项目记录到分析记录可是不能将它们转移到仓库中；

**费用项目**：包括编码以及未编码的项目，在文档摘要分开地总结。如果费用项目编码的以及有收税功能用户可以将它转移到仓库。如果项目未编码或者没有收税功能用户不能将它转移到仓库。

**票据项目**：是简单的票据，不影响会计和仓库。

**免费项目**：为收税/仓库，视为编码/未编码的项目。作为免费项目，在发票摘要中是分开地记录。按照“增值税反汇票”标记，系统表示客户可/不可付的免费项目总额

因为“编码项目”行类型自动地被选择，系统将这些数据，项目分类和项目编号自动地插入本部分。

**代码**：按照“项目条形码”，插入到“项目”，“条形码”表格。插入本代码的时候，系统自动地利用有关的项目以及项目登记的所有数据。

用户可以直接地或者通过“帮助项目”（双击于黄色部分）插入分类和编号。黄色部分表示项目登记中的所有数据。项目被插入之后，系统按照项目登记自动地检索项目的明说。如果项目有变体的话，用户在“变体”组合框得选择需要的遍体。系统自动地生成批次，或者插入批次的代码。

 

**变量**：表示选中项目的变量。选择变体对具有本配置的项目很有用。这些项目的价格与标准的价格可不同。因此，包括变体的项目价格与未变体的项目价格可不同。这个可以需要在参考销售价目表中的项目变体管理。

**计量单位**：表示项目的主要计量单位。用户也可以选择别的计量单位。

**数量**：表示主要计量单位的数量。用户可以直接地计量单位或者按照采购文档（例如采购订单）

**价格**：表示供应商登记的价目表价格。“总额”表格表示项目的参考价目表。如果供应商登记包括更多价目表（按照客户/供应商登记）采购文档可以包括属于不同价目表的项目。双击于“价目表”字段（总计表格），用户可以选择与默认提供不同的提供。插入的项目价格按照本提供。如果价目表没有，表示的数据是是项目登记的最终成本。

**增值税**：供应商登记中的“增值税”字段中的数据是最重要的。

**采购营业额**：表示项目登记表示的“一般数据”数据。如果本数据没有，记账的时候系统考虑“默认抵消的成本/利润”字段的价值。

**替代的计量单位**：表示项目登记的替代计量单位表格中的计量单位。“默认”标记设置的。

**替代数量**：表示替代计量单位的数量，系统将管理数量乘/除以在项目登记中设置的换算因素。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |

| 新的项目 | 用于插入新的项目 |
| 得接收的发票注销 | 用于注销得接收的发票 |
| 编辑营业额 | 用于编辑采购营业额的类型 |
| 删除项目 | 用于删除已插入的项目 |
| 展开/缩小项目网格 | 用于查看/缩小“总计”“数据”“变体”“批次”的表格 |
| 从条形码导入 | 用于从条形码导入 |



| RibbonFunction | RibbonMeaning |
