---
title: 从采购订单的履行
sidebar_position: 2
---

通过本程序用户可以从某个采购订单履行某个发票。

条件：订单和发票的供应商得同一。“已打印”和“确认订单日期”标记得激活。

本程序利用订单的所有数据，因此应用订单的条件，无论它们已改变了。掩码表示某个选中供应商的已打印，已确认，未履行，部分地履行的订单。用户可以：选择完全的订单（选择订单行是标记）。通过按钮将订单导入到发票，或者选择订单项目的部分，以及不同的数量。双击于订单行，用户可以选择有关行的数量（选择订单行是标记）或者编辑“数量”字段因此部分地履行订单。为了完成程序，用户得按“项目转移”

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |

| 图列 | 履行的符号图例 |
| 删除记录设置 | 用于删除记录设置 |
| 执行转移 | 用于转移选中的项目 |
| 强制履行选中的订单 | 用于强制履行某个订单 |
| 转向列表上层 | 用于转向列表上层 |
| 在列表中向下一级 | 用于在列表中向下一级 |
| 在列表中查找对象 | 用于制定列表对象的位置 |
| 选择列表中的所有对象 | 用于选择列表中的项目 |
| 用于展开/缩小选中对象的父级 | 用于用于展开/缩小选中对象的父级 |
| 展开/缩小订单面板 | 用用于展开/缩小订单面板 |
