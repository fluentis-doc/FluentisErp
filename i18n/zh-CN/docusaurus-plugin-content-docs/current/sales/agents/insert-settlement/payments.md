在这个部分中的上面，系统表示代理商登记中的默认付款设置。用户可以随便改变这些设置。在下面的网格中，用户可以估计有关的到期。我们提醒用户将到期额和结算改变调整起来。到期额和要付的额需是相同的。

 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 插入支付 | 将新行插入到付款类型中的时候，它确定光标 |
| 删除付款 | 用于删除已选择的付款设置 |
| 插入到日期 | 将新行插入到结算到期中的时候，它设置光标。 |
| 删除到日期 | 用于删除选中的到日期 |
