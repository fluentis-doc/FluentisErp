---
title: 搜索代理商的佣金结算
sidebar_position: 3
---

搜索代理商的结算程序为用户容易地找到以前输入的结算而设计的。通过这种程序用户能够看到，修改，删除而且输入新的结算。为了做这些操作，用户当然需要相当的权能。

通过筛选器，用户能够看到：单一的结算（例如按结算号码的搜索）、多个结算（例如按日期的搜索）、在某个年内插入的结算（例如按年份的搜索）、关于某个代理商的插入文档（例如按代理商密码的搜索）、包含关于某个文档（根据结算的分类可以是订单、货单、发票）的佣金的结算。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索于数据库找结算 |
| 插入结算 | 用于生成新的结算 |
| 编辑结算 | 用于编辑已插入的结算 |
| 查看结算 | 用于查看已插入的结算。通过本按钮用户不能编辑 |
| 删除结算 | 用于删除已插入的结算 |

**筛选器的区域**：包含使用户做选择的那些数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。** **

**结果网格**：结果网格表示相当于已确定的筛选器数据的结算的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的结算清单。
