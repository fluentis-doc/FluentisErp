通过本表格，用户能够搜索、查看以及删除已完成的卸载操作。这些操作符合删除仓库记录的条件。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索已卸载的POS | 用于搜索已卸载的POS。本程序的成果在结果网格中出现 |
| 删除卸载 | 用于完全地删除在卸成果网格中已选择的卸载操作。这些操作需要符合仓库记录的删除条件。 |
| 恢复POS | 用于删除在成果网格中已选择的卸载操作 |
| 恢复POS和记录 | 用于删除相当于某个操作的文档仓库记录。它们要符合删除仓库记录的条件 |
| 属性 | 用于查看卸载操作的方式用于查看卸载操作的方式 |

**筛选器域区**：筛选器域区包含使客户实施选择的数据类型的清单。筛选器的状态是“AND”。用户可以利用更多搜索的同时方式。

**结果网格**：表示相当于已确定的筛选器数据的卸载操作清单。掩吗包含两个部分：一个包含操作、另外的包含详细、POS以及由卸载操作创建的记录。通过属于本部分的主要网格，用户能够选择卸载操作以及功能。

本掩吗表示POS和POS仓库记录（双击于有关的行）
