输入某个POS需要：在[主页>表格>销售]中预定义的POS分类。按照用户确定的“POS编号”以及文档的类型，每个文档获得自己的号码。年份和日期的字段是自动地填写的。在名/性的字段能够包含输入POS的操作员。“市镇”字段包含POS 的

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | .用于保存完成的操作。如果用户将光标动到别的行，他可以获得相同的结果。 |
| 保存新的 | 用于保存现有的POS而且创建新的 |
| 卸载POS | 库和模板以后，本按钮用于卸载POS |
| 预览 | 用于预览报告 |
| 打印 | 用于打印报告 |
| 帮助POS账户 |  |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 用于管理与在使用中的掩吗有关的文档存档 |
