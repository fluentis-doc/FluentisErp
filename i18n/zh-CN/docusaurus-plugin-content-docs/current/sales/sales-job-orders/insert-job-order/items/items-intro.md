行类型给用户提供选择有不同属性的项目：

**编码的项目**：是在登记中编码的项目，用户可以将这些项目记账到分析会计中而记录到仓库中

**未编码的项目**：是明说的项目，可记账到分析会计中，可是用户不能将它们转移到仓库中

**费用的项目**：包括编码的以及未编码的项目，在文档摘要中分开地总结的，系统将编码的或者具有收税意思的项目转移到仓库中，如果某个项目未编码或者没有收税意思系统不能将它转移到仓库中

**票据项目**：是简单的票据，不影响会计和仓库。用户插入项目分类或者项目编号系统可以自动地插入这些数据。”编码项目“行类型自动地被选择的

客户简直地插入项目类型或者项目号码的时候，系统自动地插入这些数据，“编码的项目”行类型是自动地选择的

客户可以直接地或者通过“帮助项目”（双击于黄色部分）插入项目类型和项目号码。本部分表示项目登记中的数据。插入项目之后，登记自动地利用项目的明说。如果有变体的话，在变体组合框中，用户得选择需要的变体

**数量**：表示主要计量单位的数量，用户可以直接地插入计量单位或者利用采购文档中的计量单位

**价格**：由客户登记中的价目表提供的、用户在“总计”表格中可以查看项目的参考价目表。如果客户登记中的价目表是联系的，采购文档中可以包括从不同价目表来的项目（按客户/供应商登记）。双击于“价目表”字段（总计表格）之后，用户可以选择与默认的不同的价格。如果价目表没有的话，表示的数据是项目登记利用的销售价格

**增值税**：属于客户登记的增值税字段中的数据是最重要的。如果本数据没有的话，系统按项目登记中的价值。增值税是必须的

**销售营业额**：表示属于项目登记中的“普通数据”表格中的数据

**变体**：与选中项目变体有关。选择变体的可能对具有本配置的项目是有用的。这些项目的价格与标准价格可以是不同的。因此，具有变体与没有变体的项目价格可以是不同的。有可能，这个需要在参考的销售价目表中的项目变体的管理

**计量单位**：表示项目的主要计量单位，可是用户有选择另外计量单位的可能

**替代的计量单位**：表示属于项目登记的“另外的计量单位”表格中的计量单位。“默认”标记是活动的

**替代** **的数量**：表示另外计量单位的价值。管理数量（关于主要的计量单位）除/乘以转换因素=另外的计量单位价值

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：

 



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存已完成的操作。如果用户将光标移动到别的行，可以获得同一的结果 |
| 新的项目 | 用于插入新的项目 |
| 删除项目 | 用于删除已插入的项目 |
| 新的子项 | 为那些有<表格/销售/销售作业但类型>等级结构的作业单，用户可以将项目插入到这个结构中 |
| 新同级 | 为那些有<表格/销售/销售作业但类型>等级结构的作业单，用户可以将项目插入到这个结构中 |
| 展开/缩小项目的网格 | 用于展开/缩小“总计”、“数据”、“变体”、“批次”、“营业额”、“活动申请”的表格 |
| 删除折扣 | 用于删除已插入的项目的折扣 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 与在使用中的掩码有关的文档档案 |
