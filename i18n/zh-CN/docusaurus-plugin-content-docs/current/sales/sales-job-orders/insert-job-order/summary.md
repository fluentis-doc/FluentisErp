摘要表示完全文档的主要数据

 最终折扣用户可以插入利用百分比的折扣。已插入的折扣关于文档的每个项目

增值税的汇总表示某个文档的汇总，关于每个增值税的编码

费用表示客户登记包含的费用，在“费用/折扣”表格中。而且用户可以插入关于在使用中的文当的费用

最终的票据通过“帮助编码的票据”，用户可以填写本自由的字段

代理商佣金的汇总表示客户的代理商，包括有关的佣金

文档的总额



**项目的毛额**：表示所有项目的总额

**应用的折扣总计**：表示应用的折扣总计，不包括最终折扣

**项目净额**：项目的毛额–应用的折扣总计

**最终折扣的总计**：表示最终折扣的价值

**未文档的费用**：表示作为“费用”项目插入的费用价值

**有文档的费用**：表示属于费用网格的费用价值

**应纳税**：项目的净额–最终折扣+未文档的费用总额+有文档费用的总额

**增值税**：表示增值税汇总包括的价值总额

**作业单总额**：应纳税+增值税

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存已完成的操作 |
| 删除最终折扣 | 用于删除在相当网格中的最终折扣 |
| 删除费用 | 用于删除在相当网格中的费用 |
| 预览 | 用于预览某个文档 |
| 打印 | 用于打印某个文档 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 用于管理与在使用中掩码有关的文档案 |
