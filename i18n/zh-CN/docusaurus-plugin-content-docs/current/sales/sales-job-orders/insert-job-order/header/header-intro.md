插入某个销售作业单需要：在[主页>表格>销售]中预定义的作业单类型。插入客户的时候，系统自动地表示“头标”表格中的数据，按在客户登记中以前制定的数据。

在登记中制定的表示的数据是：

**交货**：表示在属于客户登记的“交货”表格中的数据（参考的表格“交货”）

**运费**：表示在属于客户登记的“交货”表格中的数据（参考的表格“运费”）

**包装**：表示在属于客户登记的“交货”表格中的数据（参考的表格“包装”）

**国家**：表示在属于客户登记的“普通数据”表格中的数据（参考的表格“国家”）

**区域**：表示在属于客户登记的“交货”表格中的数据（参考的表格是“区域“）

**价目表**：表示在属于客户登记的“价目表”表格中的数据，默认的提供是必须的（参考的表格是”销售价目表“）

**从有效日期**：选中的价目表从这个日期以来有效

**至有效日期**：选中的价目表到这个日期有效的

**语言**：表示“普通数据”表格中的数据

**货币**：表示在属于客户登记的“行政”表格中的数据（参考的表格"货币"）

**汇率**：如果公司的货币等于文档的货币、价值为1，要不然系统按照货币兑换的汇率或者欧元固定汇率中的价值

**货币日期**：在这一天，系统计算汇率价值。这个价值由公司货币与客户货币的关系造成的，按照货币转换表格中的设置

**初始票据**：用户可以插入已经在“编码记录”表格中的票据，因此用户得双击于“初始票据”然后打开“帮助编码记录”选择数据。通过右鼠标按钮，用户可以打开一个窗户然后指定很长的票据文章。如果本字段包括价值，字段的背景颜色改变

**客户的票据**：表示属于客户登记的“行政”表格中的数据

**作业单状态**：创建作业单的时候，状态为"未履行"。为了将作业单项目转移到订单中，作业单“已打印”标记以及确认作业单的日期得激活





功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存销售订单 |
| 删除付款 | 用于删除头标的付款：登记利用这些付款 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 用于管理与在用使中掩码关系的文挡档案 |
