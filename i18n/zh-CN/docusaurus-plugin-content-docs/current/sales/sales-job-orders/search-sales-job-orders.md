---
title: 搜索销售作业单
sidebar_position: 3
---

“搜索销售作业单”程序为使用户容易地搜索以前插入的作业单而设计的。通过这种程序，用户可以查看、编辑、删除作业单以及插入新的。相当的权利是必须的。

通过筛选器用户可以查看：单一的文档（例如按文档号码搜索）、多个文档（例如按日期搜索）、某个年份内插入的文档（例如按年份搜索）、关于某个供应商的文档（例如按账户/明细账目搜索）、按估计交货日期的文档或者确认的日期。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：

 



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索作业单 | 用于搜索数据库中的作业单 |
| 新的作业单 | 用于创建新的作业单 |
| 编辑作业单 | 用于编辑以前插入的作业单 |
| 查看作业单 | 用于查看以前插入的作业单。通过本按钮用户不能编辑作业单 |
| 删除作业单 | 用于删除以前插入的作业单 |
| 履行订单 | 这种自动的程序从履行销售订单 |
| 复制订单 | 用于复制某个作业单。这种程序表示本来文档的情况而且更新的数据 |
| 属性 | 用于查看作业单的履行，包括作业单的项目数据以及有关的文档数据 |
| 转换 | 用于转换文档的货币 |
| 预览 | 用于预览文档，按照已提到的筛选器标准 |
| 报告 | 用于打印文档，按照已提到的筛选器标准（作业单类型、号码、日期、客户）  |

**筛选区域**：筛选区包含使用户选择的数据清单。典型地，筛选的状态是“AND”，同时用户可以选择多种筛选的标准。

**结果网格**：表示关于已提到数据的作业单清单。指定搜索的参数之后，通货功能导航栏的“搜索”按钮在结果网格中用户可以获得需要的作业单。
