---
title: 恢复
sidebar_position: 4
---

在“恢复”中，用户可以查看会计记录的结果。通过合适的按钮用户可以删除会计记录。仅当时段还是可编辑的（总分类帐的打印、没打印的增值税登记、没打印的定期结算、没完成的账户结算、会计分录的收款或者帐单），用户可以完成本操作。双击于记录的部分使用户查看有关的会计记录。本部分包含某个查看会计错误的报告。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索会计记录 | 用于搜索已完成的会计记录清单 |
| 预览记账	 | 用于查看包含会计记录结果的报告 |
| 查看记录 | 用于打开在下面网格中已选择的会计记录。双击于网格上，用户可以获得一模一样的结果 |
| 恢复会计记录 | 用于删除已完成的完全会计记录 |
| 恢复发票 | 用于恢复已选择的单一发票 |
|  |  |
