最后是恢复的单字：通过本单字用户可以选择、查看以及删除操作。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索从送货单创建的发票 | 用于搜索有价值的货单 |
| 恢复已选择的完全创建 | 用于灰度完全的操作。如果得同时创建多个发票，通过本按钮用户可以删除所有发票而恢复送货单 |
| 创建创建报告 |  |
| 恢复发票 | 用于恢复在发票表格中选中的发票 |
| 取消”已打印“标记 |  |
