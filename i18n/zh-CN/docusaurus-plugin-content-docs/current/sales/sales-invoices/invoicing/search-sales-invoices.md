---
title: 搜索销售发票
sidebar_position: 2
---

搜索销售发票的程序为使用户容易地搜索一插入的发票而设计。通过这种程序用户可以查看、编辑、删除销售发票而且插入新的发票。相当的权利是必须的。 

通过筛选器用户可以查看：单一的文档（例如按文档号码搜索）、多个文档（例如按日期搜索）、某个年份内导入的文档（例如按年份搜索）、关于某个客户的文档（例如按帐户/明细账目搜索）、关于某个代理商或者销售作业单的插入文档（只在头标包含作业单的情况下）。而且，通过“搜索项目”按钮，用户可以搜索包含某个项目或者项目类型的文档。 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索发票 | 用于搜索于数据库中的发票 |
| 新的发票 | 用于创建新发票的按钮 |
| 编辑发票 | 用于编辑已插入的发票 |
| 查看发票 | 用于查看已插入的发票。通过这个按钮，用户不能编辑发票 |
| 删除发票 | 用于删除已插入的发票 |
| 注销发票 | 用于从已插入发票创建注销文档。用户注销发票的时候，一个包含本来发票所有项目的表格出现。用户得选择项目的转移模板而且注销标记以及要创建的发票类型。用户得将注销与本来发票的关系直接地插入到“发票参考”和“类型”参考字段。 |
| 复制发票 | 用于复制某个发票。用于复制某个文档包含本来文挡的状态和目前的数据。 |
| 转换 | 用于转换文档的价格 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  销售发票  | 用于利用打印销售发票的程序，按已选择的筛选标准（发票类型、号码、日期、客户等） |
|  销售发票清单  | 用于利用打印某个发票清单的程序，按已选择的筛选标准（发票类型、号码、客户）。用户按客户、发排号码、发票类型以及插入日期、可以分配文档。 |
|  开发票 | 用于利用打印销售发票的程序，按已选择的筛选标准（发票类型、号码、日期、客户）。用户按项目、项目类型、国家、客户刻印分配文档。 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  销售发票的会计记录  | 用于会计记录销售发票 |
|  从送货单创建  | 用于从一个或多个送货单创建销售发票 |
|  从仓库缺载的发票 | 用于从仓库缺载的某个发票 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。** **

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的发票清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的发票清单。
