通过本表格，用户可以搜索查看与消除已完成的操作。这些操作符合删除仓库记录的特别条件。 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索下载的货单。成果网格表示本程序的成果 |
| 删除 | 用于完全地删除在成果网格中已选择的卸载操作。这些操作得符合删除仓库记录的条件 |
| 恢复货单 | 用于删除在成果网格中已选择的卸载操作，关于细节网格中的货单 |
| 恢复所有 | 用于删除关于已选择的文档的仓库记录 |
| 属性 | 用于查看履行卸载操作的方式 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。 

**结果网格**：成果网格表示相当于已确定的筛选器数据的送货单/货单的清单。掩码包含两个部分：一个包括操作，其他的包括操作表格中的卸载操作生产的细节、货单、记录。在本部分主要的网格中，用户能够选择功能导航栏的功能应用于卸载操作。

通过本掩码，用户能够查看仓库的记录（双击于行）。
