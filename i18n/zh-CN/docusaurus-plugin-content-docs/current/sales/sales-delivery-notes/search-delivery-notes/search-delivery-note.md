---
title: 搜索送货单/货单
---

搜索送货单/货单的程序为容易地搜索、查看、修改以及删除已输入的送货单/货单而设计。相当的权利是必须的。

通过筛选器，用户能够查看：单一的文档（例如按文档编号搜索）、多文档（例如按日期搜索）、在一年中插入的文档、关于某个代理商或者某个销售作业单的文当（在这个情况下用户将作业单已插入到标头里）。通过“搜索项目”用户可以搜索包括某个项目或者某个项目分类的文档。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索在数据库中的送货单/货单 |
| 新货单 | 用于创建新的送货单/货单 |
| 修改货单 | 用于修改已输入的送货单/货单 |
| 查看货单 | 用于查看已输入的货单。通过本按钮用户不能修改货单 |
| 删除货单 | 用于删除已输入的货单 |
| 复制 | 用于复制货单。通过本按钮用户能够复某个文档，包括文档的本来设置和更新的数据 |
| 转换 | 用于将价格转换另外货币 |
| 冲销送货单/货单 | 用于按本来货单创建冲销文档 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。** **

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的结算的清单。搜索参数确定了之后，通过掩码中的“搜索”按钮，用户能够获得需要的结算清单。
