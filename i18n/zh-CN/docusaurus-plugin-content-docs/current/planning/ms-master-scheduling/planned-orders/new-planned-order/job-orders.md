---
title: 作业单
sidebar_position: 5
---

本表格由某个网格组成的，表示生成计划订单的生产作业单的详细。网格包括的数据是下列的：  

**作业单：**表示生产作业单的编号；

**年份：**表示生产作业单的年份； 

**预计结束日期：**表示生产作业单的预计结束；

**数量：**表示生产作业单得生产的数量； 

**分类/代码/说明/项目/变量：**表示生产作业单的分类/代码/说明/项目/变量；

**帐户/明细账目/客户的说明：**表示插入到生产作业单的帐户/明细账目/客户的合法名称。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存计划订单的编辑 |
| 插入订单 | 用于插入新的计划订单。应用自动地保存刚插入的订单然后删除所有的字段。窗户可以包括新的订单 |
| 完成订单数据 | 为计划订单的项目，插入/更新   关于物料、阶段、装备、设备、属性以及多样票据的数据。这些数据来自物料清单以及项目的加工路线 |
| 重算初始日期  | 用于更新计划订单以及加工阶段的初始日期 |
| 生成物料清单 | 用于更新计划订单项目的物料清单，利用订单包括的数据 |
| 生成加工路线 | 用于直接地更新计划订单项目的加工路线。过程利用订单包括的数据 |
| 插入物料 | 用于将新的物料插入到“物料”网格。仅当用户在计划订单的“物料”表格，本按钮存在 |
| 删除物料 | 用于删除属于“物料”表格的网格的物料。仅当用户在计划订单的“物料”表格，本按钮存在 |
