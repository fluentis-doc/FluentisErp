窗户包含佣金计算的普通因素：

**结算期间：**用户指定日期的范围；

**扣除模板：**必须的字段。为了自动地制定关于报酬的预付款的预扣的价值。如果本字段没有的话，系统不能制定代理商结算的报酬价值。

**文档类型：**代理商报酬计算的基础因素。用户可以指定代理商应计佣金的方式：按照已确认的订单、交货（就是已发布的送货单）、发票的发布、到日期（无论收账）、已完成的支付（按照已接收的金额：半发票等于半佣金）。开始创建代理商的“结算”之后，用户不能编辑本设置。用户需要创建包含新设置的新登记（因此，为了更新属于”主页>联系人“登记，使用“代理商分配”过程）。

“结算”网格不再激活。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存代理商 | 用于保存关于代理商的已完成的编辑 |
| 删除结算 | 用于删除选中的结算 |
