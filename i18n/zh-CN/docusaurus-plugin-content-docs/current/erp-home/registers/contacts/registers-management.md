---
title: 联系人的登记
sidebar_position: 2
---

搜索登记的过程为 容易地搜索于联系人登记而设计的。通过本过程用户可以查看，编辑、删除、取消联系人。用户也可以插入新的参考。当然用户需要对应的权利。

通过筛选器用户可以查看：唯一的登记（例如按照唯一的名字）更多文档（例如按国家搜索）不再有效的登记（过期登记的筛选器）以及会计的登记（会计登记的筛选器）。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将登记搜索于数据库 |
| 新的 | 用于创建新的登记 |
| 编辑 | 用于编辑已插入的登记 |
| 查看 | 用于查看选中登记的数据 |
| 删除 | 用于删除已插入的登记。用户不能删除已使用的登记 |
| 货币转换 | 用于改变登记设置的货币 |
| 复制 | 用于复制选中的登记。通过本按钮用户可以打开某个掩码。在本掩码中用户可以制定基本的因素：合法名称、标签（登记的编码）地址、邮编、城市、省。确认复制之后，用户可以编辑。登记的复制包含会计科目表  的所有明细账目，与激活的公司有关。按照每个账户的顺序编号系统制定新的明细账目 |

筛选器的区域：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

结果网格：结果网格表示相当于已确定的筛选器数据的登记的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的登记清单。
