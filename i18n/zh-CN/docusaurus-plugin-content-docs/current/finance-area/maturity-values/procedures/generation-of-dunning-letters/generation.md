生成催款信的过程使用户查看以及选择交易的部分因此识别得征求的交易组。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索于交易的完全数据库 |
| 生成催款信 | 用于生成选中的交易。下表格的按钮可以替代这个按钮 |

**筛选器的区域**：包含使用户做选择的数据类型清单，搜索完全的已插入的账单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的要求的交易的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的要求交易清单。
