---
title: 作业单的表格
sidebar_position: 6
---

仅当模板的“作业单”标记激活，作业单的表格存在。

在这个表格中系统将会计记录的经济数据分配于多样销售作业单。系统自动地将这些数据分配于插入到标头的作业单。在这个部分中，用户也可以利用   划分金额  的不同方式。

插入金额，经济应记的日期范围，金额与普通会计的平衡控制不包括自动的过程。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存记录 |
| 新记录 | 用于创建新的记录 |
| 插入的参数 | 用于利用注册的插入参数 |
| 打开登记 | 用于利用选中明细账目的登记管理或者搜索其他的登记 |
| 删除选中的明细账目 | 用于删除选中的作业单 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 与文档的管理有关的按钮 |
