---
title: 成本/利润中心的表格
sidebar_position: 5
---

如果模板有其中成本/利润中心的管理标记至少一个缉获的，  成本/利润中心的表格存在.

成本/利润表格的责任是按成本/利润中心管理记录核查的经济数据。 按照模板的设定，表格可以包括一个部分，其他部分或者这两个部分（在这个情况下有两个选项：“ 成本中心/利润中心分开管理”，用户得同时限价这两个部分，要不然过程只限价一个部分。成本中心包括成本，利润中心包括利润）。如果“机器管理”选项激活的话，表格包括某个公司车辆的限价成本/利润的网格。

装载成本/利润中心的数据可以是自动的（关于手动记录，用户将它们直接地插入到会计中）。有两个方式：  在会计科目表中，将有关的价值百分比分配于成本/利润明细账目，或者在客户/供应商登记中，一直利用百分比（在这个情况下，插入到记录中的所有经济明细账目有同一的限价方式）。限价的优先在普通会计的参数中被设定。

用户可以直接地编辑已计算的金额以及插入行。在这个最后的情况下，过程自动地表示  覆盖在会计中转移的总额的金额。过程也表示同一的应记日期范围。调整和集成的过程意味着成本/利润中心转移的调整。 

保存记录的时候，应用表示包括价值差异详细的掩码。按照模板的设定可以阻止保存的操作。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存记录 |
| 新记录 | 用于创建新的记录 |
| 插入的参数 | 用于利用注册的插入参数 |
| 打开登记 | 用于利用选中明细账目的登记管理或者搜索其他的登记 |
| 删除选中的成本中心 | 用于删除选中的成本中心 |
| 删除选中的利润中心 | 用于删除选中的利润中心 |
| 展开/缩小 | 用于展开/缩小成本/利润中心的可见化 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 与文档的管理有关的按钮 |
