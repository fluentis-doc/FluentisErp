功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索调整 | 用于搜索调整的调整分录 |
| 创建调整 | 用于创建调整分录 |
| 编辑  | 用于编辑选中的调整分录。用户选择某个调整分录之后这个按钮激活 |
| 查看 | 用于查看选中调整分录。用户选择某个调整分录之后这个按钮激活 |
| 删除 | 用于删除选中的账户关闭。用户选择某个账户关闭之后这个按钮激活 |
