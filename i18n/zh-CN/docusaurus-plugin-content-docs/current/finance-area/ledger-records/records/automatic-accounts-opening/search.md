功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索已执行的交易打开 |
| 新的打开 | 用于创建新的账户打开 |
| 编辑记录 | 用于编辑打开的记录。选择某个账户打开之后，这个按钮可以激活　 |
| 查看 | 用于查看打开的记录。选择某个账户打开之后，这个按钮可以激活　 |
| 删除 | 选中某个账户打开之后，用户可以利用这个按钮。用于删除打开记录 |
