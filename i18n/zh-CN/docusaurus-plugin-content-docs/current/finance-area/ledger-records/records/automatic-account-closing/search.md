功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索已完成的结账 |
| 新的 | 用于创建新的结账 |
| 编辑 | 选择某个结账之后，这个按钮激活。用于编辑结账的记录。 |
| 查看 | 选择某个结账之后，这个按钮激活。用于查看结账的记录。 |
| 删除 | 选择某个结账之后，这个按钮激活。用于删除结账的记录。 |
