首先，”递增操作“标记的管理非常重要。因为折旧过程按不同逻辑被管理，如果公司为它的类别利用递增的选项，为了识别数据，用户得导入这个标记。 在掩码的下部分用户得设定计算的参考年份：通常，用户得插入比当前年份之前的年份。如果X-1年份的折旧未会计记录的话，X年份的折旧计算不允许的

搜索数据的操作完成了之后，用户可以一行一行地强制应用的天日数量以及折旧的百分比。我们建议用户在单一的固定资产表格中计算这些特别的情况或者直接地编辑固定资产类别的表格。通过这个方式，迅速的计算过程包括完全行。如果用户要使用其中多样固定资产的不同百分比，可以使用“单一固定资产的百分比”参数。这个参数使用户在每个固定资产标头中制订某个得使用的折旧百分比。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索得计算的固定资产 |
| 固定资产 | 用于执行选中行计算 |
| 查看固定资产 | 用于打开固定资产的登记。这个固定资产的折扣在表格中选中 |
