标头数据保存了之后，下的表格开始激活：“资产操作“的表格包括固定资产分类账的操作详细。第一行关于固定资产的初始采购：操作的日期以及插入到有关的列子的金额（有可能固定资产数量管理）。网格的右边包括”客户/供应商明细账目“以及操作的其他详细。如果有历史固定资产的装载，系统将已计算的折旧基金插入到下面的部分，同时指定”已会计记录“状态。通过这个方式，ERP重新计算这个部分。

中间的网格表示固定资产的已计算的折旧：插入的操作可以是手动的或者自动的。插入的操作可以是手动的或者自动的。手动的包括：参考年份、行的参考连续编号（只关于不递增的类别，关于其他的类别这个字段是隐的，因为折旧与连续编号1一直有关），得使用的折旧类型：在这个阶段中，ERP表示参考的固定资产的类别，包括应用的百分比明细账目以及金额（用户可以随便编辑这些价值）。网格最后的一部分表示折旧的参考日期，作为行的参考会计年份的结束以及可扣/未可扣的金额。自动的操作涉及参考年份的定义（在网格上的字段中）以及“管理“按钮的使用，表示固定资产的类别折旧的数据。

计算折旧的金额，被上网格的”已折旧“列表示。它指定临时的金额。通过”已会计记账“标记，更高的金额应用于关于折旧类型的基金的列。手动的插入操作不执行某个会计记录（为了完成这个操作有”会计记录折旧“过程）：通常，用户将这个标记应用于历史情况的手动装载的操作。普通地，开始使用ERP的时候，系统将上年折旧的总额插入，不包括折旧的详细。 保存包括”已会计记账“行的时候。ERP问用户要不要联系已存在的会计记录。在历史数据的检索的情况下，普通的回复是”否“。”税折旧“标记包括折旧行的税管理:按照固定资产类别的标记被表示。 在一些情况下，为了核查固定资产分类账中的在普通预算使用的预期折旧和为计算税金额的税折旧，用户可以直接地使用这个标记。通过包括折旧的可扣以及不可扣金额的字段限价，用户可以管理税。这些字段的价值按照与某个固定资产或者资产类别有关的可口百分比。

最后的部分表示在第二个网格中选中的折旧限价的成本中心／销售作业单详细。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |

| 新操作 | 在资产操作的第一个网格中，这个按钮使光标激活 |

| 新折旧 | 在折旧的第二网格中，这个按钮使光标激活 |

| 插入成本中心　 | 在成本中心网格中，使光标激活 |
| 删除成本中心 | 用于删除在网格中选中的成本中心转移 |
| 计算成本中心 | 按选中的折旧行的价值计算成本中心　 |

| 文档 | 用于管理与在使用中掩码有关的文档案 |

应用：

a) 重估详细：按照在网格中设定的重估类型，用户可以将使用中固定资产的重估的价值详细插入到重估详细（重估的应纳税和金额、基金以及重估的基金总额）插入到这个掩码。
