---
title: 合并汇总的分录
sidebar_position: 9
---

在“合并汇总的分录”掩码中，至于对表外记录已经提到，通过用户自己完成的记录，用户可以完成合并汇总的操作（例如合并的投资）。这样的记录涉及某个参考的会计模板：相比于普通的记录，单一的会计行首先涉及参考公司的指示。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索记录 | 按照设置的筛选器，搜索记录 |
| 新的记录 | 用于生成与选中合并汇总有关的新表外记录 |
| 编辑选中的记录 | 用于编辑选中的表外记录 |
| 查看选中的记录 | 用于查看选中的记录 |
| 删除 | 用于删除选中的表外记录 |
| 复制记录 | 在通过按钮打开的另外的结账中，复制选中的记录。用户不能复制由自动的程序创建的记录（（例如折旧的检索、调整/集成分录等）。 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| Excel导出 | 用于将差异的数据导出到Excel |
