---
title: 搜索表外的记录
sidebar_position: 7
---

通过外表的记录搜索，用户可以查看与某个年中结账有关的操作。用户选择年中结账之前，本掩码的按钮不工作。

通过本掩码用户可以：找到以及编辑与年中结账有关的记录、删除与年中结账有关的记录、将用户创建的记录添加于年中结账（例如调查关于某个日期的仓库剩余产品）、复制在另外结账网格中选择的用户创建的记录（不包括模块程序自动地执行的记录）。

创建某个表外记录的程序与创建记录是非常相似：首先选择总账的模板然后继续执行记录（唯一的差别是年中结账不包括增值税凭证，可是，即使这两个表格类型具体地不同，本模块涉及同一的会计表格）。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索记录 | 按照设定的筛选器，搜索记录 |
| 新的记录 | 用于生成与选中的年中结账有关的新表外的记录 |
| 编辑选中的记录 | 用于编辑选中的表外记录。如果某个结账是“确认的”，有关的记录不是可编辑的 |
| 查看选中的记录 | 用于查看选中的记录 |
| 删除 | 删除选中的表外记录 |
| 复制记录 | 在通过按钮打开的另外的结账中，复制选中的记录。用户不能复制由自动的程序创建的记录（（例如折旧的检索、调整/集成分录等）。 |
