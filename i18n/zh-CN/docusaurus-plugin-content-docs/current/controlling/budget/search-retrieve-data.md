---
title: 搜索数据检索
sidebar_position: 5
---

搜索数据检索的程序为容易地搜索已插入的数据检索而设计。通过本程序用户可以查看、编辑以及删除已插入的数据检索。从数据检索用户可以完成关于某个文档 的 必须操作。相当的权利是必须的。

搜索数据检索的程序由下列特点组成的。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于数据检索的完全数据库 |
| 新的 | 用于插入新的数据检索 |
| 编辑 | 用于编辑选中的数据检索 |
| 查看 | 用于查看数据检索 |
| 删除 | 用于删除数据检索 |
