按照模型的预算限价，可以在结构的多个分组中被执行。因此，差异分析与结构细节联系，原来的单一价值的限价不需要。

加载数据有两个方法：


 1. 用户直接地制定每个层次的至少一种因素的总额
 2. 自动的方法，按照分预算的更新标记。明细账目的分预算、有活动的标记的成本中心和利润中心作为分预算模型分组的细节限价数据的本来

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 删除 | 用于删除选中的数据 |
| 用默认价值编辑期间的网格 | 用默认的价值重新计算期间细节网格 |
| Excel导出 | 用于执行从Excel导出的程序 |
| 从分预算为明细账目创建 | 用于执行从分预算为明细账目创建的程序 |
| 导入数据 | 用于执行从Excel导入会计明细账目的程序 |
