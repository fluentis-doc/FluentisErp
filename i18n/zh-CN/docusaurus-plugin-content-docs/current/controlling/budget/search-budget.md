---
title: 搜索预算
sidebar_position: 3
---

搜索预算的程序为容易地搜索已被插入的预算而设计的。通过本程序用户可以查看，编辑而删除已插入的预算。在本部分中，用户可以完成关于某个文档的所有程序，可是相当的权利是必须的。

搜索预算的程序由下列的特点而设计的。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索预算 | 用于将筛选器应用于预算的完全数据库 |
| 新预算 | 用于插入新预算的程序 |
| 编辑预算 | 用于编辑已插入的预算 |
| 查看预算 | 用于查看已插入的预算 |
| 删除预算 | 用于删除已插入的预算 |

**筛选区域**：筛选区包含使用户选择的数据清单。典型地，筛选的状态是“AND”，同时用户可以选择多种筛选的标准。

**结果网格**：表示与已制定的筛选器数据相当于的预算清单。指定搜索的参数之后，通货功能导航栏的“搜索预算”按钮，在成果网格中用户可以获得需要的预算。
