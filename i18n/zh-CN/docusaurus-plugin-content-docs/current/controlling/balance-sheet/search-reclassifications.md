---
title: 搜索重新分类
sidebar_position: 4
---

搜索重新分类的程序为容易地搜索已插入的重新分类而设计。通过这种程序用户可以查看、编辑以及删除重新分类，在搜索重新分类中用户可以完成关于某个文件的操作。相当的权利是必须的。

搜索的程序由下列的特点组成的：

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于完全的已插入的重新分类的数据库 |
| 新的 | 用于插入新的重新分类 |
| 编辑 | 用于编辑已插入的重新分类 |
| 查看 | 用于查看已插入的重新分类 |
| 删除 | 用于删除已插入的重新分类 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  打印重新分类的模型  | 用于打印重新分类的模型 |
|  打印重新分类的限价 | 用于打印重新分类的限价 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  重新分类的比较  | 用于比较重新分类 |

**筛选区域**：筛选区包含使用户选择的数据清单。典型地，筛选的状态是“AND”，同时用户可以选择多种筛选的标准。

**结果网格**：表示关于已提到数据的作业单清单。指定搜索的参数之后，通货功能导航栏的“搜索”按钮在结果网格中用户可以获得需要的作业单。
