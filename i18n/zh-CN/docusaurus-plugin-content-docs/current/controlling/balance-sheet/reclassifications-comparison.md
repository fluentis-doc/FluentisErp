---
title: 搜索重新分类的比较
sidebar_position: 5
---

搜索重新分类比较的程序，为使用户容易地搜索以前创建的比较而设计的。通过本程序用户可以容易地查看、编辑以及删除比较。在搜索中用户可以完成关于某个文档的所有的必须操作。

通过本程序用户可以将关于某个模型的多样重新分类的价值比较。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于完全的比较数据库 |
| 新的 | 用于插入新的比较 |
| 编辑 | 用于编辑已插入的比较 |
| 查看 | 用于查看已插入的比较 |
| 删除 | 用于删除已插入的比较. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  Modelli di riclassificazione | Richiama la procedura per stampare i modelli di riclassificazione. |
|  Valorizzazione riclassificazioni | Richiama la procedura per stampare le riclassificazioni di bilancio. |

AREA DI FILTRO: l'area di filtro è quella dedicata ad ospitare l'elenco dei possibili tipi di dati grazie ai quali è possibile effettuare una selezione, una ricerca sul totale delle comparazioni inserite. I Filtri sono tipicamente sempre in condizione 'AND' ed è possibile specificare più criteri di filtro contemporanei.

GRIGLIA DI RISULTATO: la griglia di risultato rappresenta l'elenco delle comparazioni che corrispondono ai dati di filtro sopra specificati. L'utente, dopo aver specificato i valori attraverso i quali desidera ottenere una ricerca comparazioni, premendo il pulsante [Ricerca] presente nella relativa Ribbon, otterrà nella griglia di risultato l'elenco delle comparazioni.
