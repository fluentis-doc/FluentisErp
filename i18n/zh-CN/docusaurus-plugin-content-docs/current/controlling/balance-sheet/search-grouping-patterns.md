---
title: 搜索分组模型
sidebar_position: 3
---

搜索分组模型的程序为容易地搜索以前插入的模型而设计。用户可以查看、编辑以及删除这些模型。在搜索模型的部分中，用户可以完成关于文档的需要的操作。相当的权利是必须的。

搜索程序由下列方式组成的：

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于已插入的模型的数据库 |
| 新的 | 用于利用插入新模型的程序 |
| 编辑 | 用于利用编辑已插入的模型的程序 |
| 查看 | 用于查看以前插入的模型 |
| 删除 | 用于执行删除以前插入的模型的程序 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  打印重新分类的模型  | 用于打印重新分类的模型 |
|  打印重新分类的限价 | 用于打印重新分类的限价 |

**筛选区域**：筛选区包含使用户选择的数据清单。典型地，筛选的状态是“AND”，同时用户可以选择多种筛选的标准。

**结果网格**：表示关于已提到数据的作业单清单。指定搜索的参数之后，通货功能导航栏的“搜索”按钮，在结果网格中用户可以获得需要的作业单。
