---
title: 帐单类型
sidebar_position: 3
---

为了填写表格需要用户的代码以及用户的说明。它们与某个参考编码器、某个账单类型以及某个移动明细账目有关。关闭客户的时候，本明细账目在账单会计过程中被利用。在支付表格中，账单类型与支付类型可有关。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照筛选器，搜索账单类型 |
| 插入 | 插入新行的时候，制定光标的位置 |
| 删除 | 删除选中的支付类型 |
| 预览 | 预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
