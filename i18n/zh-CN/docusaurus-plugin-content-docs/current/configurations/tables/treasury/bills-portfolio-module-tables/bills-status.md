---
title: 账单状态
sidebar_position: 5
---

为了填写表格，管理公司账单的状态需要用户代码以及用户的说明。预定义的状态包括：

**已颁发账单;**

**折扣的账单；**

**以到期付款为条件的账单；**

**收账后的账单；**

**贷记的账单；**

**未付账单；**

**已转移账单；**

**普通状态的账单。**

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照选中的筛选器，搜索账单状态 |
| 插入 | 插入新行的时候，制定光标的位置 |
| 删除 | 删除选中的预付类型 |
| 预览 | 预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
