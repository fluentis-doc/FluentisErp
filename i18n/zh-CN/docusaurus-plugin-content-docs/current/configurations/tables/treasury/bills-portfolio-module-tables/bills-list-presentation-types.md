为了填写本表格需要用户代码以及用户的说明。它们与某个参考的编码器有关。

掩码的其他 选项使用户决定：

- 如果清单类型作为转账到供应商账单的清单；

-创建 电子文件的时候，如果清单类型作为“受众” 直接债务清单。

创建新清单的时候，系统得表示的会计移动的明细账目：银行登记的明细账目，包括账单以及清单会计记录的最终贷记。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照选中的筛选器，搜索清单类型 |
| 插入 | 插入新行的时候，制定光标的位置 |
| 删除 | 删除选中的预付类型 |
| 预览 | 用于预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
