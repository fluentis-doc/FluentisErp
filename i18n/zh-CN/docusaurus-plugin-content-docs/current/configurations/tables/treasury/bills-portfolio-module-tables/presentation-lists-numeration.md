为了设置编号用户得设置参考年度、编号代码、编号说明以及初始编号。

在生成计数器的过程中管理的表格。它属于“工具-用处”。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照设置的筛选器，搜索清单编号 |
| 插入 | 插入新行的时候，制定光标的位置 |
| 删除 | 用于删除选中的预付类型 |
| 预览 | 用于预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
