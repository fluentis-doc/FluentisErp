---
title: 未付状态
sidebar_position: 6
---

为了填写表格，管理公司未付的状态 需要用户的代码以及用户的说明。可利用的状态是：

**未付；**

**已记账；**

**已发布的债务票据（未利用的）；**

**已记账**的债务票据（未利用的）；****

**已收账；**

**授权的（未利用的）。**

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照选中的筛选器搜索未付状态 |
| 插入 | 插入新行的时候，制定光标的位置 |
| 删除 | 用于删除选中的预付类型 |
| 预览 | 用于预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
