本部分包括下列的字段：5字的字母数字代码、有关的说明和两个标记：“作业单筛选器”标记使某个按销售作业单的筛选器对按照本类型的重新分类可用的。通过这种方式用户可以获得关于筛选器数据的重新分类；成本／利润中心的筛选器筛选重新分类的数据。

有三个标准类型：'01普通 – 经济',  ‘BEP – Break Even Point' 以及 ‘IV – 增值税指令'，按照欧洲经济共同体的第四个指令。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照设置的筛选器，用户可以搜索重新分类的类型 |
| 插入 | 插入新行的时候，设置光标的位置 |
| 删除 | 删除选中的重新分类类型 |
| 预览 | 用于预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
