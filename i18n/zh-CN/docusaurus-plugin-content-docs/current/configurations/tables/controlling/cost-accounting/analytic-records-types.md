title: 成本记录类型
sidebar_position: 1
---

本表格由记录类型的数字代码、某个说明以及类型标记组成。选项是：

- 未设置的标记：包含本设置的成本会计模板使用户实现成本会计/分析会计的记录（成本中心/利润中心/作业单）：

**-总账：**包括本设置的模板使用户使用某个总账模板，因此工作于两个层级（以及分析的层级）；

**- 仅分析：**包括本设置的模板是价值模板，包括分析会计部分的移动（成本中心/利润中心/作业单），不包括成本会计/总账的移动。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 按照设置的筛选器，本按钮搜索记录类型 |
| 插入 | 插入新行的时候，设置光标的位置 |
| 删除 | 删除选中的记录类型 |
| 预览 | 用于预览在使用中的表格 |
| 打印 | 用于打印在使用中的表格 |
