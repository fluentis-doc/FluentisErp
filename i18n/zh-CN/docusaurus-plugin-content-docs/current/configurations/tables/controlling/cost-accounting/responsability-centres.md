本表格指定包括成本／利润中心的责任中心。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 责任中心 | 责任中心的字母数字代码 |
| 说明 | 责任中心的说明 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过责任中心或者说明字段，本按钮搜索责任中心 |
| 插入 | 用于插入新的责任中心 |
| 删除 | 用于删除某个责任中心 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
