本表格指定利润中心的层级结构和特点。

普通的部分表示激活公司的利润中心结构：在会计不可计价的利润中心使用红色，生产可利用的利润中心使用粗体。 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 利润中心 | 利润中心的字母数字代码 |
| 说明 | 利润中心的说明 |
| 第二说明 | 利润中心的第二说明 |
| 物料需求计划标记 | 在生产模块中，使用户使用利润中心 |
| 可计价的标记 | 本标记激活的话，利润中心的价值可以包括会计转动。不要计价的结构层级不包括本标记。仅最低层级包括本标记 |
| 利润中心的类型 | 指定如果利润中心是直接的或者不直接的 |
| 责任中心 | 将利润中心于责任中心联系 |
| 初始日期 | 指定利润中心的初始日期 |
| 结束日期 | 指定利润中心的结束日期 |

 功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索下的利润中心 | 用于搜索下的利润中心 |
| 搜索上利润中心 | 用于搜索搜索上利润中心 |
| 新的利润中心 | 用于插入新的利润中心 |
| 利润中心的新连接 | 用于插入连接／利润中心层级 |
| 删除利润中心的连接 | 用于删除连接／利润中心层级 |
| 展开利润中心 | 用于展开利润中心的结构 |
| 折叠利润中心 | 用于折叠利润中心的结构 |
| 减小层级选项卡 | 用于减小层级选项卡 |
| 增加选项卡 | 用于增加选项卡 |
| 预览 | 用于预览利润中心 |
| 打印 | 用于打印 |
