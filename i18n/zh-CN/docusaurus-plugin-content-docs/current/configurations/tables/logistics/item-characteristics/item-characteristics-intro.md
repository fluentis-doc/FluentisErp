项目特点的表格由下列的数据组成。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于已插入的特点的完全数据库 |
| 插入 | 用于插入新的特点 |
| 删除 | 用于删除选中的特点 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印报告 |

筛选器的区域：包含使用户做选择的数据类型清单。在这个情况下，搜索的筛选器是特点的代码和说明。

结果网格：结果网格表示相当于已确定的筛选器数据的特点的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的特点清单。

**插入项目的特点。**

除了搜索系统内的特点，用户可以插入一个或多个特点。用户得击于第一个空的记录（或者按工具栏的“插入”按钮）。然后用户得制定特点以及说明的字母数字代码。
