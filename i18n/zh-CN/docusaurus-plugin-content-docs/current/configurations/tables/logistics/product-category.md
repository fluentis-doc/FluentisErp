---
title: 项目的产品类别
sidebar_position: 3
---

按照产品的逻辑，本表格管理不同类别的编码。在合适的模块中，本逻辑作为分组、统计或者用处。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于已插入的类别的完全数据库 |
| 插入 | 用于插入新的产品类别 |
| 删除  | 用于删除选中的类别 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印报告 |

**筛选器的区域：**包含使用户做选择的数据类型清单。在这个情况下搜索的筛选器是特点的代码和说明。**结果网格：**结果网格表示相当于已确定的筛选器数据的类别的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的类别清单。**插入某个产品类型；**

除了搜索系统中的类别，用户可以添加一个或多个类别。用户得击于第一个空的记录（或者按工具栏的插入按钮）。用户得制订类别的字母数字代码以及说明。
