---
title: 项目的税类别
sidebar_position: 2
---

通过本表格用户可以编码不同的类别，按照与产品有关的税逻辑。在合适的模块中，本逻辑作为分组、统计以及用处。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于税类别的完全数据库 |
| 插入 | 用于插入新的类别 |
| 删除 | 用于删除选中的类别 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印报告 |

**筛选器的区域**：包含使用户做选择的数据类型清单。在这个情况下，搜索的筛选器是类别的代码和说明。

**结果网格：**结果网格表示相当于已确定的筛选器数据的类别的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的类别清单。

**插入项目的税类别。**

为了插入新的税类别，用户得击于第一个空的记录（或者按功能导航栏的“插入”按钮）

然后用户指定税类别的字母数字代码以及说明。
