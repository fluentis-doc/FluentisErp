---
title: 采购预测季节性
sidebar_position: 1
---

通过本表格用户可以管理以前期间的采购季节的变化，按照项目和期间。通过在以前采购的当前预测中的数据检索的程序，本操作可以是自动的。本过程可以创建季节性。表格涉及三个部分：第一个包括搜索筛选器。第二个在季节性中已记录的项目。第三个，在项目部分中，对每个月度表示选中项目的采购数量。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索项目 | 用于将项目搜索于数据库 |
| 新的 | 用于将新的行插入到季节性部分中 |
| 删除选中的项目 | 用于删除选中的项目以及项目的季节性 |
| 复制 | 本过程将选中项目的季节性复制到其他的年度中 |
| 预览 | 用于预览报告 |
| 打印 | 用于打印文档，按照已制定的筛选器标准 |
