---
title: 退回类型
sidebar_position: 4
---

为了打开退回类型的表格请按照表格 > 物流 > 退回类型的路径。通过本表格用户可以设置转包退回的类型，因此按照退回的类型用户可以转动物料。

按照已插入的退回类型的代码和说明筛选的窗户，由某个网格组成。本网格包括得管理的退回类型。

网格包括的列：

**代码：**包含退回的类型；

**说明：**包括退回类型的说明；

**仓库：**通过合适的组合框，用户可以选择包含供应商的物料的仓库。通过某个转包退回，供应商可以发送自己的物料；

**仓库类型：**表示前列包含的仓库说明；

**模板：**通过合适的组合框，用户可以选择装载供应商的物料的模板；

**模板的说明：**表示前列包含的模板说明；

**下载仓库：**通过合适的组合框，用户可以选择  从那个仓库系统得下载属于合适表格的物料；

**下载仓库的说明：**表示属于前列的仓库说明；

**下载模板：**通过合适的组合框，用户可以选择下载供应商利用的物料模板，属于物料表格； 

**下载模板的说明：**表示前列包含的模板说明。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于转包退回类型的完全数据库 |
| 插入 | 用于插入新的转包退回类型 |
| 删除 | 用于删除已插入的转包退回的类型 |
| 预览 | 用于预览转包的退回类型 |
| 打印 | 按照转包的退回类型的报告的打印机，打印 |
|  |  |
