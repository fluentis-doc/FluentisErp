---
title: 位置状态
---

通过本表格，用户可以管理位置的不同状态，在合适的模块中可利用的。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于将搜索筛选器应用于已插入的位置状态的完全数据库 |
| 插入 | 用于插入新的位置类型 |
| 删除 | 用于删除选中的位置状态 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印报告 |

筛选器的区域：包含使用户做选择的数据类型清单。在这个情况下，搜索筛选器是位置状态的状态和说明。结果网格：结果网格表示相当于已确定的筛选器数据的位置状态的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的位置状态清单。**插入位置状态**

为了插入位置状态的新编码，用户得击于第一个空的记录（或者功能导航栏的“插入”按钮）。然后用户得制定类型的字母数字代码以及说明。
