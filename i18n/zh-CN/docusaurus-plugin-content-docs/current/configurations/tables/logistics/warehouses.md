为了正确地使用系统的关于物料移动的所有功能，插入仓库的表格是必须的。

仓库的表格由下列的特点组成：

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于已插入的仓库的完全数据库 |
| 插入 | 用于插入新的仓库 |
| 删除 | 用于删除选中的仓库 |
| 删除成本中心 | 用于删除选中的 成本中心 |
| 删除利润中心 | 用于删除选中的利润中心 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印报告 |

筛选器的区域：包含使用户做选择的数据类型清单。在这个情况下 搜索筛选器按照仓库的说明。

结果网格：结果网格表示相当于已确定的筛选器数据的仓库的清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的仓库清单。

**插入仓库**

除了搜索系统中的仓库，用户可以添加一个或多个仓库。用户得击于第一个空记录 （或者按工具栏的[插入]按钮）。然后用户得制定仓库代码以及说明。用户也可以制定生产地、管理的区域、仓库的有效结束日期、目的地（按照联系人登记）以及/或者销售点。

关于每个仓库，用户可以插入有关的成本以及/或者利润中心。
