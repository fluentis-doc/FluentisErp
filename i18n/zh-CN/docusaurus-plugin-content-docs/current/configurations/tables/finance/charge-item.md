sidebar_position: 31

费用条目的表格，使用户设置与会计科目表联系的条目代码。本操作有统计的目标。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 费用条目的代码 |
| 说明 | 费用条目的说明 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段，搜索费用条目 |
| 新的 | 用于插入费用条目　 |
| 编辑 | 用于编辑选中费用条目 |
| 查看 | 用于查看选中的费用条目 |
| 删除 | 用于删除选中的费用条目 |
