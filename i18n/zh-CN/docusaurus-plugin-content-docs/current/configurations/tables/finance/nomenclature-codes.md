表格使用户指定同时管理公司利用的术语代码列表。用户可以删除以及编辑存在的列表。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 术语代码 | 术语代码 |
| 说明 | 术语代码的说明 |
| 补充计量单位 | 补充计量单位 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过代码、说明和计量单位的字段，搜索某个术语代码 |
| 新的 | 用于插入新的术语代码 |
| 删除 | 用于删除选中的术语代码 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
