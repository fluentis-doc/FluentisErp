---
title: 货币汇率
sidebar_position: 4
---

属于同一数据库的公司共享的货币汇率表格使用户设置多样过程得使用的正式汇率；在本表格中，从文档/记录的日期，系统使用最后的有效汇率。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 汇率 | 普通地包含公司的货币 |
| 说明 | 货币的说明 |
| 在 | 参考的货币 |
| 说明 | 参考货币的说明 |
| 货币日期 | 汇率的参考日期 |
| 汇率值 | 关于欧元，它是汇率的相反价值 |
| 直接值 | 直接值 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个汇率。它包括按“货币”和“从日期到日期”字段筛选的可能 |
| 插入 | 用于插入新的汇率 |
| 更新汇率 | 用于更新汇率。它包括某个包含表格中的最后汇率的对话框 |
| 从文件导入 | 在家庭银行文件的情况下，从文件导入 |
| 删除 | 用于删除汇率 |
| 预览 | 用于预览汇率 |
| 打印  | 用于打印汇率 |
