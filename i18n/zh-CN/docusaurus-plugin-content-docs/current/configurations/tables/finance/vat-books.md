---
title: 增值税账簿
sidebar_position: 20
---

设置增值税账簿的表格，在激活公司内被管理。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 增值税账簿的字母数字代码 |
| 说明 | 已打印的增值税账簿的说明，普通地在正式报告内 |
| 账簿类型 | 账簿的预定类型 |
| 说明 | 账簿类型的说明 |
| 监控凭证 | 本标记阻止用户打印包含凭证缺乏或者顺序问题的账簿 |
| 待分配 | 本标记管理关于对价转账过程的账簿 |
| 增值税结算 | 在期间增值税结算报告，制定得管理的增值税登记 |
| 采购记录的参考 | 本字段对销售/对价账簿激活。它指定对价的转账的采购增值税登记 |
| 说明 | 采购记录的参考的说明 |

通过本部分，在“初始增值税凭证”字段中用户可以设置每个会计年度的凭证计数器。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 年度 | 参考的年度 |
| 初始增值税凭证 | 参考年度的初始增值税凭证 |
| 最后增值税凭证 | 最后增值税凭证字段使用户查看在登记中已打印 的最后增值税凭证 |
| 上次打印日期 | 上次打印的日期 |
| 最后已打印的页 | 最后已打印的页 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 插入 | 用于插入新的增值税账簿 |
| 插入年份 | 用于插入选中增值税账簿的新年份 |
| 删除增值税账簿 | 用于删除选中的增值税账簿 |
| 删除年 | 用于删除选中增值税账簿的年份 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
| 取消最终报告 | 用于取消取消最终报告。通过某个对话框用户可以选择要取消的账簿和期间 |
