本表格使用户实现实际折旧。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 实际折旧类型的代码 |
| 说明 | 实际折旧类型的说明 |
| 递增操作 | 管理的标记，包括实际数据（按照固定资产类别的同名的标记） |
| 计算使用的天数 | 计算使用的天数的标记。计算年度内的固定资产的使用天数，不按照会计年度 |
| 百分比 | 得应用的标准百分比 |
| 固定百分比 | 如果本标记激活的话，百分比是固定的，不能按固定资产改变 |
