---
title: 账户类型
sidebar_position: 21
---

通过账户类型表格的定义，过程知道管理明细账目代码的方式；

- 首先指定  怎么计价登记的四个类型的管理详细；

- 其次指定报表的属于部分（负债/资产/成本/利润）；

- 作为筛选器的标准，普通地在帮助账户中。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 账户类型的代码 |
| 说明 | 账户类型的说明 |
| 资产 | 包括资产的标记 |
| 负债 | 包括负债标记 |
| 成本 | 包括成本标记 |
| 利润 | 包括利润标记 |
| 订单账户 | 包括订单账户的标记 |
| 客户 | 包括客户+资产的标记 |
| 供应商 | 包括供应商+负债的标记 |
| 银行 | 包括资产或者负债+银行的标记 |
| 代理商 | 包括代理商+ 负债的标记 |
| 服务 | 包括服务的标记  - 包括本标记的账户是明细账目。过程自动地管理年中报表的调整/集成分录以及总会计的调整分录 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段，搜索账户类型 |
| 插入 | 用于插入账户的类型 |
| 删除 | 用于删除选中的账户类型 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
