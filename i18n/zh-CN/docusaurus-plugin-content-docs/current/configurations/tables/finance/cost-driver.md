---
title: 成本动因
sidebar_position: 11
---

通过本表格，用户可以设置成本／利润中心转动的重新分配代码的列表。

本部分中包括成本动因的字母数字以及说明。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 成本动因的５字代码 |
| 说明 | 成本动因的说明 |

在本第二个网格中，用户可以指定与选中成本动因有关的百分比的标准；本详细的总和，可分配于成本／利润中心，等于转动的100%。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 成本中心 | 用户将与成本动因有关的分摊标准分配于本成本中心 |
| 说明 | 成本中心的说明 |
| 利润中心 | 用户将与成本动因有关的分摊标准分配于本利润中心 |
| 说明 | 利润中心的说明 |
| 百分比 | 标准的分配百分比 |
| 部门 | 通过部门列，系统将某个转动应用于某个公司部门 |
| 说明 | 部门的说明 |
| 会计周期 | 会计周期字段使用户将分摊标准有效性与某个会计年度联系 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过代码、说明、成本中心以及利润中心的字段，搜索某个成本动因 |
| 插入 | 用于插入新的成本动因 |
| 删除 | 用于删除选中的成动因 |
| 删除详细 | 用于删除选中成本动因的详细 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
