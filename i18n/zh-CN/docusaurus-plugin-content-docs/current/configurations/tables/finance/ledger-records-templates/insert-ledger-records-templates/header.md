title: 插入总帐模板
sidebar_position: 1
---

总帐模板由某个标头以及三个部分叫属性详细、记录详细、参数详细组成。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 模板 | 由5个字组成的唯一字母数字代码 |
| 模板说明 | 模板说明 |
| 运转类型 | 记录的运转类型。它是内部的固定表格。用户不能标记它。它将不增值税的模板和增值税类型的模板分开 |
| 文档类型 | 表格已经包含数据，可是用户可以随便编辑它。在本表格中，最有意思的行包括“货记”和“变化记录”的标记。关于货记，过程监控增值税部分包括负总额 （销售/采购区域监控  货记票据的会计的符号一致性：这个意味着行不能包括正总额，同时别的行包括负总额），关于“变化记录”，在记录的“增值税”部分中，过程启动某个识别变化票据行的标记以及某个参考的日期  |
| 收账/支付记录 | 表示属于专业人员的简化会计的收账/支付记录得包含的模板 |
| 专业人 | 表示专业人的会计模板 |
| 有效期结束日期 | 从本日期以来，用户可以阻止会计模板的使用。从下的日期以来，在模板清单中，用户不能可见它 |
