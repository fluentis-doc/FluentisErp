参数的第一个部分的责任是设置表格的排序，通过合适的按钮



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 监控账户类型 | 通过本标记，过程得监控记录的明细账目与会计参数设置的对应（例如　在某个供应商模板中使用某个客户账户）。如果结果负的话，　系统通知用户 |
| 如果账户类型不兼容的话阻止记录 | 本标记使过程删除记录的明细账目，关于会计参数设置无效（例如在供应商的采购模板，使用某个客户账户） |
| 允许不平衡的会计记录 | 通过本标记用户可以保存不平衡的会计记录。 如果用户要保存不平衡的记录，他可以使用某个表示不平衡记录的报告，与总会计模块的注册联系 |
| 阻止删除包括已支付交易的记录 | 通过本标记用户不能删除包括与某个支付有关的交易的记录 |
| 如果交易和会计交易值不同的话阻止记录 | 本标记使过程阻止记录的保存，因为会计移动的价值与账目的移动价值不同（打开／关闭交易）。如果不激活的话，过程通知用户有某个差异。系统监控包括“打开交易”标记的所有明细账目 |
| 如果文档总计和增值税数据总计不同的话阻止记录 | 本标记使过程阻止记录的保存，因为增值税移动的总价值与记录标头的价值不同。如果字段不激活的话，过程通知用户 |
| 按照增值税网格的数据变化，自动地更新文档的总额 | 按照插入到记录的增值税部分总额的变化，本标记使过程更新记录标头的总额 |
| 以递减方式更新 | 仅档上标记激活，本标记也激活。本标记使过程更新记录的总额 |
| "如果凭证订单不得到尊重的话阻止记录 | 由于本标记，过程阻止记录的保存，因为凭证的时间顺序不得到尊重。尤其，仅档 某个有效的记录日期 被设置，通过本标记过程恢复凭证的编号。如果标记不激活的话，过程通知用户凭证订单的缺乏。通过默认的登记打印，用户不能打印 包括编号缺乏/不按照正确的凭证顺序 的增值税登记 |
| 成本中心/利润中心分开管理 | 仅档模板管理成本／利润中心，本标记激活。本标记将会计的经济移动分配于成本和利润中心。如果不激活的话，本操作关于成本或者利润中心 |
| 自动重算成本中心、利润中心和作业单 | 按照本模板的会计记录包括本标记：编辑某个已保存的会计记录之后，它自动地／不自动地重算成本中心、利润中心以及作业单的分析部分。　插入新记录的时候（到记录的保存）这些部分的更新一直是自动的，按照默认的设置  |
| 如果成本/利润中心值和会计交易不同的话阻止记录 | 本标记使过程阻止记录的保存，因为成本／利润中心的分析移动的总额与总会计部分的经济总额不同。如果标记不激活的话，系统通知用户有价值的差异  |
| 将增值税分组于账簿 | 本标记使过程将分配于同一会计明细账目的增值税行分组于账簿 |
| 阻止重复文档编号的保存 | 由于本默认的标记，过程阻止共享同一文档编号、同一年度以及同一的记录登记的会计记录的保存 |
| 背景颜色 | 在本字段中，用户可以设置背景颜色，因此查看按本模板的移动。为了识别移动的类型，颜色是非常有用的因为用户不再读模板的说明或者代码  |
| 文本颜色 | 在本字段中，用户可以设置文体颜色，因此查看按本模板的移动。为了识别移动的类型，颜色是非常有用的因为用户不再读模板的说明或者代码 |
| 样本 | 表示设置的颜色 |
| 斜体 | 在本字段中，用户可以设置斜体，因此查看按本模板的移动。为了识别移动的类型，斜体是非常有用的因为用户不再读模板的说明或者代码 |
| 粗体 | 在本字段中，用户可以设置粗体，因此查看按本模板的移动。为了识别移动的类型，粗体是非常有用的因为用户不再读模板的说明或者代码 |

| RibbonFunction | RibbonMeaning |
| --- | --- |
