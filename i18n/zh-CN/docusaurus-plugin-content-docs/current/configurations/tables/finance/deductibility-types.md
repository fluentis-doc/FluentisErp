通过本表格用户可以预测税的折扣类型，在固定资产类别中与 需要的折旧类型有关。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 折扣类型的代码 |
| 说明 | 折扣类型的说明 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 初始日期 | 选中的折扣类型的初始日期 |
| 结束日期 | 选中的折扣类型的结束日期 |
| 折扣百分比 | 选中的折扣类型的折扣百分比 |
