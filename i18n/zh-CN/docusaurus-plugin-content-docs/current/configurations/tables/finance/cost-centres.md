本表格定义结构的层级以及成本中心的特点。普通的部分表示激活公司的成本中心结构：在会计中不能可计价的成本中心使用红色的，在生产可利用的成本中心使用粗体。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 成本中心 | 成本中心的字母数字代码 |
| 说明 | 成本中心的说明 |
| 第二说明 | 成本中心的第二说明 |
| 物料需求计划标记 | 本标记使用户使用生产模块的成本中心 |
| 可计价的标记 | 本标记激活的话，成本中心可以包括会计的转动。不要计价的结构层级不包括本标记。仅最低的层级包括本标记 |
| 成本中心的类型 | 指定如果成本中心是直接的或者不直接的 |
| 责任中心 | 将成本中心和责任中心联系 |
| 成本动因 | 在年中结账中，将某个指定百分比的成本动因分配于成本中心 |
| 初始日期 | 指定成本中心的初始日期 |
| 结束日期 | 指定成本中心的结束日期 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索下一个成本中心 | 用于搜索下一个成本中心 |
| 搜索前一成本中心 | 用于搜索前一成本中心 |
| 新的成本中心 | 用于插入新的成本中心 |
| 插入新成本中心关联 | 用于插入某个联系／成本中心的层级 |
| 删除成本中心关联 | 用于删除连接／成本中心层级 |
| 展开成本中心 | 用于展开成本中心的层级结构 |
| 折叠成本中心 | 用于折叠成本中心 |
| 减小层级选项卡 | 用于减小层级选项卡 |
| 增加选项卡 | 用于增加选项卡 |
| 预览 | 用于预览成本中心 |
| 打印 | 用于打印 |
