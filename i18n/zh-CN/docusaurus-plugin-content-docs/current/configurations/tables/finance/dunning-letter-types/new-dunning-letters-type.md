| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 催款信类型的代码 |
| 说明 | 催款信类型的说明 |
| 交易组的催款信 | 管理 某个交易组的催款信（标准的管理）。如果标记不激活的话，每个会计交易的催款信离其它分开 |
| 利息类型 | 为了获得催款信日期的计算结果，  利息类型不是必须的 |

 设置催款信类型之后，通过“新的”按钮用户可以创建多样层级。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 催款信的代码 |
| 说明 | 催款信类型的代码 |
| 优先级 | 创建的优先级，从1到x层级 |
| 迟到天数 | 催促交易之前得过的迟到天数 |
| 天数 | 催款信有效天数。用户可以决定如果天数的计算从创建日期或者上催款信的到日期开始 |
| 从上个移动的日期 | 第一个层级之下的层级的参考日期 |
| 从创建日期 | 第一个层级普通地利用的参考日期 |
| 透支 | 系统不管理客户透支的监控类型的链接 |

保存某个催款信编号之后，用户可以插入多样语言的文章：如果登记不包括某个语言，系统默认地利用第一个被插入的文章。

文章包括4个部分：催款信的题目、标题，结论以及参考的部分（只对交易组激活）。尤其，在参考部分中，通过掩码的代码例如（8）（10）的（5）、（9）的参考（3）编号（4）到日期（6），用户可以插入某个文章。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段，搜索催款信的类型 |
| 新的类型 | 用于创建新的催款信类型 |
| 编辑类型 | 用于编辑选中的催款信类型 |
| 查看类型 | 用于查看选中的催款信类型 |
| 删除类型 | 用于删除选中的催款信类型 |
