---
title: 固定资产类别
sidebar_position: 5
---

在本表格中，用户将代码应用于公司管理的固定资产类别。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 字母数字代码 |
| 说明 | 类别的说明 |
| 账户 | 固定化的资产账户 |
| 明细账目 | 固定化的资产明细账目 |
| 说明 | 固定化的资产明细账目的说明 |
| 物料/无物料 | 指定某个资产的类别，物料/无物料 |
| 最大折旧 | 类别的所有固定资产的折旧的最大额 |
| 折旧百分比 | 固定资产价值的可折旧的最大百分比 |
| 递增操作 | 如果标记不激活的话，增加价值的每个行，被某个分开以及独立的折旧详细管理。如果标记激活的话，系统将每个增加应用于初始采购行。系统一直添加初始价值 |
| 总计折旧限制 | 在采购年度中，总计折旧限制 |
| 重新估 | 管理重新估详细的标记 |
| 余额代码 | 打印重新估工作表的余额代码 |
| 实际折旧 | 启动资产数据的管理：它包括资产会计价值的复制。它使用户利用某个表外的分开折旧逻辑 |

第二个部分的责任是 折旧详细的设置，可应用于选中的上类别。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 折旧类型 | 选中折旧类型：普通折旧、预期折旧以及普通地不利用的减速折旧 |
| 账户成本 | 成本的账户。在本账户中，用户可以核查期间折旧 |
| 成本明细账目 | 成本的账户。在本明细账目中，用户可以核查期间折旧 |
| 账户说明 | 成本明细账目的说明 |
| 账户基金 | 账户基金，包含期间折旧 |
| 明细账目的基金 | 明细账目的基金，包括期间折旧。本价值可以等于资产价值。从本价值用户可以直接地冲销 |
| 账户说明 | 基金明细账目的说明 |
| 排除 | 本标记使用户排除属于折旧的自动计算的选中折旧 |
| 百分比 | 对设置的折旧类型得使用的折旧百分比 |
| 年度编号 | 仅档它对应于预期折旧开始工作。它包括应用的三年。关于已使用的资产，仅采购年度自动地设置使用的年数量 |
| 第一年折旧 | 对所有的物料类别被表示。资产开始活动的年度，它减少折旧百分比 |
| 财政 | 预期折旧行可以包括本标记。它作为财政折旧，用户不能会计 |
| 可折扣类型 | 给折旧类型某个可折扣类型 |
| 可折扣程度说明 | 可折扣程度说明 |

在第三部分中，用户可以将按成本中心的默认百分比计价应用于类别的固定资产。资产登记中的成本中心设置比本设置有限的。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 成本中心 | 包含属于类别的固定资产的计价 |
| 说明 | 成本中心的说明 |
| 百分比 | 用于设置成本中心计价的百分比 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 在上一个表格里搜索 | 通过筛选器、类别、和说明字段，它在上一个表格里搜索 |
| 插入到上面的表格 | 用于插入新的资产类别 |
| 插入到详细表格 | 用于将详细插入到选中的上表格 |
| 插入到成本中心表格 | 用于将成本中心插入到选中的上表格 |
| 删除选中行 | 用于删除选中行（资产类别） |
| 从下一个表中删除选中行 | 用于删除在下一个表格的选中行（折旧类型详细） |
| 从成本中心表中删除选中行 | 用于删除成本中心表格的选中行 |
| 预览 | 用于预览报告 |
| 打印 | 用于打印 |
