---
title: 固定资产预定代码
sidebar_position: 18
---

通过本表格用户可以设置对数据库内的所有公司有效的预定代码。为了利用这些预定代码，管理预定代码的管理得激活。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 预定代码的字母数字代码 |
| 说明 | 预定代码的说明 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器代码、说明的字段搜索预定代码 |
| 插入 | 用于插入预定代码 |
| 删除 | 用于删除选中的预定代码 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
