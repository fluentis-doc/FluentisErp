title: 欧元固定汇率
sidebar_position: 3
---

表格表示欧元的 欧元固定汇率：如果用户设置欧元的价值，系统自动地计算相反价值，被系统计数器值的计算被利用。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 货币 | 货币代码 |
| 货币说明 | 货币说明 |
| 货币日期 | 货币日期 |
| 1 欧元 = | 1 欧元 = 参考货币 |
| 反向兑换 | 反向兑换的计算 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个汇率。按照货币和从日期到日期字段，可筛选 |
| 插入 | 用于插入新的欧元固定汇率 |
| 删除 | 用于删除某个交换欧元固定汇率 |
| 预览 | 用于预览欧元固定汇率 |
| 打印 | 用于打印欧元固定汇率 |
|  |  |
