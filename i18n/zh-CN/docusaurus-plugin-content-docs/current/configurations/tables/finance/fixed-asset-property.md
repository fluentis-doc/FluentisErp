---
title: 固定资产的属性
sidebar_position: 19
---

为了设置表格需要特点的某个代码和说明。然后系统将代码和说明分配于多样固定资产：为了获得共享某个属性的固定资产列表，它作为搜索/打印筛选器。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 固定资产属性的数字字母代码 |
| 说明 | 固定资产属性的说明 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段，搜索固定资产的属性 |
| 插入 | 用于插入固定资产的属性 |
| 删除 | 用于删除选中的固定资产属性 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
