---
title: 替代的折旧计划
sidebar_position: 16
---

通过本表格，在折旧的多年模拟，用户可以设置得计价的折旧计划。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 折旧计划的代码 |
| 说明 | 折旧计划的说明 |
| 部门 | 激活部门的管理标记 |

某个百分比按钮使类别详细的插入操作容易。本按钮应用于未插入的所有类别。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类别 | 折旧计划应用于的类别 |
| 说明 | 类别的说明 |
| 应用百分比 | 得应用的百分比 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过代码筛选器和说明字段，搜索折旧计划 |
| 新计划 | 用于插入新的折旧计划 |
| 新的详细 | 用于插入选中折旧计划的详细 |
| 删除计划 | 用于删除选中的折旧计划 |
| 删除详细 | 用于删除选中折旧计划的详细 |
| 添加缺失类别 | 自动地指定缺乏类别的百分比 |
