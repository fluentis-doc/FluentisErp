| 交易组 | 用于启动交易组的管理 |

| Cont. Enasarco a carico ditta | 只关于意大利 |
| Costo a carico ditta | 只关于意大利 |
| Debito a carico ditta | 只关于意大利 |
| 应付的税 | 应付的税的明细账目记录核查扣除债务的明细账目。记录支付的过程利用本明细账目（同时核查得付的扣除债务，按照收款的逻辑）。如果支付关于收益入的报酬的话，记录得包括关于预扣所得税的移动，同时用户得管理模块的表格因此获得证明的报告 |
| 成本记帐模板 | 成本记帐模板的标记使用户在收益人中管理成本记帐的数据。普通地本选项不是激活的 |
