title: 重估类型
sidebar_position: 29

通过本表格用户可以设置重估法律参考的代码。在固定资产表格中，本法律对重估详细的管理很有用。尤其，仅包含“重古”标记的固定资产类别 启动详细的管理。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 重估类型的代码 |
| 说明 | 重估类型的说明 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段，搜索重估类型 |
| 新的 | 用于插入新的重估类型 |
| 删除 | 用于删除选中的重估类型 |
