---
title: 支付金额类型
sidebar_position: 30
---

通过本表格用户可以设置支付金额类型，在收账管理模块利用的。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 支付金额类型的代码 |
| 说明 | 支付金额类型的说明 |
| 账户 | 不是必须的字段。通过本字段用户可以指定支付金额类型奖金／折扣核查的账户 |
| 明细账目 | 不是必须的字段。通过本字段用户可以指定支付金额类型奖金／折扣核查的账户 |
| 说明 | 明细账目的说明 |
| 已收款金额 | 用于识别已收款金额的计算影响的类型 |
| 票据 | 与核查金额类型的会计记录有关的票据 |

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器、代码和说明字段搜索支付类型 |
| 新金额类型 | 用于插入新支付金额类型 |
| 删除金额类型 | 用于删除选中的金额类型 |
| 预览 | 用于预览文档 |
| 打印 | 用于打印 |
