---
title: POS的类型
sidebar_position: 20
---

POS类型制定某个发票的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档的类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中的筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | POS类型 |
| 说明 | POS类型的说明 |
| 编号类型/说明 | 与POS类型有关的编号类型 |
| 默认 | 如果激活的话，制定默认的POS类型，创建新POS的时候被表示  |
| 票据 | POS类型的说明票据，在报告中利用 |
| 包含增值税的价格 | 如果激活的话，属于POS的价格包含增值税 |
| 预付款 | 如果激活的话，POS的类型是预付款，在普通的POS中可被冲销 |
| P.O.S服务器 | 用于读／写POS数据的服务器 |
| P.O.S数据库 | 用于读／写POS数据的数据库 |
| P.O.S表格 | 用于读／写POS数据的表格 |
| 关联文档的表格 | 用于读／写与POS有关的文档数据 |
| 计量单位 | 用于创建送货单以及发票的计量单位 |
| 增值税舍入 /增值税说明 | 增值税代码，为创建送货单以及发票被利用 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 销售点 | 创建POS销售点 |
| 基金 | 与POS有关的基金代码 |
| 货物类别 | 货物类别 |
| 增值税 | 增值税代码，在POS利用 |
| 仓库 | 属于本类型POS的货物仓库 |
| 模板 | 属于本类型POS的货物移动模板 |
| 支付类型 | 属于本类型POS的支付类型 |
| 支付解决方案 | 属于本类型POS的支付解决方案 |
| 价目表 | 普通价目表的代码，包括得插入到POS的项目价格 |
| 有效日期 | 普通价目表的有效日期 |
| 自动卸载对价结账 | 如果激活的话，打印POS的时候，系统将它从仓库下载 |
| 收银机为必须 | 如果激活的话，为了使用POS，与某个收银机的链接是必须的 |
| 查看变量 | 如果激活的话，系统可以利用项目的变量 |
| 查看属性 | 如果激活的话，系统可以使用项目的属性 |
