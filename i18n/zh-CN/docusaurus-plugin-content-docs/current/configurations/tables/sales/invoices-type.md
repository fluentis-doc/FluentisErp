---
title: 发票类型
sidebar_position: 16
---

通过发票类型用户制定发票的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 发票类型 |
| 说明 | 发票类型的说明 |
| 发票性质 | 与发票类型有关的性质 |
| 编号类型/说明 | 与发票类型有关的编号类型 |
| 从订单 | 如果激活的话，通过某个订单的履行，系统可以创建发票 |
| 直接发票 | 如果激活的话，用户得填写发票的运输数据 |
| 交易代码 | 交易代码 |
| 统计中 | 如果激活的话，属于本类型的发票被统计考虑 |
| 备考 | 如果激活的话，发票类型为备考，因此系统不能会计记录它 |
|  [模板 /模板类型](/docs/configurations/tables/finance/ledger-records-templates/insert-ledger-records-templates/header)  | 会计模板，为发票的会计记录被利用 |
|  分析模板/分析模板的说明  | 在分析会计中，被发票会计记录利用 |
| 票据 | 用户可以将发票类型的说明票据插入 |
| 监控客户 | 插入标头订单明细账目的时候。系统监控账户类型是客户的 |
| 阻止已打印的文档 | 如果激活的话，用户不能编辑已打印的发票 |
| 欧盟内 | 如果激活的话，监控如果客户的国家是欧盟内的，要不然通知用户 |
| 备考发票的履行 | 如果激活的话以及发票是备考的，系统履行订单要不然不履行订单 |
| 运输模板 | 运输模板的说明 |
| 透支 | 如果激活的话，包括本类型的发票，包括透支的监控 |
| 排除阻止 | 如果激活的话，无论超过透支限制，系统插入以及打印文档 |
| 仓库模板/仓库模板的类型 | 属于本发票类型的货物的移动模板  |
| 仓库/仓库类型 | 属于本发票累累型的货物仓库 |
| 交易分组 | 如果激活，在发票会计中系统将交易分组 |
| 未合计数量 | 如果激活的话，履行本类型的订单的时候，履行的文档表示已履行的数量，按照单一的履行行 |
| 农业增值税 | 只关于意大利 |
| 监控卸载的领料单 | 如果激活的话，从已下载的领料单，创建某个货单的时候，已下载的标记自动地开始工作 |
| 阻止领料单数量 | 在装卸单元列表创建的货单中，阻止数量以及替代的数量 |
| 计算佣金 | 如果激活的话，按照本类型的发票，系统计算代理商的佣金 |
| 断开送货单 | 如果激活的话，用户可以下载延迟的发票 |
| EDI的管理 | 如果激活的话，关于属于本类型的发票，系统可以创建EDI文件 |
| 不履行领料单 | 如果激活的话，从领料单创建发票的时候，系统不履行领料单 |
| 包含增值税的价格 | 如果激活的话，属于发票的价格包含增值税 |
| 经济应计 | 如果激活的话，发票类型包括经济应计 |
| 经济应计模板 | 经济应计模板，在发票的会计记录中被利用 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 打印的报告 | 用于制定报告，按订单类型被利用 |
| 副本份数 | 按照发票类型，用户可以制定得打印的数量 |
