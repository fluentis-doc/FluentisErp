---
title: POS编号
sidebar_position: 11
---

POS编号的表格使用户制定顺序编号，在同一的年度不同。系统将这些编号分配于一个或多个POS类型。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个编号类型 |
| 插入 | 用于插入新的编号类型 |
| 删除 | 用于删除编号的类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中的筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 年度  | 编号的参考年度 |
| 类型 | 编号类型 |
| 说明 | 编号类型的说明 |
| 顺序编号 | POS编号的初始顺序编号 |
