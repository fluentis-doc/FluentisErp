title: 销售作业单类型
sidebar_position: 14
---

通过销售作业单类型，用户可以制定某个作业单的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览某个文档 |
| 打印 | 按照选中的筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 作业单类型 |
| 作业单类型的说明 | 作业单类型的说明 |
| 层级结构 | 如果激活的花，属于作业单的项目按照某个层级结构 |
| 订单类型/订单类型说明 | 销售订单类型，为履行属于订单的作业单利用 |
