---
title: 销售订单类型
sidebar_position: 19
---

通过销售订单类型，用户可以制定某个销售订单的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 销售订单类型 |
| 说明 | 订单类型的说明 |
| 编号类型/说 | 与订单类型有关的编号 |
| 自动的作业单 | 如果激活的话，插入某个销售订单的时候，系统创建　包含与销售订单同一编号的销售作业单的标头，在订单标头内链接 |
| 自动的履行 | 如果激活的话，在送货单、领货、销售订单、用户可以履行订单。条件：已打印以及已确认的订单。激活的标记使系统自动地插入与插入订单日期同一的确认日期 |
| 阻止文档打印 | 如果激活的话，用户不能编辑某个已打印的订单 |
| D-货单类型的履行 | 从销售订单自动地创建送货单的货单类型 |
| D-发票类型的履行 | 从销售订单筛选器自动地创建销售发票的发票类型 |
| 拣货履行类型 | 在可用订单履行或者装载计划中自动地创建拣货的拣货类型 |
| 透支 | 如果激活的话，包含本订单类型的订单计算透支 |
| 排除阻止 | 如果激活的话，包含本订单类型的订单，被透支计算的阻止排除 |
| 现金流 | 如果激活的话，订单类型计算现金流 |
| 仓库/仓库说明 | 属于本订单类型的货物的使用仓库 |
| 模板/模板说明 | 属于本订单类型的货物移动的模板 |
| 未合计数量 | 如果激活的话，在本订单类型的履行中，履行的文档表示已履行的数量，按照履行的单一行 |
| 监控可用性 | 如果激活的话，属于本类型的订单计算可用性 |
| 农业增值税 | 只关于意大利 |
| 生产作业单的类型 | 在生产总计划表定义中，销售订单的生产作业单属于本类型 |
| 含增值税价格 | 如果激活的话，文档的价格包含增值税。增值税按照增值税代码以及无增值税的价格 |
| 插入类型 | 项目的自动插入，从价目表或者同一客户的最后订单发生 |
| 采购订单类型/采购订单类型的说明 | 通过选中的采购订单，从销售订单可以生成采购订单 |
| 监控客户 | 插入订单的标头明细账目的时候，系统监控账户类型是“客户”的 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 打印的报告 | 用于制定订单类型的报告 |
| 副本份数 | 用于制定订单类型的得打印的副本份数 |
