title: 送货单/货单
sidebar_position: 7
---

送货单/货单编号表格使用户在同一年度中制定不同的顺序编号。系统将这些顺序编号分配于一个或多个送货单／货单。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个编号类型 |
| 插入 | 用于插入新的编号类型 |
| 删除 | 用于删除编号的类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 年度 | 编号的参考年度 |
| 类型 | 编号类型 |
| 说明 | 编号类型的说明 |
| 顺序编号 | 送货单编号的初始的顺序编号 |
| 客户退回 | 如果激活的话，通过编号类型系统将某个编号分配于退回，因此不再监控顺序编号／日期 。系统将你们的参考添加于票据行，在送货单的基价中被生成 |
| 系列 | 如果激活的话，在会计中表示文档编号和系列 |
| 字的固定数量 | 用于做系列 |
