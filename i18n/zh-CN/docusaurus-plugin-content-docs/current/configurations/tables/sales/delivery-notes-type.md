---
title: 货单类型
sidebar_position: 13
---

用于制定送货单的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文当的类型 |
|  预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 送货单类型 |
| 说明 | 送货单类型的说明 |
| 编号类型/说明 | 与送货单类型有关的编号类型 |
| 送货单性质 | 用于制定送货单类型：交货、退回、查看等 |
| 订单 | 如果激活的话，通过某个销售订单的履行，用户可以创建送货单 |
| 价值 | 如果激活的话，送货单的报告包括价值 |
| 发票类型 /说明 | 从送货单创建发票的发票类型 |
| 运输模板 | 运输模板的说明 |
| 监控客户 | 插入订单标头明细账目的时候，系统监控账目类型是客户的 |
| 阻止文档的打印 | 如果激活的话，系统不能编辑已打印的送货单 |
| 透支 | 如果激活的话，包含本类型的送货单属于透支的监控 |
| 排除阻止 | 如果激活的话，在超过透支限制的情况下，系统不阻止文档，用户可以插入／打印文档 |
| 现金流 | 如果激活的话，送货单计算现金流 |
| 仓库/说明 | 属于本类型的送货单的货物的使用仓库 |
| 模板/说明 | 属于本类型的送货单的货物移动模板 |
| 未合计数量 | 如果激活的话，在订单履行中，履行的文档表示已履行的数量，按照履行的单一行 |
| 农业的增值税 | 只关于意大利 |
| 监控卸载的领料单 | 如果激活的话，从领料创建某个货单的时候，货单的已下载的标记开始工作 |
| 阻止领料单的数量 | 如果激活的话，被装卸货单元清单创建的货单中，阻止数量以及替代数量 |
| 允许卸载 不可用的批次 | 用于下载不可用的批次 |
| EDI的管理 | 如果激活的话，对本类型的送货单，创建EDI文件 |
| 包含增值税的价格 | 如果激活的话，属于送货单的价格包含增值税 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 打印的报告 | 按照订单类型制定得利用的报告 |
| 编号 | 按照送货单类型制定得打印的数量 |
