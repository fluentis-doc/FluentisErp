---
title: EDI文档类型
sidebar_position: 15
---

通过EDI类型用户可以指定某个EDI文档的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| Cancellazione | Pulsante per cancellare il tipo documento. |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | EDI类型的代码 |
| 说明 | EDI类型的说明 |
| 编号类型 | EDI编号类型，与EDI类型有关 |
| 订单类型 | EDI文档的销售订单类型 |
| 销售预测的类型 | 自动的，为创建EDI文档，包括周期销售预测 |
| 默认 | 如果激活的话，默认地制定EDI类型，为了创建EDI文档 |
| 票据 | 关于EDI类型的票据 |
