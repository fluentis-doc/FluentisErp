---
title: 销售营业额
sidebar_position: 6
---

会计记录文档的时候，属于销售营业额表格的数据是必须的。

为了编码营业额代码，用户得插入会计的账户明细账目、增值税类型以及金额类型。如果用户将营业额类型插入到项目登记，文档行表示本代码。本代码对文档的会计记录很有用。如果项目不包括营业额类型，系统按照供应商登记中的成本账户。属于本表格的账户－明细账目对属于文档行的成本中心很有用。

**国家的详细：**与外程序的数据交换利用。在未来，用户可以设置不同国家的成本账户以及项目的会计记录类型。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索某个文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器和打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 销售营业额的代码 |
| 说明 | 销售营业额的说明 |
| 账户/明细账目，明细账目的说明 | 成本账户，在销售文档的会计记录中利用 |
| 增值税类型 | 在文档会计记录中利用的。普通地，它的状态等于普通。为了获得别的详细，请监控在会计模板以及财政区域的会计记录中的利用 |
| 成本会计账户 /成本明细账目，成本明细账目的说明 | 在分析会计中，系统通过本账户会计记录文档 |
| 类型/说明 | 在文档的会计记录中利用的。普通地它等于１１明细账目应纳税+ 不可扣除税。为了获得别的详细，监控在会计模板以及财政区域会计记录中的使用, |
