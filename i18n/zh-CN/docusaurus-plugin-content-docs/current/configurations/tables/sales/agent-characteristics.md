---
title: 代理商特点
sidebar_position: 2
---

说明代理商的表格，在代理商登记中利用的。属于数据库的所有公司共享它。由某个代码/说明的简单表格组成。通过它，在制定的报告/统计内，用户可以将特点分配于代理商。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器字段搜索代理商的某个特点 |
| 插入 | 用于插入代理商的新特点 |
| 删除 | 用于删除某个代理商的特点，除非它已被利用的 |
| 预览 | 用于预览代理商特点 |
| 打印 | 用于打印代理商特点 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 代码 | 代理商特点的字母数字代码 |
| 说明 | 代理商特点的说明 |
