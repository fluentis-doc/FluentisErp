---
title: 折扣范围
sidebar_position: 12
---

通过折扣范围表格，用户可以设置折扣的价格范围的变异因此代理商的佣金。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 通过筛选器字段，搜索折扣范围 |
| 插入 | 用于插入新的折扣范围 |
| 删除 | 用于删除某个折扣范围，除非它已被利用 |
| 预览 | 用于预览折扣范围 |
| 打印 | 用于打印折扣范围 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 范围 | 折扣范围的数字代码 |
| 说明  | 折扣范围的说明 |
| 唯一 | 通过本标记，系统将折扣的佣金间隔应用于所有的项目 |
| 项目类别 | 通过本标记，按照项目类别，用户可以设置折扣的佣金间隔 |
| 价目表 | 通过本标记，按照销售价目表，用户可以制定折扣的佣金间隔 |
| 项目类型 | 通过本标记，按照项目类型，用户可以制定折扣的佣金间隔 |
| 特点 | 通过本标记，按照项目的特点，用户可以制定折扣的佣金间隔 |
| 产品类别 | 通过本标记，按照项目的产品类别，用户可以制定折扣的佣金间隔 |
