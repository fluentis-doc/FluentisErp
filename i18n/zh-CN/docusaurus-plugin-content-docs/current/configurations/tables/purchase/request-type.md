---
title: 申请类型
sidebar_position: 14
---

申请类型为制定某个申请的属性被利用的。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 采购申请的类型 |
| 说明 | 申请类型的说明 |
| 编号类型 /编号说明 | 与申请类型有关的编号类型 |
| 成本中心 | 创建采购申请的时候，得表示的成本中心 |
| 仓库 /仓库说明 | 创建某个采购申请的时候，得利用的仓库 |
| 监控可用性 | 如果激活的话，系统通过这些采购申请计算可用性 |
