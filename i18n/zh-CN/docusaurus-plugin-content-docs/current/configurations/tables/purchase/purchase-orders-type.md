---
title: 采购订单的类型
sidebar_position: 10
---

通过.采购订单的类型用户可以制定某个订单的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器和打印机配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 销售订单类型 |
| 说明 | 订单类型的说明 |
| 编号类型/说明 | 与订单类型有关的编号类型 |
| 字履行 | 如果激活的话，采购单的文档、货物接收以及采购发票可以履行订单 |
| 现金流 | 如果激活的话，订单类型计算现金流 |
| 仓库 / 仓库类型 | 属于本订单类型的货物的接收仓库 |
| 模板 / 模板类型 | 属于本订单类型的货物接受的移动模板 |
| 监控可用性 | 如果激活的话，属于本类型的订单计算可用性 |
| 农业增值税 | 只关于意大利 |
