---
title: 采购发票类型
sidebar_position: 9
---

通过采购发票类型用户可以制定某个采购发票的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中的筛选器以及打印机的配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 发票类型 |
| 说明 | 发票类型的说明 |
| 发票性质 | 表示发票性质 |
| 模板 /模板类型 | 为会计记录发票的会计模板 |
| 分析模板/分析模板的说明 | 为会计疾苦发票的分析模板 |
| Tran. codice | 只关于意大利 |
| 仓库 /仓库说明 | 属于这种发票类型的货物接收仓库 |
| 仓库模板  / 仓库模板类型 | 属于这种发票类型的货物移动的模板 |
| 从订单 | 包含本类型的发票可以履行采购订单 |
| 即使发票 | 在这种类型的发票中，用户得提供运输数据 |
| 欧盟内 | 如果供应商的国家是欧盟内要不然系统通知用户 |
| 在统计中 | 统计表示本类型的发票 |
| 监控供应商 | 如果激活的话，通过账户以及 供应商账户类型的明细账目，用户可以创建S采购发票；如果不激活的话用户也可以利用客户 |
| 交易分组 | 如果激活的话，在发票会计记录中系统分组交易 |
| 未分组数量 | 如果激活的话，履行属于本类型的订单的时候，履行文档表示已履行的数量，按照单一的履行行  |
| 农业的采购增值税 | 只关于意大利 |
| 待接收的发票 | 如果激活的话，属于本类型的发票是得接收的发票 |
| 待接收的发票冲销 | 如果激活的话，属于本类型的发票是得接收发票的冲销 |
| 经济应计的模板 | 发票的经济应计的会计模板 |
