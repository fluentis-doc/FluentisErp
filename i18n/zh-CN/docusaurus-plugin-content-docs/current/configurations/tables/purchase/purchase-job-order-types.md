---
title: 采购作业单的类型
sidebar_position: 8
---

通过采购作业单类型 ，用户可以制定某个作业单的属性。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于搜索文档类型 |
| 插入 | 用于插入新的文档类型 |
| 删除 | 用于删除文档类型 |
| 预览 | 用于预览文档 |
| 打印 | 按照选中筛选器以及打印机配置打印 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 类型 | 表示作业单代码 |
| 说明 | 作业单代码的说明 |
| 层级结构 | 如果激活的话，项目按照某个层级的结构 |
