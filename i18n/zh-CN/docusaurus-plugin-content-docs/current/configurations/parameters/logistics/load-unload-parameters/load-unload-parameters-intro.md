---

装载/缺载管理的过程由下列因素组成：

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于装载/缺载参数的完全数据库 |
| 插入 | 用于插入新的装载/缺载的新参数 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

**结果** **网格**：结果网格表示    相当于已确定的筛选器数据的装载/缺载参数的清单。指定搜索价值以后，通过功能导航栏中的搜索按钮，用户能够获得需要的装载/缺载参数清单。
