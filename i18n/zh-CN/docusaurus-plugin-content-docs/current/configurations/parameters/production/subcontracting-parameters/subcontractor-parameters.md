为了打开转包参数的窗户请按主页  > 参数 > 生产 > 转包参数。通过本窗户用户可以设置关于转包管理的参数，以及记录的实现方式，被模块的过程启动。 

在**物料配送到转包商**部分用户可以设置仓库以及有关的模板，卸载转包商的物料的时候利用。因此，从设置的仓库，用户得缺载输入到属于转包订单的待交货的物料表格的物料。

在**物料装载到转包商**部分中包含仓库以及有关的模板，装载物料的时候利用。因此设置的仓库包含属于待交货的物料表格的物料。 插入到模板表格 抵销模板利用这些数据。 本模板对应于缺载物料的模板（在合适的组合框设置）。

**转包商间物料配送**部分包含仓库以及有关的模板，转移物料的时候利用。其中转包商的转运掩码利用这些数据。

在**使用的物料消耗**部分用户可以设置 缺载转包商利用的物料的时候 的仓库和有关的模板。因此，从设置的仓库用户可以缺载插入到使用的物料表格的物料。 

现今**属于质量检查的产品退回**部分不激活。

在**产品退回**部分用户可以设置   装载最终产品（或者粗制品）的时候利用的仓库和有关的模板。设置的仓库包含插入到属于转包商退回的物料的退回的项目表格的产品。

在**退回的物料**部分，用户可以设置装载退回得利用的仓库以及有关的模板。因此，设置的仓库包含插入到使用的物料表格的项目，与某个退回数量有关。

在**货单类型**部分用户可以设置   销售货单类型，利用生成交货单过程的时候默认地表示的。本货单类型自动地利用在合适组合框设置的仓库和模板，属于转包窗户的“物料配送到转包商”。

插入交货单的时候**操作员代码为必须**标记使用户决定如果插入操作员的代码。

“上次插入的转包商订单编号”组合框表示最后的顺序编号，在转包订单中利用。

**建议物料数据**：在待交货的物料表格中，使用户查看过程自动地表示物料的四个不同的可能。可能是下列的：1.　从次交货到同一转包商：控制上次的转包订单，对同一的转包商生成，包含得生成的项目。 2.从物料清单结构展开级别：控制插入到得生产项目表格的项目物料清单。3. 得生产的相同项目的交货：将属于得生产的项目表格的项目插入到待交货的物料表格　4. 无物料：得交货的物料表格不表示物料。  

如果**工作的上次价格的** **建议**标记激活的话，转包订单的工作单价的箱子表示最后转包订单的价格，关于相同转包商的项目。 

**物料加工成本值**选择扭使用户决定如果工艺按照最终成本或者平均成本。因此，用户可以决定如果转包商利用的物料按照最终或者项目登记的平均成本。

版本 /选项组合框：使用户选择物料清单的版本以及有关的选项。普通地，系统默认地表示数据，与激活版本和基本选项一样。

**转包商订单中建议的物料数据**部分由五个部分组成：

**考虑可用性从**用户可以决定如果考虑从转包／生产／仓库／销售的可用性。每个标记使生成转包订单的过程，按照 ERP的单一区域的数据考虑物料的可用性。因此，如果用户选择启动采购标记可是不启动销售，系统考虑供应商、采购单／采购发票、采购申请、　的可用性，不考虑销售订单、货单以及销售发票的可用性。 

**最终产品**（标记）：用户将某个项目插入到得生产的项目表格的时候，本标记使用户决定如果项目数量按照经济批次以及／或者批次倍数；

**物料Materiali**:用户将某个项目插入到得生产的项目表格的时候，关于得交货的物料本标记使用户决定如果考虑项目的数量，按照经济批次以及／或者批次倍数。

**在项目更新中重新生成物料：**如果标记激活的话，每次用户编辑属于得生产的项目表格的某个项目行，过程重新生成关于得交货以及得利用的物料数据，得插入到订单的最后表格。

**仓库：**使用户选择如果仓库按照退回单类型或者交货单类型，同时不利用插入到物料配送到转包商以及产品退回的合适组合框的设置。

**监控可用性**标记使用户决定如果过程得控制在“考虑可用性从”部分标记的可用性。

如果**必需可用性**标记激活的话，在交货日期没有可用性的情况下，防止用户将物料插入到转包订单。

由于**无物料在交货中**标记，待交货的物料的表格不包含得生产的项目。

如果**监控物料可用性**标记激活，使过程考虑在转包退回的过程内得利用的物料的可用性。

如果**自动装载/卸载**标记激活的话，打印交货单之后，使仓库的转动（出）自动地实现。通过这个日方式用户可以避免缺载转包的交货单的阶段。

**阻止日期为公众假日的文档插入**如果激活的话，假日的时候阻止某个订单、货单或者转包退回的插入。

如果**无价目表成本为0**标记激活的话，用户可以将某个项目插入到属于转包订单的得生产的项目的表格过程不需要以项目登记的最终价格作为工艺的单价，因此单价等于0。

**监控参考的转包商**使  生成转包订单 的过程监控插入到“得生产的项目”表格的项目的优先供应商等于订单标头的供应商。
