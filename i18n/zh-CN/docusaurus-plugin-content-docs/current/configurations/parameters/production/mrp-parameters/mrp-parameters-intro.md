title: 物料需求计划参数
sidebar_position: 1
---

为了打开物料需求计划窗户请按照 [主页>参数>生产>物料需求计划参数]路线。它是数据库内的项目的物料需求计划参数的搜索窗户。合适窗户插入主要登记之后，每个项目包含某个物料需求计划参数的登记。 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于插入项目的完全数据库 |
| 编辑参数 | 用于编辑已插入项目的物料需求计划参数 |
| 查看参数 | 用于查看已插入项目的物料需求计划参数 |
| 替换信息 | 打开某个弹出的窗户。通过本窗户用户可以决定那些关于项目的数据编辑。用户也可以编辑多项目的数据，在网格中已选中 |

**筛选器的区域**：在本部分中用户可以利用一系列搜索筛选器，按项目类别、代码、说明和变量以及采购类型（采购、生产、转包）。** **

**结果网格：**双击于项目行之后，用户可以插入项目物料需求计划参数的详细窗户。窗户由某个标头（表示项目的类别、代码、说明、变量、计量单位）   以及某个包含两个表格的部分组成。 

双击于项目行用户可以插入项目的物料需求计划参数的详细窗户。

窗户由某个标头（表示项目类别、代码、说明、变量和计量单位）以及某个包含两个表格的部分组成。
