---
title: 固定资产的参数
sidebar_position: 4
---

为了打开固定资产模块的参数请按 [主页>参数>财政]路径。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存参数 | 用于保存已设置的参数 |
| 取消设置 | 用于取消设置 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 检查预编码 | 如果标记激活的话，在表格中用户得预定义预编码，生成固定资产的时候，不能自由地被设置 |
| 查看数量 | 如果本标记激活的话，用户可以查看数量 |
| 默认的计量单位 | 如果上标记激活的话，制定计量单位 |
| 单一固定资产的百分比 | 在固定资产的标头中，某个字段开始工作。通过本字段用户可以设置某个折旧百分比，比在固定资产的属于类别设置的百分比优先 |
| 部门的管理 | 如果标记激活的话，在固定资产的登记中用户可以定义需要的部门，要不然系统表示默认的 |
