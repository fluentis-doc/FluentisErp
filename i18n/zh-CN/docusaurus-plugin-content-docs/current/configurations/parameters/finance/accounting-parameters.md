会计参数属于[主页>参数>财政]

会计参数的掩码使用户制定管理 每个会计年度  的参数，在左网格中选中。

掩码有两个管理的按钮：“取消账簿的最终打印"以及”固定资产簿的最终打印“。为了利用这两个按钮，用户需要合适的权利，被ArtinfoResourceManager 可管理。在这两个情况下，用户得重置有关的计数器。

会计参数的表格使用户识别最终的管理年度，为了 生成新会计年度的过程很有用。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存关于会计管理年度的设置 |
| 插入 | 用于插入新的会计管理的新年度 |
| 删除 | 用户删除会计管理的年度 |
| 插入账户 | 用于插入授权的登记账户的类型。普通账户和税文档的明细账目的自动的替代授权。冻结或者报告不兼容的账户 |
| 删除账户 | 用于删除已插入的账户 |
| 取消最终打印 | 用于取消  的最终打印。为了利用本按钮用户需要合适的权利，被ArtinfoResourceManager可管理。最终打印取消之后，用户得重置有关的计数器  |
| 取消折旧簿 | 用于取消固定资产的最终打印。为了利用本按钮用户需要合适的权利。被ArtinfoResourceManager 可管理。取消最终打印之后用户得重置有关的计数器 |

#1.1 总表格

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 本期初始/结束 | 必须的字段。表示日期的范围。在选中的会计年度记录中，文档被可管理 |
| 开始/结束当前应计周期 | 建议的字段。表示应计的日期范围，在选中的会计年度记录中可被管理 |
| 上会计年度的关闭 | 表示上会计年度的关闭日期 |
| 账簿的最后打印的日期 | 记录当前会计年度的最后打印的日期。打开新会计年度的时候，本日期得等于上会计年度的关闭日期 |
| 会计年度的初始/结束 | 表示选中会计年度的记录日期的范围。 按照这些日期，系统将每个会计记录插入到属于的会计年度。这些日期不能重叠 |
| 账簿的最后打印页 | 记录选中会计年度的账簿的最后页。生成新会计年度的时候，本字段的价值等于0 |
| 账簿的最后打印的线 | 记录选中会计年度的账簿最后打印的线。生成新会计年度的时候，本字段的价值等于0 |
| 借记/贷方余额 | 记录选中会计年度的账簿的打印最后借记/贷方余额。生成新会计年度的时候，本字段的价值等于0 |
| 固定资产簿的打印最后页 | 记录选中会计年度的固定资产簿的最后页。生成新会计年度的时候，本字段的价值等于0 |
| 按比例百分比 | 用于设置按比例的百分比价值。如果没有选项的话，本字段可以是空或者它的价值等于100% |
| 打开／结算账户的模板 | 不必须的字段。它们使用户预设置过程得使用的模板  |
| 成本／利润中心的建议，从 | 表示成本／利润中心的会计价值顺序 |
| 增值税的会计类型 | 用于定义增值税率的默认会计类型，在会计科目表的多样明细账目 |
| 增值税的结算 | 用于设置定期增值税的周期 |
| 增值税的上舍入类型 | 设置增值税的上舍入，按照里拉或者欧元的逻辑 |
| 增值税登记 | 用于决定如果打印按照记录日期或者增值税应计的日期，在选中的时期。对那些１５日之前不开发票的用户，结果一模一样 |
| 限制的应计日期等于 | 指定如果限制的应计日期按照记录日期或者文档日期 |
| 客户／供应商的账户列表 | 在本部分中用户可以设置每个账户的登记类型。这个操作使用户将　普通账户替代税文档的明细账目，饿日且冻结或者报告不兼容的账户  |
