---
title: 采购单参数
sidebar_position: 6
---

采购单参数使基础设置正确地管理，按照每个公司的需要。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 恢复参数 | 用于恢复初始的参数 |

#1.1 总表格

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 双重计量单位管理 | 激活的时候，在采购单的项目表格中，替代计量单位的组合框和替代数量列开始工作 |
| 自动建议替代计量单位  | 仅档“双重计量单位管理”参数激活，本参数开始工作。如果用户将某个登记包含   替代计量单位表格和默认计量单位  的项目，本计量单位作为项目的自动替代计量单位，同时系统通过计量单位的缩放比例计算替代数量 |
| 查看规模 | 激活的时候，在采购单管理的项目表格中用户可以查看三个列：高度、宽度、深度（数据属于项目登记的重量/规模表格）  |
| 允许送货单有相同的编号和供应商 | 激活的时候，对相同的供应商，用户可以将有相同编号的货单插入，同时用户可以查看某个信息 |
| 批次和序列号为必须 | 激活的时候，对按批次和序列号管理的项目，在有关的表格中，用户得指定批次或者序号 |
| 在所有预定义价目表中搜索项目价格 | 本参数的功能是将价格搜索于供应商价目表。搜索操作关于在供应商登记中预定义的价目表，无论价目表无效（有效价目表的有效初始日期 <=当前日期 以及有效结束日期为0或者 >= 当前日期） |
| 允许礼品费的折扣 | 激活的时候，用户可以将折扣插入到礼品的行 |

#1.2 装载表格

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 创建有文档日期的记录 | 激活的时候，仓库记录包含货单的同一日期。在装载的表格中用户不能指定记录的日期 |
| 优先级仓库和装载模板 | 激活的时候，装载的仓库和模板被装载货单行读或者被装载表格的参数。如果不激活的话，系统利用插入到本参数表格的仓库和模板 |
| 自动装载/缺载 | 如果标记激活的话，货单货单状态是“受控”的时候， 将货单装载到仓库的过程自动地开始工作 |
| 通知虚拟项目 | 如果激活的话，装载货单的时候，系统监控如果货单包括虚拟项目；如果虚拟项目存在的话，某个问用户装载货单的信息出现 |

#1.3 导入表格在本表格中，用户可以指定文件的成名和从条形码导入项目的路线。为了打开导入的表格，请按照  货单的 管理工具兰，在项目表格中。通过本导入操作，用户将项目从文件插入到货单。 

#1.4 条形码导入的表格在本表格中，用户指定    导入属于BA_ImpBCAll  表格的参数。为了打开导入表格，按照货单的管理工具兰。通过本操作，用户可以插入从某个货单表格的项目。为了打开导入的表格，按照货单的管理工具兰，在项目表格中。通过本操作，用户可以将属于某个货单表格的项目插入。

#1.5 分析表格这些参数指定成本/利润中心的搜索优先，然后将这些中心分配于货单的项目。重新计算的标记激活的时候，通过选中的优先，在项目中，成本/利润中心被插入/更新。
