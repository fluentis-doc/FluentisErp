---
title: 销售作业单的参数
sidebar_position: 4
---

销售作业单的参数使基础设置正确地管理，按照每个公司不同的需要。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 恢复参数 | 用于恢复初始的参数 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 必需的履行 | 在销售订单利用：销售订单从一个或多个销售作业单得导入项目 |
| 必需作业单的项目履行 | 在销售订单利用：指定销售订单    得包含从销售作业单导入的项目 |
| 固定数量 | 在销售订单利用：指定在销售作业单销售订单不能履行比作业单项目 行大的数量 |
| 从项目登记中恢复属性 | 系统不再管理本参数 |
| 仅查看相关项目 | 在转包中利用：指定从子项目用户可以履行销售作业单 |
| 通过不同账户在交货送货单中作业单履行 | 在转包中利用：指定用户从与转包单不同的某个账户可以履行 |
| 双重计量单位单位管理 | 在销售作业单，替代的计量单位被管理 |
| 自动建议替代计量单位 | 自动建议替代的默认计量单位 |
