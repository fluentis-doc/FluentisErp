功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存设置 |

 

为了打开银行收据模块的参数请按[主页>参数>现金]

在有关的参数，用户可以预设置默认的因素，在模块的过程中利用的。尤其：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 银行手锯路径的创建路径 | 创建银行收据路径标准目录的设置。我们建议用户，在服务器内，创建某个共享的目录。通过这个方式，从所有的过程位置用户可以进入 |
| 客户参考 | 公司的合法名称 |
| 贷方标准说明 | 通过在挂号内的代码，用户可以编码得插入到文件的文章，作为银行收据的参考文档的说明。本文档的类型是 (3)的Ft. Nr. (2) |
| 插入最终控制字符 | 对某些电子银行的程序，文件的换行是必须的 |
| 无未付导入的参考 | 在某些电子银行的程序，按照银行收据的数据，系统自动地更新登记。导入未付的过程利用的字段，可以引起登记的不断更新。通过本标记用户可以避免本过程（因此，导入未付的过程不可利用） |
| 说明到期的管理 | 通过本标记，插入的客户说明由说明+第二说明组成 |
