功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存设置 |
| 取消设置 | 用于取消设置 |

为了打开帐单公文包的参数请按[主页>参数>现金]路径。

在帐单公文包的参数中用户可以设置默认的因素，在模块过程中被建议的。尤其：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 自动的会计记录 | 获取从发票／交易的账单的时候，用户预定义账单的会计记录 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 记帐模板 | 用于定义默认的记帐模板，在同名过程中利用的 |
| 按账户账单分组 | 用于定义账单分组的参数 |
| 临时记录 | 通过本过程被生成的记录包含临时记录的状态 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 记帐模板 | 用于定义默认的记帐模板，在同名过程中利用的 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 记帐模板 | 用于定义默认的记帐模板，在同名过程中利用 |
| 在到期记账 | 第一个选项，在多样帐单到日期分开清单的记录，添加在银行登记中设置的银行天 |
| 在到日期记账，银行货币日期=到期 | 第二个选项，在多样到日期记录多样帐单，可是银行的货币日期按照银行日期，在登记中设置的 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 记帐模板 | 用于定义默认的记账模板，在同名过程中被建议 |
| 佣金费用 | 用于定义佣金费用的明细账目，通过本过程实现的记录 |
| 应付费用 | 用于定义应付费用的明细账目，通过本过程实现的记录 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 分组小于...的总额 | 如果本字段不空或者与0不同，制定交易的阈值。如果用户设置1000，系统分组小于本价值的交易，因此到999,99. |
