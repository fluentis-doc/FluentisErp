功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存设置 |

为了打开供应商支付模块的参数请按照[主页>参数>现金]

在支付模块的参数中用户可以预设置默认的因素，被模块的过程利用的。尤其：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 生成Riba路径的 | 关于意大利 |
| ABI标准 | 设置点文件的格式。在这个情况下我们建议ABI标准。本标准按照标准的设置，被本机构定义的 |
| HR记录 | 点文件的格式设置 |
| ABI 应用中心 | 用于设置某个ABI代码以及应用中心。应用中心的责任是通信（普通地不是必须的字段）  |
| 设置在供应商登记中的参考银行 | 用户可以定义参考银行的使用，按照连接的登记默认被核查。属于银行登记的上网格中的默认参考银行复写插入到会计记录的参考银行 |
| Iban的管理 | 设置IBAN的管理的义务：在这个情况下，如果没有物料清单的某些支付，系统不生成路径  |
