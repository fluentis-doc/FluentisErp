title: 信用风险的参数
sidebar_position: 6

| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存设置 |

为了打开信用风险的参数请按[主页>参数>现金]

在信用风险的参数中用户可以预设置默认的因素，在模块的　过程中利用的。尤其：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 超过账单以到期付款为条件的过期天数 | 超过账单以到期付款为条件的过期天数 |
| 超过账单收账后的过期天数 | 超过账单收账后的过期天数 |
| 处理初始日期（帐户余额、发票、送货单） | 计算的初始日期。系统不考虑包含之前日期的文当 |
| 查看未付账目 |  通过本标记用户可以查看记账的未付总额，无论已接收的状态 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 记账余额 | 通过本设置，透支的计算按照客户的记账余额 |
| 未结的交易 | 通过本设置，透支的计算按照客户的记账余额 |
| 从临时记录考虑交易 | 如果上标记激活的话，计算也考虑从临时记录来的记录 |
| 阻止具体打印 | 如果客户的透支包含文档的阻止，本标记阻止打印文档的能力。如果本标记不激活的话，系统可以打印文档可是文档不包括“已打印”标记 |
