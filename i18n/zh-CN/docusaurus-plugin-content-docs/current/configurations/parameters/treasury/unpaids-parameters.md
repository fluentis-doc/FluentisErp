功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存 | 用于保存设置 |

 未付模块的参数属于 [主页>参数>现金]

在未付的参数中用户可以预设置默认的因素，在模块的过程中被建议的。尤其：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 临时账户 + 拒付费用 | 通过本本设置，在临时的账户，用户可以打开客户的交易，包含与起源帐单一样的总额，然后将费用插入到银行账户 |
| 客户 + 拒付费用 | 为了重新打开客户的交易，我们建议用户选择“客户+拒付费用”。总额等于起源帐单的总额，用户可以将费用插入到银行账户 |
| 重新打开与发票关联的交易 | 我们建议用户选择重新打开与发票关联的交易。通过本选项，过程打开新的交易，包含与起源发票一样的参考，文档编号和日期。在更多文档的总结帐单  的情况下，系统插入列表第一帐单，可是在交易的票据系统管理所有参考的列表 |
| 打开新交易 | 替代的选项，新交易的打开，生成新的交易。本交易的文档编号和日期按照未付帐单的编号和日期 |
| 添加相关发票的交易 | 系统不管理本字段 |
| 打开新交易 | 系统不管理本字段 |

如果用户需要某个得建议的未付费用的默认，得设置下面的部分。



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 发票类型 | 用于设置得利用的发票类型 |
| 支付类型 | 用于设置得利用的支付类型 |
| 计量单位 | 用于设置项目线的计量单位 |
| 支付的解决方案 | 用于设置支付的解决方案，因此生成有关的到期 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 开发票类型 | 用于设置销售营业额，就是 收益的抵消。通过本抵消用户可以核查操作 |
| 增值税率 | 用于设置增值税率 |

| --- | --- |
| 开发票的类型 | 系统不管理本字段 |
| 增值税率 | 系统不管理本字段 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 开发票类型 | 系统不管理本字段 |
| 增值税率 | 系统不管理本字段 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 会计模板 | 在会计未付的过程中，得利用的模板 |
| 计算拒付费用 | 用于核查银行的费用 |
| 临时账户 | 临时的明细账目。如果“临时账户+拒付费用”设置的话被会计记录利用 。我们建议用户填写本字段，因为会计记录的过程需要本字段的内容 |
| 银行临时账户 | 如果未付关于 过程不表示的帐单，用户得利用本账户（例如在利用DocFinance模块的情况下） |
| 用户的借记费用的账户 | 收益的明细账户，如果用户选择用户的借记费用的话，在会计未付  的过程中得利用明细账户 |
| 利息类型 | 未付与某个参考的利息率可有关。在本票的情况下，用户可以虚拟某个利息的计算，包括到日期和未付的核查日期 |
| 创建单个交易 | 为了某个被多样交易生成的未付打开单一的交易 |
