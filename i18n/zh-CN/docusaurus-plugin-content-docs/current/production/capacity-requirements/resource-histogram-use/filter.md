在本表格中，用户可以设置在直方图中要查看的计划和生产订单的搜索标准。表格包括设置筛选器的三个不同方式。通过属于窗户上部分的选择钮，用户可以使用“筛选器"（在这个情况下，属于窗户下部分的箱子开始工作）、”工作中心“选择钮（在这个情况下，包含工作中心的网格开始工作）以及所有的工作中心。最后的选择钮关闭两个部分，使用户自动地查看  属于公司数据库的所有工作中心的图形。    

用户可以利用下列的筛选器：

**申请类型：**在本部分的图形中，用户可以查看计划以及生产的订单，包括所有的已筛选工作中心或者有占用超出的中心（ 可用的工作数量比需要的工作数量少）或者有占用缺乏的（可用的工作比需要的工作数量高） 。 

**从日期….到日期….：**在这两个字段中，用户可以设置查看计划和生产订单的情况的时期。图形的大小按照日期的范围；

**年份 / 计划订单：** 这两个字段表示图形表示的计划订单年份和编号。通过本选择钮，用户可以利用按计划订单类型的筛选器。同时，同一部分包含的所有筛选器自动地结束工作。

**年份 /生产订单：**这两个字段表示在图形中用户要查看的生产订单的年份和编号。通过本选择钮，用户也可以利用生产订单类型的筛选器。同时，同一部分包含的所有筛选器自动地结束工作。

**年份 / 生产作业单：** 这两个字段表示在图形中用户要查看的生产订单的年份和编号。通过本选择钮，用户可以利用属于同一部分的筛选器，包括按项目的筛选器：

**分类/代码/说明：**第一组合框使用户设置项目的分类，双击于第二字段用户可以打开某个“帮助项目”。通过本工具用户可以选择单一的项目（代码和说明）；第一个组合框使用户设置项目的分类，双击于第二字段用户可以打开一个“帮助项目”。通过本工具用户可以选择单一的项目（代码和说明）；

**生产订单：这**三个标记使用户在图形中查看启动的、执行的以及/或者履行的生产订单；

**维护订单：**通过本标记，在图形中用户可以查看计划的维护订单；

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于直接地查看属于“直方图”表格的数据和图形，按照“筛选器”表格包含的筛选器 |
