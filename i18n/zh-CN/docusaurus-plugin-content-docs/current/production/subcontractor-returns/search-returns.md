为了打开搜索退回窗户请按照转包  > 退回 > 搜索退回 > 搜索退回。窗户表示 已插入的转包退回列表。用户可以编辑，查看以及插入新的转包退回。

窗户由筛选器区域和结果网格组成。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于已插入的转包退回的完全数据库 |
| 插入 | 用于插入新的转包退回 |
| 编辑 | 用于编辑某个转包退回 |
| 查看 | 用于查看某个转包退回 |
| 删除 | 用于删除已插入的转包退回  |
| 编辑属性筛选器 | 通过本窗户用户可以打开某个窗户。在这个窗户中用户可以启动属性筛选器。通过本工具，用户可以按插入的属性筛选转包退回  |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

用户可以利用下列的筛选器：

**转包商** **：**这两个字段包括退回的转包商数据：帐户、明细账目。转包商的合法名称自动地表出现。双击于第一个字段中，用户可以打开某个“帮助帐户”，通过本工具用户课题选择得筛选的转包商。普通地，转包商属于会计科目表，其中供应商；

**从送货单日期…到送货单日期：**i这些字段包括转包商的退回单日期（从....到）；

**从退回日期… 到退回日期…：**这些字段可以包括   从转包商的货物退回日期，与送货单日期不同。系统表示退回的仓库记录日期；

**退回类型：**本组合框可以包括退回类型，然后按照本标准筛选，因此查看属于选中类型的所有退回；

**退回送货单编号：**本字段包括退回送货单的编号然后按本标准筛选。货单编号的字段是字母的字段，不是数字的字段；

**作业单：**i通过合适的帮助，本字段可以包括销售作业单的编号；

**已计价：** 本标记使用户筛选已计价的退回就是已经有采购发票的退回；

**已装载：**本标记使用户筛选已装载到仓库的退回就是已经有记录退回操作的退回；

**所有：**使用户筛选所有的专包退回，无论已经装载以及/或者计价；

**搜索项目：**通过一些参数，用户可以筛选。利用的参数是下列的：

**分类/项目代码/说明：**第一个组合框使用户设置项目分类，双击于第二个字段用户可以打开某个“帮助项目”。通过本工具用户可以选择单一的项目（代码和说明）；

**作业单类型：**包括生产作业单类型。按照本标准用户可以筛选，因此查看与选中类型有关的所有退回；

**生产作业单：**表示生产作业单的编号和年份；

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的转包退回清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的转包退回清单。

网格包括的列：

**受控：**本标记使用户了解如果退回受控；如果退回未受控的话，系统不能完成的计价操作和记录操作；

****已计价：**** 通过本标记用户可以了解如果某个退回已计价；

****已装载：****用于了解如果某个退回已装载；

**货单：**表示退回单的编号； 

**送货单：** 表示退回单的日期；

**退回日期：**表示从转包商的货物退回；

**转包商：**表示转包商的帐户和明细账目；

**转包商说明：**表示退回的转包商的合法名称；

**总计：** la表示采购的总计（包括增值税率）。
