title: 从订单履行
sidebar_position: 3
---

为了打开本窗户请使用属于功能导航栏的“预计退回”按钮。通过本窗户，按照一些标准，用户可以筛选转包订单。

- 得退回的项目分类、代码、说明和变量；

- 项目退回的预计仓库；

- 转包订单类型；

- 预计退回的日期（到某个日期）；

- 转包订单的编号和年份；

- 与转包订单行有关的销售作业单；

下面的网格表示  每个 未履行/部分地履行的 转包订单的某个行；双击于订单行，用户可以展开转包商还没退回的行。用户可以选择行或者将数量插入到每个行。终于用户按属于功能导航栏的“转移”按钮。通过这种方式 ，过程将选中项目的数量转移到属于“退回项目”表格的网格。 

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索订单 | 用于将搜索筛选器应用于已插入/未履行/部分地履行的转包订单完全数据库 |
| 转移 | 将在下面窗户选中的项目转移到属于转包退回的“退回项目”表格 |
| 强制履行订单 | 用于强制履行订单 |
| 识别在列表选中的项目  | 使用户在下窗户的左部分中查看在右部分选中的项目 |
| 向上 / 向下 | 在右部分中，使用户转移到上层级/下层级 |
| 选择 | 在右部分中（包含得转移的项目），使用户选择所有的行 |
| 冻结订单以执行 | 用于冻结选中的转包订单：通过这种方式，用户不能接收属于本订单的项目 |
| 解冻订单 | 解选中的转包订单 |
