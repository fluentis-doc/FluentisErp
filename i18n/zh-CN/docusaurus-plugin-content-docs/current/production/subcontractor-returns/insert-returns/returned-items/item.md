本表格表示关于网格行的下列数据：

**分类、代码、项目说明和变量、订购数量、退回数量、剩余数量、仓库、模板、转包订单： **选中的退回行表示这些数据；

**预计退回日期：**本日期来自生成选中退回行的转包订单行；  

**生产订单：** 本生产订单的外部阶段生成 转包订单的行，被选中的退回行履行；

**装载批次：** 用户可以指定批次代码。仅当项目按批次被管理，通过本代码用户可以装载退回项目的数量；

**供应商批次：**在本字段中，用户可以指定供应商批次。通过本批次，用户得装载退回项目的数量，仅档项目按批次被管理；

**作业单：**表示与退回行有关的销售作业单。 普通地，系统撤回 属于生成退回行的转包订单行销售作业单；

**应纳税：** 表示行的总计，包括增值税的净额；

**增值税：**按照转包退回的供应商登记的增值税率，表示增值税的价值；

**交付总计：**表示交付总计，应纳税+增值税。
