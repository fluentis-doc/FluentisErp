为了插入新的转包退回请按照 转包  > 退回 > 搜索退回 > 新的退回的路径或者击于属于功能导航栏的“插入” 的图标。

首先，通过合适的“帮助帐户”（属于退回标头）用户得插入退回的转包商。指定转包商之后，用户可以将退回单的编号插入到“送货单编号”字段，将送货单日期插入到“日期”字段，将退回日期插入到“实际退回”字段（普通地与当前日期同一）以及退回类型。 

最后的字段，使  项目的仓库按照“退回类型”表格的内容被表示的。要不然，项目行表示的仓库被转包订单检索的。

退回标头也包括上列介绍的“受控”、“已计价”、“已装载”标记。用户一直手动地启动“受控”标记。用户开始执行文档价值和仓库记录的时候，系统自动地启动其他的标记。

插入标头数据之后，通过属于功能导航栏的合适“保存”按钮用户得保存文档。保存退回的同时，“预计退回”按钮开始工作。本按钮打开“从订单履行”窗户。在这个窗户中用户可以查看还包含未履行／部分地履行的行的所有转包订单。从这些行用户可以生成退回。

工作于本窗户，用户可以选择项目行然后将这些行插入到转包退回。在这个情况下，通过下列介绍的表格，用户可以关闭履行的窗户，再打开“新退回”窗户完成关于退回的数据（批次、序号、退回仓库和模板、加工成本等）。
