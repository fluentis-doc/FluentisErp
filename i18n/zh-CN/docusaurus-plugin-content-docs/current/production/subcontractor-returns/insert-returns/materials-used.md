本表格表示为了生产已插入到“退回的项目”表格的转包商得利用的物料。表格由标头（表示关于在“退回项目”表格选中行　　的数据）、网格（表示转包商使用的物料）、与上网格有关的其它网格（为了制定转包商使用的物料的批次）组成。

网格包括的列：

**分类、项目代码、变量、物料说明、项目变量的说明：**表示转包商使用的物料分离、代码、项目说明、变量以及变量的说明；

**选项：**表示项目行的选行；

**计量单位：** 表示转包商使用的物料的计量单位；

**已用数量：** 表示转包商使用的物料数量。用户将本数量从转包商仓库下载。这个可编辑的数量按照在“在转包订单得使用的物料”表格设置的单数量。数量相应于订单行，通过在“退回项目”表格选择的退回行被履行。

**退回数量：**表示转包商退回的物料数量。用户将本数量插入到退回的仓库；

**仓库：**表示退回仓库的代码，按照转包参数。仅当　用户将退回数量插入到物料行，系统自动地表示本数据。用户可以编辑本数据；

**模板：**表示退回装载的模板代码，按照转包商参数。仅当用户将退回数量插入到物料行，系统可以自动地表示本数据。用户可以编辑本数据；

**选项说明、仓库说明、模板说明：**表示在物料行中设置的选项说明、仓库说明以及模板说明；

在下面的网格中，用户可以指定转包商使用的物料批次。用户可查看的列以及网格使用规则与　属于退回项目表格的批次／序号表格的批次网格的同一。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 插入 | 用于插入新的退回：应用自动地保存刚插入的退回，同时删除窗户的所有字段内容，因此窗户可以包括新的退回 |
| 保存 | 用于保存退回的已完成的编辑 |
| 预计退回 | 用于展开“从订单履行”的窗户。将退回项目从转包订单转移　 |
| 预览 | 用于预览转包退回的报告  |
| 打印 | 通过预定义的打印机，启动转包退回的报告的打印 |
| 文档 | 用于进入文档管理同时表示与转包退回有关的文档 |
