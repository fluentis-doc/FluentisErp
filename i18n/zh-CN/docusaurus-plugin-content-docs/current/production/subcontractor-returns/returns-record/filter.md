title: 筛选器
sidebar_position: 2
---

在这个表格中，按照转包商、退回单日期（从/到）、退回单编号以及物料清单的版本，用户可以筛选转包的退回。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 将搜索筛选器应用于插入到，受控以及未记录的转包退回的完全数据库 |
| 缺载 | 用于记录退回以及有关的记录 |

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的转包退回清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的转包退回清单。

网格表示的列：

**帐户：**表示转包退回的转包商帐户；

**明细账目： **表示转包退回的转包商明细账目； 

**说明：**表示转包商的合法名称；

**退回编号：**表示转包退回的编号；

**退回日期：**表示转包退回的日期； 

窗户的下部分表示某个日期字段叫“仓库记录的日期”，默认地包含当前日期。用户可以编辑仓库记录的日期。

选择得记录的转包退回之后，通过属于功能导航栏的“缺载”按钮，用户可以执行仓库记录。同时，应用将转包商生产的项目装载到仓库，从转包商仓库 下载利用的物料，将退回装载到退回仓库。
