title: 从订单履行
sidebar_position: 4
---

为了打开本窗户请使用属于功能导航栏的“预计交货”按钮。按照下列的标准用户可以筛选转包订单：

- 得插入的项目分类、代码、说明和变量；

- 项目的仓库；

- 转包订单的类型；

- 交货的预计日期（到某个日期）；

- 转包订单的编号和年份；

- 与转包订单行有关的销售作业单；

下面的网格表示每个  还没履行或者部分地履行的转包订单行。双击于订单行之后，用户可以展开还没发送到转包商的行。用户可以选择行或者将数量插入到每个行中，终于按属于功能导航栏的 “移动”按钮。通过这种方式，过程将 在“转包货单的交货”网格选中 的项目数量移动。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索订单 | 将搜索筛选器应用于  已插入以及未履行（或者部分地履行）的转包订单的完全数据库 |
| 移动 | 将在下窗户中选中的项目移动到“送货单”表格 的网格 |
| 强制履行订单 | 用于强制履行用户在转移的转包订单行 |
| 在列表中查找选中对象 | 下面窗户的左部分表示在右部分选中的项目  |
| 向上/向下 | 在右部分中，使用户移动到上/下层级 |
| 选择 | 使用户选择包括得移动的窗户的右边的所有行 |
| 冻结订单以执行 | 用于冻结选中的转包订单：通过这个方式，属于本订单的项目不能被插入到这个订单 |
| 解冻订单 | 解冻结转包订单 |
