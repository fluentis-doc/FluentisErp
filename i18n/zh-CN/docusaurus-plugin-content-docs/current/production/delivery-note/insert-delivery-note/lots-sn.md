---
title: 批次/序号
sidebar_position: 3
---

在本表格中，关于选中的行，用户可以设置从仓库得缺载的批次。如果同一的货单行 领不同批次的话，表格包括得寄给转包商的数量；

批次网格使用户按照合适的“帮助批次”领批次。“帮助批次”的工具筛选在上网格中选中项目的可用批次。批次网格表示的数据是：批次代码类型、批次代码、供应商批次、批次类型、初始日期、到期、批次数量、位置、批次供应商的数据（帐户、明细账目和合法名称）。多样批次数量的总计得等于项目行的已交数量。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 插入 | 用于插入新的送货单：应用自动地保存 刚被插入的送货单同时  删除 所有的窗户字段的内容，因此窗户可接收新的货单 |
| 保存 | 用于保存已完成的货单编辑 |
| 预计交货 | 用于展开从订单履行的窗户。使用户将货单项目从转包订单移动 |
| 履行作业单 | 用于将项目从销售作业单移动 |
| 插入支付 | 将新的支付行插入到合适的“支付”网格 |
| 删除支付 | 在合适的“支付”表格删除某个支付行 |
| 插入物料 | 将新的物料插入到属于“交货”表格的项目网格 |
| 删除物料 | 用于删除属于“交货”表格的项目网格的物料 |
| 预览 | 用于预览送货单 |
| 打印 | 通过预定义的打印机，打印送货单的报告 |
| 文档 | 用于输入文档的管理同时查看与送货单有关的文档 |
