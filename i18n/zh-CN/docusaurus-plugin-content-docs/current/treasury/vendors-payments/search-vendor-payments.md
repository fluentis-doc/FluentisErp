功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索支付 | 在筛选器中按需要设置了相应筛选条件后，使用该按钮在供应商支付的数据库内进行搜索 |
| 新供应商支付 | 使用该按钮插入新的供应商支付 |
| 修改选中支付 | 选中并激活之前已插入的支付后，使用该按钮修改支付 |
| 查看选中支付 | 选中并激活之前已插入的支付后，使用该按钮查看支付 |
| 删除选中支付 | 选中并激活之前已插入的支付后，使用该按钮删除支付 |

 **与报告和过程相关的信息请参见相应的菜单条目**



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [打印供应商支付i](/docs/treasury/vendors-payments/reports)  | 用于打印相关的供应商支付 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [冻结应付账款](/docs/treasury/vendors-payments/procedures/block-passive-maturity-values)  | 该按钮用于管理对应付账款冻结 |
|  [自动创建供应商支付](/docs/treasury/vendors-payments/procedures/automatic-vendor-payments-creation/parameters-filter)  | 该按钮用于自动创建供应商支付 |
|  [银行支付管理](/docs/treasury/vendors-payments/procedures/payments-support-management)  | 该按钮用于管理银行支付 |
|  [记账](/docs/finance-area/professional-men/accounting/payments-accounting/filter)  | 该按钮用于供应商支付的记账 |

**筛选区域**：筛选区域包含可以对某一选中进行搜索的多种数据类型。筛选器具有高级筛选功能，可以同时指定多个筛选条件。

**结果网格**：结果网格包含按照指定条件筛选出的供应商支付。首先用户必须指定对供应商支付进行搜索的条件，然后在功能导航栏中单击“搜索”，最后在网格中将以列表形式显示筛选出的供应商支付。
