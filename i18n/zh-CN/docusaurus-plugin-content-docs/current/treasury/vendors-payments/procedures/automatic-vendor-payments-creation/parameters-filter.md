第一个表格中的掩码代表一系列搜索条件：银行的ABI/ CAB代码也可以用作筛选条件。但是，如果受益人希望获得扣减证明，则只能在受益人模块中进行到期值支付。银行明细账目的筛选可以根据登记中多家银行的支付清单和银行管理过程管理创建。

第一个选项卡包含选中行的总计字段。

功能导航栏：功能导航栏为菜单窗体，包含可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索交易 | 指定筛选后，用于激活交易搜索 |
| 开始创建由到期值支付 | 供应商支付数据选项卡中的活动按钮 |

筛选区域：筛选区域包含可以对某一选中进行搜索的多种数据类型。筛选器具有高级筛选功能，可以同时指定多个筛选条件。

结果网格：结果网格包含按照指定条件筛选出的到期值。首先用户必须指定对到期值进行搜索的条件，然后在功能导航栏中单击“搜索”，最后在网格中将以列表形式显示筛选出的到期值。
