在该部分，可以保存背书人和保证人的参考。该选项卡没有特殊的管理。

功能导航栏：功能导航栏为菜单窗体，包含可能与使用中的固定资产相关的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存账单 | 保存打开的账单 |
| 新账单 | 创建新账单 |
| 插入到期 | 激活“到期参考”选项卡时，该按钮也将被激活，可以打开到期帮助，以关联发票到期和打开的账单 |
| 删除到期参考 | 激活“到期参考”选项卡时，该按钮也将被激活，可以取消发票到期和打开账单的关联 |
| 插入交易 | 激活“到期参考”选项卡时，该按钮也将被激活，可以打开到期帮助，以关联发票到期和打开的账单 |
| 从账单删除到期值 | 激活“到期参考”选项卡时，该按钮也将被激活，可以取消发票到期和打开账单的关联 |
| 文档 | 管理与打开掩码关联的文件存档 |
