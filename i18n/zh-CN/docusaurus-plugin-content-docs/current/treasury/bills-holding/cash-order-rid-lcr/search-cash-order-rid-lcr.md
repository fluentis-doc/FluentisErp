搜索现金汇票/RID/LCR使用户轻松地搜索在账单公文包中创建的电子文件，并查看或者删除这些文件，再重新启动新文件的创建。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索现金汇票 | 搜索已创建的电子文件 |
| 查看现金汇票 | 根据设置的筛选条件搜索创建的文件 |
| 删除现金汇票 | 删除“文件已发布”的状态标记，并询问是否确认从保存路径删除文件。 |
