---
title: 清单中的账单
sidebar_position: 3
---

保存清单后，将激活多个清单管理按钮：特别是账单获取按钮，它将打开搜索掩码并搜索已发布但还不在清单中的账单。通过该掩码，可以在清单中插入拥有完整ABI/CAB代码信息的账单；还可以生成不完整账单的报告。使用该掩码执行的账单插入可以更新整个清单。如果要从清单中删除一个账单，只需选中它，然后单击删除按钮。

功能导航栏：功能导航栏为菜单窗体，包含该程序中可能需要的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存清单 | 保存打开的清单 |
|  [获取账单](/docs/treasury/bills-holding/lists-creation-and-managament/bills-acquisition)  | 打开程序，筛选待插入清单的账单 |
| 删除账单 | 从打开的清单网格中，删除选中的账单 |
| 打印预览 | 用于清单的打印预览 |
| 打印 | 开始打印 |
| 记账和结账 | 清单打印后才能激活该按钮，用于在当前日期使用模块参数中的记账模板对清单记账 |
