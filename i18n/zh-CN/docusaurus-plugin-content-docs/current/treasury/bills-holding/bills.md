title: 搜索账单
sidebar_position: 3

搜索账单的程序使用户轻松搜索已发布的应收账单，以及查看、修改或删除这些账单。授权用户通过搜索账单可以进行所有的文档操作。

账单搜索程序的组成方式如下：

功能导航栏：功能导航栏为菜单窗体，包含对已有文档而不是新建文档进行的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索账单 | 用于在已插入的账单数据库中按筛选条件搜索 |
| 新账单 | 用于插入新账单 |
| 修改账单 | 用于修改已插入的账单 |
| 查看账单 | 用于查看已插入的账单 |
| 删除账单 | 用于删除已插入的账单 |

AREA DI FILTRO: l'area di filtro è quella dedicata ad ospitare l'elenco dei possibili tipi di dati grazie ai quali è possibile effettuare una selezione, una ricerca sul totale degli effetti inseriti. I Filtri sono tipicamente sempre in condizione 'AND' ed è possibile specificare più criteri di filtro contemporanei.

GRIGLIA DI RISULTATO: la griglia di risultato rappresenta l'elenco degli effetti che corrispondono ai dati di filtro sopra specificati. L'utente, dopo aver specificato i valori attraverso i quali desidera ottenere una ricerca degli effetti, premendo il pulsante [Ricerca Effetti] presente nella relativa Ribbon, otterrà nella griglia di risultato l'elenco degli effetti desiderati.
