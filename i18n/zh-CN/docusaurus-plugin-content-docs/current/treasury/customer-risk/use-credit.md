该掩码用于保存每项计算和历史查看。掩码的计算查看列表将用红色标记差异行。

功能导航栏：功能导航栏为菜单窗体，包含对已有文档而不是新建文档进行的操作。当前选项卡的功能如下：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索 | 用于在透支计算数据库内按筛选条件搜索 |
| 新的 | 用于插入新的透支 |
| 修改 | 用于修改已插入的透支计算 |
| 查看 | 用于查看已插入的透支计算 |
| 删除 | 用于删除已插入的透支计算 |
| 预览 | 用于打印结果网格中突出的计算详细 |
| 打印 | 用于打印结果网格中突出的计算详细 |

筛选区域：筛选区域包含可以对所有插入的计算进行搜索的数据类型。筛选器具有高级筛选功能，可以同时指定多个筛选条件。

结果网格：结果网格包含按照指定条件筛选出的信用额度清单。首先用户指定对信用额度进行搜索的条件，然后在功能导航栏中单击“搜索”，最后在网格中将以列表形式显示筛选出的信用额度。
