---
title: 导入往来账户的转移
sidebar_position: 2
---

 

该掩码用于从电子文件（标准ABI）和网上银行导入往来账户的转移。

若要执行导入，首先必须根据掩码设置希望导入文件的明细账目，然后单击“执行导入”，打开电子文件的开账掩码。之后，网格中将出现新的导入行：双击该行，或者单击“查看文件”以查看导入数据的源文本文件。

导入的转移可以用于往来账户的对账。

导入往来账户转移的过程如下：

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索导入 | 用于在导入中按条件进行搜索 |
| 执行导入 | 用于选择电子文件并导入 |
| 查看文件 | 用于查看与选中导入关联的电子文件 |
| 删除导入 | 用于删除选中的导入 |

**筛选区域**：筛选区域包含可以对所有导入进行搜索的数据类型。筛选器具有高级筛选功能，可以同时指定多个筛选条件。

**结果网格**：结果网格包含按照指定条件筛选出的导入。首先用户指定对导入进行搜索的条件，然后在功能导航栏中单击“搜索”，最后在网格中将以列表形式显示筛选出的导入。
