---
title: 搜索预付款清单
sidebar_position: 5
---

搜索预付清单的过程为容易地搜索已插入的清单而设计的。通过本过程用户可以查看、编辑以及删除预付款清单。通过“搜索”用户可以完成所有的必须操作。相当的权利是必须的。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 搜索清单 | 将搜索筛选器应用于已插入的预付清单的完全数据库 |
| 新清单 | 用于插入新的预付清单 |
|  编辑清单 | 用于编辑已插入的预付清单 |
| 查看清单 | 用于查看已插入的预付清单 |
| 删除清单 | 用于删除已插入的预付清单 |

为获得关于下列的报告和过程的数据，请按照相当的菜单项。



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  [打印](/docs/treasury/advances-collections/reports)  | 用于打印预付/已收的发票清单 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
|  管理收款  | 用于管理收款 |
|  [收款预付的会计记录](/docs/treasury/advances-collections/accounting/accounting)  | 用于会计记录预付款 |

**筛选器的区域**：包含使用户做选择的数据类型清单。典型地，这些筛选器的状态是“AND”而且在同一时间内 用户可以确定多类标准的筛选器。

**结果** **网格**：结果网格表示相当于已确定的筛选器数据的预付款清单。搜索参数确定了以后，通过掩码中的搜索按钮，用户能够获得需要的预付款清单。
