---
title: 创建预付清单
sidebar_position: 1
---

预付清单的标头首先涉及得使用的清单类型：下一个的数字，年份，系统当前日期（表示日期以及发布日期）是自动地被分配。

在**银行经常帐户**字段中，用户得指定表示预付发票的银行登记的明细账目。设定清单的明说之后，用户也得指定清单的状态。状态的管理只可以是直接的。在“发布的清单”或者“表示的清单”中用户可以设定清单的状态。这个很重要数据确认之后，系统开始保存标头数据以及启用 以下的 选择按钮。通过这些按钮用户可以利用属于销售区域的发票或者属于发票的到日期：   为了管理某个清单用户只可以使用 帮助发票以及帮助到日期， 不能从这两个工具获得数据。   

掩码包括某个预览报告以及打印的按钮。

功能导航栏：功能导航栏包含格式菜单，在这里可以执行相关过程的操作。包含的功能如下表：



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 保存预付 | 完成标头数据之后，保存预付清单 |
|  [获取发票](/docs/treasury/advances-collections/create-advance-list/create-advance-list)  | 用于从销售区域获取发票。双击于选中的发票，用户可以编辑它 |
|  从到日期获取| 从发票获取单一的到日期 |
| 删除发票 | 用于删除在清单中已选中的发票   |
| 预览已插入的发票 | 用于预览清单 |
| 打印发票 | 用于打印清单 |



| RibbonFunction | RibbonMeaning |
| --- | --- |
| 文档经理 | 为文档管理关联的按钮  |
