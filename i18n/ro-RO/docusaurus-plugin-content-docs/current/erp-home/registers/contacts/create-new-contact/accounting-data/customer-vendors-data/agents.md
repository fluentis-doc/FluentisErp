title: Agenți
sidebar_position: 8
---

Acest tab este prezent doar în registre client.

Aici se inserează agenții care gestionează clientul din punct de vedere comercial.

Pot fi prezenți mai mulți agenți pentru că ar putea fi un agent și un manager de arie care primesc comisoane pentru același articol, cât și pentru că urmăresc mărci și modele diferite și deci, pe baza articolului vândut poate fi un singur agent de referință.

În câmpul **Procent** se poate insera un procent din comisionul agentului (în registrul agentului puteți vedea detaliile combinațiilor pentru calculul comisioanelor).
