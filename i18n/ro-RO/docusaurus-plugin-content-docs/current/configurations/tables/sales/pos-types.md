Tabelul se află pe calea **Configurare > Tabele > Vânzări > Tipuri POS**.  

Permite inserarea datelor noi sau căutarea celor existente pentru a le vizualiza, modifica sau șterge.

Conține o zonă de filtrare și una de rezultate. După ce ați setat toate filtrele dorite, efectuați clic pe butonul **Caută** pentru a vizualiza datele în tabelul de rezultate.

Pentru a insera datele, efectuați clic pe primul rând gol din tabel sau apăsați butonul **Nou**.

*Câmpuri specifice*

**Tip** și **Descriere**: reprezintă codul și descrierea tipului POS. 
**Default**: dacă este setat, indică tipul POS-ului implicit care este propus la crearea unui nou POS.  
**Note**: reprezintă eventuale note ale POS-ului, care vor fi utilizate în rapoarte.  
**Preț cu TVA**: dacă este setat, prețurile prezente în POS includ TVA.  
**Acont**: dacă este setat, POS-ul este de tip acont și va putea fi schimbat într-un POS normal.  
**Server**, **Database** și **Tabel**: server/Database/Tabel unde se citesc și/sau scriu datele POS-ului.    
**Tabel conexiuni documente**: tabel unde se citesc și/sau scriu datele documentelor asociate POS-urilor.  
**UM sosire**: UM utilizată la crearea de avize de vânzare și facturi din POS.  
**TVA sosire/Descriere TVA**: codul TVA utilizat la crearea de avize de vânzare și facturi din POS.  
**Punct de vânzare**: reprezintă punctul de vânzare în care sunt create POS-urile.  
**Casa**: reprezintă codul casei care va fi conectată la POS-uri.  
**Cat. merc.**: reprezintă categoria merceologică.  
**TVA**: reprezintă codul TVA care va fi utilizat în POS-uri.  
**Gestiune**: reprezintă gestiunea utilizată pentru mărfurile din POS de acest tip.  
**Șablon**: reprezintă șablonul pentru înregistrarea mărfii care se află în POS-uri de acest tip.  
**Tip plată**: reprezintă tipul de plată propus în POS-uri de acest tip.  
**Soluție plată**: reprezintă soluția de plată propusă în POS-uri de acest tip.  
**Ofertă**:  reprezintă codul ofertei generale din care sunt preluate prețurile articolelor de inserat în POS.  
**Dată valabilitate**:  reprezintă data valabilității ofertei generale.  
**Descărcare automată la închiderea zilnică**: dacă este setat, când POS-ul este listat, este automat descărcat din gestiune.  
**Necesită casă de marcat**: dacă este setat, pentru a utiliza POS-urile este necesară conexiunea la o casă de marcat pentru a lista bonul fiscal.  
**Vizualizează variante**: dacă este setat, pot fi utilizate și variantele articolului.  
**Vizualizează atribute**: dacă este setat, pot fi utilizate și atributele articolului.
