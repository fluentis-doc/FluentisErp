---
title: Tip oportunități
sidebar_position: 3
---

Este un tabel descriptiv pentru catalogarea **Tipurilor de oportunități** din CRM.  
Este comun tuturor companiilor prezente în baza de date și constă dintr-un tabel cu cod/descriere, precum și cu un *Pipeline flux* care este propus automat la inserarea oportunității.  
Se compune dintr-o zonă de filtrare și un grid cu rezultatele tipurilor inserate.
