title: Concurenților
sidebar_position: 3

Este un tabel descriptiv pentru catalogarea **Concurenților**, pe care îl puteți utiliza în **Contacte CRM**. Este comun tuturor companiilor prezente în baza de date și constă dintr-un tabel simplu cu cod/descriere.
Se compune dintr-o zonă de filtrare și un grid cu rezultatele concurenților inserați.
