Este un tabel descriptiv pentru catalogarea **Sectoarelor de utilizare**, pe care le puteți utiliza în **Contacte CRM**. Este comun tuturor companiilor prezente în baza de date și constă dintr-un tabel simplu cu cod/descriere.
Se compune dintr-o zonă de filtrare și un grid cu rezultatele categoriilor inserate.
