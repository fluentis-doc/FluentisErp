sidebar_position: 8

Este un tabel descriptiv pentru catalogarea tipurilor de note pe care le puteți utiliza în **Contacte CRM**. Este comun tuturor companiilor prezente în baza de date și constă dintr-un tabel simplu cu cod/descriere.
Se compune dintr-o zonă de filtrare și un grid cu rezultatele notelor inserate.
