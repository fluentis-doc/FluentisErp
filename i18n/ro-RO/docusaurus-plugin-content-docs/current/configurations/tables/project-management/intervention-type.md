Tabelul se află pe calea **Configurare > Tabele > Management proiecte > Tipuri intervenție**.

Permite inserarea înregistrărilor noi sau căutarea celor existente pentru a le vizualiza, modifica sau șterge.

Conține o zonă de filtrare și una de rezultate. După ce ați setat toate filtrele dorite, efectuați clic pe butonul **Caută** pentru a vizualiza datele în tabelul de rezultate.

Pentru a insera datele, efectuați clic pe primul rând gol din tabel sau apăsați butonul **Nou**. 

*Câmpuri specifice*

**Tip** și **Descriere**: reprezintă codul și descrierea tipului intervenției;  

**Tip factură**: reprezintă tipul de factură asociată;

**Stare** și **Descriere**: reprezintă starea și descrierea tipului intervenției: În curs, Terminată, Controlată, Suspendată, etc.;

**Operator**: permite alegerea operatorului implicit cu relativul **Nume** și **Prenume**;

**Cont sintetic** și **Cont analitic**: permite alegerea unui client implicit dacă intervenția este creată de la început;

**Preț cu TVA**: dacă este activ, intervenția include prețuri cu TVA;

**Extern**: dacă este activ, intervenția este externă;

**Materiale**: dacă este activ, intervenția gestionează și materiale;

**În activitate**: dacă este activ, intervenția va merge în activități;

**Cheltuieli deplasare**: dacă este activ, vor fi gestionate și cheltuielile de deplasare;

**Creează activitatea automat**: dacă este activ, schimbarea stării intervenției (din *Inserată* în *Controlată*) va genera activitatea automat;

**Proiect obligatoriu**: dacă este activ, prevede inserarea proiectului pentru acest tip de intervenție;

**Categorie activitate obligatorie**: dacă este activ, prevede inserarea categoriei activității pentru acest tip de intervenție;

**Registru instalații este obligatoriu**: dacă este activ,prevede inserarea l’anagrafica impianti pentru acest tip de intervenție;

**Resursa este obligatorie**: dacă este activ, prevede inserarea resursei pentru acest tip de intervenție;

**Progresul este obligatoriu**: dacă este activ, prevede inserarea procentului de avansare pentru acest tip de intervenție;

**Facturare vânzare**: dacă este activ, prevede inserarea facturării pentru acest tip de intervenție;

**TVA obligatoriu**: dacă este activ, prevede inserarea TVA pentru acest tip de intervenție.
