---
title: Tipuri control articole
sidebar_position: 9
---

Tabelul se află pe calea **Configurare > Tabele > Calitate > Control articol > Tipuri control articole** și permite codificarea tipurilor de control ale articolelor.

*Câmpuri specifice*      
    
**Gestiune**: reprezintă gestiunea articolului pentru care se realizează controlul;       
**Șablon descărcare articole neconforme**: dacă materialul este considerat Neconform la teste, este utilizat acest șablon pentru descărcare. Deobicei acest șablon este setat cu contrapartida de încărcare în gestiunea pentru deșeuri;             
**Șablon descărcare articole conforme**: dacă materialul este considerat Conform la teste, este descărcat din Gestiune cu acest șablon;  scaricato dal Magazzino con questa causale. Deobicei acest șablon este setat cu contrapartida de încărcare în gestiunea principală pentru că a trecut de controale.
