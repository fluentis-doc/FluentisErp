Con questi parametri vengono decise le regole di raggruppamento per la procedura:  
- **Creazione ordine di vendita da offerte**  


Cu acești parametri se stabilesc regulile de grupare pentru procedura:  
-  **Creare ordine de vânzare din oferte**.  

Este necesară selectarea coloanelor care prevăd grupările dorite pentru aceste proceduri. 
Opțiunea **Rezumat** este esențială pentru a crea un document din mai multe documente. Celelalte opțiuni se referă la grupările particulare.
