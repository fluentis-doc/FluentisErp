---
title: Parametri grupare facturi
sidebar_position: 8
---

Cu acești parametri se stabilesc regulile de grupare pentru procedurile:  
-  **Valorizare aviz de vânzare**;  
-  **Valorizare intervenții**;  
-  **Creare factură de vânzare din Liste ieşiri UDI**;  
-  **Creare factură de vânzare din Dispoziție de livrare**;  
-  **Creare factură de vânzare din POS**.  

Este necesară selectarea coloanelor care prevăd grupările dorite pentru aceste proceduri. 
Opțiunea **Rezumat** este esențială pentru a crea un document din mai multe documente. Celelalte opțiuni se referă la grupările particulare.
