---
title: Tipuri de mișcări
sidebar_position: 2
---

Tipurile de mișcări determină opțiunea activată:

**Încărcare**: funcție WMS Mobile permite crearea unei înregistrări de gestiune de încărcare;

**Descărcare**: funcție WMS Mobile permite crearea unei înregistrări de gestiune de descărcare;

**Declarare producție**: funcție WMS Mobile permite declararea producției unui produs finit sau semifabricat;

**Ieșire producție**: funcție WMS Mobile permite descărcarea din gestiune a materialelor necesare pentru producție;

**Retur producție**: funcție WMS Mobile permite declararea retururilor din producție cu înregistrarea de încărcare în gestiune;

**Rectificare de încărcare**: funcție WMS Mobile permite efectuarea unei rectificări de gestiune pozitivă;

**Rectificare de descărcare**: funcție WMS Mobile permite efectuarea unei rectificări de gestiune negative;

**Încărcare UdI**: funcție WMS Mobile și ERP permite efectuarea încărcării în gestiune a paletului;

**Descărcare palet**: funcție WMS Mobile și ERP permite efectuarea descărcării din gestiune a paletului;

**Reintrări**: funcție WMS Mobile pentru gestionarea reintrării materialelor;

**Confirmare transfer**: funcție WMS Mobile pentru confirmarea dispozițiilor de livrare pentru transfer;

**Atribuie locația**: funcție WMS Mobile care descarcă articolul dintr-o locație implicită și încarcă articolul în locația dorită;

**Mută articole**: funcție WMS Mobile permite mutarea articolelor dintr-o locație în alta. Șablonul de gestiune de inserat trebuie să fie de descărcare cu contrapartidă;

**Achiziție**: funcție WMS Mobile pentru a crea avizul de achiziție dintr-un ordin client și relativa încărcare în gestiune a articolelor;

**Mută lotul expirat**: funcție WMS Mobile permite mutarea loturilor expirate;

**Acceptă materiale gestionate pe loturi**: funcție WMS Mobile pentru a crea avizul de achiziție dintr-un ordin client și relativa încărcare în gestiune a articolelor gestionate cu loturi;

**Înlătură din UdI**: funcție WMS Mobil permite ștergerea articolelor dintr-un UdI creat anterior;

**Vânzări**: funcție WMS Mobile pentru a crea avizul de vânzare dintr-un ordin client sau manual și relativa descărcare din gestiune a articolelor;

**Dispoziție livrare**: funcție WMS Mobile pentru a crea o nouă dispoziție de livrare sau pentru a confirma unul creat anterior;

**Inventar UdI**: funcție WMS Mobile pentru a crea un inventar UdI;

**Transfer UdI**: funcție WMS Mobile pentru a crea sau confirma listele de transfer UdI;

**Încărcare UdI din creare UdI**: funcție WMS Mobile și ERP pentru a crea UDI și relativa încărcare în gestiune.
