**Default Ledger Account Style (1Edit/3Edits)** (cod FS-WPF-UIS-WS-DefaultLedgerAccountWidget): se referă la activarea, pentru conturile analitice, a widgetului unic sau a celui cu 3 câmpuri editabile.

**Default Ledger Cost Center Style (1Combo/2Edit)** (cod FS-WPF-UIS-WS-DefaultLedgerCostCenterWidget): se referă la activarea, pentru centre de cost, unui edit de filtru sau un combo-box.
