---
title: Parametri plăți furnizori
sidebar_position: 3
---

Parametrii modulului plați către furnizori se găsesc în **Parametri > Trezorerie**.

În parametrii modulului plăți furnizori se pot predefini elementele implicite care vor fi propuse în procedurile modulului.

**Cale creare fișier**: permite inserarea folderului standard pentru crearea căii electronice pentru viramente. Recomandăm crearea unui folder partajat în rețea pentru a putea fi accesat din toate locațiile procedurii.  
**Impune banca principală din registrul furnizorului**: permite utilizarea băncii principale inserate în registrul furniorului (tabel *Bancă principală*) în locul băncii din înregistrarea contabilă, asociată angajamentului deschis cu ajutorul tabelului *Tipuri plăți* din registru. Parametrul este vizibil în procedura *Creare automată plăți furnizori*. Nu este însă vizibil dacă se creează o listă nouă de plată(Plăți furnizori > Creare plată furnizor) motiv pentru care este gestionat de aici.
