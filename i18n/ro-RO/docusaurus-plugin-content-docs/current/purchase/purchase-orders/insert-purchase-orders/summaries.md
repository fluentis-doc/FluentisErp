**Discounturi finale**: pot fi inserate numai discounturile exprimate in procente. Discounturile inserate aici sunt apoi atribuite fiecarui articol din document.

**Rezumat TVA**: este propus rezumatul TVA al documentului, structurat pe cote TVA.

**Cheltuieli**: sunt propuse cheltuielile inserate in registrul furnizorului, in tab "Cheltuieli/Discounturi". De asemenea, pot fi inserate si alte cheltuielile care se refera doar la documentul curent.

**Note finale**: este un camp care poate fi completat manual sau apeland "Help note codificate ".

**Stare ordin**: este afişată starea ordinului ca în antetul ordinului; în plus este şi data executării şi data confirmării ordinului.

**Total document**

**Valoare bruta articole**: rezulta din insumarea valorilor tuturor articolelor;

**Total discounturi aplicate**: reprezinta valoarea totala a discounturilor aplicate fara discounturile numite 'finale'.

**Valoare neta articole**: Valoare bruta articole - Total discounturi aplicate;

**Total discounturi finale**: reprezinta valoarea discounturilor numite 'finale';

**Total cheltuieli nedocumentate**: reprezinta valoarea cheltuielilor inserate ca articole de tip “Cheltuieli”;

**Total cheltuieli documentate**: reprezinta valoarea cheltuielilor inserate in gridul Cheltuieli din Rezumatul documentului;

**Baza de impozitare**: Valoare neta articole - Discounturi finale + Total cheltuieli nedocumentate + Total cheltuieli documentate;

**TVA**: rezulta din insumarea valorilor cuprinse in Rezumatul TVA;

**Total**: Baza de impozitare + TVA.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Salvează | Buton pentru salvarea operaţiilor efectuate. |
| Sterge discounturi finale | Permite stergerea discounturilor din tabelul corespunzator. |
| Sterge cheltuieli | Permite stergerea cheltuielilor din tabelul corespunzator. |
| Vizualizează | Permite  vizualizarea documentului. |
| Listeaza | Permite listarea documentului. |



| Functie | Functionalitate |
| --- | --- |
| Document manager | Procedura da posibilitatea de a atasa documente. |
