---
title: Preia articole din comenzi de achiziţie
sidebar_position: 5
---

Acestă procedură permite executarea chiar şi parţială a comenzii de achiziţie. Sunt propuse toate comenzile neexecutate, neexecutate forţat şi parţial executate.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Căutare | Permite căutarea comenzilor neexecutate şi neexecutate forţat. |
| Blochează comanda  | Permite blocarea comenzilor pentru executare. |
| Deblochează comanda | Permite deblocarea comenzilor pentru executare. |
| Transfer articole | Transferă în ordin liniile de comandă selectate. |
| In sus | Urcă un nivel în listă. |
| In jos | Coboară un nivel din listă. |
| Executarea fortata comandă | Permite executarea fortata a comenzii în timpul transferării ordinului. |
| Vizualizare comandă | Permite vizualizarea comenzii. |
| Vizualizare stare comandă | Permite vizualizarea stării executării comenzii selectate. |
| Legenda | Legenda cu simbolurile utilizate. |
| Pozitionarea obiectelor din lista | Permite pozitionarea obiectelor din listă. |
| Vizualizeza/Ascunde parintele ramurei | Permite vizualizarea sau ascunderea parintelui ramurei. |
| Vizualizeaza/Ascunde arborele comenzilor | Permite vizualizarea sau ascunderea arborelui comenzii. |
| Selectează tot | Selectează toate obiectele în arborele de vizualizare a listei. |
| Șterge setările registrului | Permite ștergerea setărilor registrului. |
