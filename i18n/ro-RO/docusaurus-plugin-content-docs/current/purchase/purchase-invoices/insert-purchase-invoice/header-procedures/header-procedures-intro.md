---
title: Creare factura din ordine furnizor
sidebar_position: 2
---

Procedura ofera posibilitatea crearii unei facturi dintr-un Ordin Furnizor. Pentru ca procedura sa poata functiona este necesar ca furnizorii din ordin si factura sa fie aceiasi; ordinul trebuie sa fie aiba flag-ul Listat si Data confirmarii.

Aceasta procedura implica preluarea tuturor datelor prezente in ordin, si, ca urmare, sunt aplicate conditiile de achizitie prezente in ordin, chiar daca acestea acum sunt schimbate. In fereastra sunt propuse toate ordinele furnizorului selectat care au fost listate, confirmate, neexecutate sau partial executate. In acest moment aveti posibilitatea sa selectati ordinul in totalitate (marcarea flag-ului de pe linia ordinului) si sa importati ordinul in factura cu ajutorul butonului de transfer  sau selectati doar o parte din articolele ordinului, eventual, cu cantitati diferite. Pentru aceasta executati dublu clic pe linia ordinului, pentru a deschide liniile ordinului, si selectati intreaga cantitate a liniei respective (marcati flag-ul din casuta corespunzatoare), sau, pozitionati-va pe linia articolului si schimbati campul ‘cantitate' pentru a executa partial cantitatea. Si in acest caz transferul datelor in factura se face cu ajutorul butonului de transfer.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile sunt enumerate dupa cum urmeaza:



| Functie | Functionalitate |
| --- | --- |
| Caută ordine furnizor | Permite cautarea ordinelor furnizor. |
| Vizualizeaza situatia ordinului | Permite vizualizarea situatiei ordinului. |
| Vizualizeaza ordinul | Permite vizualizarea ordinului furnizor. |
| Legenda | Legenda cu toate simbolurile utilizate pentru executare. |
| Sterge setarile registrului | Permite stergerea setarilor registrelor. |
| Transfer articole | Permite transferarea articolelor selectate in ordin. |
| Executarea fortata a ordinului | Permite executarea fortata a ordinului. |
| Pozitia obiectului din lista in sus | Permite pozitionarea in sus a articolelor. |
| Pozitia obiectului din lista in jos | Permite pozitionarea in jos a articolelor. |
| Positionarea obiectelor din lista | Permite pozitionarea obiectelor din lista. |
| Selectarea tuturor articolelor | Permite selectarea tututor articolelor din lista. |
| Vizualizeza/Ascunde parintele ramurei | Permite vizualizarea sau ascunderea parintelui ramurei. |
| Vizualizeaza/Ascunde arborele ordinelor | Permite vizualizarea sau ascunderea arborelui ordinelor. |
| Blocheaza ordinele pentru executare | Permite blocarea ordinelor pentru executare. |
| Deblocheaza ordinele | Permite deblocarea ordinelor. |
