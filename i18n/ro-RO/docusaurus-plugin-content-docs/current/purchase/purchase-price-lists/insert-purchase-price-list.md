Pentru inserarea ofertei furnizor sunt necesare urmatoarele date: contul sintetic/analitic al furnizorului (alegerea acestuia se poate realiza cu ajutorul helpului de câmp), De la data valabilității și deviza. La data valabilității nu este obligatorie, este utilizată doar în cazul în care se dorește definirea unei campanii promoționale sau dacă se dă o scadență ofertei. Se pot insera: comanda de vânzare, notele și discountul din registru.

În gridul articole se pot insera articole cu ajutorul help-ului articole sau manual specificând clasa sau codul articolului.  Se poate insera în linie:

**Varianta articolului**, dacă, în afara codului articolului, și varianta diferențiază un articol de altul;

**Articolul furnizor**, în cazul în care codul articolului furnizorului este diferit de codul articolului societății;

**Codul variantei articolului furnizorului**, dacă în afara codului articolului furnizorului și varianta diferențiază un articol furnizor de un alt cod articol furnizor;

**Unitatea de măsură** care poate fi cea gestională a articolului sau una din unitățile sale de măsură  alternative;

**Marca articolului**, dacă este specificată;

**Cantitatea**, propusă egală cu 1, este cantitatea de referință a prețului. Dacă este diferită de 1, când se calculează prețul este împărțit pe cantitatea ofertei;

**Prețul** în deviză al ofertei.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută articole | Permite căutarea articolelor pe baza filtrului de căutare inserat. |
| Articol nou | Permite inserarea unui articol nou. |
| Salvează articole | Permite salvarea articolelor inserate în ofertă. |
| Șterge articole | Permite eliminarea articolului inserat. |
| Actualizează prețuri | Permite actualizarea prețurilor unei oferte. |
| Șterge discounturi | Permite ștergerea discounturilor inserate în liniile selectate. |



| Functie | Functionalitate |
| --- | --- |
| Document manager | Procedura da posibilitatea de a atasa documente în fereastra curentă. |

ARIA DE FILTRE: este dedicată vizualizării unei liste cu posibilele tipuri de date, prin care se poate face o selecţie multipla a ofertelor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare simultane.

GRID REZULTATE: rezultatul este o listă cu ofertele filtrate de catre utilizator cu posibilitatea de deschide, modifica sau sterge o ofertă sau de a mai adauga alte filtre pentru lista dorita.
