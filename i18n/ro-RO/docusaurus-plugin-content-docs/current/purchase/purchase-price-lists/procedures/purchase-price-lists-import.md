---
title: Import oferte furnizori
sidebar_position: 2
---

Cu ajutorul acestei proceduri se pot crea noi oferte furnizori, prin importarea documentelor de tip Excel, text (ASCII) sau XML. Procedura permite inserarea articolelor, care se află în documentele din care se execută importul direct în registrul articolelor.

Această procedura necesită:

- selectarea "Ofertei noi" și apăsarea butonului "Importă din" pentru a deschide fereastra de import în care utilizatorul va trebui să:


 1. selecteze tipul importului: din file Excel, din file text (ASCII) și din file XML;
 2. apese butonul "Deschide file" pentru a selecta documentul dorit.
 3. în câmpul "Sursă" apare calea documentului selectat.
 4. selecteze opțiunea "Prima linie conține numele coloanelor".
 5. apese butonul "Înainte".

- în această nouă fereastră utilizatorul trebuie:


 1. Să execute clic dreapta mouse în antetul tabelului “Vizualizare a 25 de câmpuri formatate” pentru a vizualiza noile nume care vor fi atribuite coloanelor;
 2. Să elimine, eventual, spațiile;
 3. Să aleagă unitatea de măsură;
 4. Să selecteze "Creează articole în registrul articole" dacă se dorește inserarea articolelor documentului direct în registrul articole.
 5. Să apese butonul "Transfer" pentru a importa articolele din document într-o ofertă.

Aplicația oferă posibilitatea de a avea imaginea importului: "Vizualizează import". Dacă articolele nu au fost transferate, utilizatorul va avea posibilitatea să selecteze, în această fereastră, datele corecte și să termine operația cu ajutorul butonului "Sfârșit".

Cu ajutorul flag-ului “Oferta selectată“ se poate alege oferta în care se va face importul  noii oferte; apăsați  butonul “Import” și alegeți datele "Oferta asupra căreia se face importul". Oferta astfel creată poate fi vizualizată în “Caută oferte furnizori”. În plus, în registrul articole, există posibilitatea să vizualizați și să completați datele articolelor create cu ajutorul procedurii de import.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea articolelor. |
| Descărcare XML | Buton pentru descărcarea documentelor XML. |
| Import | Buton pentru efectuarea importului. |
