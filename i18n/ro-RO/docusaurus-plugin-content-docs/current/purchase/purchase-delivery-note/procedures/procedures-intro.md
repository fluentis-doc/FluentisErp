RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Contabilizare aviz | Procedura de contabilizare a unui aviz inserat. |
|  [Incarcare aviz in gestiune](/docs/purchase/purchase-delivery-note/procedures/warehouse-purchase-delivery-note-record/load-delivery-notes-on-warehouse)  | Procedura de incarcare in gestiune a unui aviz deja inserat.Prin filtrelele si butonul de cautare se aduc toate avizele neicarcate. In fereastra de parametri se insereaza gestiunea si sablonul de incarcare dupa care buton'incarcare aviz'. |
|  [Creare aviz din documentul de incarcare](/docs/purchase/purchase-delivery-note/procedures/create-delivery-note-from-goods-receipt/create-delivery-note-from-goods-receipt)  | Procedura de creare aviz din documentul de incarcare. Prin filtrele  si butonul de cautare se gasesc documentele de incarcare  care nu au aviz.In fereastra de parametri se insereaza tip aviz, TVA, facturare achiziti si se apasa butonul ‘creare aviz'. |



| Functie | Functionalitate |
| --- | --- |
| Avize de achiziție | Procedura de listarea a unui aviz de achizitie deja inserat. In aceasta fereastra pot fi generate copii ale avizelor de insotire a marfii, in functie de criteriile selectate in partea de filtrare (tip aviz, numar, data, furnizor etc). |
| Lista avize de achiziție | Procedura de listare cu ajutorul unor filtre a unei liste cu avize inserate. In aceasta sectiune pot fi generate rapoarte referitoare la avizele de achizitie, in functie de criteriile selectate in partea de filtrare (tip aviz, numar, data, furnizor), oferindu-se posibilitatea ordonarii dupa furnizor, numar aviz, tip aviz sau data inserarii. |
