---
title: Creare aviz din ordine furnizor
sidebar_position: 1
---

Procedura ofera posibilitatea crearii unui A.E dintr-un Ordin Furnizor. Pentru ca procedura sa poata functiona este necesar ca furnizorii din ordin si A.E. sa fie aceiasi; ordinul trebuie sa fie aiba flag-ul Listat si Data confirmarii. Aceasta procedura implica preluarea tuturor datelor prezente in ordin, si, ca urmare, sunt aplicate conditiile de achizitie prezente in ordin, chiar daca acestea acum sunt schimbate. In fereastra sunt propuse toate ordinele furnizorului selectat care au fost listate, confirmate, neexecutate sau partial executate. In acest moment aveti posibilitatea sa selectati ordinul in totalitate (marcarea flag-ului de pe linia ordinului) si sa importati ordinul in A.E. cu ajutorul butonului de transfer sau să selectati doar o parte din articolele ordinului, eventual, cu cantitati diferite. Pentru aceasta executati dublu clic pe linia ordinului, pentru a deschide liniile ordinului, si selectati intreaga cantitate a liniei respective (marcati flag-ul din casuta corespunzatoare), sau pozitionati-va pe linia articolului si schimbati campul ‘cantitate' pentru a executa partial cantitatea. Si in acest caz transferul datelor in aviz se face cu ajutorul butonului de transfer.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Cautare ordine furnizor | Permite cautarea ordinelor furnizor. |
| Vizualizeaza situatia ordinului | Permite vizualizarea situatiei ordinului. |
| Vizualizeaza ordinul | Permite vizualizarea ordinului furnizor. |
| Legenda | Legenda tuturor simbolurilor utilizate pentru executare. |
| Sterge setarile registrului | Permite stergerea setarilor registrilor. |
| Transfer articole | Permite transferarea articolelor selectate in ordin. |
| Executarea fortata a ordinului | Permite executarea fortata a ordinului. |
| Pozitia obiectului din lista in sus | Permite pozitionarea in sus a articolelor. |
| Pozitia obiectului din lista in jos | Permite pozitionarea in jos a articolelor. |
| Pozitia obiectului în listă | Permite pozitionarea obiectelor în lista. |
| Selectarea tuturor articolelor | Permite selectarea tututor articolelor din lista. |
| Vizualizeza/Ascunde parintele ramurei | Permite vizualizarea sau ascunderea parintelui ramurei. |
| Vizualizeaza/Ascunde arborele ordinelor | Permite vizualizarea sau ascunderea arborelui ordinelor. |
| Blocheaza ordinele pentru executare | Permite blocarea ordinelor pentru executare. |
| Deblocheaza ordinele | Permite deblocarea ordinelor. |
