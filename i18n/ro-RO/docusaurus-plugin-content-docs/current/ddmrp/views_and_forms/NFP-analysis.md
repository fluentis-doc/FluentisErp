Acest raport prezintă informațiile cele mai relevante referitoare la procesul de planificare pentru buffer-ele Ddmrp.

În partea de filtrare, este important să se aleagă data pentru care trebuie să fie afișate informațiile.

Este propusă automat ultima dată la care a fost efectuat calculul NFP.

*Notă*:
Dacă în ziua curentă nu s-a efectuat acest calcul, închideți formul, efectuați calculul și apoi redeschideți-l.

*Câmpuri specifice*:

**Tipul articolului**: este tipul de articol din profilul buffer-ului liniei,  
**Limita zonă galbenă**: reprezintă suma TOY = zona roșie + zona galbenă, 
**Limita zonă verde**: reprezintă suma TOG = TOY + zona verde.

Pentru toate celelalte câmpuri, consultați help-ul procedurii NFP.
