Această fereastră este constituită din două griduri care preiau materialele care vor fi expediate terțului spre prelucrare și materialele care vor fi utilizate la reintrarea produsului prelucrat.

Materialele expediate și utilizate pot fi gestionate de operator și, în acel caz, este necesară inserirea cantității, a eventualului preț lucrare, a datei prevăzute pentru reintrare și data utilizării materialului. Materialele expediate și rezervate pot fi gestionate de operator și, în acest caz, este necesară inserarea cantității, a eventualului preț al lucrării, data prevăzută pentru reintrare, data utilizării materialului. Aceste date sunt în schimb deja valorizate în cazul în care ordinul provine din planificarea generală sau din elaborare MRP.

**Grid materiale de livrat**

**Cod articol**: indică materialul care va fi livrat pentru prelucrare; cu dublu clic se poate apela help articole cu ajutorul căruia se selectează codul articolului;

**UM utilizare**: unitate de măsură a cantității de livrat;

**Cantitate de livrat**: cantitate totală de expediat furnizorului;

****Cantitate **livrată**: cantitate totală deja livrată;

**Gestiune ieșire**: reprezintă gestiunea din care sunt extrase materialele care vor fi expediate furnizorului.

******Grid materiale de** utilizat****

**Cod articol**: indică materialul de utilizat la reintrarea articolului prelucrat; cu dublu clic se intră în help articole cu ajutorul căruia se selectează codul articolului;

**Variantă**: în acest combo se inserează varianta articolului;

**Opțiune**: în acest combo se inserează opțiunea articolului;

**UM **utilizare****: unitate de măsură la care face referire cantitatea de livrat;

**Utilizare unitară**: cantitatea de utilizat pentru fiecare unitate de articol reintrat prelucrat, în cazul gestiunii automate a planificării listelor; această informație este valorizată de sistem.

****Utilizare **totală**: cantitatea totală de utilizat pentru prelucrarea completă în cazul gestionării automate a planificării listelor da impiegare per la completa lavorazione nel caso di gestione automatica della schedulazione di distinte; această informație este valorizată de sistem.

**% pierderi**: cantitate de utilizat în plus care reprezintă pierderea;

**Data prev. utilizare**: pe baza acestei date sistemul planifică disponibilitatea necesarului.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Inserează livrarea | Permite inserarea unei noi linii în gridul Materiale de livrat; |
| Șterge livrarea | Permite ștergerea liniei selectate din gridul Materiale de livrat; |
| Inserează utilizări | Permite inserarea unei noi linii în Materiale de utilizat; |
| Șterge utilizări | Permite ștergerea liniei din gridul Materiale de utilizat. |
