---
title: Caută ordine în sistem lohn
sidebar_position: 3
---

În această fereastră pot fi căutate, vizualizate și modificate ordinele SL generate automat de procedura de lansare a ordinelor planificate de producție, în care sunt prezente faze externe și de procedura de lansare a ordinelor planificate SL. De asemenea, pot fi căutate, vizualizate și modificate ordinele SL generate manual sau se pot insera noi ordine SL.

RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea ordinelor deja inserate în baza de date. |
| Inserează | Buton pentru inserarea unui nou ordin SL. |
| Modifică | Buton pentru modificarea unui ordin inserat anterior. |
| Vizualizează | Buton pentru vizualizarea ordinelor inserate anterior. Nu permite modificarea datelor. |
| Șterge | Buton pentru ștergerea unui ordin inserat anterior. |
| Modifică filtru atribute | Butonul permite activarea sau dezactivarea căutării pe baza filtrului atribute articol. |

ARIA DE FILTRE: este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție multiplă a ordinelor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare.

GRID REZULTATE: este o listă cu datele care corespund filtrelor inserate anterior de utilizator cu posibilitatea de deschide, modifica sau sterge un ordin sau de a mai adauga alte filtre pentru lista dorita.
