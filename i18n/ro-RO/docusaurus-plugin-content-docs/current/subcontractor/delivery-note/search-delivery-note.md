#### Filtre specifice 

**Descărcate**: permite filtrarea avizelor de livrare deja descărcate din gestiune, adică avizele pentru care a fost efectuată operațiunea de *Descărcare avize*;  
**Listate**: permite filtrarea avizelor de livrare deja tipărite.  
Fereastra de *Căutare avize de livrare* se deschide cu aceste 2 opțiuni dezactivate implicit.

Din acest form este posibilă și crearea unui nou Aviz de livrare, făcând clic pe butonul [Nou](/docs/subcontractor/delivery-note/insert-delivery-note/new-delivery-note).
