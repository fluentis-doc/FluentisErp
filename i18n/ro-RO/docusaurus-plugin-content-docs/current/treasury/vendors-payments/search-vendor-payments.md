RIBBON BAR: reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută plăți | Buton pentru la căutarea, pe baza filtrelor inserate, a plății inserate anterior în baza de date; |
| Nouă plată furnizor | Buton pentru crearea unei noi plăți furnizor;  |
| Modifică plățile selectate | Buton pentru modificarea unei plăți furnizori inserate anterior, activ după selectarea unei plăți; |
| Vizualizează plățile selectate | Buton pentru vizualizarea unei plăți furnizori inserate anterior, activ după selectarea unei plăți; |
| Șterge plățile selectate | Buton pentru ștergerea unei plăți furnizori inserate anterior, activ după selectarea unei plăți. |



| Functie | Functionalitate |
| --- | --- |
|  [Raport plăți furnizor](/docs/treasury/vendors-payments/reports)  | Apelează procedura pentru listarea raportului Plăți furnizori. |



| Functie | Functionalitate |
| --- | --- |
|  [Blocare angajamente pasive](/docs/treasury/vendors-payments/procedures/block-passive-maturity-values)  | Procedură pentru gestionarea masivă a blocării angajamentelor pasive; |
|  [Creare automată plată furnizori](/docs/treasury/vendors-payments/procedures/automatic-vendor-payments-creation/parameters-filter)  | Procedură pentru crearea automată a unei plăți furnizor; |
|  [Gestiune suport plăți](/docs/treasury/vendors-payments/procedures/payments-support-management)  | Procedură pentru gestionarea băncilor plății; |
|  [Contabilizare](/docs/finance-area/professional-men/accounting/payments-accounting/filter)  | Procedură pentru contabilizarea plății furnizorului. |

ARIA DE FILTRE: aria de filtre conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecție. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: reprezintă lista cu plățile furnizor care corespunde datelor din filtrele indicate anterior. Utilizatorul, după ce a inserat valorile cu ajutorul cărora dorește să caute plățile furnizori, dacă apasă butonul [Caută Mijloace fixe] prezent în Ribbon, va obţine în gridul rezultatelor lista cu înregistrările dorite.
