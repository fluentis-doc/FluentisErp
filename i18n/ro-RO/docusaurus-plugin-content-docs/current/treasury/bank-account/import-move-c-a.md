---
title: Import mișc. C/C
sidebar_position: 2
---

Această fereastră permite executarea importului mișcărilor contului curent din fișierul telematic (standard abi) care se poate extrage din home banking.

Pentru a efectua importul, este necesară inserarea, în partea inferioară a ferestrei, a contului analitic pentru care se dorește importarea fișierului, apoi apăsând butonul ‘Execută import' va apare fereastra pentru deschiderea fișierului telematic. La sfârșitul operației, va apare în grid o nouă linie de import: cu dublu clic pe aceasta sau cu ajutorul butonului ‘Vizualizează fișier' se poate vizualiza fișierul de text inițial al datelor importate.

Mișcările astfel importate vor fi utilizate în procedura de reconciliere C/C.

Procedura Import mișcări C/C este structurată în felul următor:

RIBBON BAR: reprezintă meniul Form-ului în discuţie, și anume aria în care se pot întreprinde acțiuni legate de procedura curentă. Prezentăm în continuare lista cu funcționalitățile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută importuri | Buton pentru aplicarea filtrelor de căutare în baza de date a importurilor efectuate. |
| Execută importuri | Buton pentru selectarea fișierului telematic și pentru importarea sa. |
| Vizualizează fișier | Buton pentru vizualizarea fișierului telematic asociat importului selectat. |
| Șterge import | Buton pentru ștergerea importului selectat. |

ARIA DE FILTRE: aria de filtre conține lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua selecţia sau căutarea importurilor efectuate. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: reprezintă lista cu importurile care corespund datelor din filtrele indicate anterior. Utilizatorul, după ce a inserat valorile cu ajutorul cărora doreşte să caute importurile, dacă apasă butonul [Caută Mijloace fixe] prezent în Ribbon, va obţine în gridul rezultatelor lista cu importurile dorite.
