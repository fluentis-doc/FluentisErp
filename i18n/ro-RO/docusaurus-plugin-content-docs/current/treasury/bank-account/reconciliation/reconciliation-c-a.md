title: Reconciliere C/C
sidebar_position: 1
---

În această fereastră se reconciliază fluxurile importate din home banking cu mișcările înregistrate în contabilitate.

Sunt prezente două secțiuni: în stânga mișcările importate iar în partea dreaptă mișcările contabile. Fiecare din cele două secțiuni are o listă de filtre: pentru ambele este de semnalat, în particular, posibilitatea de a vizualiza doar mișcările nereconciliate, doar mișcările reconciliate sau toate mișcările.

După selectarea în cele două secțiuni a liniilor relative, acestea se vor putea reconcilia cu butonul ‘Reconciliază mișcări'. În caz contrar, două linii deja reconciliate se vor putea deconecta selectând și apăsând butonul ‘Deconectează mișcări'.

În fereastră este prezent un alt buton ‘Vizualizează înregistrări' care permite vizualizarea înregistrării contabile selectate în secțiunea din dreapta.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se efectuează acţiuni asociate ferestrei curente. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută import | Buton pentru căutarea listei mișcărilor importate, pe baza filtrelor inserate. |
| Caută înregistrări | Buton pentru căutarea listei înregistrărilor contabile, pe baza filtrelor inserate. |
| Reconciliază mișcări | Efectuează conexiunea mișcărilor de import cu relativele mișcări contabile selectate. |
| Deconectează mișcări | Buton pentru deconectarea mișcărilor selectate și deja reconciliate ale celor două secțiuni. |
| Vizualizează înregistrări | Buton pentru vizualizarea înregistrării contabile selectate în secțiunea din dreapta. |
