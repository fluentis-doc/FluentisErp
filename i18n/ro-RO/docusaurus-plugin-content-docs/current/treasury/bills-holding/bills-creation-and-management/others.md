În această fereastră, pot fi memorizate referințele giranților și giranților.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Salvează efect | Buton pentru salvarea efectului. |
| Efect nou | Buton pentru crearea unui nou buton. |
| Inserează scadențe | Activ când este activ tab ‘Referințe scadențe', permite apelarea helpului scadențe pentru a asocia o scadență factură la efectul ales. |
| Șterge referință scadență | Activ când este activ tab ‘Referințe scadențe', permite deconectarea scadențelor selectate din efectul ales. |
| Inserare angajamente | Activ când este activ tab ‘Referințe angajamente', permite apelarea helpului angajamente pentru a asocia un angajament la efectul ales. |
| Șterge angajamente din efect | Activ când este activ tab ‘Referințe angajamente', permite deconectarea angajamentelor selectate din efectul ales. |
| Documente | Procedura da posibilitatea de a atasa documente. |
