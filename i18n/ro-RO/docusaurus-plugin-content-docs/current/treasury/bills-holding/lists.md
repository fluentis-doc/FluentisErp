title: Caută liste
sidebar_position: 5
---

Procedura pentru căutarea listelor este conceputa pentru a permite utilizatorului să caute cu uşurinţă toate listele de prezentare emise anterior, cu scopul de a le vizualiza, edita şi şterge, dacă este necesar,in functie de drepturile alocate fiecarui utilizator.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unei înregistrări preexistente. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută liste | Permite aplicarea de filtre pentru a căuta efectele inserate în întreaga bază de date. |
| Listă nouă | Procedura de inserare nou efect. |
| Modifică liste | Procedura de modificare a unei liste deja inserate. |
| Vizualizare liste | Procedura pentru vizualizarea a unui efect deja inserat, dar nu permite utilizatorului sa modifice datele. |
| Șterge liste | Procedura pentru stergerea unui efect deja inserat. |

ARIA DE FILTRE: zona de filtrare este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție multiplă, o căutare a listelor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare.

GRID REZULTATE: rezultatul este un tabel cu listele care corespund filtrelor inserate anterior de utilizator cu posibilitatea de a deschide, modifica sau sterge o listă sau de a mai adauga alte filtre pentru  lista dorita.
