În această fereastră se pot anula contabilizările efectuate anterior. Anularea va fi posibilă doar dacă perioada se mai poate modifica (listarea registrului jurnal sau închiderea conturilor nu s-a efectuat). Cu dubluc clic în secțiunea înregistrărilor va putea fi vizualizată înregistrarea contabilă relativă. Este prezent și un raport pentru vizualizarea erorilor relevate în timpul contabilizării.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea listei cu contabilizările efectuate. |
| Restaurare contabilizare | Buton pentru restaurarea contabilizărilor selectate. |
| Restaurare înregistrări | Buton pentru restaurarea înregistrării contabile selectate. |
| Listează | Buton pentru listarea rezultatului contabilizării. |
