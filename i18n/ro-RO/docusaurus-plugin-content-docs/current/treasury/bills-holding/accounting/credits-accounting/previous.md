În această fereastră pot fi anulate operațiile efectuate anterior. Anularea poate fi efectuată doar dacă perioada se mai poate modifica (dacă nu a fost listat registrul jurnal și nu s-a facut inchiderea conturilor). Cu dublu clic în secțiunea înregistrărilor se va putea vizualiza înregistrarea contabilă relativă.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea listei cu contabilizările efectuate. |
| Restaurare contabilizare | Buton pentru restaurarea contabilizărilor selectate. |
| Restaurare înregistrări | Buton pentru restaurarea înregistrării contabile selectate. |
| Listează | Buton pentru listarea rezultatului contabilizării. |
