title: Plăți
sidebar_position: 2

Această fereastră este activă doar în cazul în care tipul listei are flag-ul ‘Efecte la furnizor'. Este dedicată identificării angajamentelor furnizori care trebuie închise cu girarea efectelor prezente în listă. Totalul  secțiunii **Plăți** va trebui să fie egal cu totalul efectelor girate: pentru a obține această eglitate este permisă inserarea manuală a unei linii pentru diferență, care va fi gestionată ca o deschidere angajament pentru furnizor dacă șablonul contbil utilizat prevede deschiderea angajamentului.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unei înregistrări preexistente. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Salvează lista | Buton pentru salvarea listei. |
| Plată nouă | Activeză cursorul într-o linie nouă pentru plată. |
| Șterge plăți | Buton pentru ștergerea plăților selectate din grid. |
| Plăți din angajamente | Deschide fereastra help angajamente din care se selecează angajamentele care trebuie închise. |
