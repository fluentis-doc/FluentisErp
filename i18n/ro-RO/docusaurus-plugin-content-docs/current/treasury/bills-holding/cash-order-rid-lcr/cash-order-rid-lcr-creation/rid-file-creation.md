title: Generare file rid
sidebar_position: 2
---

Din această fereastră se generează fișierele telematice pentru listele de prezentare RID.

Partea superioară a ferestrei este dedicată inserării filtrelor de căutare ale listelor emise, dar pentru care fișierul nu a fost încă creat.

După identificarea listei care va fi gestionată, în partea inferioară a ferestrei se afișează datele implicite așa cum au fost predefinite în parametrii: data creării și numele fișierului (setat ca [DataLista]_[Număr Lista].RID), calea fișierului, numele societății și descrierea referințelor facturii de inserat.

Utilizatorul trebuie să folosească butonul de generare pentru a crea fișierul: aplicația va controla datele obligatorii care vor fi inserate în fișier și va afișa o fereastră cu detaliile eventualelor erorilor găsite.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută liste | Buton pentru căutarea listelor pentru care va fi creat fișierul telematic. |
| Creare | Buton pentru crearea fișierului RID pentru lista selectată. |
