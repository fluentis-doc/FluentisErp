---
title: Generare file Lcr
sidebar_position: 3
---

Din această fereastră se generează fișierele telematice pentru listele de prezentare Lcr.

Partea superioară a ferestrei este dedicată inserării filtrelor de căutare ale listelor emise dar pentru care fișierul nu a fost încă creat.

După identificarea listei care va fi gestionată, în partea inferioară a ferestrei se afișează datele implicite așa cum au fost predefinite în parametrii: data creării și numele fișierului (setat ca [DataLista]_[Număr Lista].Lcr), calea fișierului, numele societății și descrierea referințelor facturii de inserat.

Utilizatorul trebuie să folosească butonul de generare pentru a crea fișierul.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută liste | Buton pentru căutarea listelor pentru care va fi creat fișierul telematic. |
| Creare | Buton pentru crearea fișierului Lcr pentru lista selectată. |
