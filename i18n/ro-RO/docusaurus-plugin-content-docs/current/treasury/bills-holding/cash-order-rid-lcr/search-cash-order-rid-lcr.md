---
title: Caută incasso documentar/Rid/Lcr
sidebar_position: 1
---

Această fereastră fost concepută pentru a permite utilizatorului căutarea simplă a fișierelor telematice create în modulul portofoliu efecte, cu scopul de a vizualiza fișierul creat sau de a șterge și restaura crearea unui nou fișier. 

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută incasso documentar | Buton pentru căutarea listei cu fișierele telematice create. |
| Vizualizează incasso documentar | Buton pentru căutarea fișierelor create pe baz filtrelor inserate. |
| Șterge incasso documentar | Anulează flag/ul de stare ‘fișier emis' și cere confirmarea ștergerii fișierelor de pe calea memorizată. |
