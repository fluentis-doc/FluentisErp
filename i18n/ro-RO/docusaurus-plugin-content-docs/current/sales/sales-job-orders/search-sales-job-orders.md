---
title: Căutare comenzi de vânzare
sidebar_position: 3
---

Fereastra de cautare comenzi de vanzare este conceputa pentru a permite utilizatorului să caute cu uşurinţă toate comenzile inserate, cu scopul de a vizualiza, edita şi şterge, in functie de drepturile alocate fiecarui utilizator. Cu ajutorul filtrelor, puteti sa gasiti: un singur document (de ex. cautare dupa numarul documentului), mai multe documente (de ex. cautare dupa data), documentele inserate intr-un anumit an (cautare dupa an), documentele inserate pentru un anumit client (cautare dupa cont sintetic/analitic), documentele dupa data de livrare prevazuta sau dupa data confirmarii.

RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Cautare comenzi | Permite  aplicarea de filtre pentru a căuta în întreaga bază de date a comenzilor inserate. |
| Noua comanda | Procedura de inserare noua comanda. |
| Modifica comanda | Procedura de modificare a unei comenzi deja inserate. |
| Vizualizare comanda | Procedura de vizualizare a unei comenzi deja inserate, dar nu permite utilizatorului sa modifice datele. |
| Sterge comanda | Procedura de stergere a unei comenzi deja inserate. |
| Executare din ordin client | Procedura automata de executare a ordinnelor client pe baza unei comenzi. |
| Duplica comanda | Procedura de duplicare a unei comenzi pe permite crearea unei copii a unui document, preluand toate conditiile documentului initial, dar cu date actuale. |
| Propietate | Procedura de vizualizare a executarii unei comenzi, cu date despre articolele din comanda cat si date despre documentele asociate. |
| Conversie | Procedura de  conversie a preturilor din document intr-o alta deviza. |
| Vizualizare | Procedura de vizualizare a unei comenzi  deja inserate in functie de criteriile selectate in partea de filtrare (tip comanda, numar, data, client, etc). |
| Listare | Procedura de listare cu ajutorul unor filtre a unei comenzi inserate in functie de criteriile selectate in partea de filtrare (tip comanda, numar, data, client). |

ARIA FILTRE: zona de filtrare este dedicata pentru a putea vizualiza o lista cu posibilele tipuri de date, prin care se poate face o multipla selecţie, a comenzilor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare.

GRID REZULTATE: rezultatul este o listă a comenzilor care se potrivesc cu filtrul specificat mai sus de catre utilizator cu posibilitatea de deschide, modifica sau sterge o comanda sau a mai adauga alte filtre pentru  lista dorita.
