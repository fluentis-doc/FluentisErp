Permite stabilirea cererilor de interventii care trebuie planificate pe baza contractului din comanda. Trebuie inserate urmatoarele date: tipul si codul instalatiei, serviciul stabilit, data pentru interventie si operatorul. Inserarea acestor date permite apoi utilizarea procedurii "Creare cereri de interventii". Deci, inserarea unei cereri de interventii in comanda nu implica crearea unei cereri de interventie efective, aceasta producandu-se doar dupa "Crearea cererii de interventii".
