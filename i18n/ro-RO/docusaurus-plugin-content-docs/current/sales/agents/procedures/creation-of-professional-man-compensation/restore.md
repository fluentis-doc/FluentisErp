sidebar_position: 3

În acest tab, utilizatorul are posibilitatea să caute, să vizualizeze și să anuleze operațiile de creare efectuate.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea plăților pe baza filtrului inserat. |
| Restaurează | Buton pentru anularea compensărilor selectate. |

ARIA DE FILTRE: zona de filtrare este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție. Exista posibilitatea să specificaţi mai multe criterii de filtrare simultane.

GRID REZULTATE: rezultatul este o listă cu operațiile de descărcare care corespunde filtrului specificat mai sus. Fereastra conține două zone: una cu plățile deja valorizate și una cu compensările relative.
