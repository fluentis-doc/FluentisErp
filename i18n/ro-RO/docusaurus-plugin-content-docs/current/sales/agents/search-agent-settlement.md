---
title: Plată agenți
sidebar_position: 3
---

Această procedură a fost concepută pentru a permite utilizatorului să caute cu ușurință toate plățile inserate, cu scopul de a le vizualiza, edita şi şterge, dacă este necesar, in funcție de drepturile alocate fiecărui utilizator.

Cu ajutorul filtrelor, puteți sa găsiți: o singură plată (de ex. căutare după numărul plății), mai multe plăți (de ex. căutare după data), plățile inserate într-un an (de ex. căutare după an), documentele inserate pentru un anumit agent (de ex. căutare după cod agent), plățile care conțin comisioanele unui anumit document (ordin, aviz sau factură).

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea plăților în baza de date. |
| Inserează plăți | Buton pentru crearea unei noi plăți. |
| Modifică plăți | Buton pentru modificarea unei plăți inserate anterior. |
| Vizualizează plăți | Buton pentru vizualizarea unei plați inserate anterior. Nu se poate modifica. |
| Șterge plăți | Buton pentru ștergerea unei plați inserate anterior. |

ARIA DE FILTRE: zona de filtrare este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție. Exista posibilitatea să specificați mai multe criterii de filtrare simultane.

GRID REZULTATE: rezultatul este o listă a plăților care se potrivesc cu filtrul specificat mai sus de către utilizator cu posibilitatea de deschide, modifica sau șterge o plată sau de a adaugă alte filtre pentru  lista dorită.
