---
title: Creare factura din lista de ambalare
sidebar_position: 3
---

Permite preluarea unui liste de ambalare in Factura. Lista de ambalare, dupa aceasta procedura, va avea statusul "descarcată”. Operatii specifice ferestrei "Executare din Lista de ambalare”: filtrare Lista de ambalare (daca doriti sa fie afisate cele cu clienti, selectati casuta "Lista de ambalare fara client"), inserare parametri, restaurare operatii.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Cautare lista de ambalare | Permite cautarea listelor de ambalare. |
| Selecteaza tot | Permite selectarea tutotor articolelor din lista. |
| Transfer articole | Permite transferarea articolelor selectate in lista de ambalare. |
