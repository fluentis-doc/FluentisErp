---
title: Creare factura din ordine clienți
sidebar_position: 1
---

Procedura ofera posibilitatea crearii unei facturi dintr-un Ordin client.

**Conditii**: clientii din ordin si factura trebuie sa fie aceiasi; ordinul trebuie sa fie aiba flag-ul Listat si Data confirmarii; compatibilitatea tipurilor de documente: in tabelul Tipuri de ordine, tipului de ordin in cauza trebuie sa-i fie asociat tipul de factura corespunzator.

Aceasta procedura implica preluarea tuturor datelor prezente in ordin, si, ca urmare, sunt aplicate conditiile de vanzare prezente in ordin, chiar daca acestea acum sunt schimbate (ex: actualizare oferte). In fereastra sunt propuse toate ordinele clientului selectat care au fost listate, confirmate si neexecutate sau partial executate. In acest moment aveti posibilitatea: sa selectati ordinul in totalitate (marcarea flag-ului de pe linia ordinului); si sa importati ordinul in factura cu ajutorul butonului  sau sa selectati doar o parte din articolele ordinului, eventual, cu cantitati diferite. Pentru aceasta executati dublu clic pe linia ordinului, pentru a deschide liniile ordinului, si selectati intreaga cantitate a liniei respective (marcati flag-ul din casuta corespunzatoare) sau pozitionati-va pe linia articolului si schimbati campul ‘cantitate' pentru a executa partial cantitatea. In urma selectiilor se apasa butonul “Transfer articole”.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:

| --- | --- |
| Cautare ordine client | Permite cautarea ordinelor client. |
| Transfer articole | Permite transferarea articolelor selectate in ordin. |
| Vizualizeaza situatia executarii ordin | Permite vizualizarea situatie ordin. |
| Vizualizeaza ordinul | Permite vizualizarea ordinului client. |
| Legenda | Legenda |
| Executarea fortata a ordinului | Permite executarea fortata a ordinului. |
| Executare ordin | Permite executarea ordinului. |
| Selectare articole | Permite selectarea tuturor articolelor. |
| Poziționează obiectul din lista | Permite selectarii articolelor din lista. |
| In sus | Permite pozitionarea in sus a articolelor. |
| In jos | Permite pozitionarea in jos a articolelor. |
| Blocheaza ordinele pentru executare | Permite blocarea ordinelor pentru executare. |
| Deblocheaza ordinele | Permite deblocarea ordinelor. |
| Sterge setarile registrului | Permite stergerea setarilor registrilor. |
| Vizualizeza/Ascunde parintele ramurei | Permite vizualizarea sau ascunderea parintelui ramurei. |
| Vizualizeaza/Ascunde arborele ordinelor | Permite vizualizarea sau ascunderea arborelui ordinelor. |
