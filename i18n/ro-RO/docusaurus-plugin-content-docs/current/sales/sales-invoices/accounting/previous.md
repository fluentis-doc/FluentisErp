În acest tab sunt vizualizate rezultatele contabilizării care pot fi anulate cu ajutorul butoanelor relative.

Anularea va fi posibilă doar dacă perioada se mai poate modifica (registru jurnal, registru TVA nelistat, lichidare periodică nelistată, închidere conturi neefectuată, angajamente contabile închise într-o încasare sau un efect). Înregistrarea poate fi vizualizată dacă utilizatorul efectuează dublu clic pe linia dorită. Este prezent și un raport care vizualizează erorile găsite în timpul contabilizării.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Caută contabilizare | Permite filtrarea si vizualizarea inregistrarilor contabile. |
| Vizualizează contabilizare | Vizualizeaza un raport pentru contabilizarea selectata. |
| Vizualizeaza inregistrarea | Vizualizeaza inregistrarea contabila selectata. |
| Restaureaza contabilizarea | Eliminarea contabilizarii asociate mai multor facturi. |
| Restaureaza factura | Eliminarea contabilizarii asociata facturi selectate. |
