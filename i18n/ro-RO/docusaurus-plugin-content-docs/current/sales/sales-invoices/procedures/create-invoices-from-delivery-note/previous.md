Aici pot fi vizualizate si filtrate operatiile care au rezultat in urma valorizarii avizelor. Cu ajutorul butonului "Restaureaza factura" puteti sa anulati factura selectata in tab-ul inferior numit "Facturi. Cu ajutorul butonului “Restaureaza inregistrarea" puteti sa anulati intreaga inregistrare, operatie care implica anularea facturilor care au fost generate in acelasi timp.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Caută facturi create din avize | Permite cautarea avizelor valorizate. |
| Restaurează crearea | Permite anularea întregii operații, și anume, în cazul creării mai multor facturi simultan, șterge toate facturile și restaurează avizele. |
| Previzualizează raport creare |  |
| Restaureaza factura | Permite anularea facturi selectate. |
| Deselectează flag listat |  |
