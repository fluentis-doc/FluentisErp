---
title: Caută facturi de vânzare
sidebar_position: 2
---

Fereastra de cautare facturi de vanzare este conceputa pentru a permite utilizatorului să caute cu uşurinţă toate facturile inserate, cu scopul de a le vizualiza, edita şi şterge, in functie de drepturile alocate fiecarui utilizator. Cu ajutorul filtrelor, puteti sa gasiti: un singur document (de ex. cautare dupa numarul documentului), mai multe documente (de ex. cautare dupa data), documentele inserate intr-un anumit an (cautare dupa an), documentele inserate pentru un anumit client (cautare dupa cont sintetic/analitic), documentele inserate pentru un anumit agent sau care se refera la o comanda de vanzare (numai in cazul in care comanda a fost inserata in antetul documentului). In plus, aveti posibilitatea sa cautati documentele care contin un anumit articol sau o anumita clasa de articole utilizand "Cautare articole".

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Cautare facturi | Permite  aplicarea de filtre pentru a căuta în întreaga bază de date a facturilor inserate. |
| Noua factura | Procedura de inserare noua factura. |
| Modifica factura | Procedura de modificare a unei facturi deja inserate. |
| Vizualizare factura | Procedura de vizualizare a unei facturi deja inserate, dar nu permite utilizatorului sa modifice datele. |
| Sterge factura | Procedura de stergere a unei facturi deja inserate. |
| Storno factura | Procedura de creare a documentului de storno din factura initiala inserata. Când utilizatorul execută stornarea, apare o fereastră care conține lista cu toate articolele facturii inițiale și utilizatorul trebuie să selecteze șablonul mișcării articolului și să selecteze flag-ul storno și tipul facturii care va fi creată. Legătura dintre factura de storno și factura inițială trebuie să fie inserată manual de utilizator în câmpul Ref. FT și Tip. |
| Duplica factura | Procedura de duplicare a unei facturi permite crearea unei copii a unui document, preluand toate conditiile documentului initial, dar cu date actuale. |
| Conversie | Procedura de conversie a preturilor din document intr-o alta deviza. |



| Functie | Functionalitate |
| --- | --- |
|  Factura de vanzare | Procedura de listarea a unei facturi de vanzare deja inserate in functie de criteriile selectate in partea de filtrare (tip factura, numar, data, client, etc). |
| Lista facturi de vanzare  | Procedura de listare cu ajutorul unor filtre a unei liste cu facturi inserate in functie de criteriile selectate in partea de filtrare (tip factura, numar, data, client), oferindu-se posibilitatea ordonarii dupa client, numar factura, tip factura sau data inserarii. |
|  Facturare vanzari | Permite generarea de rapoarte referitoare la facturile de vanzare, in functie de criteriile selectate in partea de filtrare (tip factura, numar, data, client), oferindu-se posibilitatea ordonarii dupa articol, clasa articole, tara, client. |



| Functie | Functionalitate |
| --- | --- |
|  Contabilizare facturi de vanzare | Procedura permite contabilizarea facturilor de vanzare. |
|  Creare factura de vanzare din aviz | Procedura de crearea a unei facturi de vanzare dintr-un aviz sau mai multe avize de vanzare. |
|  Descarcare facturi imediate din gestiune| Procedura de descarcare a unei facturi de vanzare din gestiune. |

ARIA DE FILTRE: zona de filtrare este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție multiplă a facturilor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare.

GRID REZULTATE: rezultatul este o listă cu facturile care corespund filtrelor inserate anterior de utilizator cu posibilitatea de a deschide, modifica sau sterge o factură sau de a mai adauga alte filtre pentru lista dorita.
