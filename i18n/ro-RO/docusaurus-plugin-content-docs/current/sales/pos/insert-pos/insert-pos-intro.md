Pentru inserarea unui POS sunt necesare urmatoarele.:

Inserare tip POS care in prealabil a fost stabilit si configurat in sectiunea de [Home>Tabele>Vanzare>Tip POS]. Fiecarui document ii este atribuit automat un numar pe baza numerotarii definite de utilizator in tabelul ‘Numerotare POS' si a tipului de document care contine acea numerotare, anul si data curenta se completeaza automat. In campul “Nume/Prenume” poate fi inserat manual numele operatorului care a inserat POS, iar in campul “Oras” se insereaza manual denumirea localitatii unde se afla magazinul. Flagul de “Listat” se activeaza automat la listarea bonului fiscal.( Pentru versiunea RO, butonul “Listare” genereaza bonul fiscal).

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Salvează | Permite salvarea operatiilor efectuate. Acelasi rezultat il puteti obtine si cand treceti pe o linie diferita de cea pe care doriti sa o salvati. |
| Salvează nou | Salveaza POS-ul curent si creaza unul nou. |
| Descarcare POS | Permite descarcarea POS, daca sunt inserate gestiunea si sablonul, pentru fiecare articol. |
| Previzualizare | Permite previzualizarea documentului. |
| Listează | Permite listarea documentului. |
