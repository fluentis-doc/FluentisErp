title: Date articol
sidebar_position: 3
---

**Articolul**: prezinta descrierea matricolei si, eventual a variantei articolului;

**Agent**: este propusa valoarea comisionului de atribuit agentului pentru vanzarea acestui articol. Procentul poate fi schimbat.

**Comanda**: se refera la comanda de vanzare asociata documentului. Aceasta poate fi asociata, in cazul ordinului client, cu ajutorul procedurii "executare din comanda" sau poate fi asociata cu ajutorul help-ului de camp (dublu clic pe banda galbena); 

**Referinta ordinului**: (numai in A.E. si Facturi) face referire la ordinul din care a fost creat A.E.;

**Marca**: este marca articolului, preluata din registrul acestuia sau din oferta din care a fost selectat articolul;

**Note**: ofera posibilitatea de a insera note pentru fiecare articol in parte.

RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Șterge agenți | Permite ștergerea agenților selectați. |
