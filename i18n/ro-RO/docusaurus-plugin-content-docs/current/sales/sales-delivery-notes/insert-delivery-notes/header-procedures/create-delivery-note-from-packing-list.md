---
title: Creare aviz din lista de ambalare
sidebar_position: 3
---

Permite preluarea unei liste de ambalare în Avizul de Vânzare. 
Lista de ambalare, după această procedură, va avea statusul "descărcată". 

*Operații specifice ferestrei "Executare din Lista de ambalare":* 
- filtrare Listă de ambalare (dacă doriți să fie afișate cele cu clienți, selectați căsuța "Listă de ambalare fără client"),   
- inserare parametri,  
- restaurare operații.

 
 *Lista functionalitatilor posibile cuprinde:*



> **Caută lista de ambalare:** Permite căutarea listelor de ambalare.  
> **Selectează tot:** Permite selectarea tuturor articolelor din listă.   
> **Transfer articole:** Permite transferarea articolelor selectate în lista de ambalare.
