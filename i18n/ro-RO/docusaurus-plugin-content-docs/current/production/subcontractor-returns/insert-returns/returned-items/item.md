În acest tab sunt fișate o serie de informații relative la linia selectată în grid. Le amintim pe următoarele:

**Dată prevăzută reintrare** ****: această dată este preluată din linia ordinului în SL din are a fost generată linia reintrării selectate;

**Ord. prod.** **: este vorba despre ordinul de producție a cărui fază externă a generat linia ordinului în SL care a fost executată cu linia reintrării selectate;

**Lot de încărcare** **: în acest câmp utilizatorul poate insera codul lotului cu care va trebui să fie încărcată cantitatea articolului reintrat, doar dacă articolul este gestionat cu loturi;

**Lot furnizor**: în acest câmp utilizatorul poate insera lotul furnizorului cu care va trebui să fie încărcată cantitatea articolului reintrat, doar dacă articolul este gestionat cu loturi;

**Comanda** **: în acest câmp este afișată comanda de vânzare asociată liniei reintrării. De obicei este apelată comanda de vânzare inserată în linia ordinului în sistem lohn din care a fost generată linia reintrării;

**Bază de impozitare** **: în acest câmp este calculat totalul liniei fără TVA;

**TVA** **: în acest câmp este calculată valoarea TVA; este considerată cota care a fost insertă în registrul furnizorului căruia îi este asociată reintrarea SL;

**Total livrare** **: în acest câmp este calculat totalul mărfii livrate.
