---
title: Executare din ordin
sidebar_position: 3
---

În această fereastră care se deschide cu ajutorul butonului 'Rentrări prevăzute' din ribbon bar, pot fi căutate ordinele SL pe baza filtrelor disponibile.

În gridul inferior este afișată o linie pentru fiecare ordin SL neexecutat încă sau parțial executat; utilizatorul, cu dublu clic pe linia ordinului, poate deschide liniile care trebuie să fie restituite de terț, le poate selecta sau poate insera cantitatea în fiecare din acestea și apoi să apese butonul 'Transferă' care se află în ribbon bar. Procedura efectuează astfel transferul cantităților articolelor selectate în gridul tabului 'Articole reintrate' din documentul cu reintrarea SL.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută ordine | Buton pentru căutarea în baza de date a ordinelor SL inserate și neexecutate încă sau parțial executate. |
| Transferă | Buton pentru transferarea articolelor selectate în fereastra inferioară în gridul tabului Articole reintrate. |
| Executare forțată ordin | Buton care permite forțarea evaziunii liniei ordinului SL pe care utilizatorul o transferă. |
| Identifică obiectul selectat în listă  | Buton care permite vizualizarea, în secțiunea din stânga a ferestrei inferioare, a articolului selectat în secțiunea din dreapta. |
| În sus / În jos | Butoane care permit mutarea pe nivelul superior / inferior din secțiunea din dreapta. |
| Selectează | Buton care permite selectarea tuturor liniilor prezente în secțiunea din dreapta a ferestrei în care sunt afișate articolele cre trebuie transferate. |
| Blochează ordinele pentru executare | Buton care permite blocare ordinului SL selectat: în acest mod nu va fi posibilă inserarea în aviz a articolelor din acest ordin. |
| Deblochează ordinele blocate | Buton care permite deblocarea ordinului SL selectat. |
