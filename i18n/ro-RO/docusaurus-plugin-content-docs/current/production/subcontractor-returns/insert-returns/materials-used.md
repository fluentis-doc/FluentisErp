title: Materiale utilizate
sidebar_position: 2
---

În acest tab sunt afișate materialele pe care terțul le-a utilizat pentru a produce ceea ce a fost inserat în tab 'Articole reintrate'. Tabul este compus dintr-un antet în care sunt afișate informațiile liniei selectate în tab 'Articole reintrate', dintr-un grid superior, în care sunt afișate materialele utilizate de terț, și dintr-un grid inferior, asociat celui superior, în care se pot insera loturile materialelor utilizate de terț.

Detaliem, în continuare, câteva coloane din grid:

**UM** **: în această coloană este afișată unitatea de măsură gestională a materialului utilizat de terț;

**Cantitate utilizată** **: în această coloană este afișată cantitatea materialului utilizat de terț și care va trebui să fie descărcată din gestiunea terțului. Această cantitate, editabilă de către utilizator, este propusă pe baza cantității unitare inserată în tab 'Materiale de utilizat' a ordinului în SL, în corespondență cu linia ordinului executată cu linia reintrării selectate în tab 'Articole reintrate'; această cantitate unitară este multiplicată pe cantitatea reintrată și se obține astfel cantitatea utilizată;

****Cantitate **restituită** **: **în această coloană este afișată cantitatea de material pe care terțul o restituie și care trebuie încărcată în gestiunea materialelor restituite;

**Gestiune** **: iîn această coloană este afișat codul gestiunii materialelor restituite, preluat din parametrii în SL; este afișat automat doar dacă se inserează cantitatea restituită în linia materialului; este un câmp editabil;

**Șablon** **: în această coloană este afișat codul șablonului pentru încărcarea materialelor restituite, preluat din parametrii în SL; este afișat automat doar dacă se inserează cantitatea restituită în linia materialului; este un câmp editabil.

În gridul inferior, utilizatorul poate insera loturile materialelor care au fost utilizate de terț. Coloanele afișate și regulile de utilizare ale gridului sunt identice cu cele din grid loturi din tab Loturi\NS care se află în tab Articole reintrate.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Inserează | Buton pentru inserarea unei reintrări noi. |
| Salvează | Buton pentru salvarea modificărilor făcute reintrării. |
| Reintrări prevăzute | Buton pentru deschiderea ferestrei de executare din ordin care permite transferarea articolelor din ordinele în sistem lohn în reintrare. |
| Vizulizează | Buton pentru vizualizarea reintrării în sistem lohn. |
| Listează | Buton pentru listarea reintrării în sistem lohn. |
| Documente | Procedura da posibilitatea de a atasa documente. |
