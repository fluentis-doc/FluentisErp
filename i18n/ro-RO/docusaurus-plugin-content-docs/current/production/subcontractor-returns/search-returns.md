---
title: Caută reintrări
sidebar_position: 2
---

Această fereastră se află în [Sistem lohn>Reintrări>Caută reintrări] și prezintă lista cu reintrările în sistem lohn inserate, permițând utilizatorului să le modifice, sa le vizualizeze sau să insereze altele noi.

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asupra unui document preexistent, în locul generării unuia nou. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea reintrărilor din sistem lohn inserate. |
| Inserează | Buton pentru inserarea unei noi reintrări din SL. |
| Modifică | Buton pentru deschiderea și modificarea unei reintrări din SL. |
| Vizulizează | Buton pentru deschiderea și vizualizarea unei reintrări din SL. |
| Șterge | Buton pentru ștergerea unei reintrări din SL inserate anterior. |
| Modifică filtru atribute | Buton pentru deschiderea unei ferestre care pemite activarea sau nu a filtrului atribute care permite filtrarea reintrărilor din sistem lohn pe baza atributelor inserate cu ajutorul codificatorului în articolele reintrării însăși. |

ARIA DE FILTRE: zona de filtrare este dedicata vizualizării unei liste cu posibilele tipuri de date, cu ajutorul cărora se poate face o selecție multiplă a reintrărilor SL inserate. Filtrele sunt în condiție  'AND', deci pot fi inserate mai multe criterii de filtrare simultan.

Utilizatorul poate folosi următoarele filtre:

**Terț** **: în aceste 2 câmpuri se inserează datele relative ale terțului din aviz: 'cont sintetic' și 'cont analitic', în timp ce numele terțului apare automat; cu dublu clic pe primul câmp se poate deschide 'Helpul conturilor' din care să se selecteze terțul. Acesta este inserat, de obicei, în planul de conturi printre furnizori;

**De la dată aviz… L**a dată aviz**** ****: în aceste câmpuri se pot insera datele avizului cu reintrări de la terț;

**De la dată reintrare… L**a dată **reintrare****** **: în aceste câmpuri se pot insera datele reintrării efective a produselor de la terț, care, de obicei, sunt diferite de data avizului; este vorba despre data în care va fi propusă înregistrarea în gestiune a reintrării;

**Nr. aviz reintrare** **: în acest câmp se poate insera numărul avizului reintrării; este un câmp text, nu este numeric;

**Valorizate** ****: acest flag permite filtrarea reintrărilor deja valorizate, și anume reintrările din care a fost deja creată o factură de achiziții în aria achiziții;

**Încărcate** **: acest flag permite filtrarea reintrărilor deja încărcate în gestiune, și anume reintrările pentru care a fost deja efectuată înregistrarea reintrării;

**Toate** **: acest flag permite filtrarea reintrărilor în sistem lohn, indiferent dacă au fost sau nu încărcate și/sau valorizate;

**Caută articole** ****: acest buton permite activarea unei secțiuni care permite filtrarea pe baza parametrilor: clasa/cod articol/descriere, tip comandă, comandă de producție.

GRIDUL REZULTATELOR: rezultatul este o listă cu avizele în SL care corespund filtrelor inserate anterior. Utilizatorul, dupa ce a inserat filtrele și a apăsat butonul Caută, va obține în grid lista cu avizele în SL dorite
