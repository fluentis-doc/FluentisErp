title: Loturi/NS
sidebar_position: 3
---

În acest tab există posibilitatea de a insera, pentru linia selectată în grid, loturile care vor fi descărcate din gestiune cu cantitatea extrasă pentru a fi expediată terțului (în cazul în care este vorba despre mai multe loturi diferite extrase cu aceeași linie de aviz).

Gridul loturilor permite utilizatorului să extragă lotul cu ajutorul helpului loturilor. Evident, suma cantităților diverselor loturi va trebui să corespundă cu cantitatea livrată a liniei articolului.

RIBBON BAR: reprezintă meniul cu butoanele de execuție, pentru fiecare operație in parte aferente fiecărei ferestre din aria in care se lucrează. Lista funcționalităților posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Inserează | Buton pentru inserarea unui nou aviz de livrare; |
| Salvează | Buton pentru salvarea modificărilor efectuate în aviz; |
| Livrări prevăzute | Buton care deschide fereastra Executare din ordine; permite transferarea articolelor din ordine SL în aviz. |
| Executare comenzi | Buton pentru transferarea articolelor din comenzile de vânzare; |
| Inserează plata | Buton pentru inserarea unei noi linii de plată în tabelul 'Plăți'. |
| Șterge plăți | Buton pentru ștergerea unei linii de plată din gridul 'Plăți'. |
| Inserează material | Buton pentru inserarea unui material nou în tabelul articolelor din tab 'Livrări'. |
| Șterge material | Buton pentru ștergerea unui materiale din gridul articolelor din tab 'Livrări'. |
| Vizualizează | Buton pentru vizualizarea raportului Aviz de livrare. |
| Listează | Buton pentru listarea Avizului de livrare. |
| Document manager | Procedura da posibilitatea de a atasa documente. |
