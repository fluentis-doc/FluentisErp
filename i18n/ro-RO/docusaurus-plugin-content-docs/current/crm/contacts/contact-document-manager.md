Questa è la quinta tab dei **Contatti CRM**.

![](/img/it-it/applications/crm/contacts/contact-document-manager.png)

In questa tab è possibile visualizzare e aggiungere degli allegati al contatto, che verranno salvati nel modulo Documentale di Fluentis.
Per aggiungere un file è sufficiente utilizzare il drag and drop, ossia selezionando il file dal percorso in cui si trova attualmente e rilasciandolo all’interno della tabella.

Si aprirà quindi la seguente maschera:

![](/img/it-it/applications/crm/contacts/new-document.png)

Nel campo **Tipo Documento**, nella parte superiore della form, è possibile selezionare la tipologia di documento che si sta caricando selezionandola tra quelle presenti, così come da [tabella](/docs/configurations/tables/general-settings/document-types).
Premendo poi il pulsante **Applica**, tale valore verrà riportato nel corrispondente campo su tutti i file che si stanno caricando in quel momento.
**Nome** e **Tipo file** vengono invece reperiti in automatico dalla procedura.
Premendo **OK**, il programma salverà il file nel Documentale di Fluentis.
