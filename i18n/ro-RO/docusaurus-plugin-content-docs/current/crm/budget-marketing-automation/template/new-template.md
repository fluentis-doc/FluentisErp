Procedura de creare a unui nou **Template** permite inserarea informațiilor (imagini, text, hyperlink și merge data) care vor fi apoi utilizate la expedierea newsletter-ului.

### Secțiune superioară

**Nume** și **Descriere**: reprezintă codul și descrierea template-ului;

**Obiect de business**: reprezintă obiectul de business Fluentis pentru care se poate utiliza acest template;

**Tip data source**: reprezintă tipul de data source pe care se bazează acest template (Simple, GroupBy și SQL Query);

**Data source**: reprezintă sursa de date care va fi utilizată de template pentru completarea Merge Data.

În continuare sunt prezente o serie de tabele cu care se pot construi un template și o pagina care afișează grafic rezultatul.
 
#### File

Acest tabel permite efectuarea operațiilor de bază ale template-ului, cum ar fi: deschiderea unei noi pagini, a unei alte pagini și vizualizarea modelului înainte de listare.  
**New**: buton pentru pagină nouă;  
**Open**: buton pentru deschiderea pagii;  
**Save As**: permite salvarea paginii cu numele inserat de utilizator;  
**Print**: permite listarea paginii;  
**Print Preview**: permite vizualizarea paginii înaintea listării ei.  
 
#### Home

Acest tabel conține butoanele care permit editarea textului modelului, identice cu cele din Microsoft Word.

#### Insert

Acest tabel conține butoanele care permit inserarea anumitor elemente în model.  
**Table**: permite inserarea în pagină a unui tabel;  
**Inline Picture**: permite inserarea unei imagini în corpul documentului;  
**Picture**: permite inserarea unei imagini în corpul documentului;    
**Bookmark**: permite inserarea unui bookmark în template;  
**Hyperlink**: permite inserarea un link ad una pagina web o ad un segnalibro del template;  
**Header**: permite inserarea antetului modelului;  
**Page Number**:permite inserarea numărului paginii în antet sau în subsolul paginii;  
**Page Count**: permite inserarea numărului total de pagini în antet sau în subsolul paginii;  
**Symbol**: permite inserarea unui simbol special în template.  

#### Merge data

Acest tabel conține butoanele care permit merge data.  
**Insert Merge Field**: permite inserarea merge data, selectându-i dintr-o lista care se bazează pe data source la care face referință modelul;  
**View Merged Data**: permite afișarea/ascunderea merge data inserați în template;  
**Show All Field Codes**: permite afișarea codurilor câmpurilor utilizate ca merge data;  
**Show All Field Results**: permite afișarea rezultatului extrapolării câmpurilor utilizate ca erge data;  
**First Record**: vă permite să vă poziționați pe primul record estrapolato dai merge data;  
**Previous record**: vă permite să vă poziționați pe înregistrarea anterioară față de cea actuală, extrapolată din merge data;  
**Next record**: vă permite să vă poziționați pe înregistrarea succesivă față de cea actuală, extrapolată din merge data;  
**Last Record**: vă permite să vă poziționați pe ultimul record extrapolat din merge data.
