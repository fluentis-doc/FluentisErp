Procedura permite definirea caracteristicilor template-ului care va fi ulterior folosit în procedura de trimitere a newsletter-ului, cum ar fi imagini, text, hyperlink și merge data. Form-ul este compus din două secțiuni.

#### Câmpuri specifice

**Nume**: corespunde codului cu care va fi identificat template-ul;  
**Descriere**: este descrierea template-ului;  
**Obiect Business**: este obiectul de business din Fluentis pentru care poate fi utilizat template-ul;  
**Tip de data source**: reprezintă tipologia sursei de date pe care se bazează template-ul (Simple, GroupBy și SQL Query);  
**Data Source**: reprezintă sursa de date care va fi utilizată de template pentru completarea merge data.

Partea inferioară a form-ului prevede o serie de tab-uri cu diferite funcții pentru a construi template-ul și foaia de lucru în care se construiește efectiv template-ul apoi se vizualizează grafic rezultatul:

![](/img/it-it/applications/crm/budget-marketing-automation/template-file.png)

#### File  

Acest tab permite efectuarea operațiunilor de bază ale template-ului, cum ar fi deschiderea unei noi foi de lucru și previzualizarea pentru tipărire a template-ului.  
**New**: deschide o nouă foaie de lucru;  
**Open**: deschide foaia de lucru;  
**Save As**: permite salvarea pe PC a foii de lucru;  
**Print**: permite tipărirea foii de lucru;  
**Print Preview**: permite afișarea foii de lucru pe ecran, pentru a o previzualizare înainte de listare;  
**Undo**: șterge ultima modificare a template-ului;  
**Redo**: restabilește ultima modificare ștearsă a template-ului.  

![](/img/it-it/applications/crm/budget-marketing-automation/file.png)

#### Home

Acest tab conține butoanele pentru funcționalitățile de bază privind editarea textului din template, precum dimensiunea fontului, liste, alinierea textului etc.  
Acesta reia funcțiile de bază ale celor mai populare editoare de text, precum Microsoft Word.  

#### Insert

Acest tab conține butoanele pentru inserarea anumitor elemente în template, cum ar fi tabele, imagini și hyperlink.  
**Tabel**: permite inserarea unui tabel în foaia de lucru;  
**Inline Picture**: permite inserarea unei imagini în corpul documentului;  
**Picture**: permite inserarea unei imagini în corpul documentului;  
**Bookmark**: permite inserarea unui bookmark în interiorul template-ului;  
**Hyperlink**: permite inserarea unui link către o pagină web sau către un bookmark din template;  
**Header**: permite inserarea antetului template-ului;  
**Footer**: permite inserarea subsolului template-ului;  
**Page Number**: indică numărul paginii în antet sau în subsolul template-ului;  
**Page Count**: indică numărul total de pagini al template-ului în antet sau în subsol;  
**Text Box**: permite inserarea unei casete de text în template;  
**Symbol**: permite inserarea unui simbol special în template .  

#### Merge data  

**Insert Merge Field**: permite inserarea merge data selectându-le dintr-o listă bazată pe Data Source la care face referire template-ul;  
**View Merged Data**: permite ascunderea/vizualizarea merge data inserate în template;  
**Show All Field Codes**: permite afișarea codurilor câmpurilor utilizate ca merge data;  
**Show All Field Results**: afișează rezultatul extragerii câmpurilor utilizate ca merge data;  
**First Record**: permite poziționarea pe primul record extras din merge data;  
**Previous record**: permite poziționarea pe recordul anterior față de cel curent extras din merge data;  
**Next record**: permite poziționarea pe următorul record față de cel curent extras din merge data;  
**Last Record**: permite poziționarea pe ultimul record extras din merge data.
