**Variante**: dacă articolul selectat are variante, va fi necesar să selectaţi varianta dorită din acest combo box. Alegerea variantei este utilă pentru articolele cu această configuraţie specială, care pot avea un preţ diferit de cel standard setat. Ca urmare, preţul articolului cu variante poate fi diferit de preţul articolului fără variante. Acest lucru poate necesita gestionarea variantelor articolului în oferta de referinţă.
