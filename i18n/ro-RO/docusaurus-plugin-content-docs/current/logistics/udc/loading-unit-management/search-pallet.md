---
title: Caută UdI
sidebar_position: 2
---

Această procedura a fost concepută pentru a permite utilizatorului să caute rapid paleți inserați anterior cu scopul de a le vizualiza, modifica sau șterge. Toate aceste operații sunt posibile dacă utilizatorul are drepturile necesare.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile cuprinde:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea paleților în baza de date. |
| UdI nou | Buton pentru inserarea unui UdI nou. |
| Șterge UdI | Buton pentru ștergerea paletului selectat. |
| Nou articol UdI | Buton pentru inserarea în paletul selectat a unuia sau mai multor articole. |
| Șterge articol UdI | Buton pentru ștergerea unuia sau mai multor articole din paletul selectat. |
| Inserare multiplă UdI | Buton pentru inserarea mai multor articole într-un UdI (este o fereastră care permite inserarea rapidă a articolelor). |
| Inserare ordine în UdI | Buton pentru inserarea articolelor unui ordin client într-nu UdI. Această procedură crează o legătură între UdI și ordinul client. |



| Functie | Functionalitate |
| --- | --- |
| Restituit client | Buton pentru crearea avizului pornind de la o dispoziție de livrare. |
| Încărcare UdI | Buton pentru crearea facturii pornind de la o dispoziție de livrare. |
| Descărcare UdI | Buton pentru crearea unui POS pornind de la o dispoziție de livrare. |
| Vizualizează înregistrarea de încărcare | Buton deschiderea înregistrării de încărcare a paletului. |
| Vizualizează înregistrarea de descărcare | Buton deschiderea înregistrării de descărcare a paletului. |
| Restaurare încărcare UdI | Buton pentru restuararea înregistrării de încărcare dacă paletul este încărcat. |
| Restaurare descărcare UdI | Buton pentru restuararea înregistrării de descărcare dacă paletul este descărcat. |
| Adaugă/elimină articole | După inserarea utilizatorului, care în parametrii de încărcare/descărcare are drepturile relative la procedura 'Elimină UdI' asociată la un șablon de descărcare, această procedură permite eliminarea unei cantități dintr-un UdI și descărcarea ei directă, sau descărcarea și încărcarea aceleiași cantități într-un alt UdI, în acel caz utilizatorul trebuie să aibă drepturilor relative la procedura de 'Încărcare UdI' asociată la un șablon de încărcare. |
| Mută UdI | Buton pentru transferul paletului dintr-o locație în alta. |
| Vizualizează | Buton pentru vizualizarea raportului. |
| Listează | Buton pentru listarea raportului; activează flag-ul listat în UdI. |
| Restaurează raport | Dezactivează flag-ul listat în UdI. |

ARIA DE FILTRE: conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecţie/căutare a paleților din gestiune. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRIDUL REZULTATELOR: este afișată lista cu paleții care corespund datelor din filtrele inserate sus. Utilizatorul, după ce a inserat filtrele și a apăsat butonul [Caută] prezent în ribbonul relativ, va obține, în gridul rezultatelor, lista cu paleții doriți.

GRIDUL DETALIILOR: acest grid afișează conținutul paletului selectat în gridul rezultatelor.
