---
title: Caută dispoziție de livrare
sidebar_position: 1
---

Această procedură a fost concepută pentru a permite utilizatorului să caute cu uşurinţă toate dispozițiile de livrare introduse anterior, cu scopul de a le vizualiza, edita sau şterge, dacă este necesar, in functie de drepturile alocate fiecarui utilizator. 

RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Caută dispoziții de livrare | Buton pentru căutarea în baza da date a dispozițiilor de livrare inserate. |
| Nouă dispoziție de livrare | Buton pentru inserarea unei noi dispoziții de livrare. |
| Modifică dispoziția de livrare selectată | Buton pentru modificarea dispoziției de livrare inserate. |
| Vizualizează dispoziția de livrare selectată | Buton pentru vizualizarea dispoziției de livrare inserate. |
| Șterge dispoziția de livrare selectată | Buton pentru ștergerea dispoziției de livrare inserate. |
| Descarcă dispoziția de livrare | Buton pentru executarea descărcării din gestiune a unei dispoziții de livrare. |



| Functie | Functionalitate |
| --- | --- |
| Aviz | Buton pentru crearea avizului pornind dintr-o dispoziție de livrare. |
| Factură | Buton pentru crearea facturii pornind dintr-o dispoziție de livrare. |
| Pos | Buton pentru crearea POS-ului pornind dintr-o dispoziție de livrare. |

ARIA DE FILTRE: conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecţie/căutare a dispozițiilor de livrare inserate. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: este afișată lista cu dispozițiile de livrare care corespund datelor din filtrele inserate sus. Utilizatorul, după ce a inserat filtrele și a apăsat butonul [Caută] prezent în ribbonul relativ, va obține, în gridul rezultatelor, dispozițiile de livrare dorite.
