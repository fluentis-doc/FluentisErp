RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Vizualizează | Buton pentru vizualizarea raportului. |
| Listează | Buton pentru listarea raportului. |

ARIA DE FILTRE: conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecţie/căutare a listelor de ieșiri prezente în aplicație. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare. În plus, este posibilă căutarea dispozițiilor de livrare descărcate din gestiune și/sau executate cu avizul și/sau executate cu factura și/sau executate cu lista de ambalare.

Este posibilă listarea unui raport standard sau personalizat (care se alege din combo Dispunere în tabel) configurat anterior în Gestionare rapoarte.
