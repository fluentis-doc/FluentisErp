# Inserare inventare fizice

## Introducere

Formularul se deschide prin calea **Logistică > Inventare fizice**, selectând butonul **Nou**.  
De aici este posibilă crearea unui nou inventar fizic și gestionarea listelor de inventar aferente.

## Procedură

1. Introduceți **Număr**, **Dată**, **Descriere**, eventuale **Note** și **Operator**.  
2. Odată ce diferențele de inventar au fost efectuate, activați bifa **Închis** pentru a marca inventarul ca închis, acesta nemaifiind luat în considerare pentru procedura de aliniere a stocurilor.  
3. Completați tabelul cu **Listele de inventar**, indicând:
   - **Tipul** de listă  
   - **Operatorul**  
   - **Gestiunea** de referință  
4. Opțional, se poate seta o **Dată de închidere** pentru a exclude lista din calculul diferențelor de inventar.  
5. După introducerea listelor de inventar, acestea trebuie completate. Pentru a face acest lucru, selectați o listă de inventar și faceți dublu clic pentru a accesa fereastra de 
[Modificare inventar articole](/docs/logistics/physical-inventory/inventory-management/inventory-lists-management).

## Rezumat și detalii suplimentare

Procedura de introducere a inventarelor fizice permite crearea unui inventar, definirea listelor aferente și gestionarea completării acestora până la eventuala închidere. Listele de inventar constituie baza pentru gestionarea diferențelor și actualizarea stocurilor.
