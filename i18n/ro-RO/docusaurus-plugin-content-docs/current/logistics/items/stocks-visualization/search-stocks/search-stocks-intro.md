---
title: Vizualizare stocuri
sidebar_position: 1
---

Această procedură permite utilizatorului să vizualizeze stocurile de gestiune cu scopul de a controla gestiunea și corespondența dintre stocul faptic și cel scriptic.

RIBBON BAR: reprezintă meniul cu butoanele de executie, pentru fiecare operatie in parte aferentă fiecarei ferestre din aria in care se lucreaza. Lista functionalitatiilor posibile conține:



| Functie | Functionalitate |
| --- | --- |
| Recalculare stocuri | Buton pentru recalcularea stocurilor după importarea mișcărilor din exterior. |
| Caută | Buton pentru căutarea stocurilor pe baza filtrelor inserate. |



| Functie | Functionalitate |
| --- | --- |
| Generare înregistrări | Buton pentru crearea unei înregistrări de gestiune. Este prezent doar în tab-urile: gestiune, comenzi de producție și detali stoc articole. |
| Vizualizare | Buton pentru vizualizarea unui raport standard sau a unui raport personalizat. Este prezent doar în tab Detaliu stoc articol. |
| Listare | Buton pentru listarea unui raport standard sau a unui raport personalizat. Este prezent doar în tab Detaliu stoc articol. |

ARIA DE FILTRE: conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecţie/căutare a stocurilor din gestiune. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: este afișată lista cu articolele care corespund datelor din filtrele inserate sus. Utilizatorul, după ce a inserat filtrele cu care dorește să caute stocurile și a apăsat butonul [Caută] prezent în ribbonul relativ, va obține, în gridul rezultatelor, lista cu articolele dorite.
