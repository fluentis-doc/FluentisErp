### Script

Acest tab este împărțit în:  
> - **Definiții**: conține definiția scriptului, posibilitatea de a schimba platforma scriptului, căutarea și substituirea funcției și diverse utilități legate de script (ex. versionare, îmbunătățire, duplicare, compilare/salvare, notă, etc.).  
> - **Referințe script**: conține toate referințele scriptului.
> - **General**: afișează parametrii standard ai scriptului: *Cod, Descriere, Categorie, Obiect Business, flag Dezactivat, Versiune* etc.

### Parametri

Permite definirea parametrilor utilizabili în ambele platforme de coding, *Script Editor* și *Blockly*. Dacă se utilizează cea din urmă, este disponibil blocul *getTaskParameter* (care se află în *Toolbox*, categoria *Task Utility*), care permite obținerea parametrilor creați anterior în tab.

### Execution logs

Afișează toate log-urile de execuție referitoare la scripturi, cu diverse niveluri de importanță (ex. Informație, Eroare, Avertisment, etc.). Log-urile pot fi șterse prin selectarea rândurilor din tabel și apăsarea butonului *CANC*.
