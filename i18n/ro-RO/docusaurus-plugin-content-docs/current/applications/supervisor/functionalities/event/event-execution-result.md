---
title: Rezultat executare eveniment
sidebar_label: Rezultat executare eveniment
sidebar_position: 3
---

În form-ul [Eveniment](event-intro) sunt afișate câteva date utile despre execuția *Evenimentului*.  
În tabelul principal al ferestrei sunt prezente câmpurile *Rezultatul ultimei execuții* și *Durata maximă*, iar informațiile detaliate (erori, informații și loguri) despre execuția *Evenimentului* pot fi vizualizate selectând și extinzând rândul.
