---
title: Caută lista de avans
---

Procedura de căutare liste de avans a fost concepută pentru a permite utilizatorului să caute ușor listele de avans inserate anterior, cu scopul de a le vizualiza, modifica și, eventual, șterge. Din căutare se pot face toate operațiile necesare, cu condiția ca utilizatorul să aibă drepturi.

Procedura de căutare a listelor de avans este astfel constituită:

RIBBON BAR: reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm  în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută liste | Buton pentru aplicarea filtrelor de căutare în baza de date a listelor de avans inserate anterior. |
| Listă nouă | Apelează procedura pentru inserarea unei noi liste de avans. |
| Modifică lista | Apelează procedura pentru modificarea unei liste de avans inserate anterior. |
| Vizualizează lista | Apelează procedura pentru vizualizarea unei liste de avans inserate anterior. |
| Șterge liste | Apelează procedura pentru ștergerea unei liste de avans inserate anterior. |



| Functie | Functionalitate |
| --- | --- |
|  [Rapoarte](/docs/treasury/advances-collections/reports)  | Apelează procedura de listare relativă la Lista facturi anticipate/încasate. |



| Functie | Functionalitate |
| --- | --- |
|  Gestiune încasări | Buton pentru deschiderea gestiunii încasărilor. |
|  [Contabilizare avans încasări](/docs/treasury/advances-collections/accounting/accounting)  | Buton pentru crearea înregistrării contabile de avans și încasări. |

ARIA DE FILTRE: conţine lista cu posibilele tipuri de date cu ajutorul cărora se poate efectua o selecţie/căutare a solicitărilor inserate. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: sunt afișate listele de avans care corespund datelor din filtrele inserate sus. Utilizatorul, după ce a inserat filtrele și a apăsat butonul [Caută] prezent în ribbonul relativ, va obține, în gridul rezultatelor, listele de avans dorite.
