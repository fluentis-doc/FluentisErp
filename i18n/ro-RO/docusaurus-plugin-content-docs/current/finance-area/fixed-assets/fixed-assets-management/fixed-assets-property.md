sidebar_position: 5

Ultimul tab standard se referă la ‘Proprietăţile' mijlocului fix: este posibilă atribuirea unei liste cu proprietăţi diferite mijlocului fix în uz, cu o eventuală notă explicativă.

RIBBON BAR: Reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Nouă proprietate | Activează cursorul pentru inserarea unei proprietăţi. |
| Şterge proprietate | Şterge proprietatea selectată în grid. |
