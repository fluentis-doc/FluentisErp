---
title: Creare file Intrastat
---

Cu această procedura se va crea fișierul electronic pentru comunicarea declarațiilor intrastat: se selectează modelele perioadei de prezentare și se apasă butonul pentru crearea fișierului, care va deschide o fereastră pentru salvarea fișierului  prevăzut Schimburi.cee. În cazul în care în declarație sunt prezente doar date relative la bunuri, fișierul va avea structura veche ‘EUROA', dacă are și servicii va fi aplicată noua structură ‘EUROX' prevăzută din 2010. Nu este prevăzută blocarea declarațiilor pentru care a fost creat fișierul electronic.

RIBBON BAR: reprezintă meniul form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asociate procedurii utilizate. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Caută declarațiile intrastat pentru care doriți să generați fișierul electronic. |
| Exportă în fișier | Buton pentru crearea și exportarea fișierului electronic, cu salvarea fișierului într-un director ales de utilizator. |
