title: Intra1 quater
sidebar_position: 5
---

Secțiunea quater a declarațiilor intrastat conține detaliile perioadei, relative la prestarea de servicii. Secțiunea este completată automat de procedură, prin preluarea datelor din tipuri de factură cu natura tranzacției  *S – servicii* sau din înregistrările contabile (negenerate din achiziții/vânzări) întodeauna cu linii de tip TVA *Serviciu*. Este permisă modificarea și inserarea manuală.

Opțiunile sunt:

> **Salvează** - Buton cu ajutorul căruia se salvează declarația.  
> **[Crează automat](/docs/finance-area/declarations/intrastat/automatic-creation-intrastat1/automatic-creation)** - Buton care apelează procedura de preluare a datelor din aria vânzări.  
> **[Crează din înregistrări](/docs/finance-area/declarations/intrastat/create-from-records-intrastat1/create-from-records-intrastat1-intro)** - Buton care apelează procedura de preluare a datelor din înregistrările contabile.  
> **Crează detaliu nou** - Buton pentru crearea manuală a unei noi linii detaliu.  
> **Șterge** - Buton pentru ștergerea liniei selectate.
