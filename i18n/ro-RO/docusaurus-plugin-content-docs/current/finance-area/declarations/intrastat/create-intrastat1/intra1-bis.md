Secțiunea bis a declarațiilor intrastat conține detaliile perioadei, relative la vânzarea bunurilor. Secțiunea este completată de procedură automat, însă este permisă modificarea și inserarea manuală. În partea inferioară a secțiunii este prezent butonul de grupare.

Optiunile sunt:

> **Salvează** - Buton cu ajutorul căruia se salvează declarația;  
> **[Creare automată](/docs/finance-area/declarations/intrastat/automatic-creation-intrastat1/automatic-creation)** - Buton care apelează procedura de preluare a datelor din aria vânzări;  
> **[Creare din înregistrări](/docs/finance-area/declarations/intrastat/create-from-records-intrastat1/create-from-records-intrastat1-intro)** - Buton care apelează procedura de preluare a datelor din înregistrările contabile;  
> **Crează detaliu nou** - Buton pentru crearea manuală a unei noi linii detaliu;  
> **Șterge** - Buton pentru ștergerea liniei selectate.
