Procedura **Închidere automată conturi** gestionează închiderea bilanțului și succesiva redeschidere: este vorba doar despre înregistrarea contabilă.

În fereastra *Închidere automată conturi* se face clic pe butonul pentru o nouă închidere.

Opțiuni disponibile:

> **Salvează** - Permite salvarea antetul închiderii conturilor; devine activă după inserarea descrierii.
