---
title: Model F24
sidebar_position: 1
---

Procedura este dedicată creării formatului ministerial al declaraţiei F24, pentru expedierea sa electronică. Acest model este dedicat comunicării diverselor impozite vărsate lunar de companie.

Setarea de bază este codificarea, în modulul personal din aria administrativă, a subiectului semnatar al declaraţiei. În particular, trebuie indicate codul de încărcare şi datele despre rezidenţă, domiciliu, CNP, nume şi prenume.

RIBBON BAR: reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate procedurii curente. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru aplicarea filtrelor de căutare în baza de date a modelelor inserate. |
| Nou | Apelează procedura pentru inserarea unui model nou. |
| Modifică | Apelează procedura pentru modificărea unui model inserat anterior. |
| Vizualizare | Apelează procedura pentru vizualizarea unui model inserat anterior. |
| Şterge | Execută procedura pentru ştergerea unui model inserat anterior. |
