sidebar_position: 2

Partea superioară a ferestrei este dedicată inserării filtrelor de căutare. După ce, în partea inferioară a ferestrei, au fost inserate datele pentru înregistrarea competenţei şi eventuala înregistrare cu statusul provizoriu, este suficient să apăsaţi butonul pentru a efectua contabilizarea; dacă a fost efectuat totul corect, liniile selectate vor dispare din grid iar in rezumate veţi putea vizualiza rezultatul operaţiei.

RIBBON BAR: reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate procedurii curente. Prezentăm în continuare lista cu funcţionalităţile asociate procedurii curente:



| Functie | Functionalitate |
| --- | --- |
| Caută remuneraţii | Caută lista remuneraţiilor de contabilizat. |
| Contabilizare remuneraţii | Execută contabilizarea remuneraţiilor pentru liniile selectate. |
| Modifică remuneraţie | Deschide pentru modificare remuneraţia selectată. |
