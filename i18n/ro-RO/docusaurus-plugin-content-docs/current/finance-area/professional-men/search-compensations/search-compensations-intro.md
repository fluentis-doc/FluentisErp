---
title: Caută remuneraţii
sidebar_position: 1
---

Procedura Caută remuneraţii a fost concepută pentru a permite utilizatorului căutarea remuneraţiilor inserate anterior, cu scopul de a le vizualiza, modifică sau ştergei, cu condiţia ca utilizatorul sa aibă drepturile necesare.

Procedura Caută efecte este structurată în felul următor:

RIBBON BAR: reprezintă meniul Form-ului în discuţie. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută remuneraţii | Buton pentru aplicarea filtrelor de căutare în baza de date a remuneraţiilor. |
| Nouă remuneraţie | Apelează procedura pentru inserarea unei noi remuneraţii. |
| Modifică remuneraţii | Apelează procedura pentru modificărea unei remuneraţii inserate anterior. |
| Vizualizare remuneraţii | Apelează procedura pentru vizualizarea unei remuneraţie inserate anterior. |
| Şterge remuneraţii | Execută procedura pentru ştergerea unei remuneraţie inserate anterior. |
| Vărsăminte reţinere | Deschide fereastra cu datele vărsămintelor. |



| Functie | Functionalitate |
| --- | --- |
| Raport certificare reţineri | Apelează procedura pentru ştergerea certificatelor periodice ale reţinerilor. |
|  [Raport Situaţie reţineri](/docs/finance-area/professional-men/reports/withholding-tax-situation)  | Apelează procedura pentru ştergerea situaţiei reţinerilor vărsate/de vărsat pentru plăţile din perioada selectată. |



| Functie | Functionalitate |
| --- | --- |
| Modello F24 | Apelează procedura pentru gestionarea datelor Modelului F24. |
|  [Contabilizare remuneraţii](/docs/finance-area/professional-men/accounting/compensation-accounting/compensations-accounting)  | Apelează procedura pentru efectuarea contabilizării remuneraţiilor. |
|  [Contabilizare enasarco](/docs/finance-area/professional-men/accounting/enasarco-accounting/enasarco-accounting-intro)  | Apelează procedura pentru efectuarea contabilizării impozitelor datorate autorităţii Enasarco. |
|  [Contabilizare plăţi](/docs/finance-area/professional-men/accounting/payments-accounting/payments-accounting-intro)  | Apelează procedura pentru efectuarea contabilizării plăţilor către profesionişti, care va releva reţinerile şi eventualele cheltuieli Inps. |

ARIA DE FILTRE: aria de filtre conţine lista cu posibilele tipuri de date, cu ajutorul cărora se poate efectua selecţia sau căutarea efectelor inserate. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: reprezintă lista cu remuneraţiile care corespund datelor din filtrele indicate anterior. Utilizatorul, după ce a inserat valorile cu ajutorul cărora doreşte să caute remuneraţiile, dacă apasă butonul [Caută remuneraţii] prezent în Ribbon, va obţine în gridul rezultatelor lista cu remuneraţiile dorite.
