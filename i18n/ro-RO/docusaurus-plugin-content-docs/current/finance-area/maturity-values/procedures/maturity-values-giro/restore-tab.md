În acest tab pot fi afișate sau anulate înregistrările efectuate cu ajutorul butoanelor.

RIBBON BAR: reprezintă meniul Forum-ului în discuţie, mai precis aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea transferului angajamentelor efectuate anterior. |
| Restaurează transfer angajamente | Buton pentru vizualizarea și/sau restaurarea transferului angajamentelor. |
