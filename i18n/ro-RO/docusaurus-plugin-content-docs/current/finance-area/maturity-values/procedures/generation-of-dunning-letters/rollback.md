Tabul pentru restaurare permite anularea creării: același rezultat se obține intrând în caută solicitări și ștergând direct de aici.

RIBBON BAR: reprezintă meniul form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru cautarea creării solicitărilor. |
| Restaurează tot | Buton pentru ștergerea tuturor solicitărilor creării selectate. |
| Restaurează selecția | Buton pentru ștergerea doar a anumitor solicitări ale creației selectate. |

ARIA DE FILTRE: conține posibilele tipuri de date cu ajutorul cărora se poate efectua o selecție. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRID REZULTATE: este lista cu solicitările create care corespund datelor din filtrele indicate anterior. Utilizatorul, după ce a inserat valorile cu ajutorul cărora dorește să obțină căutarea solicitărilor, dacă apasă butonul [Caută] din relativul ribbon, va obține, în gridul rezultatelor, lista cu solicitările create.
