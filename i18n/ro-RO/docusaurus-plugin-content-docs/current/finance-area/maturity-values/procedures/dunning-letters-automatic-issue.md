---
title: Creare automată solicitări
sidebar_position: 2
---

În această fereastră se poate insera o listă de clienți pentru care nu se crează solicitări, o listă de angajamente excluse din solicitări și tipul de solicitare implicită de creat.

După salvarea acestor setări se va activa butonul pentru crearea automată: cu acesta aplicația va crea solicitările pentru toate angajamentele care satisfac criteriile de selecție setate.

RIBBON BAR: reprezintă meniul form-ului în discuţie. Mai precis, aria în care se pot efectua acţiuni asociate la procedura utilizată. Prezentăm în continuare lista cu funcţionalităţile posibile:



| Functie | Functionalitate |
| --- | --- |
| Salvează | Buton pentru salvarea generării solicitărilor. |
| Generare solicitări | Buton pentru efectuarea generării solicitărilor angajamentelor care satisfac criteriile de selecție inserate. |
| Caută client de exclus | Buton pentru căutarea clienților pe care doriți să-i excludeți din generare automată solicitări. |
| Șterge clienți | Buton pentru ștergerea clienților din listă. |
| Caută angajamente de exclus | Buton pentru ștergerea clienților din listă. |
| Caută angajamente de exclus | Caută angajamente de exclus	Buton pentru căutarea angajamentelor pe care doriți să le excludeți. |

ARIA DE FILTRE: este dedicată inserării filtrelor pentru selecția datelor. În general, filtrele sunt în condiţie “AND” şi există posibilitatea inserării simultane a mai multor criterii de filtrare.

GRIDUL REZULTATELOR: este lista cu solicitările care corespund datelor din filtrele indicate anterior. Utilizatorul, după ce a inserat filtrele de căutare, și după ce a apăsat butonul de căutare, va obține în gridul rezultatelor lista cu solicitările dorite.
