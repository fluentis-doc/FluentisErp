---
title: Caută angajamente
sidebar_position: 2
---

Această filă vă permite să căutați angajamentele prin câmpurile din antet, care reprezintă zona de filtrare a datelor care vor fi propuse și să editați fiecare angajamentindividual cu un dublu clic al mouse-ului.

Alte câmpuri utile pentru filtrul de căutare sunt disponibile expanzand anajamentul selectat.

În mod implicit, angajamentele deschise sunt propuse (deoarece câmpul Stare al filtrului este completat cu intrarea Deschide angajament când fila este deschisă).

Prin efectuarea căutării (comanda Căută în bara de meniu), angajamentele vor fi afișate cu toate informațiile relative.

În special, pentru fiecare linie corespunzătoare angajamentului există o comandă (+) care vă permite să vizualizați informații suplimentare și ascunse inițial: acestea sunt orice plăți legate de anjamentul închis sau parțial deschis.


**DOMENII SPECIFICE PENTRU FORMULAREA SUBIECTULUI**:

**În plată**: indică faptul că angajamentul a fost deja introdus într-un buletin de plată al furnizorului, dar buletinul de plată nu este încă contabilizat și, prin urmare, angajamentul nu a fost încă închis;

**Contabilitate**: indică faptul că angajamentul este legat de o înregistrare contabilă;

**Care nu pot fi plătite**: indică faptul că angajamentul a fost blocat de utilizator, astfel încât acesta să nu fie plătibil. Blocul poate fi setat direct în momentul creării sau ulterior atât din fila  [Angajemente](/docs/finance-area/ledger-records/records/ledger-record)  a înregistrării contabile care generează angajamentul, cât și prin modificarea angajamentului însuși din formularul în cauză.



| **Funcție** | **Semnificație** |
| --- | --- |
| **Caută** | Începe căutarea potrivirilor în funcție de datele introduse în filtru; |
| **Angajament Nou** | Permite inserarea unui nou angajament; |
| **Modifică angajament** | Permite selectarea angajamentului dorit pentru a modificat informațiile din interiorul acestuia; |
| **Vizualizează** | Permite viziualizarea anjamentul dorit, fără a putea salva anumite modificări; |
| **Șterge angajamentele selectate** | Șterge rândul selectat; |
| **Schimbă pozitie angajament** | Acesta afișează un formular de căutare pentru poziția angajamentului, permițându-vă să atribuiți statutul selectat meciului; |
| **Actualizează situația angajamentelor** | Reîmprospătați starea angajamentului pentru vizualizarea corectă; |
| **Subdivide angajamente** | Deschide un formular de tip pop-up care vă permite să împărțiți angajamentul selectat în mai multe angajamente cu suma și termenul dorit. În mod clar, suma angajamentelor rezultate trebuie să coincidă cu suma angajamentului inițial. |
| **Modifică înregistrarea** | Permite introducerea directă a modificării înregistrării contabile legate de angajamentul selectat. Dacă angajamentul selectat nu este legat de o înregistrare contabilă, butonul va fi dezactivat. |
