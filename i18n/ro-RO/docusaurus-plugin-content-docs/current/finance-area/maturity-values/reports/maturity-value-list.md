---
title: Listă angajamente
sidebar_position: 3
---

Din acest formular este posibilă imprimarea unei listăe de angajamente deschise (sau angajamente într-una din celelalte stări posibile) în conformitate cu parametrii de filtrare stabiliți.

Funcționarea formularului este aceeași ca și pentru  [alte tipăriri ale formularului la care se face trimitere](/docs/finance-area/maturity-values/reports/customer-vendor-due-register).
