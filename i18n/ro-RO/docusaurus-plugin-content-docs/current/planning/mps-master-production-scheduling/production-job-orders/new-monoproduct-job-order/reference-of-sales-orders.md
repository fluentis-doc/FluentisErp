sidebar_position: 3

Gridul este read only și este populat doar când comanda a fost generata dintr-o linie a unui ordin client. Descriem în continuare coloanele:

**Tip/An/Număr/Nr. linie/Cantitate/Comandă**: în aceste coloane sunt afișate datele relative ale ordinului client în care există linia detaliată aici;

**UM util.**: este afișată unitatea de măsură utilizată în linia ordinului client care este detaliată aici;

**Prev. livrare**: este afișată data prevăzută pentru livrare care a fost inserată în linia ordinului client care este detaliată aici;

**Client/Descriere client**:  în aceste coloane sunt afișate datele relative ale clientului căruia îi aparține ordinul client și în care există linia detaliată aici;
