Fereastra comenzilor se deschide din [Planificare>Comenzi>Comenzi]. Această fereastră permite căutarea comenzilor de producție deja create sau crearea noilor comenzi. 

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie în partea aferentă fiecărei ferestre din aria in care se lucreaza. Lista functionalitatiilor contine:



| Functie | Functionalitate |
| --- | --- |
| Caută | Buton pentru căutarea comenzilor de producție în baza de date. |
| Inserează comanda monoprodus | Buton pentru inserarea unei noi comenzi de producție monoprodus. |
| Inserează comanda multiprodus | Buton pentru inserarea unei noi comenzi de producție multiprodus. |
| Modificare comandă | Buton pentru modificarea unei comenzi de producție inserate anterior. |
| Vizualizează comanda | Buton pentru vizualizarea unei comenzi de producție inserate anterior. |
| Șterge comanda | Buton pentru ștergerea unei comenzi de producție inserate anterior. |
| Modifică data comenzii | Buton pentru modificarea datei prevăzute pentru sfârșitul unei sau mai multe comenzi de producție. |
| Modifică filtru atribute | Butonul deschide o fereastră care permite activarea sau nu a filtrului atribute care permite filtrarea comenzilor de producție pe baza atributelor inserate cu ajutorul codificatorului în articolele care fac obiectul comenzii. |

ARIA DE FILTRE: Zona de filtrare este utilizata pentru a specifica o serie de valori ale unor campuri de date, dupa care se poate face o filtrare multipla a comenzilor inserate. Exista posibilitatea să specificaţi mai multe criterii de filtrare.

**Tip com** **.**:  se poate insera tipul comenzii de producție;

**De la număr ….. La număr**: se poate insera intervalul de numere după care vor fi căutate comenzile;

**De la an...La an**: se poate insera intervalul temporal considerând anul de creare al facturii;

**Articol**: în combo se poate insera clasa articolului, iar cu dublu clic in al doilea și al treilea câmp se deschide help articole cu ajutorul căruia se selectează articolul după care se dorește filtrarea comenzilor de producție;

**Tip comandă**: se poate alege filtrarea după comanda monoprodus, comanda multiprodus sau toate;

**Ordin client**: Tip: se poate insera tipul ordinului, descierea și numărul relativ. În acest caz vor fi afișate toate comenzile de producție asociate numărului ordinului pentru care a fost inserat filtru;

**Comandă**: cu dublu clic pe acest câmp se deschide helpul comenzilor cu ajutorul căruia se poate selecta comanda de vânzare la care sunt asociate comenzile de producție pe care le căutați;

**Client**: în aceste câmpuri se inserează 'Contul', 'Contul analitic', 'Numele clientului'; cu dublu clic pe primul câmp se poate deschide 'Help conturi' cu ajutorul căruia se poate alege clientul;

**De la...La data prevăzută pentru sfârșitul lucrării**: se poate insera intervalul de date prevăzute pentru sfârșitul lucrării; cu ajutorul flagurilor relative se pot afișa comenzile întârziate și/sau scadente;

**Tip aprovizionare**: cu ajutorul flagurilor se poate selecta tipul aprovizionării comenzii: achiziții, producție sau sistem lohn;

**Stare comandă**: cu ajutorul flagurilor se poate filtra statusul comenzii care vor fi afișate (Neexaminate, Planificate, Lansate, În execuție, Executate, Arhivate).

GRIDUL REZULTATELOR: Rezultatul este o listă a comenzilor de producție care se corespund filtrelor specificate mai sus. Utilizatorul, după ce a inserat filtrele de căutare, dacă apasă butonul Caută prezent în Ribbon, va obține în gridul rezultatelor lista cu comenzile dorite.

**Tip cerere**: poate fi Dependentă dacă este o comandă generată automat cu proceudra Definire MPS, sau Independentă, dacă este o comandă generată manual;

**Tip aprv. comandă**: afișează tipul aprovizionării comenzii, care pot fi de producție, de achiziție sau sistem lohn, în baza tipului de aprovizionare inserat pentru articolul care face obiectul comenzii în parametri săi MRP;

**Stare comandă**: afișează starea comenzii de producție (neexaminată, planificată, lansată, în execuție, executată, arhivată);

**Dată sugerată**: afișează data pe care planificarea generală o sugerează ca dată prevăzută pentru sfârșitul comenzii de producție, dacă este inserată data blocării în parametrii relativi înaintea executării procedurii de planificare;

**Cantitate**: afișează cantitatea de produs din articol;

**Comanda**: afișează eventuala comandă de vânzare asociată liniei comenzii de producție;

**Descriere articol**: afișează descrierea articolului de produs.
