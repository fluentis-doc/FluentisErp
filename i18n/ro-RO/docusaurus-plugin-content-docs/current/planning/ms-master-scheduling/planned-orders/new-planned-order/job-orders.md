În această fereastră sunt afișate detaliile comenzilor de producție din care a fost generat ordinul planificat.

RIBBON BAR: reprezinta meniul cu butoanele de executie, pentru fiecare operatie in parte aferente fiecarei ferestre din aria in care se lucreaza. Lista functionalitatilor posibile sunt enumerate dupa cum urmeaza:



| Functie | Functionalitate |
| --- | --- |
| Salvează | Butonul permite salvarea modificărilor efectuate ordinului planificate. |
| Inserează ordinul | Apelează procedura pentru inserarea unui nou ordin planificat. |
| Completare date ordin | Apelează procedura care permite inserarea și/sau actualizarea, pentru articolul inserat în ordinul planificat, a tuturor datelor relative la materiale, faze, montaj, echipamente, atribute și note, preluându-le din lista componentelor și ciclul d lucru al articolului. |
| Recalculează dată început  | Apelează procedura care permite executarea unei actualizări a datei pentru începerea ordinului planificat și, eventual, a fazelor de lucru ale acestuia în urma unei modificări a timpilor de lucru și/sau a cantității de produs de articol. |
| Generează lista componentelor | Apelează procedura care permite actualizarea directă a listei componentelor articolului utilizând datele prezente în ordin. |
| Generează ciclul de lucru | Apelează procedura care permite actualizarea directă a ciclului de lucru a articolului ordinului planificat, utilizând datele prezente în ordin. |
| Inserează materiale | Apelează procedura pentru inserarea unui nou material în gridul din tab Materiale. Butonul este prezent doar dacă se află în tab Materiale din ordinul planificat. |
| Șterge materiale | Apelează procedura pentru ștergerea unui material din gridul tabului Materiale. Butonul este prezent doar dacă se află în tab Materiale din ordinul planificat. |
