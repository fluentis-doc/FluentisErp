---
title: Generazione file rid
sidebar_position: 2
---

Da questa maschera si generazioni i tracciati telematici per le distinte di presentazione RID.

La parte superiore della maschera è dedicata alla definizione dei filtri di ricerca delle distinte emesse ma per le quali il file non sia stato ancora create.

Una volta individuata la distinta da gestire, alla base della maschera si possono visualizzare e gestire i dati di default così come predefiniti nei parametri: la data di creazione del file, il suo nome (impostato come [DataDistinta]_[NumeroDistinta].RID), il percorso di creazione del file, il nome della società e la descrizione dei riferimenti fattura da inserire. Alla base della maschera sono presenti i flag di gestione del ritorno a capo finale del file e l'esclusione della gestione dei riferimenti per gli insoluti.

Per eseguire la creazione del file è sufficiente premere il pulsante di generazione: il programma procederà ad un controllo dei dati obbligatori da inserire nel tracciato (ma non della correttezza del dato obbligatorio inserito), ritornando nel caso una maschera con i dettagli degli errori individuati.

RIBBON BAR: rappresenta il menu della Form in oggetto, ossia l'area nella quale è possibile effettuare delle azioni collegate alla procedura in uso. La lista delle funzionalità possibili è di seguito rappresentata:



| Funzione | Significato |
| --- | --- |
| Ricerca distinte | Ricerca la lista delle distinte per le quali creare il file telematico. |
| Creazione | Esegue la creazione del tracciato Rid per la distinta selezionata. |
