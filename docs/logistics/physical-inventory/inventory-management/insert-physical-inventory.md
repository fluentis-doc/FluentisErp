# Inserimento inventari fisici

## Introduzione

La form si apre tramite il percorso **Logistica > Inventari fisici**, selezionando il pulsante **Nuovo**.  
Da qui è possibile creare un nuovo inventario fisico e gestirne le relative liste inventariali.

## Procedura

1. Inserire **Numero**, **Data**, **Descrizione**, eventuali **Note** e l’**Operatore**.  
2. Una volta eseguite le differenze inventariali, attivare il flag **Chiusa** per rendere chiuso l'inventario che non verrà quindi più preso in considerazione per la procedura di allineamento giacenze.  
3. Completare la griglia delle **Liste inventariali**, indicando:
   - **Tipo** lista  
   - **Operatore**  
   - **Magazzino** di riferimento  
4. Facoltativamente è possibile impostare una **Data chiusura** per escludere la lista dal calcolo delle differenze inventariali.  
5. Una volta inserite le liste inventariali è necessario compilarle. Per fare questo, selezionare una lista inventariale e fare doppio clic per accedere alla finestra di modifica [Modifica inventari articoli](/docs/logistics/physical-inventory/inventory-management/inventory-lists-management).


## Riepilogo e approfondimenti

La procedura di inserimento inventari fisici consente di creare un inventario, definire le relative liste e gestirne la compilazione fino all’eventuale chiusura. Le liste inventariali costituiscono la base per la gestione delle differenze e per l’aggiornamento delle giacenze.
