# Attivare visualizzazione Fluent per MES

Per utilizzare la modalità **Fluent**, progettata per ottimizzare la visualizzazione **bordo macchina** di Fluentis MES, è necessario configurare correttamente le impostazioni dell’interfaccia.

## Procedura di attivazione

1. Cliccare sull’icona **ingranaggio** situata nella parte sinistra dello schermo.
2. Aprire la sezione **Impostazioni**.
3. Selezionare **Visualizzazione Fluent**.

Al termine della procedura, Fluentis verrà aperto con la visualizzazione Fluent.
