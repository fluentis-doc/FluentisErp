### Qualità

#### Documenti di origine esterna e interna

**Documento di origine esterna** e **Documento di origine interna**: Sono state apportate alcune variazioni alle informazioni contenute nel corpo dell'email utilizzata per la distribuzione dei documenti.

```
Spett. _destinatario_  
   
In allegato troverà, in formato digitale, copia del documento in oggetto.   
A disposizione per eventuali chiarimenti si rendessero necessari.   
  
Cordiali saluti.
```
