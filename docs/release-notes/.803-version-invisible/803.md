### Pianificazione


#### Ordini pianificati

**Rilascio ordini pianificati**: Nel caso in cui venga fatto il ripristino di una ordine pianificato di acquisto, per il quale sia già stato creato un ordine di acquisto, viene visualizzato un messaggio di errore.


### WMS

#### Generale

**Framework**: Rilascio della nuova app sviluppata in .NET MAUI. L'app Xamarin non sarà più supportata.
